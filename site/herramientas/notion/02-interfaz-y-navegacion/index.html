    Módulo 02: Interfaz y Navegación¶

    🎯 Objetivos del módulo¶

    Al terminar este módulo vas a:

    • ✅ Dominar la navegación de Notion
    • ✅ Entender los tipos de bloques
    • ✅ Crear páginas y subpáginas
    • ✅ Usar las funciones de búsqueda
    • ✅ Organizar tu contenido con arrastrar y soltar

    📱 Conociendo la interfaz¶

    Cuando abrís una página en Notion, la interfaz tiene estas partes:

    ┌─────────────────────────────────────┐
│  ←  │ Título de la página    │ ⋮ │  ← Menú de página
├─────────────────────────────────────┤
│                                     │
│     [Contenido de la página]        │
│                                     │
│     Escribí "/" para ver bloques    │
│                                     │
│                                     │
├─────────────────────────────────────┤
│  [+]  │ Bloques  │ Comment  │ Compartir │
└─────────────────────────────────────┘

    Partes de la interfaz:¶

    Parte Función
    Flecha ← Volver al nivel anterior
    Título Nombre de la página (editabl)
    Menú ⋮ Opciones de la página
    [+] Agregar bloque rápidamente
    Bloques Selector visual de bloques
    Comment Agregar comentarios
    Compartir Invitar personas

    🧱 Los bloques: La base de todo¶

    En Notion todo es un bloque. Una página, un texto, una imagen, una tabla... todo es un bloque que podés mover, editar o eliminar.

    Tipos de bloques básicos:¶

    Texto simple¶

    • Escribí normalmente y presioná Enter
    • Se crea un párrafo

    Título grande (H1)¶

    • Escribí "/" seguido de "h1" o seleccioná "Heading 1"
    • O usá el atajo: ## al inicio de la línea

    Subtítulo (H2)¶

    • Escribí "/" + "h2"
    • O usá: ### al inicio

    Subtítulo menor (H3)¶

    • Escribí "/" + "h3"
    • O usá: #### al inicio

    Cómo agregar un bloque:¶

    1. Método 1: Hacé clic en el ícono "+" que aparece a la izquierda cuando pasás el cursor
    2. Método 2: Escribí "/" en una línea vacía y escribí el tipo de bloque
    3. Método 3: Tocá el botón "+" en la barra inferior

    Tip: Escribir "/" es el método más rápido. Probá escribir:

    • /h1 → Título
    • /h2 → Subtítulo
    • /todo → Lista de tareas
    • /table → Tabla
    • /image → Imagen
    • /divider → Línea separadora

    📂 Crear páginas y subpáginas¶

    Crear una página nueva¶

    Método 1: Desde cualquier lugar

    1. Tocá el botón "+" azul en la esquina inferior derecha
    2. Se crea una página en blanco
    3. Ponele nombre y empezá a escribir

    Método 2: Dentro de una página

    1. Escribí "/" en una línea
    2. Buscá "Page" o "Página"
    3. Seleccioná
    4. Se crea una subpágina

    Método 3: Desde el sidebar

    1. Tocá "Nuevo" o el ícono "+" junto a una carpeta
    2. Elegí "Página"

    Diferencia entre Página y Subpágina¶

    Workspace CM Belen
├── 📁 Clientes
│   ├── 📄 Joyería Lumina     ← Subpágina de Clientes
│   └── 📄 Accesorios María   ← Subpágina de Clientes
├── 📄 Calendario Editorial    ← Página principal
└── 📄 Tareas de la semana   ← Página principal

    Las subpáginas aparecen como children de una página padre. Podés tener hasta infinitos niveles de profundidad.

    🔍 Búsqueda: Encontrar cualquier cosa¶

    Notion tiene una búsqueda muy potente.

    Abrir búsqueda:¶

    • Tocá la barra de búsqueda arriba de la pantalla
    • O usá el atajo: Cmd/Ctrl + K (en teclado)

    Qué podés buscar:¶

    • Títulos de páginas
    • Contenido dentro de las páginas
    • Comentarios
    • Nombres de personas
    • Bloques específicos

    Ejemplo práctico:¶

    Si buscás "joyería", Notion te va a mostrar:

    • Páginas con "joyería" en el título
    • Páginas que contienen "joyería" en su contenido
    • Resultados organizados por tipo

    ↕️ Organizar: Arrastrar y soltar¶

    Mover bloques¶

    1. Tocá y mantené el ícono de arrastre (⋮⋮) a la izquierda del bloque
    2. Arrastrá hacia arriba o abajo
    3. Soltá en el lugar deseado

    Mover páginas¶

    1. En el sidebar, tocá y mantené el nombre de la página
    2. Arrastrá para:
    3. Cambiar el orden
    4. Moverla dentro de otra página (se convierte en subpágina)
    5. Sacarla de una página (la hacés página principal)

    Cambiar el ancho de las columnas¶

    En tablas o grupos:

    1. Arrastrá la línea entre columnas
    2. Soltá cuando esté del ancho que querés

    🎛️ Menú de página (ícono ⋮)¶

    Cuando tocás el menú de una página, vas a ver:

    Opción Función
    Duplicar Crear una copia exacta
    Mover a Mover la página a otro workspace
    Eliminar Mandar a la papelera
    Agregar a favoritos Aparecerá en acceso rápido
    Bloquear Evitar cambios accidentales
    Historial Ver versiones anteriores
    Agregar icono Poner un emoji o imagen
    Agregar portada Imagen de encabezado
    Copiar enlace Para compartir un link directo

    Tip pro: Agregá un icono y portada a cada cliente. Te ayuda a identificarlos rápido cuando tenés muchos.

    🏷️ Filtros y vistas (Favoritos y Propiedades)¶

    Agregar a favoritos¶

    Cuando marcás una página como favorita:

    • Aparece en la sección "Favoritos" del sidebar
    • Acceso mucho más rápido

    Para marcar como favorito:

    1. Tocá el menú ⋮ de la página
    2. Seleccioná "Agregar a favoritos"

    Propiedades de página¶

    Las propiedades son como "metadatos" de la página.

    Propiedades útiles para CM:

    • Estado: Activo / Pausado / Finalizado
    • Cliente: Nombre del cliente
    • Fecha de inicio: Cuándo arrancaste
    • Próxima reunión: Fecha y hora
    • Presupuesto: Cuánto cobra por ese cliente

    Para agregar propiedades:

    1. Tocá el nombre de la página
    2. Buscá "Propiedades"
    3. Agregá las que necesites

    📋 Plantillas rápidas¶

    Notion tiene plantillas prediseñadas. Accedés con:

    1. Tocá "Nuevo" o el botón "+"
    2. Buscá "Templates" o "Plantillas"
    3. Elegí una categoría:
    4. Proyecto
    5. Tareas
    6. Meeting notes
    7. Guía
    8. etc.

    Plantillas útiles para CM:¶

    • Meeting notes: Para guardar notas de reuniones con clientes
    • Task list: Para hacer seguimiento de pendientes
    • Project tracker: Para proyectos grandes
    • Calendar: Como alternativa visual al calendario editorial

    ✅ Checklist de navegación¶

    □ Puedo crear una página nueva
□ Sé usar "/" para agregar bloques
□ Puedo crear subpáginas
□ Usé la búsqueda para encontrar algo
□ Organicé páginas arrastrando
□ Agregué una página a favoritos
□ Agregué icono a una página
