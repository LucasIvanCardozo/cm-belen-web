    Módulo 06: Gestión de Tareas¶

    🎯 Objetivos del módulo¶

    Al terminar este módulo vas a: - ✅ Crear un sistema de tareas eficiente - ✅ Priorizar trabajo por cliente y deadline - ✅ Usar fechas límite y recordatorios - ✅ Hacer weekly reviews productivas - ✅ Automatizar con templates

    ✅ Por qué necesitás un sistema de tareas¶

    Como CM manejando 2-3 clientes, vas a tener: - Diseñar posts y reels - Escribir captions - Responder mensajes - Programar contenido - Hacer reportes - Reuniones semanales - Investigación y aprendizaje

    Sin sistema:
❌ Se te escapan cosas
❌ Trabajás en modo reactivo (siempre apagando incendios)
❌ No sabés qué hacer primero
❌ Terminás el día sin saber si hiciste bastante

Con sistema:
✅ Vés todo de un vistazo
✅ Sabés qué hacer primero (prioridades claras)
✅ Terminás con sensación de progreso
✅ No se te escapa nada

    🗂️ Crear tu base de tareas¶

    Paso 1: Crear la página¶

    1. Creá una página llamada "✅ Mis Tareas"
    2. Dentro, creá un bloque "Tabla" o "Board" (tablero kanban)

    Paso 2: Elegir entre Tabla o Board¶

    Tabla = mejor para ver todo junto, filtrar, ordenar Board = mejor para visualizar estados (Por hacer → Haciendo → Hecho)

    Recomendación: Usá los dos. Tené una vista de Tabla para todo y una vista de Board para tu semana.

    Paso 3: Propiedades de la base de tareas¶

    Propiedad Tipo Descripción
    Nombre de tarea Título Qué hay que hacer
    Cliente Relación Links a tu base de clientes
    Fecha límite Fecha Cuándo debe estar listo
    Prioridad Select 🔴 Alta / 🟡 Media / 🟢 Baja
    Estado Select Por hacer / En progreso / Hecho / Bloqueado
    Tipo Select Diseño / Copy / Estrategia / Admin / Reunión
    Tiempo estimado Número Minutos que lleva
    Etiquetas Multi-select Libre (urgente, follow-up, etc.)

    📋 Sistema semanal de trabajo¶

    Estructura de tu semana:¶

    MIS TAREAS SEMANALES
════════════════════

📅 LUNES
────────
• Revisar métricas de la semana anterior
• Planificar contenido de la semana
• Briefing con cliente X
• Diseñar posts semana

📅 MARTES
─────────
• Batch diseño: 5 posts
• Escribir copies
• Programa contenido en Later
• Responder mensajes

📅 MIÉRCOLES
────────────
• Revisión de campaigns
• Crear reels
• Community management
• Seguimiento pendientes

📅 JUEVES
─────────
• Batch diseño: 3 posts + 2 reels
• Reunión semanal con cliente Y
• Actualizar reportes
• Aprendizaje (1 hora)

📅 VIERNES
──────────
• Revisión final de semana
• Enviar reporte a clientes
• Planificar próxima semana
• Tareas pendientes de la semana

📅 SÁBADO-DOMINGO
─────────────────
• Descanso (¡importante!)
• Opcional: Inspiráte, planificá ideas
• NO: No hagas trabajo pesado

    🎯 Sistema de prioridades¶

    Matriz de Eisenhower:¶

                        URGENTE              NO URGENTE
              ┌──────────────────────┬──────────────────────┐
   IMPORTANTE │     Hacé esto         │   Agendá esto        │
              │     AHORA            │   Pero hacelo        │
              ├──────────────────────┼──────────────────────┤
NO IMPORTANTE │     Delegá esto      │     Eliminá esto     │
              │     o reducilo       │     No vale la pena  │
              └──────────────────────┴──────────────────────┘

    Cómo aplicar en Notion:¶

    PRIORIDAD ALTA (🔴) = Importante + Urgente
→ Hacer hoy, como primera cosa
→ Ejemplo: Cliente pidiendo cambio urgente

PRIORIDAD MEDIA (🟡) = Importante + No urgente  
→ Agendar para esta semana
→ Ejemplo: Diseñar posts de la próxima semana

PRIORIDAD BAJA (🟢) = No urgente + No importante
→ Hacer si te queda tiempo
→ Ejemplo: Organizar archivos

    📅 Tareas recurrentes¶

    Configurá estas tareas para que se repitan automáticamente:

    Diario:¶

    □ Revisar métricas de ayer (5 min)
□ Revisar notificaciones de clientes (10 min)
□ Respondé mensajes pendientes (15 min)

    Semanal:¶

    □ Weekly review (30 min, viernes)
□ Planificar contenido próxima semana (1 hora)
□ Reuniones con clientes (según agenda)

    Mensual:¶

    □ Hacer reportes de resultados (2 horas)
□ Revisión de estrategia (1 hora)
□ Facturación (30 min)

    Cómo crear tareas recurrentes:¶

    1. Creá una tarea normal
    2. Buscá la propiedad "Repetir" o "Recurrence"
    3. Configurá la frecuencia:
    4. Diariamente
    5. Semanalmente
    6. Mensualmente
    7. Días específicos de la semana

    📊 Weekly Review: Tu ritual semanal¶

    Por qué es importante:¶

    La weekly review es el ritual que separa a los CM organizados de los que reactúan. Toma 30 minutos y te da claridad para toda la semana.

    Cómo configurar el Board:¶

    1. En tu base de tareas, creá una nueva vista
    2. Seleccioná "Board"
    3. Agrupá por "Estado"
    4. Cada columna será un estado de tarea

    🔗 Vínculos con otras bases¶

    Tu sistema de tareas se conecta con todo lo demás:

    Mis Tareas
├── Relación con Clientes → Sabés qué tarea es para quién
├── Relación con Calendario → Coordinás con fechas de publicación
├── Relación con Reportes → Seguís el progreso de reportes

    Filtros útiles:¶

    Para ver solo tareas de un cliente:
→ Filtro: Cliente = [Nombre del cliente]

Para ver solo tareas de esta semana:
→ Filtro: Fecha límite = Esta semana

Para ver solo tareas urgentes:
→ Filtro: Prioridad = Alta

    ⏱️ Time blocking¶

    Concepto:¶

    En lugar de solo listar tareas, asignás bloques de tiempo específicos en tu agenda.

    Ejemplo:¶

    LUNES
─────
09:00 - 09:30  │ Revisión emails y métricas
09:30 - 11:30  │ BATCH DISEÑO: Posts para Later
11:30 - 12:00  │ Break
12:00 - 13:00  │ Escribir copies de la semana
13:00 - 14:00  │ Almuerzo
14:00 - 15:00  │ Community management (responder comentarios)
15:00 - 16:00  │ Reunión con Joyería Lumina
16:00 - 17:00  │ Revisión y ajustes finales

    En Notion:¶

    Podés usar una base de Time Blocks: - Propiedad: Hora de inicio - Propiedad: Hora de fin - Propiedad: Actividad - Propiedad: Cliente (relación)

    💡 Tips para gestión de tareas¶

    Productividad:¶

    • Hacé las tareas más duras primero (no la más fácil)
    • Usá el tiempo de espera (cola, transporte) para tareas pequeñas
    • Bacheá (agrupá tareas similares y hacelas seguidas)

    Organización:¶

    • Revisá tus tareas cada mañana (5 minutos)
    • Actualizá estados inmediatamente cuando cambie
    • Archivá las tareas hechas al final de cada semana

    Herramientas complementarias:¶

    • Later tiene su propio sistema de tareas → usalo solo para publicación
    • WhatsApp → para comunicación, no para trackear tareas
    • Notion → tu fuente de verdad central
