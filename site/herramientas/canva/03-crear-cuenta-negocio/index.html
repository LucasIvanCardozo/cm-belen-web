    ✨ 03. Crear tu Cuenta de Negocio: Brand Kit¶

    🎯 Objetivo¶

    Al terminar esta sección, vas a tener configurado tu Brand Kit en Canva con los colores, fuentes y logo de tu marca de joyería. Esto te permitirá crear diseños consistentes sin tener que elegir colores y fuentes cada vez.

    🏪 ¿Qué es un Brand Kit y Por Qué lo Necesitás?¶

    Un Brand Kit (Kit de Marca) es un conjunto de elementos que definen la identidad visual de tu marca:

    • Colores de marca
    • Fuentes de marca
    • Logo de la marca
    • Paletas predefinidas

    ¿Por qué importa para joyería?¶

    En el nicho de joyería y lujo, la consistencia visual es crítica. Tus clientes esperan una experiencia premium. Si tus posts tienen colores diferentes cada vez, fuentes que cambian, o un estilo visual inconsistente, la marca pierde valor percibido.

    Ejemplo:

    ❌ MAL (sin Brand Kit):
Post 1: Fondo dorado, tipografía script rosa, logo circular
Post 2: Fondo negro, tipografía sans-serif azul, logo cuadrado
Post 3: Fondo blanco, tipografía serif verde, sin logo

→ La marca parece amateur, los seguidores no la reconocen

✅ BIEN (con Brand Kit):
Post 1, 2, 3, 4...: Los mismos 3 colores, las mismas 2 fuentes,
                     mismo estilo de logo, mismo enfoque visual

→ La marca es reconocible, transmite profesionalismo y lujo

    🔧 Crear tu Brand Kit en Canva¶

    Nota: El Brand Kit completo está disponible en Canva Pro. Si usás la versión gratuita, podés crear algo similar manualmente creando una plantilla base con tus colores y fuentes guardados.

    Paso 1: Acceder al Brand Kit¶

    1. Tocá Perfil (👤).
    2. Buscá "Brand Kit" en el menú.
    3. Si tenés Canva Pro, aparecerá una sección especial.
    4. Si tenés Free, verás opciones limitadas.

    Paso 2: Agregar tus colores de marca¶

    1. En Brand Kit → Colores → "Agregar color".
    2. Ingresá el código HEX de tu color (ej: #D4AF37 para dorado).
    3. Nombra el color: "Dorado Joyería", "Negro Elegante", etc.
    4. Repetí para cada color de tu marca.

    Colores típicos para joyería de lujo:

    Color Código HEX Uso
    Negro #1A1A1A Fondos, texto principal
    Blanco #FFFFFF Fondos, espacio
    Dorado #D4AF37 Acentos, detalles premium
    Dorado claro #F5E6C8 Fondos suaves, brillos
    Gris elegante #4A4A4A Textos secundarios
    Champagne #F7E7CE Fondos cálido-lujosos

    Paso 3: Agregar tus fuentes¶

    1. Brand Kit → Fuentes → "Agregar fuente".
    2. Buscá la fuente que usás en tu marca.
    3. Marcá una como "Principal" (título).
    4. Marcá otra como "Secundaria" (cuerpo de texto).

    Fuentes recomendadas para joyería/lujo:

    Estilo Fuentes en Canva Personalidad
    Títulos premium Playfair Display, Montserrat, Cinzel Elegancia, sofisticación
    Body text Lato, Open Sans, Montserrat Legibilidad, modernidad
    Acentos/Logos Great Vibes, Parisienne, Allura Romance, feminidad (según el estilo)

    Pro tip: Limitá tu marca a máximo 2 fuentes: una para títulos (display) y una para texto (body). Mezclar más fuentes hace que el diseño se vea desordenado, especialmente en joyería donde el minimalismo es clave.

    Paso 4: Subir tu logo¶

    1. Brand Kit → Logo → "Subir logo".
    2. Elegí el archivo desde tu galería (PNG con fondo transparente es ideal).
    3. Ajustá si es necesario.

    Formato ideal para logos: - PNG con fondo transparente - Tamaño mínimo: 500 × 500 px - Fondo blanco o transparente (no usar JPG con fondo blanco)

    📝 Aplicar el Brand Kit a tus Diseños¶

    Una vez configurado, usar tu Brand Kit es fácil:

    Opción A: Desde una plantilla¶

    1. Crear nuevo → Explorar → buscar "joyería" o "lujo".
    2. Abrir la plantilla.
    3. Tocá el ícono de Brand Kit (probablemente un pincel o paleta).
    4. Seleccioná tu marca.
    5. La plantilla adopta tus colores y fuentes automáticamente.

    Opción B: Desde un diseño existente¶

    1. Abrí tu diseño.
    2. Tocá un texto o elemento de color.
    3. En el panel de color, buscá el ícono de "Marca" o "Mi marca".
    4. Seleccioná el color de tu Brand Kit.

    Opción C: Configurar antes de empezar¶

    1. Crear nuevo.
    2. Antes de agregar elementos, tocá "Aplicar marca" en el menú superior.
    3. Seleccioná tu Brand Kit.
    4. Todos los elementos nuevos usarán tu marca.

    Pro tip: En diseños para joyería, usá el color de fondo con sabiduría: - Fondo negro → transmite lujo y hace brillar las piezas doradas/-plateadas - Fondo blanco → limpieza y modernidad (bueno para plata y zirconias) - Fondo dorado/champagne → calidez y romanticismo (bueno para oro rosa) - Fondo transparente → máxima flexibilidad, se adapta a cualquier fondo de Instagram

    🖼️ Cómo Crear un Logo si No Tenés Uno¶

    Si todavía no tenés un logo profesional, podés crear uno básico en Canva:

    Opción 1: Usar un diseño existente¶

    1. Crear → Diseño personalizado → tamaño cuadrado (500×500).
    2. Agregar texto con el nombre de tu marca.
    3. Elegir una fuente elegante.
    4. Agregar un ícono decorativo (estrella, diamante, línea).
    5. Exportar como PNG con fondo transparente.

    Opción 2: Usar generador de logo de Canva¶

    1. En la pantalla de inicio, buscar "Logo" o "Generador de logo".
    2. Responder las preguntas (estilo, colores, nombre).
    3. Canva genera opciones.
    4. Elegir y personalizar.

    Pro tip: Para joyería, el logo ideal es: - Nombre de la marca en tipografía elegante (o solo el nombre si es muy conocido) - O un símbolo/discreto junto al nombre - Tipografía: Playfair Display, Montserrat, o similar - Evitar logos muy complejos — en formato pequeño (perfil de Instagram) se pierden los detalles

    ❌ Errores Comunes¶

    Error 1: Usar muchos colores¶

    Problema: Agregar 6+ colores al Brand Kit. Solución: Limitá a 3-5 colores máximo. En joyería, menos es más. Una paleta de 3 colores (negro, dorado, blanco) es más potente que una de 6 colores mezclados.

    Error 2: Mezclar fuentes incompatibles¶

    Problema: Poner Playfair Display + Comic Sans + Impact en el mismo diseño. Solución: Usá máximo 2 fuentes de estilos contrastantes. Si usás serif para títulos, usá sans-serif para body. Mantené coherencia.

    Error 3: Logo con fondo blanco en PNG¶

    Problema: Exportar el logo como PNG con fondo blanco sólido. Solución: Cuando exportés, elegí "Fondo transparente" o usá una herramienta para eliminar el fondo blanco antes de subirlo a Canva.

    Error 4: No guardar como plantilla¶

    Problema: Cada vez hacés el diseño desde cero. Solución: Una vez que tengas un diseño con tus colores de marca, guardalo como plantilla propia (menú ⋮ → "Guardar como plantilla").

    📌 Resumen¶

    • El Brand Kit centraliza tus colores, fuentes y logo para uso consistente.
    • Elegí 3-5 colores que representen tu marca de joyería.
    • Usá máximo 2 fuentes: una para títulos, una para body text.
    • El logo debe ser PNG con fondo transparente.
    • Aplicá tu Brand Kit a cada diseño para mantener coherencia visual.

    ✏️ Mini-Acción¶

    1. Creá un diseño nuevo (1080×1080).
    2. En el menú superior, buscá "Aplicar marca" o el ícono de paleta.
    3. Si tenés Pro, aplicá tu Brand Kit.
    4. Si tenés Free, configurá manualmente:
    5. Fondo: #1A1A1A (negro elegante)
    6. Texto: nombre de tu marca en Playfair Display
    7. Acento: una forma decorativa en dorado #D4AF37
    8. Guardá este diseño como "Plantilla Base Joyería".
    9. Duplicá el diseño para usarlo como base para futuros posts.
