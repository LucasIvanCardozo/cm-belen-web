    🔤 09. Texto y Tipografía¶

    🎯 Objetivo¶

    Al terminar esta sección, vas a saber trabajar con texto en Canva: elegir fuentes apropiadas para joyería, crear jerarquía visual, aplicar efectos, y diseñar textos que transmitan elegancia y profesionalismo.

    🔤 Tipos de Texto en Canva¶

    Cuando agregás texto, tenés tres opciones principales:

    Tipo Uso Tamaño típico
    Encabezado (Heading) Títulos principales, nombres de producto 80-150px
    Subencabezado (Subheading) Subtítulos, descripciones cortas 40-70px
    Cuerpo (Body) Texto largo, descripciones detalladas 20-35px

    Cómo agregar texto:¶

    1. Tocá 🔤 Texto en la barra de herramientas.
    2. Elegí el tipo:
    3. "Agregar encabezado" → texto grande
    4. "Agregar subencabezado" → texto mediano
    5. "Agregar texto" → texto pequeño
    6. "Agregar más texto" → otro bloque

    Pro tip: Tocá dos veces rápido sobre cualquier texto en el canvas para editarlo directamente. Este es el atajo ms rápido para cambiar textos.

    🎨 Fuentes Recomendadas para Joyería¶

    Categoría 1: Fuentes Elegantes/Serif (Premium, sofisticación)¶

    Fuente Estilo Mejor para
    Playfair Display Clásico, con serifs Títulos de lujo, nombres de marca
    Montserrat (variaciones bold) Moderno-elegante Títulos de productos
    Cinzel Romano, majestuoso Nombres de colección, fechas
    Bodoni Moda Editorial, moda Títulos fashion/lujo
    Cormorant Delicado, романтический Textos de colección, citas
    Lora Elegante, legible Descripciones de producto

    Categoría 2: Fuentes Limpias/Sans-Serif (Modernidad, claridad)¶

    Fuente Estilo Mejor para
    Montserrat Limpio, versátil Casi todo
    Lato Profesional, legible Body text, descripciones
    Poppins Friendly, moderno CTAs, textos cortos
    Raleway Elegante, minimalista Títulos modernos
    Proza Libre Legible, warmth Body text

    Categoría 3: Fuentes Decorativas (Solo uso limitado)¶

    Fuente Uso Cuidado
    Great Vibes Firmas, fechas, decoraciones Solo una o dos palabras
    Parisienne romantique, elegante Solo uso decorativo
    Allura Soft, femininity Solo como acento
    Pinyon Script Firma romántica Solo firmas

    Pro tip para Belen: Limitá tu diseño a máximo 2 fuentes: una serif elegante para títulos y una sans-serif limpia para body text. Esto mantiene la coherencia y transmite profesionalismo.

    📐 Jerarquía Visual con Texto¶

    La jerarquía le dice al usuario dónde mirar primero.

    Cómo crear jerarquía:¶

    ┌─────────────────────────────────────┐
│                                     │
│   💎 ANILLO AURORA                  │  ← Jerarquía 1: Título (ms grande, ms importante)
│                                     │
│   Colección Primavera 2026          │  ← Jerarquía 2: Subtítulo (ms pequeño, ms claro)
│                                     │
│   ████████████████████████████████  │  ← Separador visual
│                                     │
│   $890                              │  ← Jerarquía 3: Precio (destacado, pero no tan ms como el título)
│                                     │
│   Diseño exclusivo · 18K Oro        │  ← Jerarquía 4: Detalles (ms pequeño, informativo)
│                                     │
└─────────────────────────────────────┘

    Reglas de jerarquía:¶

    Regla Por qué
    Tamaño más grande = más importante El ojo va primero a lo ms grande
    Color más oscuro = más importante Contraste crea énfasis
    Posición superior = más importante Arriba = primero
    Solo un elemento главный Evita competencia visual

    🎨 Estilizar Texto¶

    Panel de opciones de texto:¶

    Cuando seleccionás texto, aparece este panel:

    ┌─────────────────────────────────────┐
│  Tipo: Encabezado        ▼         │  ← Cambiar tipo
│  Fuente: Playfair Display  ▼        │  ← Cambiar fuente
│  Tamaño: 72           ←──●───→     │  ← Ajustar tamaño
│                                     │
│  ════════════════════════════════ │
│  B   I   U   S̶   A̶   ≡   ⋮  AA     │  ← Formato rápido
│  Bold Ital Underline Strike Align More Font
│                                     │
│  Color: ██████  Fondo: ░░░░░        │  ← Colores
│                                     │
│  Efectos: Sombra  Desenfoque        │
│  Espaciado: Interletra             │
└─────────────────────────────────────┘

    Opciones de formato:¶

    Ícono Función Uso
    B Negrita Títulos, énfasis
    I Cursiva Nombres, citas
    U Subrayado rarely usado en diseño
    S̶ Tachado Precios antigos
    A̶ Mayúsculas Títulos, labels
    ≡ Alineación Left, center, right
    ⋮ Ms opciones Spacing, decorations
    AA Mayúsculas / Minúsculas Transformar texto

    Tamaño de fuente:¶

    Tipo de contenido Tamaño recomendado (1080×1080)
    Título principal 80-120px
    Subtítulo 40-60px
    Precio 60-80px
    Body text 25-35px
    Hashtags 20-25px
    Logo/marca Variable según diseño

    Pro tip: En Canva móvil, usá el slider de tamaño y arrastrá para ver el texto cambiar en tiempo real. Si el texto se ve pequeño en la preview de tu celular, probablemente también se verá pequeño en Instagram. Aproximadamente: lo que ves en tu celular a 1:1 es lo que se verá en el post.

    🌈 Color de Texto¶

    Cómo cambiar el color:¶

    1. Seleccioná el texto.
    2. Tocá el cuadrado de color (ej: blanco ██████).
    3. Aparecerá el selector de color:
    4. Paleta predefinida → colores de Canva
    5. Brand Kit → colores de tu marca
    6. Selector de color → elegí cualquier color
    7. Código HEX → ingresá el color manualmente

    Colores recomendados para texto:¶

    Tipo de texto Colores recomendados
    Títulos sobre fondo oscuro Blanco #FFFFFF, Dorado #D4AF37, Champagne #F7E7CE
    Títulos sobre fondo claro Negro #1A1A1A, Gris oscuro #4A4A4A, Dorado #D4AF37
    Body text Gris claro #6B6B6B, Blanco sobre oscuro
    Precios Dorado #D4AF37, Blanco, Negro

    Regla de contraste:¶

    Fondo Mejor color de texto
    Negro Blanco, dorado, champagne
    Blanco Negro, gris oscuro
    Dorado/Champagne Negro, blanco
    Foto oscura Blanco, dorado (con sombra)
    Foto clara Negro (con sombra o outline)

    Pro tip: Si ponés texto sobre una foto, agregá una sombra de texto o un outline para mejorar la legibilidad. El contraste entre texto e imagen debe ser alto.

    ✨ Efectos de Texto¶

    Sombra (Shadow)¶

    1. Seleccioná el texto.
    2. Tocá Efectos → Sombra.
    3. Ajustá:
    4. Desenfoque: qué tan suave es la sombra (0 = nítida, más = ms difusa)
    5. Color: generalmente negro o blanco semi-transparente
    6. Opacidad: qué tan visible es la sombra
    7. Desplazamiento: qué tan lejos está la sombra del texto

    Configuración recomendada para joyería:

    Sombra suave:
- Desenfoque: 4-8
- Opacidad: 30-50%
- Color: Negro #000000 (o blanco si el fondo es oscuro)
- Desplazamiento: 2-4px

    Outline (Contorno)¶

    1. Seleccioná el texto.
    2. Buscá "Contour" o "Outline" en opciones avanzadas.
    3. Ajustá grosor y color.

    Usos: Texto sobre fotos complejas, texto blanco sobre fondo claro.

    Otros efectos disponibles:¶

    Efecto Uso
    Desenfoque Texto decorativo de fondo
    Glow Efecto neón, moderno
    Relieve Efecto 3D
    Distorsión Texto creativo (usar con cuidado)

    📏 Espaciado de Texto¶

    Interletra (Letter Spacing)¶

    Controla el espacio entre letras.

    Valor Efecto Uso
    -2 a 0 Ms compacto Títulos cortos, elegancia
    0 Normal Texto general
    +1 a +3 Ms aireado Títulos largos, moderno
    +5 a +10 Mucho espacio Estilo minimalista

    Pro tip: Para títulos en joyería, usá interletra +1 a +3. Esto le da aire y sofisticación al texto. Evitá interletra negativa en fuentes muy pequeñas porque se vuelve ilegible.

    Interlineado (Line Height)¶

    Controla el espacio entre líneas.

    Valor Efecto Uso
    0.8 a 1.0 Ms compacto Textos cortos
    1.2 a 1.5 Normal Texto general
    1.5 a 2.0 Ms espacioso Quotes, textos largos

    Espacio entre letras y palabras:¶

    Ajuste Qué controla
    Letter Spacing Espacio entre cada letra
    Word Spacing Espacio entre palabras
    Line Height Espacio entre líneas
    Paragraph Spacing Espacio entre párrafos

    📝 Texto en Multiple Líneas¶

    Centrar texto verticalmente¶

    1. Seleccioná el bloque de texto.
    2. Tocá el ícono de alineación ≡.
    3. Elegí cómo alinear horizontalmente (left, center, right).
    4. Para centrar verticalmente: tocá ⋮ (ms opciones) → buscá alineación vertical.

    Texto en columnas¶

    1. Agregá un bloque de texto.
    2. Tocá ⋮ → "Columnas" o "Text columns".
    3. Elegí el número de columnas (2 o 3).
    4. Ajustá el espaciado entre columnas.

    Agregar línea decorativa¶

    1. Tocá 🎨 Elementos → "Líneas".
    2. Elegí una línea simple.
    3. Posicioná la línea sobre o debajo del texto.
    4. Ajustá el grosor y color.

    🚫 Qué Evitar en Texto¶

    Error Por qué Solución
    Demasiadas fuentes Causa ruido visual Máximo 2 fuentes
    Texto ms grande de lo necesario Pierde elegancia Lo justo, no lo ms
    Todos los textos igual de importantes Sin jerarquía Tamaños diferentes
    Colores que no contrastan Ilegible Contraste alto siempre
    Texto ms de 3 líneas No se lee en celular Reducir, usar ms slides si es necesario
    Fuentes decorativas en body text Ilegible Solo títulos cortos
    Palabras sin mayúsculas consistentes Luc amateur Usar mayúsculas al inicio de palabras importantes

    🔄 Transformar Texto¶

    El ícono AA te permite transformar texto:

    Transformación Ejemplo Uso
    Mayúsculas HOLA Títulos, labels
    Minúsculas hola Estilo casual
    Primera mayúscula Hola Texto normal
    Oración Hola Texto general
    versalitas HOLA Títulos elegantes

    ❌ Errores Comunes¶

    Error 1: Usar Comic Sans o fuentes genéricas¶

    Problema: Transmite amateurismo. Solución: Usá fuentes elegantes como Montserrat, Playfair Display, Lato. Evitá Arial, Comic Sans, Times New Roman para diseño de joyería.

    Error 2: Texto ms grande que la joya¶

    Problema: El texto compite con el producto. Solución: La joya es la protagonista. El texto es secundario. Si tenés que elegir, achicá el texto.

    Error 3: No revisar en el teléfono¶

    Problema: Se ve bien en Canva desktop pero mal en celular. Solución: Siempre mirá la preview en tu celular. Si no se lee claramente, ajustá.

    Error 4: Mezclar estilos de texto¶

    Problema: Negrita, cursiva, mayúsculas, todo junto. Solución: Usá solo negrita para énfasis. La cursiva puede funcionar para nombres o citas. No mezcles todo.

    Error 5: Texto sobre fondos complejos¶

    Problema: Ilegible por falta de contraste. Solución: Agregá overlay oscuro semi-transparente (30-40% negro) entre la foto y el texto. O poné el texto en zonas ms simples.

    📌 Resumen¶

    • Usá máximo 2 fuentes: serif elegante para títulos, sans-serif para body.
    • Creá jerarquía visual con tamaño y color.
    • El título principal debe ser el elemento ms grande y ms contrastado.
    • Sombra de texto mejora legibilidad sobre fotos.
    • Interletra +1 a +3 da aire y sofisticación.
    • Evitá Comic Sans, fuentes genéricas, y ms de 3 líneas de texto.

    ✏️ Mini-Acción¶

    1. Creá un diseño 1080×1080.
    2. Agregá una foto de una joya como fondo.
    3. Agregá texto con la siguiente jerarquía:
    4. Título: nombre de la pieza (Playfair Display, 90px, blanco, sombra suave)
    5. Subtítulo: colección (Montserrat, 40px, dorado)
    6. Precio: "$890" (Montserrat Bold, 60px, blanco)
    7. Detalle: "18K Oro Amarillo | Zafiro" (Lato, 25px, blanco 70%)
    8. Agregá una línea decorativa dorada entre el título y el subtítulo.
    9. Agregá tu @usuario en la esquina inferior.
    10. Aplicá sombra a todo el texto.
    11. Exportá y verificá que se lea claramente en tu celular.

    El texto bien diseñado transforma un diseño bueno en uno profesional. La tipografía es la voz de tu marca.
