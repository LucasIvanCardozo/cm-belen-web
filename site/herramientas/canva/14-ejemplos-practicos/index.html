    💎 14. Ejemplos Prácticos para Joyería¶

    🎯 Objetivo¶

    Al terminar esta sección, vas a tener plantillas y guías específicas para los tipos de contenido ms usados en marcas de joyería. Cada ejemplo incluye estructura, colores, fuentes y pasos para crear exactamente ese tipo de post.

    📸 Ejemplo 1: Post de Producto Individual¶

    Cuándo usarlo¶

    Cuando tenés una pieza nueva o quieres destacar un producto específico.

    Estructura¶

    ┌─────────────────────────────────────┐
│  [LOGO - esquina superior izq]      │
│                                     │
│  ═══════════════════════════════════│
│                                     │
│         [FOTO DEL PRODUCTO]         │
│         (Centro, ocupando 65%)       │
│                                     │
│  ═══════════════════════════════════│
│                                     │
│          💍 ANILLO SOLITARIO        │
│          Colección Aurora           │
│                                     │
│          ─────────────              │
│                                     │
│             $890                    │
│          18K Oro Amarillo            │
│          Zafiro azul de Ceilán       │
│                                     │
│  ────────────────────────────────── │
│                                     │
│              @joyeriaaurora          │
│                                     │
└─────────────────────────────────────┘

    Colores¶

    Elemento Color Código HEX
    Fondo Negro elegante #1A1A1A
    Título Blanco #FFFFFF
    Subtítulo Dorado #D4AF37
    Precio Blanco #FFFFFF
    Detalles Blanco 70% #FFFFFF con 70% opacidad
    Línea Dorado #D4AF37

    Fuentes¶

    Elemento Fuente Estilo
    Título producto Playfair Display Bold
    Colección Montserrat Regular, uppercase
    Precio Montserrat Bold
    Detalles Lato Regular

    Pasos para crear¶

    1. Nuevo diseño → 1080×1080
    2. Fondo → Color negro #1A1A1A
    3. Subir foto del producto → centrar y ajustar
    4. Línea decorativa → horizontal, dorada, poner arriba del texto
    5. Nombre del producto → Playfair Display, blanco, centrado
    6. Colección → Montserrat, dorado, uppercase
    7. Línea → dorada, separar nombre y precio
    8. Precio → Montserrat Bold, blanco, grande
    9. Detalles → Lato, blanco 70%, pequeño
    10. Logo → esquina inferior, @tuMarca en dorado o blanco

    🎠 Ejemplo 2: Carrusel Educativo (5 Tips)¶

    Cuándo usarlo¶

    Para dar contenido de valor: tips de cuidado, cómo elegir piezas, guías educativas.

    Estructura completa¶

    Slide 1 (Portada): 

    ┌─────────────────────────────────────┐
│                                     │
│  [FONDO: Negro #1A1A1A]              │
│                                     │
│              ✨                      │
│                                     │
│    CÓMO CUIDAR TUS                 │
│    JOYAS DE ORO                    │
│                                     │
│    5 Tips que necesitas saber       │
│                                     │
│         → Deslizá                   │
│                                     │
└─────────────────────────────────────┘

    Slides 2-5 (Cada tip): 

    ┌─────────────────────────────────────┐
│                                     │
│  [FONDO: Negro #1A1A1A]             │
│                                     │
│     💎  TIP 1                       │
│     ─────────────────               │
│                                     │
│     [ILUSTRACIÓN O FOTO]            │
│     (ícono de joyero, o foto)        │
│                                     │
│     No dormir con tus joyas         │
│     El roce constant puede          │
│     dañar losengaces y doblar       │
│     los cadenas                    │
│                                     │
│                                     │
└─────────────────────────────────────┘

    Slide 6 (CTA): 

    ┌─────────────────────────────────────┐
│                                     │
│  [FONDO: Negro #1A1A1A]             │
│                                     │
│        💎💎💎💎💎                   │
│                                     │
│     Guardá este post para           │
│     no perder estos tips            │
│                                     │
│     Compartí con alguien            │
│     que lo necesite                  │
│                                     │
│     @joyeriaaurora                   │
│     #cuidadojoyas #joyeriadelujo    │
│                                     │
└─────────────────────────────────────┘

    Pasos para crear¶

    1. Nuevo diseño → 1080×1080
    2. Crear el slide 1 completo (portada)
    3. Duplicar el slide 5 veces
    4. Editar cada slide:
    5. Cambiar el número del tip
    6. Cambiar la ilustración/foto
    7. Cambiar el texto del tip
    8. Editar el slide final:
    9. Cambiar a mensaje de CTA
    10. Agregar hashtags

    Tips de contenido¶

    Tip # Tema Ejemplo
    1 Almacenamiento "Guardá cada pieza por separado"
    2 Limpieza "Usá agua tibia y jabón suave"
    3 Actividades "Quitá las joyas para hacer ejercicio"
    4 Quimicos "Evitá el contacto con perfume y crema"
    5 Revisión "Llevá tus piezas a revisar annually"

    🎬 Ejemplo 5: Cover de Reel Tutorial¶

    Cuándo usarlo¶

    Para reels educativos o de demostración.

    Estructura¶

    ┌─────────────────────────────────────┐
│                                     │
│  [FOTO DEL PRODUCTO / RESULTADO]    │
│  (Close-up atractivo)               │
│                                     │
│  ────────────────────────────────   │
│                                     │
│  CÓMO LIMPIAR                      │
│  TUS JOYAS DE ORO                   │
│  EN CASA                           │
│                                     │
│  ────────────────────────────────   │
│                                     │
│  [NOMBRE DE MARCA]                  │
│                                     │
│  💎Tutorial | 2 min                 │
│                                     │
└─────────────────────────────────────┘

    Pasos para crear¶

    1. Nuevo diseño → 1080×1920
    2. Agregar foto del producto o resultado final
    3. Agregar overlay oscuro sutil (20-30% negro) sobre la foto
    4. Texto "CÓMO" → Playfair Display, blanco, uppercase
    5. Nombre de la acción → Montserrat Bold, dorado
    6. Nombre de marca → pequeño, esquina inferior
    7. "Tutorial | X min" → pequeño, debajo de la marca

    📚 Ejemplo 6: Post Colección¶

    Cuándo usarlo¶

    Para mostrar varias piezas de una colección o línea.

    Estructura (Grid 2×2)¶

    ┌─────────────────────────────────────┐
│                                     │
│  ┌───────────┬───────────┐         │
│  │           │           │         │
│  │  [FOTO 1] │  [FOTO 2] │         │
│  │           │           │         │
│  ├───────────┼───────────┤         │
│  │           │           │         │
│  │  [FOTO 3] │  [FOTO 4] │         │
│  │           │           │         │
│  └───────────┴───────────┘         │
│                                     │
│  COLECCIÓN AURORA                   │
│  Primavera 2026                     │
│                                     │
│  ✨ 4 piezas · Oro 18K             │
│                                     │
│  @joyeriaaurora                     │
│                                     │
└─────────────────────────────────────┘

    Pasos para crear¶

    1. Nuevo diseño → 1080×1080
    2. Agregar Grid → buscar "2x2" o "Cuadrícula"
    3. Agregar fotos en cada celda
    4. Ajustar cada foto con pellizcar y arrastrar
    5. Texto debajo del grid: nombre de colección, descripción
    6. Logo → esquina inferior

    📐 Ejemplo 8: Post Antes/Después¶

    Cuándo usarlo¶

    Para mostrar la diferencia de calidad o el resultado de una limpieza/reparación.

    Estructura¶

    ┌─────────────────────────────────────┐
│                                     │
│  ┌───────────┬───────────┐         │
│  │           │           │         │
│  │  ANTES   │ DESPUÉS  │         │
│  │           │           │         │
│  │  [Foto]   │  [Foto]   │         │
│  │           │           │         │
│  │           │           │         │
│  └───────────┴───────────┘         │
│                                     │
│  ¿Ves la diferencia?                │
│  Así se ve la calidad real          │
│                                     │
│  ✨ Joyería artesanal              │
│  💎 Hecho a mano                   │
│                                     │
│  @joyeriaaurora                     │
│                                     │
└─────────────────────────────────────┘

    Tips para este formato¶

    • Usá fotos con la misma iluminación y ángulo
    • Centrá las fotos para que se alineen bien
    • Poné "ANTES" y "DESPUÉS" bien visible
    • El mensaje debe ser claro: calidad, transformación

    🗓️ Calendario de Contenido Sugerido¶

    Para una semana típica:¶

    Día Tipo de contenido Formato
    Lunes Post de producto 1080×1080
    Martes Story interactiva Pregunta/Poll
    Miércoles Carrusel educativo 5 slides
    Jueves Reel/Tutorial Cover para reel
    Viernes Quote motivacional 1080×1080
    Sábado Detrás de cámaras Story
    Domingo Repost de cliente 1080×1080

    Ajustá según tu estrategia:¶

    • Si publicás menos: priorizá posts de producto y carruseles educativos.
    • Si tenés más tiempo: mezclá stories interactivas y contenido de valor.

    📋 Checklist Antes de Publicar¶

    □ ¿La foto es de alta resolución?
□ ¿Los colores son de mi paleta de marca?
□ ¿Las fuentes son las correctas?
□ ¿El texto se lee claramente?
□ ¿El logo está posicionado correctamente?
□ ¿Los márgenes son seguros (150px)?
□ ¿Exporté en calidad alta?
□ ¿Revisé la imagen en mi celular?

    📌 Resumen¶

    • Tené plantillas para cada tipo de contenido: producto, carrusel, story, cover, quote.
    • Mantené coherencia visual entre todos los posts.
    • Usá formatos específicos para cada plataforma.
    • Creá contenido educativo para posicionar tu marca.
    • Mezclá producto + interacción + valor.

    ✏️ Mini-Acción¶

    1. Creá las 4 plantillas siguientes:
    2. Post producto individual
    3. Carrusel educativo (5 tips)
    4. Story lanzamiento (3 slides)

    5. Cover reel tutorial

    6. Guardá cada una como plantilla con nombre descriptivo.

    7. Usá una plantilla esta semana para publicar tu primer post.

    8. Evaluá qué funcionó y qué ajustar.

    La clave es tener las plantillas listas para no perder tiempo. Una vez que tenés las bases, solo cambiás contenido y publicás.
