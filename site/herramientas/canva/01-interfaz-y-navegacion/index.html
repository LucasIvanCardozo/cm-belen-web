    📱 01. Interfaz y Navegación de Canva Móvil¶

    🎯 Objetivo¶

    Al terminar esta sección, vas a conocer cada parte de la interfaz de Canva en tu celular, entender cómo moverte entre funciones, y saber dónde encontrar lo que necesitás para crear contenido de joyería sin perder tiempo buscando opciones.

    📲 La Pantalla Principal¶

    Cuando abrís Canva, lo primero que ves es la pantalla de inicio (o "Home"). Esta es tu base de operaciones.

    🧭 La Barra Inferior de Navegación¶

    En la parte baja de la pantalla tenés cinco íconos principales:

    Ícono Nombre Para qué sirve
    🏠 Home Inicio Volver a la pantalla principal
    ➕ Crear Crear Nuevo diseño desde cero o plantilla
    🔍 Explorar Explorar Buscar plantillas, fotos, elementos
    💼 Proyectos Proyectos Ver todos tus diseños guardados
    👤 Perfil Perfil Configuración, cuenta, ajustes 
    ┌─────────────────────────────────────┐
│                                     │
│           (contenido)               │
│                                     │
├─────────────────────────────────────┤
│  🏠    ➕    🔍    💼    👤         │
│ Inicio  Crear Explorar Proyectos Perfil
└─────────────────────────────────────┘

    Pro tip: Tocá y mantené presionado el botón ➕ para acceder a opciones adicionales como "Importar" (subir tu propia foto) o "Escanear documento".

    🎨 El Editor de Diseños (La Pantalla Principal de Trabajo)¶

    Cuando abrís o creás un diseño, aparece el editor. Aquí es donde vas a pasar la mayor parte del tiempo.

    Partes del editor:¶

    ┌─────────────────────────────────────┐
│ ← Volver    Nombre del diseño    ⋮  │  ← Header del editor

│                                     │

    Herramientas del editor:¶

    Herramienta Ícono Función
    Fotos 📷 Agregar imágenes desde tu galería o Canva
    Texto 🔤 Agregar títulos, subtítulos, body text
    Elementos 🎨 Formas, stickers, líneas, decorativos
    Grids 📐 Marco para colocar fotos ordenadamente
    Medios 🖼️ Videos y GIFs

    Pro tip: La barra inferior cambia según lo que tengas seleccionado. Si tocás un texto, aparecerán opciones de edición de texto. Si tocás una foto, aparecerán opciones de edición de imagen.

    🔧 El Menú de Tres Puntos (⋮)¶

    Arriba a la derecha del editor hay un menú con opciones adicionales:

    Opción Para qué sirve
    Deshacer Volver atrás un paso
    Rehacer Repetir lo que deshiciste
    Duplicar Copiar el diseño completo
    Mover a carpeta Organizar tus diseños
    Compartir Enviar el diseño o invitar colaboradores
    Descargar Exportar en diferentes formatos
    Configurar página Cambiar tamaño del canvas
    Eliminar Borrar el diseño
    Ver versión anterior Si usaste magia de diseño

    Pro tip: El gesto de "agitar" tu teléfono activa el deshacer/rehacer rápidamente. Probá agitar el celular hacia abajo después de un cambio.

    🖐️ Gestos Táctiles Básicos¶

    Gesto Acción
    Toque simple Seleccionar elemento
    Toque prolongado Abrir menú contextual
    Arrastrar Mover elemento
    Pellizcar Acerca/aleja el zoom del canvas
    Doble toque Editar texto directamente
    Arrastrar desde esquina Cambiar tamaño (mantené shift para mantener proporción)
    Deslizar con dos dedos Rotar elemento

    🔄 Navegación entre Páginas (Diseños de varias páginas)¶

    Para diseños con varias páginas (como carruseles), aparecen miniaturas en la parte inferior:

    ┌─────────────────────────────────────┐
│                                     │
│           (canvas)                  │
│                                     │
├─────────────────────────────────────┤
│  1  2  3  4  5  6  ...             │  ← Miniaturas de páginas
│  ▲                                │
│  Página actual                     │
└─────────────────────────────────────┘

    Funciones de páginas múltiples:¶

    • Agregar página: Tocá el ➕ al final de las miniaturas
    • Duplicar página: Toque prolongado en miniatura → "Duplicar"
    • Eliminar página: Toque prolongado → "Eliminar"
    • Reordenar: Arrastrá la miniatura a nueva posición

    ⚙️ Configuración Rápida desde Perfil¶

    Desde el ícono de perfil (👤) accedés a:

    Sección Para qué
    Configuración de la cuenta Idioma, zona horaria, email
    Preferencias de la app Calidad de subida, guardado automático
    Canva Pro Suscripción, funciones desbloqueadas
    Brand Kit (Pro) Configurar colores, logos, fuentes de marca
    Plantillas guardadas Tus favoritos
    Historial de diseños Diseños eliminados recoverable
    Ayuda Tutoriales, soporte

    Pro tip: En Configuración → Preferencias de la app, active "Guardar automáticamente cada 30 segundos". Te salva de perder trabajo si se cierra la app.

    ❌ Errores Comunes¶

    Error 1: Perderse en la app¶

    Problema: No saber cómo volver al inicio después de abrir un diseño. Solución: Tocá la flecha ← en la esquina superior izquierda. Siempre te lleva al menú anterior.

    Error 2: No encontrar funciones¶

    Problema: Buscar opciones donde no están. Solución: Recordá que el menú cambia según lo que tengas seleccionado. Si no ves opciones de texto, tocá primero el texto.

    Error 3: TOCAR sin querer y perder configuración¶

    Problema: Un toque accidental cambia el elemento seleccionado. Solución: Usá el botón "Deshacer" (↶) inmediatamente. O configurá "Seleccionar y luego editar" en preferencias.

    Error 4: No saber si se guardó¶

    Problema: Cerrar la app sin saber si el trabajo se perdió. Solución: Verificá que aparezca "Guardado" en el título del diseño. Si no aparece, tocá cualquier parte del canvas para que se active el auto-guardado.

    📌 Resumen¶

    • La Home es tu base: recentres, plantillas, acceso rápido.
    • La barra inferior tiene 5 secciones: Inicio, Crear, Explorar, Proyectos, Perfil.
    • El editor tiene herramientas contextuales que cambian según lo que seleccionás.
    • Los gestos táctiles (pellizcar, arrastrar, doble toque) son tu principal forma de interactuar.
    • El menú ⋮ tiene funciones avanzadas como deshacer, duplicar, descargar.
    • Para carruseles, las miniaturas inferiores te permiten navegar entre páginas.

    ✏️ Mini-Acción¶

    1. Abrí Canva en tu celular.
    2. Andá a Crear → Diseño personalizado.
    3. Buscá el ícono de "Tamaños" o "+ Tamaño" en la esquina inferior.
    4. Experimentá cambiando el tamaño del canvas entre diferentes formatos (cuadrado, historia, banner).
    5. Explorá la barra de herramientas sin miedo — no vas a romper nada.

    El objetivo es familiarizarte con la interfaz. En las siguientes secciones vamos a crear contenido real.
