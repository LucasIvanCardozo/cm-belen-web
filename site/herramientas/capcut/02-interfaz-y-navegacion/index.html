    Módulo 02: Interfaz y Navegación¶

    🎯 Objetivo del módulo¶

    Al terminar este módulo vas a:

    • Entender cada parte de la pantalla de edición
    • Saber dónde está cada herramienta
    • Moverte fluidamente entre funciones
    • No sentirte abrumada cuando abras CapCut

    📱 La pantalla de edición: Desglose completo¶

    Cuando creás un nuevo proyecto o abrís uno existente, vas a ver esta pantalla:

    ┌─────────────────────────────────────────────┐
│  ← Volver    [Nombre del proyecto]   Exportar│  ← Barra superior
├─────────────────────────────────────────────┤
│                                             │
│                                             │
│          ┌─────────────────────┐            │
│          │                     │            │
│          │    VISTA PREVIA     │            │  ← Área de preview
│          │    (tu video)       │            │
│          │                     │            │
│          └─────────────────────┘            │
│                                             │
│  00:15 ▶️ ▐▐ ◀️ ▶️ 🔊  ↻                     │  ← Controles de reproducción
├─────────────────────────────────────────────┤
│  ════════════════════════════════════════    │  ← Línea de tiempo

    🎬 La barra de herramientas (lo más importante)¶

    Esta es la barra que vas a usar el 90% del tiempo. Cada botón abre un mundo de opciones:

    🔴🔴🔴 HERRAMIENTAS PRINCIPALES¶

    Ícono Nombre Para qué sirve
    ✂️ Cortar/Editar Cortar, dividir, eliminar clips. MUY USADO
    ↩️ Deshacer Volver atrás si te equivocaste
    ↪️ Rehacer Volver a hacer lo que deshiciste
    📐 Recortar Cambiar el tamaño del video (zoom, pan)
    🖼️ Rotar Girar el video 90° a la vez
    🎯 Ajustar Brillo, contraste, saturación, temperatura
    🔍 Zoom Acercar o alejar la vista del timeline

    🟡🟡🟡 HERRAMIENTAS SECUNDARIAS¶

    Ícono Nombre Para qué sirve
    📝 Texto Agregar textos, subtítulos, títulos
    🎵 Audio Agregar música de fondo, efectos de sonido
    🎤 Voz en off Grabar tu voz sobre el video
    🖼️ Sticker Emojis, stickers decorativos
    📦 Overlay Agregar otro video/capa encima
    🎨 Filtros Filtros de color preestablecidos
    💫 Efectos Efectos visuales (destellos, partículas, etc.)
    🔀 Transiciones Cambios entre clips

    🟢🟢🟢 HERRAMIENTAS AVANZADAS¶

    Ícono Nombre Para qué sirve
    🧩 Collage Poner varios videos/fotos juntos
    🪞 Espejo Dar vuelta el video horizontalmente
    ⏱️ Velocidad Hacer el video más rápido o lento
    🔁 Bucle Repetir una sección
    🗑️ Eliminar fondo Quitar el fondo de una imagen (¡GRATIS!)
    🎚️ Ecualizador Ajustar frecuencias de audio

    ⏱️ El Timeline (Línea de tiempo)¶

    El timeline es donde armás tu video. Es como una línea horizontal donde cada clip está representado por un rectángulo.

    Estructura del timeline:¶

    Pista de VIDEO:
┌────────────┬────────────┬────────────┬────────────┐
│  Video 1   │  Video 2   │  Foto 1    │  Video 3   │
│  0s        │  5s       │  12s      │  18s       │
└────────────┴────────────┴────────────┴────────────┘
     ████████████

Pista de TEXTO:
┌────────────────────────┐
│  "Joyas que enamoran"  │
└────────────────────────┘
     ██████████████████

Pista de AUDIO:
┌────────────────────────────────────────────────┐
│  🎵 Música de fondo                            │
└────────────────────────────────────────────────┘
     ████████████████████████████████████████

    Cómo leer el timeline:¶

    Elemento Significado
    Ancho del rectángulo Duración del clip (más ancho = más largo)
    Colores Diferentes tipos de contenido (video, texto, audio)
    Barra vertical El momento exacto donde está "parada" la cabeza de reproducción
    Números Tiempo en segundos (00:00, 00:05, etc.)

    Acciones con el timeline:¶

    Acción Cómo hacerlo
    Seleccionar un clip Tocar el clip (se pone azul/celeste)
    Mover un clip Mantener presionado y arrastrar
    Acercar/alejar Pellizcar con dos dedos o usar el slider de zoom
    Ir a un momento Tocar la barra de reproducción arriba del timeline
    Cortar un clip Tocar el clip → tocar "Cortar"

    👆 Gestos básicos (MUY IMPORTANTE)¶

    En CapCut, los gestos reemplazan muchos botones. Dominarlos te hace mucho más rápida:

    Gestos en el área de preview:¶

    Gesto Acción
    Un dedo: tocar Reproducir/pausar
    Un dedo: arrastrar Mover el video en preview (cuando está miniaturizado)
    Dos dedos: pellizcar Zoom in/out en el preview
    Dos dedos: arrastrar Mover el video cuando está ampliado

    Gestos en el timeline:¶

    Gesto Acción
    Tocar clip Seleccionar
    Doble toque clip Abrir editor de ese clip específico
    Arrastrar borde Cortar el principio o final
    Mantener presionado Mover el clip
    Pellizcar Zoom del timeline

    📐 El Preview (Vista previa)¶

    El preview te muestra cómo se ve tu video en tiempo real. Cosas importantes:

    Controles del preview:¶

    🔊 Volumen: ajustá el volumen del preview
◀️▶️ Botón play: reproducir/pausar
↻ Rotar preview: girar la vista 90°
🔍 Zoom: acercar el preview
⬜ Pantalla completa: ver en full screen

    El botón de reproducir:¶

    Estado Significado
    ▶️ Blanco Listo para reproducir
    ⏸️ Blanco con rayas Pausado
    🔴 Rojo Grabando
    🔄 Girando Cargando

    🎛️ Los diferentes "editores"¶

    CapCut tiene diferentes lugares donde editás según lo que estés tocando:

    1. Editor de proyecto (principal)¶

    Cuando abrís un proyecto, es la pantalla completa con timeline.

    2. Editor de clip individual¶

    Cuando hacés doble tap en un clip, se abre un mini-editor solo para ese clip.tiene:

    • ✂️ Cortar/Recortar
    • 🎯 Ajustar (brillo, contraste, etc.)
    • 🎨 Filtros
    • 🔀 Transiciones
    • ⏱️ Velocidad
    • 🪞 Rotar/Voltear

    3. Editor de texto¶

    Cuando editás texto, aparece la barra de herramientas de texto con:

    • Tipografía
    • Tamaño
    • Color
    • Posición
    • Animación
    • Efectos

    4. Editor de audio¶

    Para música y efectos de sonido:

    • Volumen
    • Fade in/out
    • Velocidad
    • Efectos

    ❓ Cuando estás perdida: El botón "Deshacer"¶

    Regla de oro: Si no sabés qué hiciste o te arrepentiste:

    1. Buscá el botón ↩️ "Deshacer" (siempre está visible arriba)
    2. Tocalo todas las veces que necesites
    3. ¡No pasa nada! Podés deshacer hasta 50 acciones

    Truco: Mantené presionado "Deshacer" para ver el historial completo y volver a cualquier punto.
