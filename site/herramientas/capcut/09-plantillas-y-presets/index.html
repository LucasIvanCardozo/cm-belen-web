    Módulo 09: Plantillas y Presets¶

    🎯 Objetivo del módulo¶

    Aprender a usar plantillas para crear videos rápido y mantener una identidad visual consistente. También vas a aprender a crear tus propios presets para reutilizar ajustes de color y texto.

    Por qué es importante: - Las plantillas te ahorran 50-70% del tiempo de edición - Mantienen tu marca consistente - Te permiten publicar más contenido sin sacrificar calidad

    📋 ¿Qué son las plantillas?¶

    Definición simple¶

    Una plantilla es un proyecto prediseñado que podés usar como base. Ya tiene: - Texto Placeholder - Música - Transiciones - Filtros - Duración predefinida

    Vos solo reemplazás las fotos/videos y personalizás el texto.

    Tipos de plantillas en CapCut¶

    Tipo Descripción Dónde encontrarla
    Plantillas de video Estructura completa con animaciones "Plantillas" en pantalla principal
    Presets de color Ajustes de color guardados En ajustes de filtros
    Estilos de texto Combinaciones de fuente+color+animación En herramientas de texto

    🎨 Plantillas de video (el poder de CapCut)¶

    Cómo acceder a las plantillas¶

    1. En la pantalla principal de CapCut, tocá "Plantillas" o "Templates"
    2. Vas a ver miles de plantillas organizadas por categoría

    Categorías útiles para joyería¶

    Categoría Para qué sirve
    Trending Lo más popular ahora mismo
    Product Showcase Exhibición de productos
    Fashion Moda y accesorios
    Luxury Estilo premium
    Jewelry Específico para joyería
    Sale / Promo Ofertas y promociones
    Before & After Comparaciones
    Slideshow Presentación de fotos
    Intro / Outro Aperturas y cierres

    Cómo usar una plantilla¶

    Paso 1: Elegir la plantilla 1. Navegá por las categorías o buscá lo que necesites 2. Tocá la plantilla para previsualizarla 3. Tocá "Usar plantilla" o "Apply"

    Paso 2: Personalizar 1. CapCut abre la plantilla con placeholder de fotos/videos 2. Tocá "Reemplazar" en cada placeholder 3. Seleccioná tu foto/video de la galería 4. Personalizá el texto (tocá el texto placeholder y escribí el tuyo)

    Paso 3: Ajustar - Cambiá colores si querés - Ajustá duración de clips - Agregá o quitá elementos

    Paso 4: Exportar - Cuando esté listo, exportá normalmente

    ✨ Crear y guardar presets de color¶

    Los presets de color te permiten guardar tus ajustes favoritos para aplicarlos rápidamente a otros videos.

    Cómo guardar un preset¶

    Paso 1: Ajustá los colores 1. Editá un video con los ajustes que te gusten 2. Encontraste tu combinación perfecta: 

    Brillo: +8
Contraste: +15
Saturación: +12
Temperatura: +10

    Paso 2: Guardar el preset 1. En el menú de ajustes, buscá el ícono de "+" o "Guardar" o "Save" 2. Ponele un nombre, ej: "Joyería dorada premium" 3. Tocá "Guardar"

    Paso 3: Usar el preset 1. En otro video, andá a Ajustes 2. Buscá "Presets" o "Guardados" 3. Tocalo para aplicarlo instantáneamente

    Presets recomendados para guardar¶

    Preset: Oro cálido¶

    Brillo: +8
Contraste: +15
Saturación: +12
Temperatura: +10
    Para: Videos de oro amarillo, ambiente cálido

    Preset: Plata limpia¶

    Brillo: +5
Contraste: +18
Saturación: +8
Temperatura: -5
    Para: Videos de plata, acero, piedras blancas

    Preset: Piedras vibrantes¶

    Brillo: +5
Contraste: +12
Saturación: +18
Temperatura: 0
    Para: Videos de piedras de colores (rubí, esmeralda, zafiro)

    Preset: Blanco puro¶

    Brillo: +15
Contraste: +10
Saturación: +5
Temperatura: +5
    Para: Fondos blancos, fotos de catálogo

    📝 Crear estilos de texto guardados¶

    Si usás siempre el mismo estilo de texto, podés guardarlo.

    Cómo guardar un estilo de texto¶

    1. Agregá texto a cualquier proyecto
    2. Formatealo como quieras (fuente, color, tamaño, animación)
    3. En el menú de texto, buscá "Guardar estilo" o "Save style"
    4. Ponele nombre: "Título dorado" o "Subtítulo elegante"

    Estilos de texto para joyería¶

    Nombre Fuente Color Fondo Animación

    CTA elegante Montserrat Bold Blanco Ninguno Fade in
    Detalle técnico Poppins Gris Ninguno Ninguna
    Marca de agua Montserrat Blanco 70% Ninguno Ninguna

    🏗️ Crear tu propia plantilla desde cero¶

    Si ninguna plantilla existente te sirve, creá la tuya.

    Paso a paso¶

    1. Creá un proyecto nuevo
    2. Armá la estructura ideal:
    3. Agregá clips placeholder (fotos genéricas)
    4. Configurá transiciones
    5. Agregá texto placeholder con tus fuentes preferidas
    6. Agregá tu música de marca

    7. Configurá duración

    8. Guardá como plantilla:

    9. Tocá los tres puntitos ⋮ o ...
    10. Buscá "Guardar como plantilla" o "Save as template"

    11. La plantilla aparece en tu sección personal

    12. Usala:

    13. Andá a "Mis plantillas" o "Creados por mí"
    14. Encontrá tu plantilla
    15. Tocá "Usar"

    Qué incluir en tu plantilla ideal¶

    Para joyería, tu plantilla debería tener:
✓ 4-6 espacios para fotos/videos
✓ Texto de título placeholder
✓ Texto de subtítulo placeholder
✓ Espaço para CTA
✓ Música de fondo (o dejar vacío para variar)
✓ Transiciones consistentes (o ninguna)
✓ Filtro/preset preaplicado
✓ Duración total: 15-30 segundos

    💾 Guardar presets de filtros¶

    Además de los ajustes de color, podés guardar combinaciones de filtros.

    Cómo guardar preset de filtro¶

    1. Aplicá un filtro base
    2. Ajustá la intensidad
    3. Tocá el ícono de "+" junto al filtro
    4. Ponele nombre

    Combinaciones de filtro populares¶

    Nombre Filtro base + intensidad Ajustes adicionales
    Lujoso Clear + 30% Contraste +10
    Romántico Rose + 25% Saturación +8
    Moderno B&W + 15% Contraste +15
    Natural Natural + 40% Temperatura +5

    🔄 Flujo de trabajo eficiente con plantillas¶

    Método 1: Batch editing (edición por lotes)¶

    Ideado para: Crear contenido semanal consistentemente

    LUNES: 
1. Crear 5 proyectos desde plantillas
2. Solo reemplazar fotos/videos
3. Personalizar texto para cada producto
4. Guardar como borrador

MIÉRCOLES:
1. Revisar los 5 proyectos
2. Hacer ajustes finales
3. Exportar los 5
4. Programar para la semana

    ⚠️ Limitaciones de las plantillas¶

    Limitación Solución
    No podés cambiar estructura Elegí plantilla que se acerque a lo que necesitás
    Texto puede estar limitado Agregá más texto manualmente después
    Música no siempre se puede quitar Usá la música o silenciarla
    Puede tener marca de agua Buscá plantillas sin marca o pagá versión Pro
    No siempre editable Podés guardar como proyecto nuevo y editar más
