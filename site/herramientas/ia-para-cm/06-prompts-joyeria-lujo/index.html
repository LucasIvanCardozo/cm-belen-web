    06 — Prompts para Joyería y Lujo¶

    Tu Biblioteca Personal de Prompts Optimizados¶

    Qué vas a aprender¶

    • Cómo escribir mejores prompts
    • Prompts específicos para joyería
    • Prompts para diferentes tipos de contenido
    • Guardar y organizar tu biblioteca
    • Iterar y mejorar tus prompts

    1. La anatomía de un buen prompt¶

    Estructura básica:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                          │
│   UN BUEN PROMPT TIENE:                                 │
│                                                          │
│   1. ROL → Quién sos (contexto)                        │
│   2. TAREA → Qué querés                                │
│   3. CONTEXTO → Detalles importantes                   │
│   4. FORMATO → Cómo lo querés                         │
│   5. RESTRICCIONES → Qué evitar                        │
│                                                          │
└─────────────────────────────────────────────────────────┘

    Ejemplo de prompt bien estructurado:¶

    ROL: Sos un copywriter especializado en marketing de 
     joyería de lujo para Latinoamérica.

CONTEXTO: Estoy escribiendo captions para una joyería 
boutique en Buenos Aires que vende piezas de oro 18k 
para mujeres profesionales 30-45 años.

TAREA: Generá 5 captions para un post de collar nuevo.

FORMATO: Cada caption debe tener:
         - Hook en la primera línea
         - 100-150 palabras
         - Tono elegante pero accesible
         - Incluir un CTA sutil

RESTRICCIONES: Evitar cliché romántico básico, 
               frases genéricas como "hermoso" o "lindo".

    2. Prompts para Captions¶

    2.1 Captions de producto¶

    Prompt: 

    Sos copywriter de lujo para una joyería boutique en 
Argentina especializada en piezas de oro 18k.

Escribí [NÚMERO] captions para [TIPO DE PRODUCTO].

Contexto:
- Audience: [DESCRIPCIÓN]
- Tono: [TONO DESEADO]
- Precio point: [PUNTO DE PRECIO]

Formato:
- Hook diferente en cada uno
- Longitud variada
- Incluir CTA

Evitar: [COSAS A EVITAR]

    2.2 Captions emotivos¶

    Prompt: 

    Generá captions que cuenten historias emotivas sobre 
joyería.

Para cada uno:
- Creá una mini-historia (2-3 oraciones)
- Conectá la historia con el producto
- Incluí insight sobre por qué las joyas importan
- Terminate con reflexión o pregunta

Tema: [TEMA ESPECÍFICO]
Tono: [TONO]

    2.3 Captions educativos¶

    Prompt: 

    Creá captions educativos sobre joyería que agreguen 
valor real al follower.

Incluí:
- Un dato interesante o trivia
- Información útil sobre materiales/cuidado
- Por qué importa para el consumidor
- CTA que invite a guardar/comentar

Tema: [TEMA]
Nivel de conocimiento: principiante/intermedio/avanzado

    3. Prompts para Estrategia¶

    3.1 Análisis de cuenta¶

    Prompt: 

    Analizá esta estrategia de contenido para Instagram 
de una joyería y respondé:

1. ¿Qué tipo de contenido funciona mejor?
2. ¿Cuál es el balance entre producto/engagement/branding?
3. ¿Qué está faltando?
4. ¿Qué 3 cambios recomiendas para mejorar engagement?
5. ¿Qué hashtags debería usar más?

[DESCRIPCIÓN O URL DE LA CUENTA]

    3.2 Planificación mensual¶

    Prompt: 

    Planificá el contenido de Instagram para [TIPO DE JOYERÍA] 
para [MES A PLANIFICAR].

Incluí:
- [NÚMERO] posts para el mes
- Distribución por tipo de contenido
- Temas específicos para cada semana
- Campaña o highlight del mes
- Hashtags principales

Goals: [OBJETIVOS DEL MES]

    3.3 Análisis de métricas¶

    Prompt: 

    Tengo estos datos de mi cuenta de Instagram de joyería:
[PEGAR MÉTRICAS]

Analizame:
1. ¿Qué está funcionando bien?
2. ¿Qué necesita mejorar?
3. ¿Qué debería hacer diferente el próximo mes?
4. ¿Qué benchmarks debería buscar?

Sé específico con recomendaciones.

    4. Prompts para Ideas de Contenido¶

    4.1 Banco de ideas general¶

    Prompt: 

    Generá [NÚMERO] ideas de contenido para Instagram de 
una joyería.

Para cada idea incluí:
- Tipo de post (foto, reel, carrusel, video)
- Tema/título
- Brief de qué mostrar
- Por qué funcionaría (rationale)

Incluí variedad:
- Producto directo
- Lifestyle
- Educativo
- Behind the scenes
- Storytelling
- Engagement
- User generated

    4.2 Ideas para campañas¶

    Prompt: 

    Brainstorming de campañas para [TEMPORADA/EVENTO] 
para una joyería.

Para cada campaña:
- Nombre de campaña
- Concepto/idea central
- 5 posts del content plan
- Mensajes clave
- Call to action
- Hashtags de campaña

[TEMPORADA/EVENTO]
[JOYERÍA TIPO]

    4.3 Contenido para cada pilar¶

    Prompt: 

    Generá 10 ideas de contenido para el pilar "[NOMBRE DEL PILAR]" 
de una joyería.

Cada idea debe:
- Ser ejecutable por 1 persona con celular
- Diferente de las otras
- Resonar con audiencia [DESCRIPCIÓN]
- Poder escalarse a serie si funciona

    5. Prompts para Storytelling¶

    5.1 Historias de marca¶

    Prompt: 

    Creá contenido de storytelling para una joyería familiar 
[DESCRIPCIÓN BREVE DE LA HISTORIA].

Generá:
1. 3 posts de "cómo empezó la marca"
2. 2 posts de valores/misión
3. 2 posts de behind the scenes
4. 2 posts de testimonios/historias de clientes

Cada uno con:
- hook
- desarrollo
- cierre
- CTA sugerido

    5.2 Storytelling de producto¶

    Prompt: 

    Contá la historia detrás de [PRODUCTO TIPO/PIEZA ESPECÍFICA].

Incluí:
- Origen del diseño o inspiración
- Proceso de creación
- Qué lo hace especial
- Para quién es ideal
- Momento perfecto para usarlo

Formato: mini-post de storytelling (200 palabras)

    5.3 Historias de clientes¶

    Prompt: 

    Creá [NÚMERO] posts ficticios de "historias de clientes" 
para una joyería.

Cada historia debe:
- Ser relatable para la audiencia
- Conectar emocionalmente
- Naturalmente integrar el producto
- Terminar con reflexión

Contexto: [TIPO DE JOYERÍA, AUDIENCIA]

    6. Prompts para Joyería Específicos¶

    6.1 Para oro¶

    Prompt: 

    Soy CM de una joyería que vende joyería de oro 18k y 24k.
Necesito contenido sobre el oro.

Generá ideas para:
1. Posts educativos sobre tipos de oro
2. Cómo cuidar joyería de oro
3. Por qué invertir en oro
4. Tendencias en joyería de oro
5. El valor sentimental del oro

Audience: [DESCRIPCIÓN]

    6.2 Para plata¶

    Prompt: 

    Soy CM de una joyería especializada en plata 925.
Necesito contenido sobre plata.

Ideas para:
1. Beneficios de la plata 925
2. Cómo identificar plata real
3. Estilos populares en plata
4. Cuidado y mantenimiento
5. Combinaciones con outfits

Audience: [DESCRIPCIÓN]

    6.3 Para piedras¶

    Prompt: 

    Creá contenido educativo sobre [TIPO DE PIEDRA] para una 
joyería.

Incluí:
- Origen/historia de la piedra
- Significado y simbolismo
- Qué buscar en joyería con esta piedra
- Cuidado específico
- Tendencias actuales

Audience: [DESCRIPCIÓN]

    7. Prompts para Temporadas/Campañas¶

    7.1 San Valentín¶

    Prompt: 

    Planificá una campaña de San Valentín para una joyería.

Incluí:
- Concepto/nombre de campaña
- 3 fases (anticipación, día D, post)
- [NÚMERO] posts por fase
- Mensajes clave
- CTA principales
- Hashtags de campaña
- Ideas para stories

Audience: [DESCRIPCIÓN]

    7.2 Día de la Madre¶

    Prompt: 

    Campaña para Día de la Madre para una joyería boutique.

Target: [DESCRIPCIÓN DE AUDIENCIA]

Generá:
- Concepto de campaña
- 10+ ideas de contenido
- Mensajes según relación (hija/hijo que regala, autoregalo,etc)
- CTA para venta
- Hashtags

    7.3 Navidad/Fin de año¶

    Prompt: 

    Estrategia de contenido para Navidad/Fin de año para 
joyería.

Incluí:
- Timeline de campaña (cuándo empezar, peak, post-navidad)
- Tipos de contenido por fase
- mensajes para regalo vs autoregalo
- Estrategias de urgencia/scarcity
- Ideas para contenido evergreen post-navidad

    8. Prompts para Engagement¶

    8.1 Preguntas y polls¶

    Prompt: 

    Generá [NÚMERO] ideas para aumentar engagement en Instagram 
de una joyería.

Incluí:
- Preguntas para captions
- Ideas para stories (polls, quizzes, preguntas)
- Desafíos o trends adaptados a joyería
- Contenido que invite a comentar

Audience: [DESCRIPCIÓN]

    8.2 User generated content¶

    Prompt: 

    Estrategia para generar user generated content (UGC) 
para una joyería.

Ideas para:
- Cómo invitar a clients a compartir
- Qué pedir que compartan
- Cómo usar UGC en mi feed
- Incentivos para compartir
- Hashtags de comunidad

[JOYERÍA TIPO]

    10. Tips para mejorar prompts¶

    Iteration:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                          │
│   ITERAR = OBTERNER MEJORES RESULTADOS                 │
│                                                          │
│   Prompt v1 → Revisar resultado → Ajustar → Prompt v2  │
│                                                          │
│   Preguntas para iterar:                                │
│   ├── ¿El resultado fue lo que esperaba?               │
│   ├── ¿Faltó algo específico?                        │
│   ├── ¿Sobró algo?                                    │
│   ├── ¿El tono era correcto?                          │
│   └── ¿Cómo lo mejoro?                                │
│                                                          │
└─────────────────────────────────────────────────────────┘

    Phrases útiles para ajustar:¶

    • "Hacé esto más corto/más largo"
    • "Cambiá el tono a más [profesional/casual/emotivo]"
    • "Agregá más detalle sobre [tema]"
    • "Enfocá más en [aspecto específico]"
    • "Eliminá [cosa específica]"
    • "Dame solo [cantidad] opciones"
    • "Incluí un elemento humorístico"
    • "Hacé esto más aspiracional/relatable"

    11. Ejercicio práctico¶

    Tu misión:¶

    1. 📱 Abrí ChatGPT
    2. 💬 Copiá este prompt:
    Sos copywriter especializado en marketing de joyería 
de lujo para LatAm.

Escribí 10 captions para un post sobre un collar de 
oro 18k con dije de luna.

Contexto:
- Audience: mujeres profesionales 30-45 años
- Tono: elegante, aspiracional, con calidez
- Precio: premium

Formato:
- Hook creativo en cada uno
- 100-150 palabras
- CTA sutil al final
- Un toque de misterio o poetry

Evitar: clichés románticos básicos
    1. ⏱️ Esperá la respuesta
    2. ✏️ Elegí tu favorito y personalizalo
    3. 📝 Agregalo a tu banco de captions
