    Módulo 8 — Finanzas Personales¶

    Objetivos de Aprendizaje¶

    Al finalizar este módulo vas a: - ✅ Entender la diferencia entre ingreso e ingreso disponible - ✅ Crear un presupuesto como freelancer - ✅ Construir tu fondo de emergencia - ✅ Planificar para impuestos - ✅ Separar finanzas personales y del negocio

    1. La realidad financiera del freelancer¶

    REALIDAD:

Como freelancer, vos sos tu propio departamento financiero.
No hay nadie que te retenga impuestos,
que te pague vacaciones, o que te asegure un ingreso fijo.

TUS RESPONSABILIDADES:
✓ Impuestos (pagás vos)
✓ Jubilación (ahorrás vos)
✓ Vacaciones (pagás vos)
✓ Enfermedad (sin ingreso si no trabajás)
✓ Fondo de emergencia (construís vos)

    2. Ingreso vs Ingreso Disponible¶

    INGRESO = Lo que cobrás
INGRESO DISPONIBLE = Lo que realmente te queda

CÁLCULO:

Ingreso mensual (ejemplo): $1000 USD
- Impuestos (~20-35%): -$250 USD
- Costos del negocio: -$100 USD
- Comisiones plataformas: -$50 USD
= Ingreso disponible: $600 USD

REGLA DE OPORTUNIDAD:
"Ingreso disponible" es lo que podés gastar.
NO cuentes con el bruto.

REGLA DEL 10-20-30:
• 10% → Ahorro/Jubilación
• 20% → Impuestos (guardá)
• 30% → Costos del negocio (software, etc.)
• 40% → Gastos personales

    3. Presupuesto para freelancers¶

    3.1 Cómo armar presupuesto¶

    PASO 1: CALCULÁ TUS GASTOS FIJOS
• Alquiler/Gastos del hogar
• Servicios (luz, agua, internet)
• Celular
• Transporte
• Seguro (si tenés)
• Suscripciones (Netflix, Spotify, etc.)

PASO 2: GASTOS VARIABLES
• Alimentación
• Entretenimiento
• Ropa
• Salidas
•Imprevistos

PASO 3: GASTOS DEL NEGOCIO
• Software (Canva Pro, Later, etc.)
• Hardware (si necesitás)
• Capacitación
• Networking/Eventos
• Comisiones de plataformas

PASO 4: IMPUESTOS
• Guardá 20-30% de cada ingreso
• No lo toques hasta pagarlos

PASO 5: AHORRO
• Fondo de emergencia: 3-6 meses de gastos
• Jubilación: 10% mínimo

    4. Fondo de emergencia¶

    4.1 Por qué es crucial¶

    FONDO DE EMERGENCIA:
Dinero guardado para cuando NO hay ingreso.

COMO FREELANCER:
• Los meses malos existen
• Los clientes se van
• Los proyectos terminan
• Las enfermedades pasan

SIN FONDO:
→ Pánico
→ Aceptar clientes malos
→ Deudas
→ Decisiones desesperadas

CON FONDO:
→ Tranquilidad
→ Capacidad de decir "no"
→ Espacio para negociar mejor
→ Paz mental

    4.2 Cómo construirlo¶

    META: 3-6 meses de gastos

EJEMPLO:
Gastos mensuales: $800 USD
Fondo mínimo: $2400 USD (3 meses)
Fondo ideal: $4800 USD (6 meses)

CÓMO CONSTRUIRLO:

1. AUTOMATIZÁ:
   "De cada ingreso, 20% va a fondo de emergencia."

2. EMPEZÁ CHICO:
   Primer objetivo: $500 USD
   Segundo objetivo: $1000 USD
   Tercero: $2000 USD
   ...

3. NO LO TOCÁS PARA NADA:
   Fondo de emergencia = NO existe
   hasta que sea una emergencia real.

4. RECONSTRUÍ INMEDIATO:
   Si lo usás, reconstruí en cuanto puedas.

    5. Planificación de impuestos¶

    IMPORTANTÍSIMO:
El dinero de impuestos NO es tuyo.
Es del gobierno. Guardalo desde el día uno.

REGLA SIMPLE:
De cada pago que recibís, inmediatamente:
→ 25% → Cuenta de impuestos (NO TOCAR)
→ 10% → Fondo de emergencia
→ 5% → Jubilación

AHORRÁ PARA:
• Monotributo (Argentina)
• VAT/IVA (según país)
• Impuesto a las ganancias
• Declaraciones trimestrales/anuales

HERRAMIENTAS:
• Planilla de Google Sheets
• Notion con base de datos
• App de finances (Mint, YNAB)

MANTENÉ REGISTRO:
• Fecha de cada ingreso
• Fecha de cada gasto
• Facturas emitidas
• Facturas recibidas

    6. Separación de finanzas¶

    6.1 Cuentas separadas¶

    CUENTA PERSONAL:
→ Para gastos diarios
→ Sueldo de freelancer que transferís

CUENTA DEL NEGOCIO:
→ Para todos los ingresos del negocio
→ Para todos los gastos del negocio

BENEFICIOS:
✓ Clarity de qué es tuyo y qué es del negocio
✓ Más fácil para impuestos
✓ Aspecto más profesional
✓ Mejor control financiero

COMO EMPEZAR:
• Abrí una cuenta bancaria separada
• Transferí tu "sueldo" mensualmente
• Todo ingreso del negocio va a esa cuenta
• Todo gasto del negocio sale de esa cuenta

    6.2 Cuánto pagarte a vos mismo¶

    MÉTODO "SUELDO DE FREELANCER":

1. Calculá tus gastos personales mensuales
   Ejemplo: $800 USD

2. Agregá gastos del negocio
   Ejemplo: $200 USD

3. Agregá ahorro deseado
   Ejemplo: $100 USD

4. TOTAL = Tu "sueldo" mínimo
   Ejemplo: $1100 USD

CUANDO COBRÁS:
1. Recibís pago
2. Separás 25% para impuestos
3. Separás 10% para ahorro
4. El resto = disponible para gastos
5. De lo disponible, transferís tu "sueldo"

EJEMPLO:
Cobraste: $1000 USD
Impuestos: -$250 USD
Ahorro: -$100 USD
Disponible: $650 USD
Transferís a tu cuenta: $650 USD

    7. Errores comunes¶

    ❌ Error 1: No separar dinero de impuestos¶

    MAL:
"Cobré $1000, gasté en lo que quise."

PROBLEMA:
Llega fin de año, debés $300 en impuestos
y no tenés.

BIEN:
"Cobré $1000, guardé $250 para impuestos.
Gasté $750."

REGLA:
"Si no guardás los impuestos desde el día uno,
vas a tener problemas después."

    ❌ Error 2: Gastar todo lo que ganás¶

    MAL:
"Cobré $2000 este mes. ¡A vivir!"
*No queda nada para meses malos*

BIEN:
"Cobré $2000.
$400 para impuestos.
$200 para ahorro.
$200 para negocio.
$1200 disponibles para gastos."

REGLA:
"Vivir por debajo de tus medios.
Ahorrar para los meses malos."

    ❌ Error 3: No llevar registro¶

    MAL:
"¿Cuánto gané este año? Ni idea..."

PROBLEMA:
Para impuestos necesitás números.
Si no llevás registro, es un caos.

BIEN:
"Tengo una planilla donde registro
cada ingreso y cada gasto.
Al final del año, tengo todo organizado."
