    Módulo 8 — Medir la Salud de tu Comunidad¶

    Objetivos de aprendizaje¶

    Al finalizar este módulo vas a:

    • ✅ Identificar métricas de comunidad que importan
    • ✅ Medir la salud general de tu comunidad
    • ✅ Crear dashboards de seguimiento
    • ✅ Tomar decisiones basadas en datos de comunidad
    • ✅ Mejorar continuamente según métricas

    1. Por qué medir la comunidad¶

    SIN MEDICIÓN:
→ No sabés si estás avanzando
→ No sabés qué funciona
→ Dependés de intuición
→ Podés estar mejorando o empeorando sin saberlo

CON MEDICIÓN:
→ Sabés exactamente dónde estás
→ Podés ver qué mejora qué
→ Tomás decisiones informadas
→ Puedes iterar y mejorar

REGLA:
"Lo que no se mide, no se mejora."

    2. Las métricas que importan¶

    Métricas de cantidad¶

    CUANTITY ≠ QUALITY
Pero necesitás ambas.

SEGUIDORES:
→ No es la métrica más importante
→ Pero es baseline
→ "¿Está creciendo mi reach?"

MONITOREO:
→ Seguimiento semanal
→ ¿Crece o decrece?
→ ¿Por qué?

HERRAMIENTA:
→ Meta Business Suite → Audience

    Métricas de engagement¶

    ENGAGEMENT = INTERACCIÓN REAL

CUÁLES MEDIR:

1. ENGAGEMENT RATE
   → (likes + comentarios + guardados) / Seguidores x 100
   → Mide qué % interactúa
   → Benchmark para joyería: >3% es bueno

2. COMENTARIOS POR POST
   → Promedio semanal
   → ¿Suben o bajan?

3. GUARDADOS (SAVES)
   → Contenido guardable = valioso
   → Saves > Likes en importancia

4. COMPARTIDOS (SHARES)
   → Contenido que la gente quiere compartir
   → Alcance orgánico

5. DM RECIBIDOS
   → ¿La gente te contacta?
   → Más DMs = más confianza

    Métricas de comunidad específica¶

    MÉTRICAS PROPIAS DE COMUNIDAD:

1. MIEMBROS ACTIVOS
   → ¿Cuántos interactúan regularmente?
   → Define "activo" (comentó al menos 1 vez/mes)

2. ENGAGEMENT ENTRE MIEMBROS
   → ¿Responden entre ellos?
   → ¿Se refieren entre ellas?
   → ¿Hay conversaciones sin vos?

3. TASA DE RETENCIÓN
   → ¿La gente sigue o se va?
   → Followers que se van / Total que había

4. MIEMBROS VIP
   → ¿Cuántos son clientes frecuentes?
   → ¿Cuántos son promotores?

5. RECOMENDACIONES ORGÁNICAS
   → ¿Te mencionan sin que lo pidas?
   → ¿Generás conversación?

6. PARTICIPACIÓN EN EVENTOS
   → ¿Cuánta gente viene a lives?
   → ¿Cuánta gente participa en rituales?

    Cómo medir engagement entre miembros¶

    CHECK: ¿Los miembros interactúan entre ellos?

CÓMO SABERLO:
1. Leé los comentarios de posts recientes
2. ¿Hay respuestas a otros comentaristas?
3. ¿Alguien taggea a otra persona?

INDICADORES POSITIVOS:
✓ "Yo tengo uno igual! @fulanita también tiene uno"
✓ Respuestas entre desconocidos
✓ Mención de "nosotras" o "todas"

INDICADORES A MEJORAR:
✗ Solo interacción con la marca
✗ Sin respuestas entre commenters
✗ Gente que no interactúa con otros

    4. Benchmarking¶

    Benchmarks para joyería¶

    ENGAGEMENT RATE:
• < 1% = Bajo, hay que mejorar
• 1-3% = Normal/Average
• 3-5% = Bueno (¡trabajá para esto!)
• > 5% = Excelente

COMENTARIOS POR POST:
• 0-5 = Bajo
• 5-20 = Normal
• 20-50 = Bueno
• 50+ = Excelente

SAVES RATE:
• < 0.5% = Bajo
• 0.5-1% = Normal
• 1-3% = Bueno
• > 3% = Excelente

CRECIMIENTO SEMANAL:
• < 0% = Estancado/perdiendo
• 0-2% = Normal
• 2-5% = Bueno
• > 5% = Excelente (sostenido es raro)

    Cómo comparar¶

    NO TE COMPARES CON MARCAS GRANDES.

COMPARATE CON:
→ Tu cuenta de hace 3 meses
→ Cuentas similares en tamaño
→ Promedio de tu nicho

EJEMPLO:
❌ "Tiffany tiene 2M de seguidores, yo tengo 500"
   → Comparación injusta

✅ "Mi ER es 2.5%, hace 3 meses era 1.8%"
   → Progreso real

✅ "Cuentas de joyería de mi tamaño tienen ER de 2-3%"
   → Estamos en rango

    5. Análisis de tendencias¶

    Qué buscar¶

    PATRONES A IDENTIFICAR:

1. TENDENCIA DE ENGAGEMENT
   → ¿Sube o baja?
   → ¿Hace cuánto?

2. QUÉ FUNCIONA
   → ¿Qué tipo de contenido tiene mejor ER?
   → ¿Qué horarios?
   → ¿Qué temas?

3. PATRONES DE CRECIMIENTO
   → ¿Qué publicación trajo más seguidores?
   → ¿Hubo pico de seguidores? ¿Por qué?

4. COMPORTAMIENTO ESTACIONAL
   → ¿Mejora en ciertas épocas?
   → ¿Baja en otras?

5. MIEMBROS QUE SE VAN
   → ¿Por qué? (si podés saberlo)
   → ¿Hay patrón?

    Cómo hacer análisis mensual¶

    ANÁLISIS MENSUAL:

SEMANA 1 DEL MES:
→ Revisá el mes anterior completo
→ Completá dashboard mensual
→ Identificá mejores y peores momentos
→ Escribí 3 aprendizajes clave

EJEMPLO DE APRENDIZAJE:
"Aprendí que los posts sobre cuidado de joyas
tienen 2x más saves que los de producto.
Debo hacer más contenido educativo."

    6. KPIs de comunidad específicos¶

    Para clientes¶

    SI TRABAJÁS PARA UN CLIENTE,
REPORTÁ ESTO:

NPS (Net Promoter Score):
→ "¿Recomendarías esta marca?"
→ Escala 0-10
→ 9-10 = promotores

SENTIMENT:
→ ¿Los comentarios son positivos, negativos, neutrales?
→ % de cada uno

MENCIÓN DE MARCA:
→ ¿Cuántas veces se menciona sin que la marca lo pida?
→ ¿En stories? ¿En posts? ¿En comentarios?

COMMUNITY HEALTH SCORE:
→ Combina todas las métricas
→ Del 1 al 10
→ Seguimiento mensual

    Fórmula simple¶

    COMMUNITY HEALTH SCORE:

ER: ___% (ponderación 30%)
Saves Rate: ___% (ponderación 20%)
Crecimiento: ___% (ponderación 20%)
NPS: ___ (ponderación 20%)
Sentiment positivo: ___% (ponderación 10%)

SCORE TOTAL = Promedio ponderado
→ 8-10 = Saludable
→ 5-7 = Necesita mejorar
→ < 5 = Problemas serios

    7. Acciones según métricas¶

    QUÉ HACER SEGÚN EL DIAGNÓSTICO:

SI ER BAJA PERO CRECIMIENTO ALTO:
→ Contenido atractivo pero no engancha
→ Hacer más contenido conversacional
→ Preguntar más en posts

SI ER ALTA PERO CRECIMIENTO BAJO:
→ Contenido engancha pero no reachea
→ Mejorar hashtags
→ Colaborar más
→ Meer contenido compartido

SI AMBAS BAJAN:
→ Revisar estrategia completa
→ ¿Qué cambió?
→ Puede ser algoritmo o oferta

SI AMBAS SUBEN:
→ Continuar haciendo lo que funciona
→ Documentar qué estás haciendo bien
→ Escalar lo que funciona

SI DMs SUBEN:
→ La comunidad confía más
→ Seguir haciendo lo mismo
→ Quizás ofrecer más interacción directa

SI DMs BAJAN:
→ ¿Se resolvieron bien los pedidos?
→ ¿Hay algún problema sin resolver?
→ ¿La atención está disponible?

    8. Herramientas de medición¶

    HERRAMIENTAS:

META BUSINESS SUITE:
→ Analytics de IG y FB
→ Gratis
→ Lo básico

NOTION O GOOGLE SHEETS:
→ Tu propio dashboard
→ Podes trackear lo que necesites
→ Personalizado

CROWDTANGLE (META):
→ Monitoreo de menciones
→ Análisis competitivo
→ Gratis (para ciertos usos)

MANUAL TRACKING:
→ Simplemente lleva registro
→ En una hoja de cálculo
→ Anota observaciones

PARA EMPEZAR:
Solo necesitas Meta Business Suite + Notion.
No necesitás herramientas caras.

    EJERCICIO 1: Tu dashboard actual

   → Engagement rate promedio: ___
   → Saves promedio: ___
   → Comentarios/post promedio: ___
   → Seguidores totales: ___
   → Crecimiento semanal: ___

2. ANALIZÁ UN POST DESTACADO
   → ¿Cuál fue tu mejor post?
   → ¿Qué tenía?
   → ¿Qué vas a replicar?

3. VERIFICÁ ENGAGEMENT ENTRE MIEMBROS
   → En tus últimos 5 posts
   → ¿Hay respuestas entre commenters?
   → ¿Sí o no?

RESULTADO:
Dashboard con números + aprendizajes

    10. Resumen¶

    PUNTOS CLAVE:

✓ Métricas de cantidad:
  → Seguidores (baseline)
  → Crecimiento

✓ Métricas de engagement:
  → ER (benchmark: >3%)
  → Comentarios, saves, shares
  → DMs recibidos

✓ Métricas de comunidad:
  → Miembros activos
  → Engagement entre members
  → Retención
  → Recomendaciones orgánicas

✓ Benchmarking:
  → Comparate con tu pasado
  → No con marcas grandes
  → Con cuentas similares

✓ Acciones según diagnóstico:
  → Diagnóstico → Acción concreta
  → Medí → Ajustá
