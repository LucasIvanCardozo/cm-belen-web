    Módulo 6 — Gestión Multi-Cliente¶

    Organizate para No Perder la Cabeza¶

    Objetivos del Módulo¶

    Al finalizar este módulo vas a: - ✅ Entender cómo organizar múltiples clientes - ✅ Crear sistemas de gestión eficientes - ✅ Priorizar tareas correctamente - ✅ Evitar que un cliente te robe todo el tiempo

    1. La Realidad del Freelance¶

    CUANDO EMPEZÁS:

1 cliente: "Es fácil, tengo todo bajo control"
3 clientes: "¿Dónde puse esto? ¿Cuándo era la reunión?"
5 clientes: "¡AYUDA!"
10 clientes: "Necesito un sistema O MUERО"

LA VERDAD:
A partir de 2-3 clientes,
necesitás sistemas. No dependas de tu memoria.

    4. Gestión de Tiempo¶

    TÉCNICA: TIME BLOCKING

Bloques específicos para cada cliente:

🟢 Cliente A (Joyería): 8-12am
🟡 Cliente B (Lifestyle): 1-4pm
🔵 Admin/Personal: 4-6pm

REGLAS:
1. No cambiar bloques sin razón
2. El cliente sabe cuándo sos disponible
3. Cada bloque tiene una tarea específica

    5. Cuántos Clientes Puedes Tener¶

    CÁLCULO SIMPLE:

Si trabajás 40 horas/semana:
- Tiempo efectivo de trabajo: 30h (25% admin)
- 30 horas ÷ horas por cliente = clientes máximos

EJEMPLO:
CM Junior: 5h/cliente/mes
→ 30h ÷ 5h = 6 clientes máximo

CM Senior: 8-10h/cliente/mes (más eficiencia)
→ 30h ÷ 8h = 3-4 clientes

PARA JOYERÍA:
- Cada cliente necesita contenido de alta calidad
- Más fotos, más revisión
- Factor multiplicador: 1.3x el tiempo
→ Menos clientes, más calidad

    7. Cuando un Cliente Te Absorbe¶

    SEÑAL:
"Trabajo 40 horas solo para Cliente A
y tengo 3 más que apenas atiendo."

SOLUCIÓN:

1. HABLAR CLARO
   "María, valoro mucho trabajar con vos.
   Quiero darte el mejor servicio, pero
   actualmente tengo X proyectos.
   Necesitás: [más presupuesto / menos scope]."

2. REDUCIR SCOPE
   "De X posts, podemos ir a Y posts
   para que pueda darle la calidad que merece."

3. SUBIR PRECIO
   "Para mantener este nivel de atención,
   necesito ajustar el precio a $X."

4. DECIR NO
   "No puedo tomar más trabajo de este tipo
   actualmente. Te aviso cuando tenga espacio."

    8. Herramientas Recomendadas¶

    PARA ORGANIZACIÓN:

• Notion (gratis): Base de datos de clientes
• Trello (gratis): Tableros de tareas
• Google Calendar: Scheduling
• Google Drive: Archivos compartidos
• WhatsApp Business: Komunikasi

PARA JOYERÍA ESPECÍFICO:
• Notion: Template de cliente joyero
• Pinterest: Referencias visuales
• Later/Buffer: Programación multi-cuenta

    9. Ejercicio Práctico¶

    EJERCICIO: Prepará tu sistema

PASO 1: Creá estructura de carpetas
   □ Una carpeta por cliente
   □ Subcarpetas: info, marcas, calendario, etc.

PASO 2: Creá template de cliente
   □ Info-general.md
   □ Adaptalo para joyería

PASO 3: Planificá tu semana
   □ Time blocks
   □ Días para contenido
   □ Días para admin

PASO 4: Configurá herramientas
   □ Notion o Trello
   □ Calendario con recordatorios

    10. Resumen¶

    ✓ Más de 2 clientes = necesitás sistemas
✓ Carpeta por cliente + info-general.md
✓ Time blocking = clave para organización
✓ Regla: no cambiar bloques
✓ Matriz de urgencia vs importancia
✓ Saber decir "no" o "reduzcamos scope"
✓ Joyería requiere más tiempo = menos clientes
