    Módulo 02: Anatomía de un Caption Perfecto¶

    Las 5 Partes que Todo Caption Exitoso Debe Tener¶

    🎯 Objetivos de aprendizaje¶

    Al terminar este módulo vas a:

    1. Identificar las 5 partes de un caption efectivo
    2. Escribir un hook que detenga el scroll
    3. Desarrollar el cuerpo del caption con claridad
    4. Crear CTAs que generen acción
    5. Aplicar la estructura a tus propios captions

    1. Por qué la estructura importa¶

    Imaginate un edificio sin cimientos. Se cae. Un caption sin estructura se "cae" también: la gente no lee, no engagea, no actúa.

    ┌─────────────────────────────────────────────────────┐
│                                                     │
│   UN CAPTION BIEN ESTRUCTURADO ES COMO:            │
│                                                     │
│   🎣 Una caña de pescar                              │
│   ├── Anzuelo (Hook) — Captura la atención         │
│   ├── Caña (Contexto) — Da información            │
│   ├── Línea (Transición) — Mantiene el interés     │
│   ├── Carnada (CTA) — Provoca la acción           │
│   └── Winnaret (Comunidad) — Construye relación    │
│                                                     │
└─────────────────────────────────────────────────────┘

    2. Las 5 partes de un caption¶

    Vista rápida:¶

    1. 🎣 HOOK (Anzuelo)
   → La primera línea que detiene el scroll
   → Debe ser intrigante, sorprendente o relatable

2. 📖 CUERPO (Contexto)
   → La información principal
   → Tu mensaje, historia o propuesta de valor

3. 🔗 CONTEXTO-ADICIONAL (Opcional)
   → Detalles, testimonios, información extra
   → Solo si el cuerpo lo necesita

4. 📢 CTA (Llamada a la acción)
   → Qué querés que haga el lector
   → Debe ser clara y específica

5. 💬 CIERRE (Opcional)
   → Pregunta, reflexión o invitación a comentar
   → Genera engagement adicional

    3. PARTE 1: El Hook (Anzuelo)¶

    ¿Por qué es crucial?¶

    📊 EN 3 SEGUNDOS SE DECIDE TODO

0-1 seg:   ¿Me detengo a leer?
1-2 seg:   ¿Me aporta algo?
2-3 seg:   ¿Lo guardo o paso de largo?

TU HOOK DECIDE SI CONTINÚA LEYENDO

    Tipos de Hook que funcionan en 2026:¶

    A) Hook de pregunta¶

    "¿Sabías que el 85% de las mujeres guarda los anillos
de plata para siempre?"

    B) Hook de estadística¶

    "De cada 10 novias, 7 eligen plata sobre oro.
¿Vos qué preferís?"

    C) Hook de historia¶

    "El anillo que ves acá belonged a mi bisabuela.
Ahora belongs a mí."

    D) Hook de afirmación¶

    "No es solo un collar. Es el momento en que dijiste
'me lo merezco'."

    E) Hook de números¶

    "3 razones por las que la plata 925 es mejor que el oro
para el día a día"

    F) Hook de provocación¶

    "Dejá de comprar anillos. Empezá a invertir en piezas."

    Ejercicio: Reescribí estos hooks¶

    Original: "Nuevo collar disponible en nuestra tienda"

    Hook mejorado: "Ese collar que vas a usar TODOS los días ✨"

    4. PARTE 2: El Cuerpo¶

    El cuerpo debe:¶

    ✓ Resolver lo que el hook prometió
✓ Ser fácil de escanear (frases cortas)
✓ Usar espacios en blanco estratégicamente
✓ Mantener el tono de la marca
✓ Conectar emocionalmente

    Estructura del cuerpo según longitud:¶

    Caption corto (1-3 oraciones):¶

    Hook → Mensaje principal → CTA

    Caption medio (4-8 oraciones):¶

    Hook → Información principal
→ Detalle de valor
→ CTA

    Caption largo (9+ oraciones):¶

    Hook → Introducción
→ Desarrollo
→ Prueba/testimonio
→ cierre
→ CTA

    Ejemplo de cuerpo efectivo para joyería:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ 🎣 HOOK:                                                 │
│ "Ese momento cuando una joya se convierte en             │
│  tu mejor acompañante"                                  │
│                                                         │
│ 📖 CUERPO:                                              │
│ Cada pieza de nuestra colección Cosmos está              │
│ diseñada para la mujer que no para.                      │
│                                                         │
│ ¿El detalle? El baño en rodio que usamos                 │
│ hace que no se oscurezca. Ni con el agua.              │
│ Ni con el perfume. Ni después de años.                   │
│                                                         │
│ Porque vos tenés suficientes cosas                      │
│ de las que ocuparte.                                    │
│ Tu joya no tiene que ser una de ellas.                  │
│                                                         │
└─────────────────────────────────────────────────────────┘

    5. PARTE 3: El CTA (Llamada a la Acción)¶

    CTAs que convierten vs. CTAs que fallan¶

    ❌ CTAs que nadie hace ✅ CTAs que funcionan
    "Comprá ahora" genérico "Escribinos para consulta sin cargo"
    "Dale like" "Guardá este post para cuando necesites un regalo"
    "Compartí" "Etiquetá a alguien que merece una joya este finde"
    "Click en el link" "Descubrí toda la colección en el link de bio"
    "Visitá nuestra tienda" "Pasá a saludarnos en [dirección]"

    CTAs según el objetivo:¶

    📊 SEGÚN LO QUE QUERÉS LOGRAR:

💬 Engagement: "Contame en los comentarios..."
❤️ Guardar: "Guardá para cuando necesites inspiración"
📤 Compartir: "Etiquetá a alguien especial"
💬 Consultar: "Escribinos, te asesoramos sin compromiso"
🛒 Comprar: "Link en bio para hacerla tuya"
📱 DM: "Mandame un mensaje privado"

    CTAs para joyería de lujo:¶

    ┌─────────────────────────────────────────────────────┐
│                                                     │
│ LUXO NO PIDE. INVITA.                               │
│                                                     │
│ ❌ "Comprá ahora antes de que se agote"             │
│ ✅ "Esta pieza fue creada en edición limitada.     │
│     Si es para vos, vas a saberlo."                │
│                                                     │
│ ❌ "No te lo podés perder"                         │
│ ✅ "¿Sos de las que guardan las cosas             │
│     especiales para 'después'? Esta es para        │
│     usar AHORA."                                   │
│                                                     │
└─────────────────────────────────────────────────────┘

    6. PARTE 4: El Cierre¶

    El cierre es tu oportunidad de:¶

    • Generar comunidad
    • Invitar a la conversación
    • Dejar una impresión memorable

    Cierres que funcionan:¶

    💬 "¿Y a vos? ¿Qué joya guardarías para siempre?

💬 "Contame tu historia con las joyas. Las replies
   más hermosas las compartimos.

💬 "Doble tap si reconocés a alguien que merece
   una joya este finde 💎

💬 "Guardá. Pero también contame: ¿preferís
   oro o plata? 👇

    8. Ejercicio práctico¶

    Tu tarea:¶

    Tomá los 3 captions que escribiste en el módulo anterior (anillo de plata con piedra de luna) y analizalos:

    CHECKLIST DE ANATOMÍA:

□ ¿Tenés un HOOK claro en la primera línea?
□ ¿El HOOK atrae a tu público objetivo?
□ ¿El CUERPO resuelve lo que el hook prometió?
□ ¿Las oraciones son cortas y fáciles de leer?
□ ¿Usaste espacios para que sea fácil escanear?
□ ¿Hay un CTA claro?
□ ¿El CTA es específico, no genérico?
□ ¿El CIERRE invita a la conversación?
□ ¿El tono matchea con joyería de lujo?
□ ¿Incluíste hashtags relevantes al final?

    Ahora reescribí los 3 captions incorporando lo que faltaba.

    9. Tarea de práctica avanzada¶

    Opción A: Mejora un caption existente¶

    1. Buscá un caption de una cuenta de joyería que no te convenza
    2. Reescribilo usando la estructura de 5 partes
    3. Explicá por qué tu versión es mejor

    Opción B: Crea un caption nuevo¶

    1. Elegí una pieza de una tienda de joyería (puede ser de Instagram)
    2. Escribí un caption completo usando las 5 partes
    3. Include: hook, cuerpo, contexto adicional, CTA, cierre

    Guardá ambas versiones para tu portfolio.

    📝 Resumen del módulo¶

    PUNTOS CLAVE:

✓ Todo caption efectivo tiene 5 partes:
  Hook → Cuerpo → (Contexto) → CTA → (Cierre)

✓ El HOOK es lo más importante —
  detiene el scroll en 3 segundos

✓ Hooks efectivos: pregunta, estadística,
  historia, afirmación, números, provocación

✓ El CTA debe ser específico y matching
  con tu objetivo

✓ El cierre invita a la conversación
  y genera engagement

✓ Estructura sin espacios = difícil de leer
