    Módulo 03: Fórmulas Ganadoras para Captions¶

    AIDA, PAS, Storytelling y Más¶

    🎯 Objetivos de aprendizaje¶

    Al terminar este módulo vas a:

    1. Conocer las 4 fórmulas principales de copywriting
    2. Aplicar la fórmula AIDA paso a paso
    3. Dominar la fórmula PAS para vender
    4. Usar storytelling para conectar emocionalmente
    5. Elegir la fórmula correcta según el contexto

    1. ¿Por qué usar fórmulas?¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   LAS FÓRMULAS NO LIMITAN TU CREATIVIDAD.              │
│   LA STRUCTURAN.                                        │
│                                                         │
│   Cuando sabés la estructura,                          │
│   podés enfocarte en lo que importa:                   │
│   → El mensaje correcto                                 │
│   → Las palabras exactas                                 │
│   → La emoción precisa                                  │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Analogía del chef: Un chef novato sigue recetas al pie de la letra. Un chef experto conoce la estructura (sazón, técnica, presentación) y puede crear recetas propias. Las fórmulas son tu receta base.

    2. Fórmula 1: AIDA¶

    La fórmula más clásica del copywriting¶

    AIDA = Atención → Interés → Deseo → Acción

    Desglose:¶

    🎣 ATENCIÓN
   → Captar la mirada, detener el scroll

💡 INTERÉS
   → Mantener la atención, dar información relevante

✨ DESEO
   → Generar wanting, mostrar el valor

📢 ACCIÓN
   → Provocar el siguiente paso

    Ejemplo completo con AIDA:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ 🎣 ATENCIÓN:                                           │
│ "El anillo que vas a usar el día de tu casamiento      │
│  no tiene que ser el más caro."                        │
│                                                         │
│ 💡 INTERÉS:                                            │
│                                                                            │
│ En Lunar trabajamos con plata 925 porque es el metal    │
│ más noble para pieles sensibles.                      │
│ Pero lo que realmente nos diferencia es el diseño.      │
│                                                                            │
│ ✨ DESEO:                                              │
│                                                                            │
│ Cada pieza de nuestra colección Matrimonio incluye     │
│ grabado personalizado sin costo adicional.              │
│ Porque lo que vas a usar todos los días merece         │
│ ser verdaderamente tuyo.                               │
│                                                                            │
│ 📢 ACCIÓN:                                             │
│                                                                            │
│ Visitá nuestra tienda esta semana para ver              │
│ la colección completa.                                 │
│ Reservas por DM — link en bio.                         │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Plantilla AIDA para joyería:¶

    🎣 [ATENCIÓN — Statement provocador o pregunta que engancha]

💡 [INTERÉS — Información sobre el producto/colección]

✨ [DESEO — Beneficio emocional, no solo características]

📢 [ACCIÓN — Qué hacer ahora, específico y claro]

    3. Fórmula 2: PAS¶

    Problema → Agitación → Solución¶

    Esta fórmula es poderosa para cuando querés vender directamente.

    ❌ PROBLEMA
   → Identificá el pain point del cliente

🔥 AGITACIÓN
   → Agitá ese problema, hacelo más real

✅ SOLUCIÓN
   → Presentá tu producto como la solución

    Ejemplo completo con PAS:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ ❌ PROBLEMA:                                            │
│ "Te pasa que comprás una joya, la usás 3 veces,         │
│  y después se oscurece o te da alergia?"                │
│                                                         │
│ 🔥 AGITACIÓN:                                          │
│                                                                            │
│ La mayoría de las joyas en el mercado tienen baño      │
│ de apenas 2-3 capas de metal precioso.                 │
│                                                                             │
│ Eso significa que después de un mes de uso,             │
│ ya empiezan a cambiar de color.                         │
│                                                                             │
│ Y cuando te das cuenta, ya pasó el evento especial.     │
│                                                                             │
│ ✅ SOLUCIÓN:                                          │
│                                                                            │
│ En Lunar usamos plata 925 maciza con baño de rodio     │
│ de 10 capas.                                           │
│                                                                             │
│ ¿El resultado? Joyas que se ven igual de lindas         │
│ después de 2 años de uso diario.                        │
│                                                                             │
│ Galería Lunar — donde cada pieza perdura.              │
│                                                                             │
│ Visitá nuestra bio para conocer la colección.          │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Plantilla PAS para joyería:¶

    ❌ [PROBLEMA — Pain point que tu cliente reconoce]

🔥 [AGITACIÓN — Por qué ese problema es peor de lo que piensa]

✅ [SOLUCIÓN — Tu producto como respuesta, con prueba/beneficio]

    4. Fórmula 3: Storytelling¶

    Contá una historia que venda¶

    El storytelling no es "contar la historia de la marca". Es usar estructuras narrativas para generar conexión emocional.

    STORYTELLING = Situación → Conflicto → Resolución

    Elementos clave del storytelling:¶

    📖 SITUACIÓN
   → Creá el escenario, hacé que se identifiquen

⚡ CONFLICTO
   → Presentá el problema, duda o decisión

🎯 RESOLUCIÓN
   → Mostrá cómo la joya ayuda/resuelve

    Ejemplo completo con Storytelling:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ 📖 SITUACIÓN:                                           │
│ "Mi abuela tenía un collar de perlas que nunca usaba.   │
│  Lo guardaba en el cajón, envuelto en un pañuelo        │
│  de seda que ya estaba amarillo."                       │
│                                                         │
│ ⚡ CONFLICTO:                                          │
│                                                                            │
│ "Cuando pregunté por qué no lo usaba, me dijo:          │
│ 'Las perlas de tu abuelo están bien guardadas.          │
│  Este es más para ocasiones especiales.'"                │
│                                                                            │
│ "El problema es que las ocasiones especiales              │
│  nunca llegaban."                                     │
│                                                                            │
│ 🎯 RESOLUCIÓN:                                        │
│                                                                            │
│ "Por eso diseñamos esta colección.                      │
│                                                                            │
│ Piezas tan versátiles que podés usar todos los días.   │
│ Porque la verdadera elegancia no se guarda.             │
│ Se vive."                                             │
│                                                                            │
│ Collares, pulseras y anillos que van del trabajo       │
│ al after. Del mediodía a la cena. De lo cotidiano      │
│ a lo especial.                                          │
│                                                                            │
│ Esta semana: 20% off en toda la colección.               │
│ Porque vos también merecés usar lo bonito.              │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Plantilla Storytelling para joyería:¶

    📖 [SITUACIÓN — Un momento o escena que se relacional]

⚡ [CONFLICTO — La duda, decisión o problema]

🎯 [RESOLUCIÓN — Cómo la joya resolve la situación + CTA]

    5. Fórmula 4: BAB (Before-After-Bridge)¶

    Antes → Después → Puente¶

    Excelente para mostrar transformación o beneficio claro.

    BEFORE = Tu cliente ANTES de usar tu producto
AFTER = Tu cliente DESPUÉS de tener tu producto
BRIDGE = Cómo llegar de A a B (tu producto/servicio)

    Ejemplo completo con BAB:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ 📷 ANTES:                                               │
│ "Antes de encontrar Lunar, mi joyero era un desastre.   │
│                                                                            │
│ Tenía 15 anillos distintos que nunca usaba,              │
│ un collar que me daba alergia, y una pulsera            │
│ que se había oscurecido después del primer lavado      │
│ de manos."                                             │
│                                                         │
│ ✨ DESPUÉS:                                            │
│                                                                            │
│ "Ahora tengo 4 piezas que uso rotación.                 │
│                                                                            │
│ Un anillo de plata 925 que no se oscurece.              │
│ Un collar que puedo usar en la ducha.                   │
│ Una pulsera que ya tiene 2 años y sigue igual."         │
│                                                                            │
│ 🌉 PUENTE:                                             │
│                                                                            │
│ "¿El secreto? Dejé de buscar jewelry que se viera bien  │
│ en la foto y empecé a buscar jewelry que se sintiera    │
│ bien todos los días."                                  │
│                                                                            │
│ En Lunar ayudamos a encontrar LA pieza correcta.        │
│ No la más cara. La que realmente vas a usar.            │
│                                                                            │
│ Escribinos. Te asesoramos sin compromiso.              │
│                                                         │
└─────────────────────────────────────────────────────────┘

    6. Cuándo usar cada fórmula¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ 📊 ELECCIÓN DE FÓRMULA SEGÚN CONTEXTO:                  │
│                                                         │
│ ┌──────────────────┬────────────────────────────────┐  │
│ │ CONTEXTO         │ FÓRMULA RECOMENDADA             │  │
│ ├──────────────────┼────────────────────────────────┤  │
│ │ Producto nuevo   │ AIDA o Storytelling            │  │
│ │ Venta directa   │ PAS                             │  │
│ │ Testimonio      │ Storytelling o BAB              │  │
│ │ Contenido valor │ AIDA (sin CTA fuerte)           │  │
│ │ Campaña oferta  │ PAS                             │  │
│ │ Marca personal  │ Storytelling                    │  │
│ │ Behind scenes   │ Storytelling                    │  │
│ └──────────────────┴────────────────────────────────┘  │
│                                                         │
└─────────────────────────────────────────────────────────┘

    7. Combinación de fórmulas¶

    Podés mezclar fórmulas para resultados más fuertes:¶

    AIDA + Storytelling:

🎣 ATENCIÓN (con historia):
   "La primera vez que usé un anillo de plata..."

💡 INTERÉS (con AIDA):
   "...nunca más quise saber de otro metal."

✨ DESEO (con PAS):
   "Porque la plata 925 no oscurece, no da alergia,
   y se ve igual de linda después de años."

📢 ACCIÓN:
   "Encontrá tu pieza para siempre."

    8. Ejercicio práctico: Aplicá las 4 fórmulas¶

    Producto: Collar con dije de luna para regalo de cumpleaños¶

    Escribí 4 versiones distintas del mismo caption, una con cada fórmula:

    📝 MIS CAPTIONS:

1. AIDA:
   [Tu caption aquí]

2. PAS:
   [Tu caption aquí]

3. Storytelling:
   [Tu caption aquí]

4. BAB:
   [Tu caption aquí]

    Preguntas para reflexionar:¶

    • ¿Cuál te salió más natural?
    • ¿Cuál creés que va a generar más engagement?
    • ¿Cuál usarías para un cliente específico?

    9. Tarea de práctica¶

    Nivel básico:¶

    Elegí 3 productos distintos de una tienda de joyería y escribí un caption usando AIDA para cada uno.

    Nivel avanzado:¶

    Elegí 1 producto y escribí los 4 captions (uno por fórmula). Luego probalos en tus stories o guardalos para comparar resultados.

    📝 Resumen del módulo¶

    PUNTOS CLAVE:

✓ AIDA: Atención → Interés → Deseo → Acción
  → Mejor para productos nuevos o contenido de valor

✓ PAS: Problema → Agitación → Solución
  → Mejor para venta directa y ofertas

✓ Storytelling: Situación → Conflicto → Resolución
  → Mejor para conexión emocional y marca

✓ BAB: Antes → Después → Puente
  → Mejor para testimoniales y transformaciones

✓ Combiná fórmulas según el contexto y objetivo

✓ La práctica hace que las fórmulas se vuelvan naturales
