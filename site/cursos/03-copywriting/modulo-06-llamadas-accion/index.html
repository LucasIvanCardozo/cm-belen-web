    Módulo 06: Llamadas a la Acción (CTAs)¶

    Cómo Hacer que la Gente Realmente Actúe¶

    🎯 Objetivos de aprendizaje¶

    Al terminar este módulo vas a:

    1. Entender por qué la mayoría de los CTAs fallan
    2. Crear CTAs específicos para cada objetivo
    3. Aplicar CTAs para joyería de lujo sin parecer desesperado
    4. Medir qué CTAs funcionan mejor

    1. La verdad sobre los CTAs¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   EL 90% DE LOS CM ESCRIBEN CTAs COMO ESTO:          │
│                                                         │
│   "Link in bio"                                         │
│   "DM for more"                                         │
│   "Visit our page"                                      │
│                                                         │
│   Y SE PREGUNTAN POR QUÉ NADIE ACTÚA.                │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Por qué fallan:¶

    ❌ "Link in bio" 
   → No dice QUÉ van a encontrar

❌ "DM for more" 
   → No da INCENTIVO para hacerlo

❌ "Visit our page" 
   → No dice POR QUÉ les importa AHORA

    2. Anatomía de un CTA efectivo¶

    Los 4 elementos:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   1. 🎯 QUÉ hacer (claro)                            │
│      → "Escribinos" no "Para más info"                │
│                                                         │
│   2. 💎 PARA QUÉ (incentivo)                          │
│      → "Escribinos para consulta SIN CARGO"           │
│                                                         │
│   3. ⏰ CUÁNDO (urgencia sutil)                       │
│      → "Esta semana" / "Edición limitada"             │
│                                                         │
│   4. 😊 CÓMO (fácil)                                 │
│      → "Tocá aquí abajo" / "Mandanos un DM"          │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Ejemplo de CTA completo:¶

    ❌ DÉBIL: "Link in bio"

✅ FUERTE: "Encontrá tu pieza ideal → tocá el link
           de bio. Esta semana tenemosenvío gratis
           en toda la colección."

    3. CTAs por objetivo¶

    Objetivo: Engagement (comentarios)¶

    ❌ "Comment below"
✅ "¿Cuántos outfits armás por semana?
    Contame en los comentarios 👇"

❌ "Leave a comment"
✅ "Si tuvieras que elegir UNA joya para usar
    el resto de tu vida, ¿cuál sería? 👇"

❌ "Let us know"
✅ "¿Oro o plata? (No hay respuesta incorrecta.
    Solo reveladora.) 👇"

    Objetivo: Guardar (save)¶

    ❌ "Save this post"
✅ "Guardá este post para cuando necesités
    un regalo de último momento 🎁"

❌ "Save for later"
✅ "Guardá: esta es la guía de regalos
    por precio que vas a necesitar"

❌ "Save"
✅ "Guardá para cuando no sepas qué regalar"

    Objetivo: Compartir (share)¶

    ❌ "Share with a friend"
✅ "Etiquetá a alguien que merece una joya
    este fin de semana 💎"

❌ "Tag a friend"
✅ "¿Quién en tu vida merece una de estas?
    Etiquetala para que lo vea 👇"

❌ "Repost"
✅ "Si conocés a alguien que está buscando
    una joya especial, compartile esto"

    Objetivo: Mensaje/DM¶

    ❌ "DM us"
✅ "Mandanos un DM con tu idea y te mandamos
    un presupuesto sin compromiso ✉️"

❌ "Message us"
✅ "Escribinos por privado. Te ayudamos a
    encontrar la pieza perfecta — sin presión 📩"

❌ "Inbox us"
✅ "¿Tenés dudas? Escribinos. Respondemos
    en menos de 2 horas 💬"

    Objetivo: Compra¶

    ❌ "Buy now"
✅ "Hacé tuya esta pieza antes de que se agote.
    Link en bio →"

❌ "Shop the link"
✅ "Descubrí toda la colección en el link de bio.
    Envío a todo el país 🚚"

❌ "Get yours"
✅ "Llevátela. Merecés una joya que te haga
    sentir así todos los días. Link en bio →"

    4. CTAs para joyería de lujo¶

    El dilema del lujo:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   ⚠️ LUJO vs. VENTA DIRECTA:                         │
│                                                         │
│   Si sos muy pushy → Parece tienda de barrio          │
│   Si sos muy passivo → Nadie compra                   │
│                                                         │
│   EL ARTE ESTÁ EN EL MEDIO:                          │
│   Invitar, no pedir.                                  │
│   Sugerir, no exigir.                                  │
│   Hacer fácil, no obligar.                            │
│                                                         │
└─────────────────────────────────────────────────────────┘

    CTAs de lujo que funcionan:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ 🏆 INVITACIÓN A DESCUBRIR:                           │
│                                                         │
│ "Si esta pieza es para vos, lo vas a saber."          │
│                                                         │
│ "Para quienes reconocen lo excepcional."              │
│                                                         │
│ "Esta edición fue creada para ellas.                  │
│  Si sos una de ellas, conocé más en el link."         │
│                                                         │
│ ─────────────────────────────────────────────────────  │
│                                                         │
│ 🏆 INVITACIÓN A CONSULTAR:                           │
│                                                         │
│ "Si tenés preguntas sobre materiales, tallas           │
│  o disponibilidad, escribinos.                         │
│  Te respondemos con gusto. Sin presión."              │
│                                                         │
│ "Querés ver más? Mandanos un mensaje privado.         │
│  Te mostramos toda la colección."                     │
│                                                         │
│ ─────────────────────────────────────────────────────  │
│                                                         │
│ 🏆 INVITACIÓN A RESERVAR:                            │
│                                                         │
│ "Stock limitado. Reservá la tuya antes                │
│  de que se agote. Escribinos por DM."                │
│                                                         │
│ "Esta pieza no se repone.                             │
│  Si te gustó, no la dejes para después."             │
│                                                         │
└─────────────────────────────────────────────────────────┘

    CTAs que NO usar en lujo:¶

    ❌ "COMPRÁ AHORA ANTES DE QUE SE AGOTE!!!!"
❌ "Última oportunidad!!!"
❌ "OFERTA IMPERDIBLE!!!"
❌ "2x1!!!"
❌ "Hoy o nunca!!!"

→ Todo esto dice "desesperación", no "lujo"

    5. CTAs según la plataforma¶

    Instagram Feed:¶

    → Usá 1 CTA claro
→ Mencioná el beneficio
→ Incluí el "cómo" (link en bio)

EJEMPLO:
"Encontrá la tuya → link en bio"

    Instagram Stories:¶

    → Podés usar hasta 3 CTAs
→ Usá stickers de Link (hasta 4)
→ Haz preguntas que inviten a responder

EJEMPLO:
"Pulsá 'Message' si querés que te ayude a elegir 👇"

    Carruseles:¶

    → CTA solo en el ÚLTIMO slide
→ No distraigas durante la lectura
→ El contenido tiene que hablar primero

EJEMPLO:
"¿Cuántas de estas conocías?
Contame abajo 👇"

    Reels:¶

    → El video ya es el hook
→ El caption es para la acción
→ Short and sweet

EJEMPLO:
"Guardá para después + etiquetá a alguien
que necesita ver esto 💎"

    6. Secuencia de CTAs¶

    Para posts de venta:¶

    1️⃣ Hook: Generá atención
2️⃣ Cuerpo: Mostrá el valor
3️⃣ CTA principal: La acción más importante
4️⃣ CTA secundario: Acción alternativa

    Ejemplo de secuencia:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ [HOOK — Captaste atención]                             │
│                                                         │
│ "El collar que se ve acá tiene una historia."         │
│                                                         │
│ [CUERPO — Mostraste valor]                             │
│                                                         │
│ "Plata 925 con baño de rodio que no se oscurece.      │
│  Piedra de luna natural que cambia con la luz.          │
│  Diseño que va con todo y con todos."                  │
│                                                         │
│ [CTA PRINCIPAL]                                        │
│                                                         │
│ "Llevátelo: link en bio →"                           │
│                                                         │
│ [CTA SECUNDARIO]                                       │
│                                                         │
│ "¿No estás segur@? Escribinos.                        │
│  Te ayudamos a elegir la pieza correcta. 📩"         │
│                                                         │
└─────────────────────────────────────────────────────────┘

    7. Testing de CTAs¶

    Cómo medir qué CTA funciona mejor:¶

    📊 MÉTRICAS A SEGUIR:

CTA de Engagement:
→ Comentarios por post
→ Compará posts con vs. sin CTA específico

CTA de Guardado:
→ Guardados por post
→ Guardados / Alcance = Save Rate

CTA de DM:
→ Mensajes recibidos por post
→ Tracking: "Dijiste [palabra clave] en los comentarios"

CTA de Compra:
→ Tráfico desde link en bio
→ Código de descuento específico por CTA
→ UTM parameters para trackear

    Test A/B simple:¶

    POST A:
CTA: "Link en bio"
RESULTADO: X comentarios

POST B:
CTA: "Guardá este post para después + link en bio"
RESULTADO: Y comentarios + Z guardados

→ El que tenga mejores métricas, usá más seguido

    8. Ejercicio práctico: Mejora tus CTAs¶

    Tomá 5 de tus captions existentes y:¶

    CHECKLIST DE CTA:

□ ¿Tu CTA dice claramente QUÉ hacer?
□ ¿Da un incentive para hacerlo?
□ ¿Es específico y no genérico?
□ ¿Está alineado con el objetivo del post?
□ ¿Es fácil de seguir desde el celular?

AHORA REESCRIBÍLOS:

Post 1:
CTA actual: _______________
CTA mejorado: _____________

Post 2:
CTA actual: _______________
CTA mejorado: _____________

Post 3:
CTA actual: _______________
CTA mejorado: _____________

    9. Tarea de práctica¶

    Nivel básico:¶

    Elegí 3 posts que tengas o de otras cuentas. Analizá los CTAs. Reescribí los que son débiles con CTAs más fortes.

    Nivel avanzado:¶

    Durante 1 semana, usá un CTA diferente en cada post. Al final de la semana, revisá qué CTAs generaron más engagement. Creá tu "top 3" de CTAs winners.

    📝 Resumen del módulo¶

    PUNTOS CLAVE:

✓ CTAs efectivos tienen 4 partes:
  QUÉ hacer + PARA QUÉ + CUÁNDO + CÓMO

✓ El 90% de los CTAs fallan porque son genéricos

✓ CTAs por objetivo:
  → Engagement: preguntas específicas
  → Guardar: "guardá para cuando..."
  → Compartir: etiquetar a alguien
  → DM: beneficio + sin presión
  → Compra: urgencia sutil + link

✓ En lujo: invitar > exigir

✓ Testeá CTAs regularmente para mejorar
