    Módulo 08: IA como Asistente de Escritura¶

    Potenciá tu Creatividad, No la Reemplaces¶

    🎯 Objetivos de aprendizaje¶

    Al terminar este módulo vas a:

    1. Entender cómo usar IA sin perder tu voz
    2. Aplicar prompts efectivos para copywriting de joyería
    3. Crear un banco de prompts personalizado
    4. Evitar los errores de usar IA de más

    1. La regla de oro de IA + Copywriting¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   🤖 IA = ASISTENTE, NO AUTOR                        │
│                                                         │
│   La IA te ayuda a:                                   │
│   → Generar ideas cuando estás en blanco              │
│   → Mejorar la estructura de tus textos                │
│   → Encontrar palabras que no se te ocurrían          │
│   → Traducir emociones en vocabulario                 │
│                                                         │
│   La IA NO te ayuda a:                                │
│   → Sentir lo que tu cliente siente                  │
│   → Contar TU historia de marca                      │
│   → Decidir qué emoción usar                        │
│   → Sustituir tu criterio                            │
│                                                         │
└─────────────────────────────────────────────────────────┘

    2. Cómo integrar IA en tu workflow¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   WORKFLOW SUGERIDO:                                  │
│                                                         │
│   1️⃣ BRIEFING (vos)                                  │
│      → ¿Qué producto? ¿Qué objetivo?                 │
│      → ¿Qué emoción? ¿Qué público?                   │
│                                                         │
│   2️⃣ IA: GENERAR OPCIONES                            │
│      → Pedile 3-5 versiones de hooks                 │
│      → Pedile diferentes ángulos                    │
│                                                         │
│   3️⃣ VOS: EVALUAR + ELEGIR                          │
│      → ¿Cuál resuena con la marca?                  │
│      → ¿Cuál usarías así?                             │
│                                                         │
│   4️⃣ IA: MEJORAR                                    │
│      → Pedile que mejore tu opción elegida          │
│                                                         │
│   5️⃣ VOS: AJUSTAR + APROBAR                         │
│      → Agregá tu toque personal                       │
│      → Verificá que sea authentic                   │
│                                                         │
└─────────────────────────────────────────────────────────┘

    3. Prompts efectivos para copywriting¶

    Prompt 1: Generar hooks¶

    📝 PROMPT:

"Soy Community Manager de una tienda de joyería de lujo
en [ciudad]. Necesito 5 hooks diferentes para Instagram
para un post sobre [producto].

El tono es [tono: elegante/cercano/aspiracional].
El público objetivo es [descripción].
Incluí al menos uno que sea provocador.

Ejemplo de hooks que me gustan:
[incluí 1-2 ejemplos si tenés]
"

    Variaciones: - Cambiá "joyería de lujo" por "boutique de accesorios" - Cambiá el tono según la marca - Ajustá el público objetivo

    Prompt 2: Reescribir en diferente estilo¶

    📝 PROMPT:

"Tengo este caption y necesito que lo reescribas
en 3 tonos diferentes:

[pegar caption original]

1. Tono elegante y sofisticado (como marcas de lujo)
2. Tono cercano y conversacional (como si hablaras con una amiga)
3. Tono aspiracional (que genere deseo sin ser pushy)

Para cada versión, mantén la información sobre el producto
pero cambiá cómo se presenta."

    Prompt 3: Generar ideas de contenido¶

    📝 PROMPT:

"Soy CM de una joyería especializada en [tipo: plata 925 / oro 18k / piedras preciosas].
Necesito 20 ideas de contenido para Instagram para el mes de [mes].

Incluí:
- 5 ideas de producto (posts de venta)
- 5 ideas educativas (tips, cuidados, etc.)
- 5 ideas emocionales (storytelling, conexiones)
- 5 ideas de detrás de cámaras o comunidad

Cada idea incluye: tipo de post, caption hook, y tema específico."

    Prompt 4: Mejorar un CTA específico¶

    📝 PROMPT:

"Tengo este CTA para un post de joyería:
[pegar CTA]

Necesito que me des 5 alternativas que:
1. Sean específicas y claras
2. Generen urgencia sin ser desesperadas
3. Incluyan un beneficio claro
4. Sean appropriate para jewelry de lujo
5. Funcionen para el público objetivo: [descripción]"

    Prompt 5: Análisis de captions exitosos¶

    📝 PROMPT:

"Analiá este caption y decime:
1. ¿Cuál es el hook?
2. ¿Qué emociones utiliza?
3. ¿Es efectivo? ¿Por qué?
4. ¿Qué podrías mejorar?

Caption:
[pegar caption]"

    4. Prompts específicos para joyería¶

    Para producto con piedra:¶

    📝 PROMPT:

"Quiero un caption para un collar con piedra de
[mineral: luna / turquesa / jade / etc.].

Incluí:
- La leyenda/mito asociado a la piedra
- Por qué esta piedra es especial
- Un hook emocional que conecte con [emoción: autoamor / regalo especial / conexión]

Tono: [elegante / cercano / aspiracional]"

    Para colección limitada:¶

    📝 PROMPT:

"Tengo una colección de joyería limitada a 50 piezas.
Necesito un caption que transmita:
- Exclusividad sin ser snob
- Urgencia sutil
- Valor real (no solo "es cara")
- Invitación a actuar ahora

Tono: lujo accesible, no intimidante"

    Para cuidado de joyas:¶

    📝 PROMPT:

"Necesito un caption educativo sobre cómo cuidar
joyas de [material: plata 925 / oro / acero].

El tono debe ser:
- Útil pero no condescendiente
- Profesional pero accesible
- Con un CTA al final (visitar tienda / escribirnos)

Incluí al menos 3 tips específicos."

    5. Qué NO dejar que haga la IA¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   🚫 NUNCA DEJES QUE LA IA:                           │
│                                                         │
│   1. Escriba el caption completo sin tu revisión     │
│      → Siempre ajustá y hacé tuyo el texto           │
│                                                         │
│   2. Use jerga o frases genéricas de internet       │
│      → "¡No te lo podés perder!" = NO               │
│                                                         │
│   3. Invente datos o estadísticas falsas             │
│      → Verificá todo lo que te dé                    │
│                                                         │
│   4. Copie el estilo de otra marca                  │
│      → Tu marca tiene voz propia                     │
│                                                         │
│   5. Ignore el contexto cultural o local             │
│      → LatAm tiene sus propias formas de expresarse  │
│                                                         │
│   6. Reemplace tu criterio sobre qué es mejor        │
│      → Vos conocés tu marca mejor que nadie          │
│                                                         │
└─────────────────────────────────────────────────────────┘

    6. Cómo mantener tu voz authentic¶

    CHECKLIST DE VOZ AUTHENTIC:

□ ¿Suena como vos? (leé en voz alta)
□ ¿Tiene tu tono de marca?
□ ¿Usás palabras que realmente decís?
□ ¿Tiene algo personal (un detalle, una historia)?
□ ¿Es too much perfecto? (si sí, agregá algo imperfecto)
□ ¿Pasaría el "test del amigo"? 
  → ¿Tu amigo reconocería tu estilo?

    Si el texto de IA suena genérico:¶

    TÉCNICA: "Desperfect" lo

Agregá:
- Un detalle específico
- Un dato real sobre el producto
- Una opinión personal
- Un toque de humor
- Una referencia cultural o local

    8. Herramientas de IA recomendadas¶

    🤖 HERRAMIENTAS PARA COPYWRITING:

1. ChatGPT (chat.openai.com)
   → Mejor para: hooks, ideas, rewriting
   → Ventaja: Rápido, versátil
   → Costo: Gratis (versiones antiguas) / Plus $20/mes

2. Claude (claude.ai)
   → Mejor para: análisis, struktur, refinado
   → Ventaja: Más nuanced, mejor para análisis
   → Costo: Gratis / Pro $20/mes

3. Gemini (gemini.google.com)
   → Mejor para: búsqueda de información
   → Ventaja: Integra con Google
   → Costo: Gratis

4. Perplexity (perplexity.ai)
   → Mejor para: investigar tendencias
   → Ventaja: Cita fuentes
   → Costo: Gratis / Pro $20/mes

💡 RECOMENDACIÓN:
Usá ChatGPT o Claude para escritura.
Perplexity para investigación.

    9. Ejercicio práctico: Integrar IA en tu workflow¶

    Tu tarea de esta semana:¶

    📝 DIA 1-2: Experimentá

1. Elegí 3 productos distintos
2. Usá IA para generar 3 hooks para cada uno
3. Evaluá cuáles te gustan y cuáles no
4. Anotá qué prompts funcionaron mejor

📝 DIA 3-4: Implementá

1. Elegí un post completo
2. Usá IA para ayudarte con:
   - Hook
   - Cuerpo (opcional)
   - CTA
3. Reescribí con tu toque personal
4. Publicá

📝 DIA 5-7: Ajustá

1. Revisá los resultados
2. ¿Qué funcionó? ¿Qué no?
3. Ajustá tus prompts según los resultados
4. Guardá tus mejores prompts

    10. Tarea final del curso¶

    Proyecto: Tu primer caption profesional¶

    📝 REQUISITOS:

□ Usar una fórmula de las aprendidas (AIDA, PAS, Storytelling)
□ Incluir una emoción principal
□ Tener un hook fuerte
□ Incluir un CTA claro
□ Usar hashtags estratégicos (5-8)
□ Ser auténtico (no genérico)

📝 FORMATO:

Producto: [Cuál es]
Fórmula usada: [Cuál]
Emoción principal: [Cuál]
Hook: [Tu hook]
Caption completo:
[Tu caption aquí]

CTA usado: [Tu CTA]
Hashtags:
[5-8 hashtags]

Por qué funciona:
[Tu análisis de por qué este caption debería funcionar]

    Este es tu primer trabajo demostrable. Guardalo para tu portfolio.
