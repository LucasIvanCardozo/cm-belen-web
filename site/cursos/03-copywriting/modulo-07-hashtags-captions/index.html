    Módulo 07: Hashtags Estratégicos¶

    Más Allá de #Joyas #Plata #Oro¶

    🎯 Objetivos de aprendizaje¶

    Al terminar este módulo vas a:

    1. Entender cómo funcionan los hashtags en 2026
    2. Aplicar la estrategia de 3 niveles
    3. Crear tu propia biblioteca de hashtags para joyería
    4. Evitar errores comunes con hashtags

    1. La verdad sobre los hashtags en 2026¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   📊 LO QUE LA GENTE PIENSA:                           │
│   "Los hashtags ya no sirven"                          │
│                                                         │
│   📊 LO QUE REALMENTE SUCEDE:                         │
│   "Los hashtags MAL USADOS no sirven"                  │
│                                                         │
│   Los hashtags siguen funcionando.                      │
│   Lo que cambió es CÓMO usarlos.                       │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Cómo funcionan los hashtags ahora:¶

    ALGORITMO 2026:
├── Hashtags = señal de contenido
├── Permiten que te encuentren SIN seguirte
├── Tienen límite efectivo: 3-5 por post
├── Más no es mejor — es más ruido
└── La relevancia > la cantidad

    2. La estrategia de 3 niveles¶

    El sistema de niveles:¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│  NIVEL 1: POPULARES (20%)                              │
│  ├── 1M - 10M de publicaciones                        │
│  ├── Gran alcance, mucha competencia                   │
│  └── Para ser descubiertos por nuevos ojos             │
│                                                         │
│  NIVEL 2: DE NICHO (50%)                              │
│  ├── 10K - 1M de publicaciones                        │
│  ├── Balance alcance + relevancia                     │
│  └── Para attract tu público objetivo                 │
│                                                         │
│  NIVEL 3: ESPECÍFICOS (30%)                          │
│  ├── Menos de 10K publicaciones                        │
│  ├── Menor alcance, mayor conversión                  │
│  └── Para attracts personas que SÍ van a comprar      │
│                                                         │
└─────────────────────────────────────────────────────────┘

    Ejemplo para collar de plata:¶

    #joyas (N1 - 12M) — Gran alcance
#plataplata925 (N2 - 85K) — Tu público
#colgantesplata (N2 - 45K) — De nicho
#joyeriaboutique (N3 - 3K) — Específico
#joyasconencargo (N3 - 1.5K) — Alta intención
#joyasargentinas (N3 - 8K) — Tu marca país

    3. Cuántos hashtags usar¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│   📊 RECOMENDACIÓN 2026:                              │
│                                                         │
│   INSTAGRAM: 5-8 hashtags máximo                       │
│   ├── 1-2 de nivel popular                            │
│   ├── 3-4 de nicho                                    │
│   └── 2-3 específicos                                 │
│                                                         │
│   NO SON 20 NI 30. SON 5-8 BIEN ELEGIDOS.            │
│                                                         │
│   PIEDAD: Los 20 hashtags parecen desesperación.       │
│                                                         │
└─────────────────────────────────────────────────────────┘

    ¿Por qué no más?¶

    ❌ +20 hashtags = spammy = reduce engagement
❌ +20 hashtags = dilución de relevancia
❌ +20 hashtags = parece automatización

✅ 5-8 hashtags = estratégico
✅ 5-8 hashtags = relevante
✅ 5-8 hashtags = profesional

    6. Errores comunes con hashtags¶

    ┌─────────────────────────────────────────────────────────┐
│                                                         │
│ ❌ ERROR 1: Todos populares o todos específicos        │
│    → Necesitás el balance de los 3 niveles            │
│                                                         │
│ ❌ ERROR 2: Hashtags irrelevantes                      │
│    → "#moda" en post de jewelry no = sentido         │
│                                                         │
│ ❌ ERROR 3: Siempre los mismos hashtags               │
│    → Cambiá según el post y contenido                │
│                                                         │
│ ❌ ERROR 4: Hashtags genéricos                        │
│    → #joyas es muy broad. #plataplata925 es mejor   │
│                                                         │
│ ❌ ERROR 5: Poner hashtags sin leer el post           │
│    → Cada hashtag tiene que relacionarse             │
│      con lo que mostrás                              │
│                                                         │
│ ❌ ERROR 6: 20+ hashtags                             │
│    → 5-8 bien elegidos > 20 cualquiera               │
│                                                         │
└─────────────────────────────────────────────────────────┘

    7. Estrategia por cuenta nueva vs. establecida¶

    Cuenta nueva (< 1,000 seguidores):¶

    ✓ Usá más hashtags populares para ser descubierto
✓ Usá hashtags locales para construir comunidad
✓ Probá diferentes combinaciones
✓ Enfocate en relevancia sobre cantidad

EJEMPLO:
#joyas #jewelry #fashion #plataplata925
#joyeriauruguaya #joyasuy #montevideo

    Cuenta establecida (1,000+ seguidores):¶

    ✓ Enfocate en hashtags de nicho y específicos
✓ Usá hashtags de marca (#nombredemarca)
✓ Probá hashtags de competidores
✓ Monitoreá cuáles traen más engagement

EJEMPLO:
#joyeria #plataplata925 #colgantesplata
#joyasuyboutique #joyasinfin #lujoboutique

    8. Herramientas para investigar hashtags¶

    📱 HERRAMIENTAS:

1. Instagram Search
   → Escribí el hashtag y vedé cuántas publicaciones tiene
   → También vedé hashtags relacionados

2. Hashtag Generator apps
   → Hashtagify
   → Display Purposes
   → All Hashtag

3. Análisis manual
   → Buscá el hashtag en Instagram
   → Vedé qué posts aparecen primero
   → Si son similares al tuyo, es buen hashtag

    9. Cómo organizar tus hashtags¶

    Opción 1: En el caption (separados por línea)¶

    ...tu caption termina aquí.

#joyas #plataplata925 #colgantes
#joyeriaboutique #joyasinfin
#joyasparasí #regaloperfecto

    Opción 2: En el primer comentario¶

    Recomendado para posts largos.
En el caption solo va el CTA.
Todos los hashtags en el primer comentario.

    Opción 3: Mix (los importantes en caption, resto en comment)¶

    ...tu caption + CTA.

#joyas #plataplata925

(continúa en comentarios 👇)

    11. Tarea de práctica¶

    Nivel básico:¶

    Elegí 5 posts de ejemplo y asignales hashtags usando la estrategia de 3 niveles. Verificá que no repitas más de 3 hashtags entre posts.

    Nivel avanzado:¶
