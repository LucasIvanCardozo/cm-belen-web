    Módulo 8 — Herramientas de Análisis¶

    Objetivos de Aprendizaje¶

    Al finalizar este módulo vas a: - ✅ Conocer las mejores herramientas gratuitas para analytics - ✅ Elegir la herramienta correcta según tu necesidad - ✅ Usar múltiples herramientas en combinación - ✅ Optimizar tu flujo de trabajo de análisis - ✅ Saber cuándo una herramienta vale la inversión

    2. Meta Business Suite (Profundo)¶

    MÁS DETALLES:

FUNCIONES AVANZADAS QUE PODÉS USAR:

1. EXPORTAR DATOS
   → Insights → Exportar → CSV
   → Podés abrir en Excel/Google Sheets
   → Útil para análisis más detallados

2. COMPARAR PERÍODOS
   → Elegí dos períodos para comparar
   → Ve mejoras o caídas lado a lado

3. FILTRAR POR TIPO DE CONTENIDO
   → Solo reels, solo carruseles, etc.
   → Analizá qué formato funciona mejor

4. PROGRAMAR PUBLICACIONES
   → Análisis + Programación en un solo lugar
   → Planificá desde el calendario

5. ALERTAS AUTOMÁTICAS
   → Configurá notificaciones cuando metricas suben/bajan

    3. Google Analytics (si tenés website)¶

    CUÁNDO USARLO:
Solo si tenés una tienda online o website linked a tu IG.

QUÉ MIDE:
• Tráfico desde Instagram a tu website
• Comportamiento de visitantes
• Conversiones (ventas)
• Keywords que traen tráfico

CONFIGURACIÓN BÁSICA:
1. Creá cuenta de Google Analytics (gratis)
2. Agregá el código a tu website
3. Vinculá con tu IG (si es posible)

PARA JOYERÍA E-COMMERCE:
• Medí: Visitas desde IG → Compras realizadas
• Calculá: Tasa de conversión IG → Website

EJEMPLO:
• 1,000 visitas desde IG
• 25 compras
• Tasa de conversión: 2.5%
• Valor promedio de compra: $150
• Ingresos generados: $3,750

¡Esto demuestra ROI real de tu gestión!

    4. Later (Free tier)¶

    LATER - PROGRAMACIÓN + ANALYTICS

LÍMITE FREE:
• 1 perfil social
• 30 publicaciones/mes
• Análisis básicos

FUNCIONES:
✓ Calendario visual de publicaciones
✓ Programación de contenido
✓ Análisis de mejores horarios
✓ Preview de feed
✓ Hashtag suggestions

PARA QUÉ SIRVE EN ANÁLISIS:
• Ver cuándo publicás (calendario visual)
• Identificar gaps en tu calendario
• Planificar contenido estratégico
• Ver distribución de tipos de contenido

PARA QUÉ NO SIRVE:
• Análisis profundos de engagement
• Datos detallados de audiencia

¿VALE LA PENA?
→ Para empezar: Sí, el calendario visual ayuda
→ Cuando crezcás: Considerá paid tier o Metricool

    5. Metricool (Free tier)¶

    METROOL - ANÁLISIS + PROGRAMACIÓN

LÍMITE FREE:
• Análisis de 2 perfiles
• 30 programaciones/mes
• Reportes básicos

FUNCIONES:
✓ Análisis de múltiples redes
✓ Comparación temporal
✓ Mejores horarios
✓ Reportes schedule
✓ Análisis de hashtags

PARA QUÉ SIRVE EN ANÁLISIS:
✓ Más visual que Meta Business Suite
✓ Comparación de períodos más fácil
✓ Programación included
✓ Reportes automáticos por email

¿VALE LA PENA vs. Later?
→ Metricool tiene mejor análisis que Later
→ Later tiene mejor calendario visual
→ Metricool es mejor para métricas
→ Later es mejor para programación

RECOMENDACIÓN:
Empezá con Meta Business Suite + Later free.
Si necesitás más análisis, Metricool es buena opción.

    6. Iconosquare (para análisis profundo)¶

    ICONOSQUARE - ANÁLISIS PROFUNDO

LÍMITE FREE: 30 días de prueba gratis

FUNCIONES:
✓ Análisis de engagement
✓ Análisis de hashtags
✓ Análisis de Stories
✓ Análisis de audience
✓ Comparaciones detalladas
✓ Scheduling

PARA QUÉ SIRVE:
→ Análisis más detallado que Meta Business Suite
→ Análisis de Stories con más métricas
→ Comparación con competidores (si ponés sus handles)
→ Reports más visuales

VALE LA PENA LA INVERSIÓN?
• Si tenés clientes paying: Sí, vale $15-30/mes
• Si es para tu cuenta personal: Probablemente no

BENCHMARK:
• Solo hacé upgrade si necesitás features específicas
• Ejemplo: Comparar con 5 competidores a la vez

    7. Herramientas específicas¶

    PARA HASHTAGS:

1. HASHTIFY (Gratis)
   → Análisis de hashtags
   → Suggestion de hashtags relacionados
   → Hashtag performance tracking

2. ALLHASHTAG (Gratis)
   → Generador de hashtags
   → Ranking de hashtags populares
   → Análisis de performance

PARA UBICACIÓN:
→ Instagram Insights ya muestra ubicación de audiencia
→ Solo necesitás eso para la mayoría de los casos


PARA ANALIZAR COMPETIDORES:

1. PHANTOMBUZZER (Free tier)
   → Analizá cualquier cuenta pública
   → Métricas sin necesidad de ser owner

2. SOCIALBLADE (Gratis)
   → Estadísticas de crecimiento
   → Histórico de seguidores
   → Estimaciones de engagement


PARA CAPTIONS:

→ CO-SCHEDULE HEADLINE ANALYZER (Gratis)
   → Analiza tu caption antes de publicar
   → Score de engagement esperado
   → Sugerencias de mejora

    10. Cuándo pagar por herramientas¶

    ¿CUÁNDO VALE LA PENA PAGAR?

PAGÁ SI:

✓ Tenés 3+ clientes pagados
   → Necesitás reportar a múltiples personas
   → Herramientas como Metricool paid valen la pena

✓ Necesitás programación avanzada
   → Más de 30 posts/mes
   → Múltiples perfiles

✓ Querés análisis de competidores
   → Benchmark contra 5+ cuentas

✓ Necesitás reportes automáticos
   → Enviar reportes por email automáticamente
   → Ahorra tiempo

NO PAGUÉS SI:

✗ Es para tu cuenta personal
✗ Empezando como CM freelance
✗ Menos de 3 clientes

MI RECOMENDACIÓN:
Empezá con herramientas gratuitas.
Paga solo cuando SIENTAS el dolor de no tener la feature.

    12. Ejercicio: Configurá tu stack¶

    EJERCICIO: Armá tu kit de análisis

PASO 1: Checklist
   □ ¿Ya tenés Meta Business Suite instalado?
   □ ¿Ya tenés Google account (para Sheets)?
   □ ¿Querés usar Later o Metricool?

PASO 2: Configurá Meta Business Suite
   → Si no lo hiciste: descargá la app
   → Vinculá tu cuenta de Instagram
   → Explorá todas las secciones

PASO 3: Exportá tus primeros datos
   → Insights → Últimos 30 días
   → Exportar a CSV
   → Guardá en Google Drive

PASO 4: Creá tu primera tabla
   → Abrí Google Sheets
   → Importá tu CSV
   → Organizá las columnas

PASO 5: Compartí con alguien
   → Mostráselo a un amigo o a vos misma
   → Pedí feedback sobre claridad

ENTREGA:
• Stack de herramientas configurado
• Primeros datos exportados
• Dashboard básico en Sheets
