    Módulo 6 — Crear Reportes para Clientes¶

    Objetivos de Aprendizaje¶

    Al finalizar este módulo vas a:

    • ✅ Estructurar un reporte profesional de CM
    • ✅ Incluir las métricas correctas
    • ✅ Presentar datos de forma clara y actionnable
    • ✅ Evitar reportes que nadie lee
    • ✅ Hacer reportes que demuestren tu valor

    1. Por qué los reportes importan¶

    UN BUEN REPORTE:

✓ Demuestra tu valor como CM
✓ Justifica el dinero que te pagan
✓ Genera confianza con el cliente
✓ Identifica áreas de mejora
✓ Construye relación profesional

UN MAL REPORTE:

✗ Es una lista de números sin contexto
✗ Nadie lo lee porque es aburrido
✗ No lleva a acciones
✗ No demuestra valor
✗ El cliente pregunta "¿y qué hacemos con esto?"

    2. Estructura de un reporte profesional¶

    SECCIONES DEL REPORTE:

1. RESUMEN EJECUTIVO (30 segundos de lectura)
   → Los highlights más importantes
   → "TL;DR" del mes

2. MÉTRICAS PRINCIPALES
   → Los números más importantes
   → Comparación con período anterior

3. RENDIMIENTO DE CONTENIDO
   → Qué tipo funcionó mejor
   → Posts destacados

4. INSIGHTS Y APRENDIZAJES
   → Qué aprendimos de los datos
   → Por qué pasaron las cosas

5. RECOMENDACIONES
   → Qué hacer el próximo mes
   → Acciones concretas

6. PRÓXIMOS PASOS
   → Tareas pendientes
   → Objetivos del próximo período

    3. Resumen Ejecutivo (la sección más importante)¶

    RESUMEN EJECUTIVO:

El resumen ejecutivo es lo primero que ve el cliente.
Dedele 5 minutos máximo. Debe responder:
• ¿Cómo fue el mes?
• ¿Qué funcionó?
• ¿Qué necesitamos mejorar?

PLANTILLA:

┌─────────────────────────────────────────────────────────────┐
│ 📊 RESUMEN EJECUTIVO - Mayo 2026                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ ESTADO GENERAL: ✅ Positivo                                │
│                                                             │
│ HIGHLIGHTS DEL MES:                                         │
│ • Alcance creció 23% vs. mes anterior                        │
│ • Engagement rate de 4.2% (meta: 4%)                      │
│ • Mejor publicación: Reel de Día de la Madre                │
│   → 15K alcance, 8% engagement                            │
│                                                             │
│ ÁREAS DE MEJORA:                                           │
│ • Guardados por debajo de meta (2.1% vs 3%)                │
│ • Accion: Agregar más contenido educativo                   │
│                                                             │
│ ACCIÓN PRIORITARIA:                                          │
│ • Preparar campaña de San Valentín (Febrero)               │
│                                                             │
└─────────────────────────────────────────────────────────────┘

REGLA: Si el cliente solo lee esta sección, igual se lleva
       lo más importante.

├─────────────────────────────────────────────────────────────┤
│                                                             │

│    Los reels representan el 62% del alcance total.          │

    10. Formato del reporte¶

    OPCIONES DE FORMATO:

1. DOCUMENTO DE TEXTO (Google Docs, Word)
   ✓ Fácil de crear y editar
   ✓ Rápido de enviar por email
   ✗ Menos visual

2. PDF PRESENTABLE (Canva)
   ✓ Se ve profesional
   ✓ Fácil de compartir
   ✗ Más tiempo de creación

3. VIDEO (Loom, Zoom)
   ✓ Muy personal
   ✓ Puedes explicar en detalle
   ✗ No todos quieren ver videos

4. COMBINADO (Recomendado)
   → Documento PDF con lo esencial
   → Llamada/video de 15 min para explicar
   → Ambos canales abiertos para preguntas

MI RECOMENDACIÓN PARA PRINCIPIANTES:
Empezá con documento Google Docs.
Cuando tengas más confianza, pasá a PDF de Canva.

    11. Ejercicio: Crea tu primer reporte¶

    EJERCICIO: Armá un reporte para tu cuenta personal

SI NO TENÉS CLIENTE:
→ Usa tu propia cuenta de Instagram
→ Tratá como si te pagaran a vos misma
→ Este ejercicio te prepara para cuando tengas cliente real

PASO 1: Recolectá datos
   → Meta Business Suite → Insights → Últimos 30 días
   → Anotá: Alcance, Engagement, Seguidores, Guardados

PASO 2: Identificá highlights
   → ¿Qué metricas mejoraron?
   → ¿Cuál fue tu mejor post?
   → ¿Qué tipo de contenido funcionó mejor?

PASO 3: Identificá áreas de mejora
   → ¿Qué metricas bajaron?
   → ¿Qué contenido no funcionó?
   → ¿Qué podríamos mejorar?

PASO 4: Escribí el reporte
   → Usa la plantilla de arriba
   → Incluye: Resumen ejecutivo + Métricas + Insights + Recomendaciones

PASO 5: Compartilo
   → Enviáselo a vos misma o a un amigo
   → Pedí feedback sobre claridad
   → Ajustá para la próxima vez

ENTREGA: Tu primer reporte de CM

    12. Resumen¶

    PUNTOS CLAVE:

✓ El reporte tiene 6 secciones:
  1. Resumen Ejecutivo

  6. Próximos Pasos
