    Módulo 2 — Estética y Grid para Joyería¶

    Objetivos de Aprendizaje¶

    Al finalizar este módulo vas a:

    • ✅ Entender qué es el "grid" y por qué importa
    • ✅ Conocer los diferentes patrones de grid
    • ✅ Aplicar coherencia visual a tu cuenta de joyería
    • ✅ Crear un perfil que atraiga clientes de lujo
    • ✅ Planificar tu grid con anticipación

    1. Qué es el Grid de Instagram¶

    Definición simple¶

    El grid de Instagram es la vista de 3x3 publicaciones que aparece
cuando alguien entra a tu perfil. Es tu carta de presentación visual.

PRIMERA IMPRESIÓN = ¿Me quedo o me voy?

El grid es lo primero que ven potenciales clientes cuando te descubren.
Tienes 3 SEGUNDOS para convencerlos de seguirte.

PARA JOYERÍA: Esto es CRUCIAL porque:
• Tu producto ES visual
• La calidad del producto se percibe visualmente
• El lujo se transmite a través de la estética

    2. Por qué la Estética Importa en Joyería¶

    EL LUJO SE VE, NO SE EXPLICA

Cuando alguien llega a tu perfil de joyería, busca:
✓ Coherencia → "¿Esta marca sabe lo que hace?"
✓ Calidad → "¿Las fotos son profesionales?"
✓ Estilo → "¿Esto refleja lo que yo busco?"
✓ Confianza → "¿Puedo confiar en comprar aquí?"

Si tu grid es un desastre, el mensaje es:
"Esta joyería no sabe presentarse. ¿Cómo van a saber hacer joyas?"

PARA JOYERÍA, EL GRID DEBE DECIR:
"Nosotros tomamos esto en serio. Cada detalle importa.
Si así se ven las fotos, imaginate las joyas."

    3. Los 6 Patrones de Grid¶

    3.1 Grid Checkerboard (Tablero de Ajedrez)¶

    PATRÓN:
Alternas entre dos tipos de contenido en cada fila.

Ejemplo:
[ Foto clara ] [ Foto oscura ] [ Foto clara ]
[ Foto oscura ] [ Foto clara ] [ Foto oscura ]
[ Foto clara ] [ Foto oscura ] [ Foto clara ]

PARA JOYERÍA:
✓ Alternar: producto solo vs lifestyle
✓ Alternar: fondo claro vs fondo oscuro
✓ Alternar: pieza única vs styled shot

VENTAJAS: fácil de mantener, alto contraste visual
DESVENTAJAS: puede sentirse repetitivo

EJEMPLO REAL:
Row 1: [Collar dorado] [Collar en modelo] [Collar macro]
Row 2: [Modelo con collar] [Collar packaging] [Close-up detalles]

    3.2 Grid con Columnas (Column Layout)¶

    PATRÓN:
Cada columna tiene un tipo de contenido específico.

Ejemplo:
[ Producto ]  [ Lifestyle ]  [ Detalle ]
[ Producto ]  [ Lifestyle ]  [ Detalle ]
[ Producto ]  [ Lifestyle ]  [ Detalle ]

COLUMNA 1: Solo producto, fondo blanco
COLUMNA 2: Modelo/influencer usando la pieza
COLUMNA 3: Close-up, detalles, packaging

PARA JOYERÍA:
✓ Columna 1: Joyas en fondo limpio
✓ Columna 2: Joyas en contexto (outfit, lifestyle)
✓ Columna 3: Detalles, macros, packaging

VENTAJAS: muy organizado, fácil de planificar
DESVENTAJAS: puede ser rígido

    3.3 Grid de Líneas (Row Stripes)¶

    PATRÓN:
Cada fila es un "bloque" temático.

Ejemplo:
Row 1: [ Post 1 ] [ Post 2 ] [ Post 3 ] → TODOS de la misma campaña
Row 2: [ Post 4 ] [ Post 5 ] [ Post 6 ] → TODOS lifestyle
Row 3: [ Post 7 ] [ Post 8 ] [ Post 9 ] → TODOS de producto

PARA JOYERÍA:
✓ Row 1: Línea San Valentín (todo rosa/rojo)
✓ Row 2: Behind the scenes (todo más casual)
✓ Row 3: Producto del mes (todo serie nueva)

VENTAJAS: permite campaigns visuales claras
DESVENTAJAS: requiere planificación anticipada

    3.4 Grid Orgánico/Mixto¶

    PATRÓN:
No hay patrón estricto, pero hay coherencia general.

Ejemplo:
Posts individuales tienen relación en tono, paleta, calidad
pero no hay estructura rígida.

PARA JOYERÍA:
✓ Misma paleta de colores general
✓ Misma calidad de edición
✓ Mismo estilo de captions
✓ Pero variety en el contenido

VENTAJAS: flexible, natural
DESVENTAJAS: requiere ojo entrenado para mantener coherencia

CUÁNDO USARLO: Cuando ya tenés identidad de marca clara

    3.5 Grid con Foto Destacada¶

    PATRÓN:
Una foto "hero" o "feature" rompe el patrón de forma intencional.

Ejemplo:
[ Foto 1 ] [ Foto 2 ] [ ⭐ HERO ]
[ Foto 4 ] [ Foto 5 ] [ Foto 6 ]
[ Foto 7 ] [ ⭐ HERO ] [ Foto 9 ]

PARA JOYERÍA:
✓ Una foto especialmente stunning cada 6-9 posts
✓ Foto de campaña o lanzamiento
✓ Foto que quieras que se destaque

VENTAJAS: genera impacto, crea momentos
DESVENTAJAS: requiere contenido excepcional regularmente

    3.6 Grid Blanco Total¶

    PATRÓN:
Todas las fotos tienen fondo blanco o muy limpio.

Ejemplo:
[ Producto A ] [ Producto B ] [ Producto C ]
[ Producto D ] [ Producto E ] [ Producto F ]

PARA JOYERÍA:
✓ Ideal para joyería minimalista
✓ Transmite claridad y enfoque
✓ Funciona muy bien para oro amarillo y plata

VENTAJAS: ultra profesional, timeless
DESVENTAJAS: poco flexible para variety

EJEMPLO: Tiffany & Co, Mejuri,Mejuri usan versiones de esto

    4. Cómo Elegir tu Patrón¶

    Guía de decisión¶

    ┌─────────────────────────────────────────────────────┐
│ PREGUNTA                          RESPUESTA = PATRÓN│
├─────────────────────────────────────────────────────┤
│ ¿Tenés mucho contenido lifestyle?  → Checkerboard   │
│ ¿Querés organizar por categoría?    → Columnas      │
│ ¿Tenés campaigns específicas?        → Rows          │
│ ¿Tu marca es muy establecida?       → Orgánico      │
│ ¿Tu joyería es minimalista/dorada? → Blanco total   │
│ ¿Tenés contenido "hero" regular?    → Foto Destacada │
└─────────────────────────────────────────────────────┘

PRO TIP: Podés combinar patrones.
Ejemplo: Columnas + a veces hero photo

    5. Los 5 Pilares de Estética para Joyería¶

    5.1 Paleta de Colores¶

    PARA JOYERÍA DE LUJO:

PALETA RECOMENDADA:
┌─────────────────────────────────────────────────────┐
│ COLORES PERMITIDOS:                                │
│ ✓ Blanco (universal, limpio)                       │
│ ✓ Negro (dramático, contraste)                     │
│ ✓ Oro/Dorado (si es tu producto)                  │
│ ✓ Tonos neutros (beige, gris, nude)               │
│ ✓ Colores apagados (dusty rose, sage green)       │
│                                                     │
│ COLORES A EVITAR:                                  │
│ ✗ Neon o fluorescentes                             │
│ ✗ Multicolor sin propósito                         │
│ ✗ Fotos con muchos colores compitiendo             │
│ ✗ Filtros que distorsionan el color real            │
└─────────────────────────────────────────────────────┘

EJEMPLO DE PALETA PARA CUENTA DE ORO:
• Blanco cálido (#FFF8F0)
• Negro profundo (#1A1A1A)
• Oro (#D4AF37)
• Champán (#F7E7CE)

    5.2 Calidad Consistente¶

    REGLA DE ORO: TODAS las fotos deben verse del mismo "nivel".

QUÉ INCLUYE:
• Iluminación consistente (o editarla para que lo parezca)
• Misma resolución (mínimo 1080px)
• Mismo estilo de edición (o preset de color)
• Fondo coherente (o edición que lo unifique)

COMPROBAR:
Antes de publicar, mirá tu grid en tu celular.
¿Todas las fotos parecen de la "misma marca"?

SI LA RESPUESTA ES NO:
→ Editá las fotos para que tengan la misma temperatura
→ Usá el mismo preset en todas
→ Uniformizá los fondos

    5.3 Tipografía en Captions¶

    PARA JOYERÍA DE LUJO:

CAPTION STYLE:
• Font: Sans-serif limpio, o serif elegante
• Tamaño: Legible en móvil (mínimo 14px si es texto en foto)
• Color: Negro o blanco sobre la imagen
• Cantidad: Mínimo, como acento
•alineación: Centro o izquierda (nunca justificado)

HERRAMIENTAS:
• Canva (text tool)
• Over (app específica)
• Phonto (iPhone)
• Vont (iPhone)

PROHIBIDO:
✗ Comic Sans o fonts cursivas genéricas
✗ Texto que compita con el producto
✗ Demasiado texto que tape la joya
✗ Fonts de colores brillantes

    5.4 Presets de Edición¶

    QUÉ ES UN PRESET: Configuración guardada de edición que aplicás
a todas las fotos para unificar el look.

PARA JOYERÍA:
PRESET "ORO CÁLIDO":
• Exposición: +0.3
• Contraste: +10
• Temperatura: +5 (más cálido)
• Tinte: +3 (ligeramente dorado)
• Sombras: +5
• Highlights: -10

PRESET "PLATA FRÍA":
• Exposición: +0.2
• Contraste: +15
• Temperatura: -8 (más frío)
• Tinte: 0
• Sombras: +3
• Highlights: -5

DÓNDE GUARDAR:
• Lightroom Mobile (preset guardado en cuenta)
• VSCO (preset guardado)
• Snapseed (guardado como "look")

APLICAR SIEMPRE ANTES DE PUBLICAR

    5.5 Fondo y Espacio Negativo¶

    PARA JOYERÍA:

FONDO IDEAL:
✓ Blanco puro (joyería clara)
✓ Negro (joyería oscura o con piedras de color)
✓ Gris medio (versátil)
✓ Tono nude/beige (lifestyle)

ESPACIO NEGATIVO:
El espacio alrededor del producto NO ES DESPERDICADO.
Es parte del diseño.

┌─────────────────────────────┐
│                             │
│         ┌───┐               │
│         │ 💎│               │ ← Joya centrada
│         └───┘               │
│                             │ ← Espacio negativo
│                             │
└─────────────────────────────┘

REGLA: El producto debe tener "aire" para respirar.
No lo tapes con otros elementos.

    6. Cómo Planificar tu Grid¶

    6.1 Herramientas para planificar¶

    HERRAMIENTAS RECOMENDADAS:

📱 apps de PLANNING DE GRID:

1. Plann (iOS/Android)
   → Planificador visual de grid
   → Tiene versión gratuita
   → Precio: gratis / $15USD/mes pro

2. Preview (iOS/Android)
   → Grid planner + caption manager
   → Precio: gratis / $8USD/mes

3. UNUM (iOS/Android)
   → Grid planner simple
   → Precio: gratis / $5USD/mes pro

4. Later (Desktop)
   → Scheduling + grid preview
   → Precio: gratis / $18USD/mes

MANUAL:
También podés usar Pinterest o simplemente
hacer screenshot de tu grid y organizar en una carpeta.

    6.2 Proceso de planificación¶

    PASO A PASO:

1️⃣ REUNÍ TODO TU CONTENIDO
   → fotos listas para publicar
   → organiza por fecha/prioridad

2️⃣ DEFINÍ EL PATRÓN DE GRID
   → Elegí uno de los 6 patrones
   → O combiná los que mejor funcionen

3️⃣ ARMA EL GRID VIRTUAL
   → Usá Plann o similar
   → Arrastrá las fotos al orden correcto
   → Ajustá hasta que fluy bien

4️⃣ REVISÁ QUE TENGA SENTIDO
   → ¿Se ve cohesivo?
   → ¿Hay variedad suficiente?
   → ¿Los cambios entre filas no son muy bruscos?

5️⃣ AJUSTÁ Y PUBLICÁ
   → Hacé los cambios necesarios
   → Publicá según el calendario
   → Repite semanalmente

TIEMPO ESTIMADO: 30-45 minutos semanal

    6.3 Timing de publicación¶

    NO PUBLIQUES EN ORDEN DE CREACIÓN.
PUBLICA EN ORDEN DE GRID.

Ejemplo:

Semana planificada:
[Post A] [Post B] [Post C] → Publicás A primero
[Post D] [Post E] [Post F] → Después B, después C
[Post G] [Post H] [Post I] → Y así sucesivamente

HERRAMIENTA: Later o Meta Business Suite
→ Programá los posts en el orden del grid
→ Verificá que el orden se mantenga

    7. Estética por Tipo de Joyería¶

    Para Oro¶

    PALETA: Dorados, blancos cálidos, nudes
FONDO: Blanco, negro, o dorado suave
EDICIÓN: Cálida, brillante, emphasize el oro
LUZ: Que resalte el brillo del metal

EJEMPLO DE GRID:
Row 1: [Oro en blanco] [Modelo dorado] [Macro oro]
Row 2: [Outfit con oro] [Oro packaging] [Detalle cierre]
Row 3: [Oro vs plata] [Oro lifestyle] [Colección completa]

    Para Plata¶

    PALETA: Blancos fríos, grises, azules suaves
FONDO: Blanco puro, negro, o gris medio
EDICIÓN: Fría, nítida, emphasize el brillo plateado
LUZ: Que capture los reflejos

EJEMPLO DE GRID:
Row 1: [Plata en negro] [Plata modelo] [Macro plata]
Row 2: [Plata casual] [Plata detallada] [Packaging minimal]
Row 3: [Plata lifestyle] [Colección plata] [Plata + piedra]

    Para Piedras de Color¶

    PALETA: Neutros que no compitan con la piedra
FONDO: Depende del color de la piedra
EDICIÓN: Que preserve/elogie el color de la gema
LUZ: Careful, las piedras son traicioneras con luz

EJEMPLO DE GRID:
Row 1: [Esmeralda en blanco] [Detalle esmeralda] [Modelo esmeralda]
Row 2: [Zafiro en negro] [Colección zafiro] [Macro zafiro]
Row 3: [Amatista lifestyle] [Comparativa] [Gift guide esmeralda]

    Para Piedras Preciosas (Alto Lujo)¶

    PALETA: Ultra minimalista, neutros puros
FONDO: Negro profundo o blanco puro
EDICIÓN: Cine, editorial, dramática
LUZ: Studio lighting ideal

EJEMPLO DE GRID:
Row 1: [Diamante negro] [Diamante macro] [Diamante modelo]
Row 2: [Joyas apiladas] [Colección completa] [Detalle craftsmanship]
Row 3: [Editorial completa] [Detalle cierre] [Packaging lujo]

    8. Errores Comunes¶

    ❌ ERROR 1: Grid Inconsistente¶

    PROBLEMA: Algunas fotos se ven increíbles, otras amateur.

SOLUCIÓN:
→ Establecé un preset de edición y úsalo SIEMPRE
→ Tené iluminación estándar para sesiones
→ Revisá cada foto antes de publicar
→ Invierte en sesiones de fotos profesionales periódicamente

    ❌ ERROR 2: Demasiada Variedad¶

    PROBLEMA: Cada post parece de una marca diferente.

SOLUCIÓN:
→ Definí 3-5 colores/tipos de fondo MÁXIMO
→ Mantené la misma calidad de edición
→ Reducí los "moody pics" vs "product shots"

    ❌ ERROR 3: Ignorar el Cover de Reels¶

    PROBLEMA: Los thumbnails de reels arruinan el grid.

SOLUCIÓN:
→ Elegí siempre el frame del video que vaya con tu grid
→ O usá un cover personalizado con tu estética
→ Revisá cómo se ve TU GRID incluyendo reels

    ❌ ERROR 4: No pensar en el scroll¶

    PROBLEMA: Cuando alguien scrollea tu grid, no hay flujo.

SOLUCIÓN:
→ Planificá cómo se ve la TRANSICIÓN entre posts
→ Evitá que dos posts similares estén juntos
→ Intentá que cada 3 posts haya un "cambio" visual
