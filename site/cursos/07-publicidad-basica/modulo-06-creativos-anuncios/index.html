    Módulo 6 — Creativos de Anuncios¶

    Objetivos de Aprendizaje¶

    Al finalizar este módulo vas a: - ✅ Entender qué hace que un ad de joyería funcione - ✅ Crear fotos que generen clicks y ventas - ✅ Escribir textos que convenzan - ✅ Diseñar videos cortos efectivos - ✅ Implementar CTAs claros - ✅ Testear diferentes creatividades

    1. Por qué el creativo es crucial¶

    EL CREATIVO (foto/video + texto) ES LO QUE TU AUDIENCIA VE.

Puedes tener:
• El mejor targeting del mundo
• El presupuesto perfecto
• La oferta ideal

PERO SI TU FOTO ES MALA:
→ Nadie para el scroll
→ Nadie hace click
→ Gastás dinero erre reach a nadie

REALIDAD:
El 80% del éxito de un ad depende del creativo.
Por eso los mejores marketers testean creatividades constantemente.

    2. Tipos de creativos para joyería¶

    FORMATOS QUE FUNCIONAN PARA JOYERÍA:

1. 📸 FOTO DE PRODUCTO LIMPIA
   → Joya sola, fondo blanco
   → Mostrá el producto claramente

2. 🌟 FOTO LIFESTYLE
   → Joya en contexto de uso
   → Modelo usando la joya
   → Día a día, aspiracional

3. 🎬 VIDEO CORTO
   → Mostrar la joya en movimiento
   → Detalles del craftsmanship
   → "POV: When you find THE piece"

4. 📱 REELS/PANTALLAS
   → Antes/después
   → Transformaciones
   → Comparaciones

5. 🎨 CARRUSEL
   → Múltiples vistas del producto
   → Diferentes ángulos
   → Colección completa

6. 📖 CAROUSEL DE STORYTELLING
   → Historia en imágenes
   → Emoción en cada slide

    3. Fotografía para ads¶

    3.1 Principios de fotos que venden¶

    REGLAS DE ORO:

1. LUZ ES TODO
   → Luz natural suave (junto a ventana)
   → Evitar sombras duras
   → Brillo en piedras = valor percibido más alto

2. FONDO SIMPLE
   → Fondo blanco o neutro
   → Que la joya sea la protagonista
   → Sin distracciones

3. FOCO EN EL PRODUCTO
   → Joya clara y nítida
   → Sin elementos competiendo
   → Detalles visibles

4. EMOCIÓN
   → Si hay modelo: que se note alegría
   → Contexto aspiracional
   → "Quiero ser esa persona"

5. CALIDAD PERCIBIDA
   → Foto debe verse profesional
   → Esto transmite valor
   → "Si se ve cara, es cara"

    3.2 Tipos de fotos por objetivo¶

    PARA ALCANCE (que la gente pare el scroll):
→ Fotos impactantes
→ Colores vibrantes
→ Detalles llamativos
→ Gemas brillando

PARA ENGAGEMENT (que la gente guarde/comente):
→ Fotos "deseables"
→ "Quiero esto"
→ Estética aspiracional
→ Que inspire

PARA VENTAS (que la gente compre):
→ Foto del producto clara
→ Muestra el producto bien
→ Incluye precio o valor
→ CTA claro

    3.3 Errores de foto comunes¶

    ❌ MAL:
• Foto oscura (no se ve el producto)
• Fondo cargado (distrae)
• Calidad baja (celular sin luz)
• Joya fuera de foco
• Demasiados productos juntos

✅ BIEN:
• Luz natural suave
• Fondo simple (blanco, gris, negro)
• Producto claro y nítido
• Detalle que muestre calidad
• Solo la joya protagonista

    4. Texto del anuncio (Copy)¶

    4.1 Estructura del copy para ads¶

    ESTRUCTURA EFECTIVA:

[HOOK - Gancho]
↓ Primeras palabras que paran el scroll
"Este collar lo cambió todo para mí."

[PROBLEMA]
↓ Qué problema resuelve
"Buscaba algo especial para mi graduación..."

[SOLUCIÓN]
↓ Cómo tu joya resuelve eso
"...y cuando lo vi, supe que era este.
Plata 925 con piedra de luna."

[BENEFICIO EMOCIONAL]
↓ Por qué importa
"Cada vez que lo uso, siento que puedo lograr lo que me proponga."

[CTA]
↓ Qué hacer ahora
→ [Link en bio / Toca para ver más]

[SOCIAL PROOF]
↓ Por qué confiar
"Más de 500 mujeres ya lo tienen."

    4.2 Tips para copy efectivo¶

    PARA JOYERÍA:

✓ HABLA A LA EMOCIÓN, NO AL PRODUCTO
  ❌ "Collar de plata 925 con zirconia"
  ✓ "El collar que vas a usar cada vez que quieras sentirte especial"

✓ USA "YO" Y TESTIMONIOS
  ❌ "Este collar es hermoso"
  ✓ "Compré este collar hace 3 meses y no puedo dejar de usarlo"

✓ GENERÁ URGENCIA (sin mentir)
  ❌ "¡ÚLTIMAS UNIDADES!" (si no es cierto)
  ✓ "Solo quedan 3 de este modelo. Después no lo fabricamos más."

✓ PRECIO = VALOR, NO GASTO
  ❌ "$85 USD"
  ✓ "Una inversión en vos misma que va a durar generaciones"

✓ SPECIFICO > VAGO
  ❌ "Joyas hermosas"
  ✓ "Anillo de plata con piedra de ónix natural"

    4.3 Longitud del copy¶

    PARA FEED (Instagram):
• Primera línea = lo más importante (no cortés)
• Ideal: 100-150 palabras máximo
• Si es más largo, usá "más" para expand

PARA STORIES ADS:
• Máximo 125 caracteres por pantalla
• 3-5 pantallas máximo
• Cada pantalla = un mensaje

PARA CARRUSEL:
• Primera slide = gancho
• Cada slide = un punto
• Última slide = CTA

PARA VIDEO:
• Los primeros 3 segundos = gancho
• Mínimo texto (la gente está viendo, no leyendo)

    5. Llamadas a la acción (CTA)¶

    5.1 CTAs efectivos para joyería¶

    CTAs QUE FUNCIONAN:

PARA VENTA DIRECTA:
• "Comprar ahora"
• "Ver colección"
• "Consigue el tuyo"
• "Shop now"

PARA LEAD:
• "Enviar mensaje"
• "Escribinos por WhatsApp"
• "Reservá tu turno"
• "Consultá disponibilidad"

PARA ENGAGEMENT:
• "Guardá para después"
• "Comentá tu opinión"
• "¿Cuál preferís?"
• "Share con alguien que lo necesite"

PARA TRÁFICO:
• "Ver más"
• "Conocé más"
• "Descubrí la colección"

    5.2 Errores de CTA¶

    ❌ NO USES:
• "Click aquí" (genérico)
• "Más información" (aburrido)
• "Submit" (robot)
• "Leer más" (sin emoción)

✅ USA:
• Verbos de acción claros
• Lo que van a obtener
• Emoción o valor
• Urgencia cuando aplique

    6. Testing de creatividades¶

    6.1 Qué testear¶

    ELEMENTOS PARA TESTEAR:

1. IMAGEN
   → Foto producto vs Lifestyle
   → Fondo claro vs oscuro
   → Con modelo vs sin modelo

2. VIDEO
   → Corto (15s) vs Largo (30s)
   → Con audio vs Sin audio
   → Diferentes thumbnails

3. TEXTO
   → Diferentes hooks
   → Diferentes longitudes
   → Diferentes tonos (formal vs casual)

4. CTA
   → "Comprar ahora" vs "Ver colección"
   → Con precio vs Sin precio

    6.2 Cómo testear¶

    MÉTODO SIMPLE:

SEMANA 1:
Ad A: Foto producto + Copy emocional
Ad B: Foto lifestyle + Copy emocional
Ad C: Foto producto + Copy de oferta

RESULTADO DESPUÉS DE 7 DÍAS:
→ ¿Cuál tuvo mejor CTR?
→ ¿Cuál tuvo mejor ROAS?
→ ¿Cuál tuvo mejor engagement?

GANADOR:
→ Seguí corriendo solo el ganador
→ Creá variaciones del ganador

REGLA:
"No te cases con una creatividad.
Deja que los datos te digan cuál funciona."

    7. Especificaciones técnicas¶

    TAMAÑOS DE IMAGEN:

Feed de Instagram:
• Cuadrado: 1080 x 1080 px
• Vertical: 1080 x 1350 px
• Horizontal: 1080 x 566 px

Stories/Reels:
• 1080 x 1920 px (9:16)
• Mínimo 600 x 1067 px

Carrusel:
• 1080 x 1080 px por imagen
• Mínimo 2, máximo 10 imágenes

VIDEO:
• Duración: 1 segundo - 60 minutos
• Ideal para ads: 15-30 segundos
• Formato: MP4 o MOV
• Peso máximo: 4GB

TEXTOS:
• Headline: 40 caracteres
• Primary text: 125 caracteres (visible antes del "más")
• Description: 30 caracteres
• CTA: Automático de Meta

    8. Ejercicio práctico¶

    EJERCICIO 1: Crea 3 variaciones de un ad

PRODUCTO:
Anillo de oro 18k con esmeralda
Precio: $280 USD

VARIACIÓN A - FOCUS EN PRODUCTO:
• Foto: Anillo solo en fondo negro
• Texto: Describe el producto en detalle
• CTA: "Ver colección"

VARIACIÓN B - FOCUS EN EMOCIÓN:
• Foto: Modelo usando el anillo (lifestyle)
• Texto: Emoción de quien lo usa
• CTA: "Consigue el tuyo"

VARIACIÓN C - FOCUS EN OFERTA:
• Foto: Anillo + caja de regalo
• Texto: Por qué es buen momento para comprar
• CTA: "Reservalo ahora"

ESCRIBÍ el copy para cada uno.
Vas a testearlos en tu próxima campaña.

    9. Resumen del módulo¶

    PUNTOS CLAVE:

✓ El creativo es 80% del éxito del ad
✓ Foto: luz natural, fondo simple, producto protagonista
✓ Copy: habla a la emoción, no solo al producto
✓ CTA: claro, con verbo de acción
✓ Testea siempre: mínimo 2-3 variaciones
✓ Los datos te dicen qué funciona, no tu opinión
