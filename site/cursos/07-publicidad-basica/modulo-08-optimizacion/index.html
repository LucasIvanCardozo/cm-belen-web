    Módulo 8 — Optimización Básica¶

    Objetivos de Aprendizaje¶

    Al finalizar este módulo vas a: - ✅ Entender qué es optimización de campaigns - ✅ Identificar cuándo y qué optimizar - ✅ Aplicar técnicas de optimización - ✅ Saber cuándo pausar vs continuar - ✅ Implementar mejora continua

    1. Concepto de optimización¶

    OPTIMIZAR = Hacer que algo funcione MEJOR con los mismos recursos.

EJEMPLO:
• Antes: $100 USD → 3 ventas (CPA $33)
• Después de optimizar: $100 USD → 6 ventas (CPA $17)
• Mismo gasto, mejor resultado = optimización

    2. Cuándo optimizar¶

    REGLAS DE DEDO:

1. MÍNIMO 7 DÍAS DE DATOS
   → No optimices antes de 7 días
   → Meta necesita tiempo para aprender

2. MÍNIMO 50 EVENTOS
   → Si tenés menos de 50 clics/leads/ventas
   → No hay suficiente data para decidir
   → Seguí esperando

3. CUANDO VES PATRÓN CLARO
   → Un ad funciona 3x mejor que otro
   → Un interés genera 2x más ventas
   → Hay una audiencia que no convierte

TIEMPO DE OPTIMIZACIÓN:
• Campaña nueva: esperar 7 días
• Primer test: esperar hasta 50 eventos
• Cambio de público: esperar 5-7 días después

    3. Qué optimizar¶

    ORDEN DE PRIORIDAD:

1. AUDIENCIA
   → Si el problema es que nadie convierte
   → El público objetivo puede ser incorrecto

2. CREATIVO
   → Si las personas ven el ad pero no interactúan
   → El creativo no está conectando

3. PRESUPUESTO
   → Si todo funciona pero querés más resultados
   → Solo escalar cuando funciona bien

4. BIDDING/PUJA
   → Si el costo es muy alto
   → Ajustar estrategia de puja

JERARQUÍA:
AUDIENCIA > CREATIVO > PRESUPUESTO > PUJA
(Fixea lo grande primero)

    4. Optimización de audiencias¶

    Señales de problema con audiencia¶

    PROBLEMA: Audiencia incorrecta

SEÑALES:
• Alto alcance pero bajo CTR (< 0.5%)
• Muchos clics pero 0 ventas
• Costo por resultado muy alto
• Meta dice "audiencia limitada"

ACCIONES:
1. Ensanchar intereses (agregar más)
2. Cambiar demografía
3. Probar nueva audiencia fría
4. Aumentar alcance geográfico
5. Remover exclusiones si hay muchas

    Optimizar basándose en data¶

    SI TENÉS DATA:

Ad Set A (Interés: Joyería): 5 ventas, $25 CPA
Ad Set B (Interés: Moda): 2 ventas, $50 CPA
Ad Set C (Interés: Regalos): 1 venta, $75 CPA

ACCIÓN:
→ Aumentar presupuesto en Ad Set A (+50%)
→ Reducir presupuesto en B (-50%)
→ Pausar C y testear algo nuevo

PATRÓN:
El que gana se lleva más.
El que pierde se reduce o pausa.

    5. Optimización de creativos¶

    Señales de problema con creativo¶

    PROBLEMA: Creativo no conecta

SEÑALES:
• Alto CTR pero bajo engagement post-click
• Mucho alcance pero nadie guarda/comenta
• Personas hacen clic pero no compran

ACCIONES:
1. Testear nueva foto/video
2. Cambiar hook (primeras palabras)
3. Cambiar oferta/CTA
4. Probar formato diferente (foto vs video)

    Proceso de optimización de creativos¶

    SEMANA 1:
Ad A: Foto producto + Copy A → 5 ventas, $20 CPA
Ad B: Video + Copy A → 3 ventas, $35 CPA
Ad C: Foto lifestyle + Copy B → 8 ventas, $12 CPA

RESULTADO:
→ C es ganador (menor CPA)
→ A es segundo
→ B se pausa

SEMANA 2:
→ Crear variaciones de C
→ C1: Misma foto, diferente CTA
→ C2: Mismo copy, diferente foto
→ C3: Nuevo hook

CONTINUAR:
→ Loop de test → optimize → repeat

    6. Optimización de presupuesto¶

    Cuándo aumentar presupuesto¶

    SEÑALES DE QUE PODÉS ESCALAR:
✓ CPA estable o bajando por 7+ días
✓ ROAS > 3x consistente
✓ Todas las ventas son rentables
✓ Meta dice "límite de distribución"

CÓMO ESCALAR:
1. Aumentar 20-30% por semana
   → No duplicar de golpe
   → Meta necesita adaptarse

2. Esperar 3-5 días después de cada aumento
   → Ver si el CPA se mantiene

3. Si el CPA sube más de 20%
   → Volver al presupuesto anterior
   → Investigar por qué subió

    Cuándo reducir/pausar¶

    SEÑALES DE QUE DEBÉS PAUSAR:
✗ CPA subió significativamente (30%+)
✗ ROAS < 1.5x por 7+ días
✗ Meta dice "audiencia agotada"
✗ No hay mejora después de optimizaciones

ACCIONES:
1. Pausar el ad/ad set bajo rendimiento
2. No eliminar (puede servir para data)
3. Crear algo nuevo para reemplazar

    7. Checklist de optimización semanal¶

    CHECKLIST SEMANAL:

□ ¿Tengo al menos 7 días de data?
□ ¿Tengo al menos 50 eventos para decidir?
□ ¿Cuál es mi CPA vs benchmark?
□ ¿Cuál es mi ROAS?
□ ¿Hay algún ad outperforming claramente?
□ ¿Hay algún ad underperforming?
□ ¿Qué voy a hacer con los winners?
□ ¿Qué voy a hacer con los losers?
□ ¿Debo aumentar presupuesto?
□ ¿Debo pausar algo?
□ ¿Qué voy a testear esta semana?

ACCIONES POSITIVAS (+) vs NEGATIVAS (-):
Winners → +Presupuesto / +Duplicar
Losers → -Presupuesto / Pausar
Nuevos tests → $20-30 para probar

    8. Errores de optimización¶

    ❌ Error 1: Cambiar todo a la vez¶

    MAL:
"Llevo 3 días. Voy a cambiar el público,
el creativo, el presupuesto y el objetivo."

PROBLEMA:
Si cambiás todo, no sabés qué funcionó.

BIEN:
"Hice un cambio a la vez.
Espero 7 días para ver el impacto."

    ❌ Error 2: No documentar cambios¶

    MAL:
"Cambié cosas hace 2 semanas. No me acuerdo qué."

PROBLEMA:
No podés aprender de tus decisiones.

BIEN:
"Documenté cada cambio en una tabla:
Fecha | Qué cambié | Por qué | Resultado"

    ❌ Error 3: Pausar muy rápido¶

    MAL:
"No hubo ventas en 2 días. Pauso."

PROBLEMA:
2 días no es suficiente data.
Meta todavía está aprendiendo.

BIEN:
"7 días sin ventas con suficiente tráfico.
Ahora sí, evaluó pausar."

    ❌ Error 4: Ignorar lo que funciona¶

    MAL:
"Tengo un ad con ROAS 5x. Lo dejo igual."

PROBLEMA:
Si funciona, duplicá. Dinero en la mesa.

BIEN:
"Tengo ROAS 5x. Duplico presupuesto
y pruebo variaciones para escalar."

    9. Ejercicio práctico¶

    EJERCICIO: Optimizá una campaña ficticia

SEMANA 1 - RESULTADOS:

Campaña "Collares San Valentín"
Presupuesto: $10/día ($70 total)

Ad Set A (Joyería): $20 gastado, 4 ventas, $5 CPA
Ad Set B (Moda): $25 gastado, 2 ventas, $12.50 CPA
Ad Set C (Regalos): $25 gastado, 1 venta, $25 CPA

Ad 1 (Foto): 3 ventas
Ad 2 (Video): 2 ventas
Ad 3 (Carrusel): 2 ventas

PREGUNTAS:

1. ¿Qué Ad Set está funcionando mejor?
2. ¿Qué Ad Set pausarías?
3. ¿Cómo redistribuirías el presupuesto?
4. ¿Qué creatividad testearías más?

ESCRIBÍ tu plan de optimización.

    10. Resumen del módulo¶

    PUNTOS CLAVE:

✓ Esperá 7+ días y 50+ eventos antes de optimizar
✓ Orden de prioridad: Audiencia > Creativo > Presupuesto > Puja
✓ Winners: + presupuesto
✓ Losers: - presupuesto o pausar
✓ Solo escalar cuando ROAS > 3x
✓ Documentá cada cambio para aprender
