    Módulo 5 — Setup Casero para Fotografía de Joyería¶

    Objetivos de Aprendizaje¶

    Al finalizar este módulo vas a:

    • ✅ Crear un lightbox casero por menos de $10
    • ✅ Construir un sistema de reflectores con foam board
    • ✅ Preparar fondos profesionales con materiales accesibles
    • ✅ Armar tu estación de fotografía permanente
    • ✅ Optimizar un espacio pequeño para fotografiar

    1. Por qué hacer tu propio setup¶

    COMPARACIÓN:

resultados profesionales si se usa correctamente.

    2. El Lightbox Casero Perfecto¶

    2.1 Materiales necesarios¶

    PRESUPUESTO MÍNIMO ($5-10):
┌────────────────────────────────────────────────────┐
│ ✓ 1 caja de cartón grande (zapatos no, más grande)│
│ ✓ 1 sábana blanca fina o papel blanco             │
│ ✓ 1 rollo de cinta de papel (masking tape)        │
│ ✓ 1 cúter o tijeras                               │
└────────────────────────────────────────────────────┘

PRESUPUESTO MEJORADO ($15-25):
┌────────────────────────────────────────────────────┐
│ ✓ Todo lo anterior                                │
│ ✓ 2-3 foam boards blancos (papelería)            │
│ ✓ 1 foam board negro                             │
│ ✓ 1 pequeña lámpara LED de mesa                   │
│ ✓ Trípode de celular ($10-15)                    │
└────────────────────────────────────────────────────┘

    2.2 Cómo construirlo paso a paso¶

    PASO 1: Elegí tu caja
───────────────────────
• Tamaño mínimo: 40cm x 40cm x 40cm
• Ideal: 60cm x 60cm x 60cm
• Puedes usar 2 cajas unidas para más espacio

PASO 2: Prepará la caja
───────────────────────
• Cortá un lado de la caja (la "puerta" para la cámara)
• Cortá los otros 4 lados en ángulo (como pirámide invertida)
• Dejá la base intacta

PASO 3: Forrá con blanco
─────────────────────────
• Forrá TODOS los lados internos con papel blanco
• Usá cinta de papel para asegurar
• El blanco debe llegar hasta el borde de la "puerta"

PASO 4: Reforzá la base
────────────────────────
• Forrá el piso también de blanco
• Tiene que verse un continuo blanco sin uniones visibles

    2.3 Versión ultrabarata: sábana + silla¶

    OPCIONES SIN CAJA:

OPCIÓN A: Silla + sábana
─────────────────────────
1. Buscá 2 sillas del mismo alto
2. Tirá un sábana blanca sobre ambas
3. Formó un arco natural
4. Colocó la joya sobre una superficie blanca
5. Disparó desde arriba

OPCIÓN B: Pared + piso
───────────────────────
1. Ubicá una silla cerca de una pared
2. Forrá la pared con papel blanco
3. Extendé el papel hasta el piso
4. Curveó el papel en la unión pared-piso
5. Creó una curva continua
6. Colocó la joya en el "soplo" de la curva

OPCIÓN C: Ventana como fondo
─────────────────────────────
1. Cerrá la cortina de la ventana
2. Usá solo luz natural que entra
3. Colocó un reflector blanco al otro lado
4. Joya centrada frente a la ventana
5. Dispara de frente a la ventana

    3. Sistema de Reflectores DIY¶

    3.1 Qué es un reflector y por qué importa¶

    PROBLEMA: La luz directa crea sombras duras.

SOLUCIÓN: El reflector "llena" las sombras con luz reflejada.

TIPOS DE REFLECTORES:
┌──────────────┬──────────────┬────────────────┐
│ Color        │ Efecto       │ Cuándo usarlo  │
├──────────────┼──────────────┼────────────────┤
│ Blanco       │ Luz suave     │ Luz fill básica│
│ Negro        │ Absorbe luz  │ Definir sombras│
│ Plata        │ Luz intensa  │ Brillo máximo  │
│ Oro          │ Luz cálida   │ Tono dorado    │
│ Gris         │ Neutral      │ Referencia     │
└──────────────┴──────────────┴────────────────┘

PARA JOYERÍA: Blanco es el más usado.
               Oro para joyería en oro.
               Negro para definir sombras en plata.

    3.2 Cómo hacer reflectores con foam board¶

    MATERIALES:
• Foam board blanco (cartulina gruesa) - $1-2 c/u
• Foam board dorado (o papel aluminio)
• Foam board negro

TAMAÑOS RECOMENDADOS:
• Pequeño: 20cm x 30cm (para joyería chica)
• Mediano: 30cm x 45cm (para la mayoría)
• Grande: 45cm x 60cm (para sesiones completas)

CÓMO ARMARLOS:

REFLECTOR BLANCO:
1. Tomá el foam board blanco
2. Cortá si es necesario
3. Listo para usar

REFLECTOR DORADO:
1. Forrá foam board blanco con papel aluminio (lado brillante hacia afuera)
2. Alisá sin arrugas
3. Fijá con cinta

REFLECTOR NEGRO:
1. Comprá foam board negro
2. Cortá a tamaño deseado

SET COMPLETO (6 reflectores):
┌─────────────────────────────────────────┐
│ ✓ 2 reflectores blancos grandes         │
│ ✓ 2 reflectores blancos medianos       │
│ ✓ 1 reflector dorado                     │
│ ✓ 1 reflector negro                     │
│                                         │
│ Inversión total: $8-15                  │
└─────────────────────────────────────────┘

    3.3 Cómo usar los reflectores¶

    POSICIÓN DE LUZ NATURAL + REFLECTORES:

         ☀️ SOL/VENTANA
            ↓
    ┌───────────────┐
    │   LIGHTBOX    │
    │   ┌───────┐   │
    │   │ JOYA  │   │
    │   └───────┘   │
    │   ↑           │
    │   │ REFLECTOR │  ← Blanco, frente a la luz
    │   │ (fill)    │
    └───────────────┘
            ↑
         📱 CÁMARA


CONFIGURACIÓN 1: Luz lateral suave
────────────────────────────────────
• Luz principal: ventana a 45° de la joya
• Reflector blanco: lado opuesto, "llenando" sombras
• Cámara: frente a la joya
• Ideal para: oro, plata, piedras claras


CONFIGURACIÓN 2: Luz desde arriba
────────────────────────────────────
• Luz natural de ventana alta
• Joya en superficie reflectante blanca
• Cámara: a nivel de la joya
• Ideal para: anillos, aros, productos pequeños


CONFIGURACIÓN 3: Backlight (contra luz)
────────────────────────────────────────
• Luz detrás de la joya
• Reflector blanco EN FRENTE (bounce)
• Cámara: lateral o 45°
• Ideal para: joyas con piedras (genera brillo)
• CUIDADO: Puede sobreexponer

    4. Fondos Profesionales¶

    4.1 Tipos de fondo para joyería¶

    OPCIONES DESDE LO MÁS SIMPLE A LO MÁS ELABORADO:

1. BLANCO PURO
   • Sábana blanca planchada
   • Papel blanco fotográfico
   • Foam board grande
   ➜ Para: fotos de producto limpio

2. GRIS NEUTRO
   • Cartulina gris
   • Tela gris (felpa, terciopelo)
   ➜ Para: joyería donde el blanco competiría

3. TEXTURIZADO NATURAL
   • Mesa de madera
   • Tela de lino
   • Mármol (pueden ser fotos del material)
   ➜ Para: lifestyle, contexto

4. OSCURO ELEGANTE
   • Tela negra
   • Marble negro
   • Tela terciopelo negro
   ➜ Para: lujo, joyería premium

5. COLORIDO MÍNIMO
   • Verde salvia
   • Rosa pálido
   • Azul cielo
   ➜ Para: posts específicos, no grid

6. TEXTURA CREATIVA
   • Arena
   • Flores secas
   • Hojas
   ➜ Para: campañas estacionales

    4.2 Cómo crear un fondo curvo (infinito)¶

    LA TÉCNICA DEL "INFINITO":
Es la base profesional donde no se ve el horizonte.

PASO A PASO:
1. Conseguí una superficie larga (mesa o piso)
2. Forrá con papel blanco hasta 1 metro después del borde
3. En el borde, curvá el papel hacia arriba suavemente
4. Fijá con cinta en la pared
5. El resultado: transición suave de horizontal a vertical

    4.3 Fondos específicos para joyería¶

    PARA ORO:
• Blanco frío (evita amarillos)
• Negro (contraste máximo)
• Dorado sutil (para unity shots)
• Gris medio

PARA PLATA:
• Blanco puro (resalta el brillo)
• Negro (genera drama)
• Azul claro (refresca)
• Gris perla

PARA PIEDRAS DE COLOR:
• Blanco (para mostrar el color real)
• Negro (para gemas oscuras: zafiro, amatista)
• Tono complementario suave (para fotos creativas)

PARA PERLAS:
• Blanco rosado (suaviza)
• Gris claro
• Negro (para perlas blancas grandes)

    5. Tu Estación Permanente¶

    5.1 Elegir el lugar perfecto¶

    REQUISITOS DEL LUGAR:
✓ Luz natural constante (ventana disponible)
✓ Espacio mínimo: 1.5m x 1.5m
✓ Superficie plana para colocar equipos
✓ Fácil de limpiar (para joyas pequeñas)
✓ Acceso a toma corrientes (para cargar celular)
✓ Pared libre para colgar reflectores

LUGARES IDEALES:
• Cerca de una ventana grande
• Escritorio cerca de pared blanca
• Mesa en habitación con buena luz
• Balcón cubierto (con cuidado de polvo)

EVITAR:
• Luz directa del sol al mediodía (muy dura)
• Fondo desordenado
• Lugares con mucho tránsito
• Cerca de fuentes de calor o humedad

    6. Tips de Optimización¶

    6.1 Para espacios pequeños¶

    PROBLEMA: "Solo tengo un rincón chiquito"

SOLUCIÓN: Setup minimalista vertical

┌────────┐
│VENTANA │
│   ↓    │
│ ┌────┐ │
│ │FOAM│ │ ← Reflector sujeto con pinza
│ └────┘ │
│        │
│  [JOYA]│ ← Sobre foam board blanco
│        │
│  📱    │ ← Cámara a mano
└────────┘

META: Que entre en 60cm x 60cm
CONSEJO: Cuanto más simple, mejor resultado

    6.2 Para luz inconsistente¶

    PROBLEMA: "Mi luz cambia todo el tiempo"

SOLUCIONES:

1. FOTOGRAFÍAR EN EL MISMO MOMENTO
   • Todas las fotos del día a la misma hora
   • Anotar qué hora funcionó mejor
   • Repetir cuando funcione

2. USAR LAMPARA COMO LUZ CONSTANTE
   • Lámpara LED de mesa ($10-15)
   • Posicionar como si fuera ventana
   • Combinar con reflector blanco
   • Resultado: luz controlable

3. LUGAR CON LUZ DIFUSA
   • Ventana con cortina translúcida
   • Luz más suave y constante
   • Ideal para joyería (evita puntos de luz)

    6.3 Quick setup para emergencia¶

    SETUP MÍNIMO EN 2 MINUTOS:
1. Silla + sábana blanca = fondo
2. Foam board blanco = reflector
3. Mesa + teléfono = done
4. Buscá luz natural de ventana
5. Photograph!

EQUIPO MÓVIMO:
• 1 sábana blanca
• 1 foam board
• Tu teléfono
= $3 y listo

    7. Errores Comunes¶

    ❌ NO hacer:¶

    1. FONDO SUCIO O ARrugado
   → Las líneas se ven en la foto
   → Arruga el look profesional

2. SOMBRAS DURAS SIN RELLENO
   → La joya pierde detalle en las sombras
   → Siempre agregar un reflector

3. PAPEL QUE SE TERMINA EN EL BORDE
   → Se ve donde termina el fondo
   → Extender 30cm más allá de lo necesario

4. DEMASIADOS ELEMENTOS
   → El fondo compite con la joya
   → Menos es más: 1 fondo, 1 joya

5. OLVIDAR LIMPIAR LA JOYA
   → Huellas, polvo, pelusa
   → Limpiar antes de cada sesión

6. NO FIJAR EL TELÉFONO
   → Fotos movidas = inútiles
   → SIEMPRE usar trípode o apoyo estable

    ✅ SÍ hacer:¶

    1. PLANCHÁ EL FONDO
   → Fondo liso = resultado profesional

2. REVISÁ CADA FOTO EN EL MOMENTO
   → Error in situ = fácil corregir
   → Error después = tener que repetir todo

3. HACE UN CHECKLIST ANTES DE EMPEZAR
   □ Luz correcta
   □ Fondo listo
   □ Reflectores posicionados
   □ Trípode estable
   □ Joya limpia

4. TOMÁ FOTOS DE REFERENCIA
   □ De frente
   □ De ángulo
   □ Close-up del detalle
   □ Con flash (por las dudas)

5. ETIQUETÁ LAS FOTOS INMEDIATAMENTE
   → "Anillo_Oro_24052026_001.jpg"
   → Así no perdés nada
