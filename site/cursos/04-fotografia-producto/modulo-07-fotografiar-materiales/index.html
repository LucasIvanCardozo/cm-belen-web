    Módulo 07 — Fotografiar Materiales: Oro, Plata, Piedras y Perlas¶

    Cada material tiene sus secretos¶

    "Fotografiar oro no es lo mismo que fotografiar plata. Un zafiro no se ilumina como un diamante. Conocer estas diferencias es lo que separa amateur de profesional."

    1. La Física de Fotografiar Metales¶

    Por qué los metales son difíciles:¶

    METALES = REFLECTIVOS

Lo que pasa:
→ El metal refleja TODO
→ Ventanas, tu cara, el techo, el suelo
→ Si hay algo undesired cerca, se va a reflejar

El problema:
→ Hot spots (puntos muy brillantes)
→ Reflejos distractores
→ Zonas muy oscuras (donde no llega luz)

La solución:
→ Controlar TODA la luz en la habitación
→ Usar difusores en TODOS lados
→ Posicionar cuidadosamente

    Los tipos de metal más comunes:¶

    METALES EN JOYERÍA:

ORO AMARILLO:
→ Reflectividad media
→ Color cálido
→ Mejor con luz cálida suave

ORO BLANCO (ROSE GOLD):
→ Similar a plata pero con tono
→ Muy reflectivo
→ Cuidado con tonos fríos

PLATA / PLATA 925:
→ MUY reflectivo
→ Tono neutro
→ Cualquier cosa se refleja

ACERO QUIRÚRGICO:
→ Similar a plata
→ Menos brilho que plata Sterling
→ Más fácil de fotografiar

METALESoxidados / NATURALES:
→ Menos reflectivos
→ Se ven mejor con luz direta suave
→ Mais texture visible

    2. Fotografiando Oro Amarillo¶

    El desafío específico:¶

    ORO AMARILLO:

PROBLEMA:
→ El oro refleja la luz de forma "caliente"
→ Los brillos tienden a ser blancs o amarillos
→ Puede verse "apagado" si la luz es muy fría

OBJETIVO:
→ Capturar el brilho sin perder el color dorado
→ Que se vea brilhante pero no "lavado"
→ Mantener la calidez del oro

    Técnica para oro:¶

    CONFIGURACIÓN:

LUZ:
→ Luz natural cálida (hora dorada ideal)
→ O día nublado (luz suave neutra)
→ Evitar luz muy fría (sombra azul)

EXPOSICIÓN:
→ -0.5 a -0.7 (oscurecer un poco)
→ Proteger los brillos del oro
→ Si está muy oscuro, ajustar en edición

FONDO:
→ Blanco: clássico e limpo
→ Negro: hace контрастировать el oro
→ Gris: sophistication

ÁNGULOS:
→ 45° para mostrar la superficie
→ Backlight suave para realzar brilho

SOMB Braz:
→ Suaves, no muy marcadas
→ Añaden profundidad al oro

    Tips para oro:¶

    ORO - TIPS:

✓ Usá reflectores blancos para llenar sombras
✓ Luz lateral suave para mostrar texture
✓ Exposición ligeramente negativa para proteger brillos
✓ El backlight (luz desde atrás) realza el brilho
✓ Un poco de textura visible = calidad

⚠️ EVITAR:
→ Luz directa que cree hot spots
→ Exposición positiva (quema los brillos)
→ Fondo muy brilhante (competencia visual)
→ Flash direto

    3. Fotografiando Plata¶

    El desafío específico:¶

    PLATA:

PROBLEMA:
→ La plata es EL MÁS reflectivo
→ Muestra absolutamente todo
→ Incluso una pared pintada se puede reflejar
→ Los brillos pueden ser extremosi

OBJETIVO:
→ Iluminación uniforme
→ Mínima cantidad de puntos brillantes
→ Que se vea limpia y brillante

    Técnica para plata:¶

    CONFIGURACIÓN:

LUZ:
→ Luz MUY difusa (día nublado ideal)
→ O luz muy grande (ventana grande + diff)
→ La clave:diffundir todo lo possível

EXPOSICIÓN:
→ -0.7 a -1.0 (oscurecer bastante)
→ La plata reflete mucha luz
→ Necesitás compensar

FONDO:
→ Negro: contrasta perfecto con plata
→ Blanco: puede ser muito brillante
→ Gris: intermedioelegante

ÁNGULOS:
→ Frontal suave
→ Evitar backlight (crea reflection excesiva)

REFLECTORES:
→ Usar reflector blanco PARA LLENAR sombras
→ Importante: que el reflector esté limpio

    Tips para plata:¶

    PLATA - TIPS:

✓ La iluminación más difusa possível
✓ Exposición negativa para controlar reflejo
✓ Fondo negro para максимум contraste
✓ Reflector branco para llenar sombras
✓ Limpiar MUY bien la prata antes (manchas se ven)

⚠️ EVITAR:
→ CUALQUIER cosa cerca que no quieras que se vea refletida
→ Sol direto (crea spotis horribles)
→ exposición positiva (quema todo)
→ Fondos coloridos (se van a reflejar)

    4. Fotografiando Oro Blanco y Rose Gold¶

    Específicamente:¶

    ORO BLANCO:

→ Similar a prata pero con tono ligeramente azulado
→ MUY reflectivo
→ La iluminación fría puede hacerlo ver gris
→ Mejor con luz нейтральная

ROSE GOLD:

→ Tono cálido pero más suave que oro amarelo
→ Reflectivo como ouro blanco
→ Luz cálida puede realzar el tono rosado
→ Luz fría puede hacerlo ver más blanco

    Técnica:¶

    ORO BLANCO:
→ Usar luz нейтральная (no muy cálida ni fría)
→ Exposición: -0.5 a -0.7
→ Fondo: negro o gris escuro
→ Similar a prata pero menos crítico

ROSE GOLD:
→ Luz cálida suave (para realzar el tono)
→ O нейтральная para точность
→ Exposición: -0.5 a -0.7
→ Fondo: branco оu negro (depende del efecto)

    5. Fotografiando Piedras Preciosas¶

    Categorías:¶

    PIEDRAS = TRANSPARENCIA

DIAMANTES:
→ Transparente puro
→ Refleja y transmite luz
→ Los "4Cs" importan (corte, color, claridad, quilates)
→ Requiere iluminación très específica

ZAFIROS, RUBÍES, ESMERALDAS:
→ Translúcidos a transparentes
→ Tienen color intenso
→ La luz a través de la piedra
→ Muestran el fondo un poco

TURQUESAS, OPALAS, CUARZOS:
→ Opacos a translúcidos
→ Color ms uniforme
→ Menos complicado que transparente

PÉROlas:
→ Semi-opacas con brilho único (nacre)
→ No transmissão de luz
→ Reflections iridiscentes
→ Muy delicadas

    6. Fotografiando Diamantes¶

    El desafío:¶

    DIAMANTES - REy DE LA FOTOGRAFÍA JOYA:

PROBLEMAS:
→ Transparente = ves todo el fondo
→ Refleja todo = necesitas control TOTAL
→ Brilho extremo = exposição muy difícil
→ Los facetass crean mille reflejos
→ Un mal ángulo puede hacer ver "sucio"

OBJETIVO:
→ Que el diamante parezca BRILLANTE
→ No necessarily un reflejo perfecto
→ Capturar el "fire" (destellos de color)
→ Que se vea limpio y claro

    Técnica para diamantes:¶

    CONFIGURACIÓN:

LUZ:
→ DIFUSA pero con puntos de luz directed
→ Específicamente: necesitás puntos de luz pequeños
→ Un grande, suave + uno pequeño, directo
→ Así se crean los reflejos interesantes

EXPOSICIÓN:
→ -0.7 a -1.0 (bastante oscura)
→ Los brillos se van a quemar facilmente
→ Mejor underexposed que overexposed

FONDO:
→ Negro ABSOLUTO
→ Así los reflejos del borde destacan
→ Cualquier cosa puede arruinar la foto

ÁNGULOS:
→ Tocar el punto de luz para ver el reflejo
→ Mover hasta encontrar el "sweet spot"
→ Generalmente 45° con ligera variación

SOMB Braz:
→ Mínimas
→ El diamond no muestra sombras de la misma forma

    Tips para diamantes:¶

    DIAMANTES - TIPS:

✓ Múltiples fuentes de luz (difusa + puntual)
✓ Fondo negro
✓ Exposición negativa
✓ Mover el anillo hasta encontrar el reflejo perfecto
✓ Tomar MUCHAS fotos (el ángulo es critical)
✓ Limpiar-perfectamente antes de disparar

⚠️ EVITAR:
→ Un solo источник de luz
→ Fondo que no sea negro
→ Flash direto
→ exposición positive
→ Huellas o polvo (se nota MUCHO)

    7. Fotografiando Piedras de Color (Zafiros, Rubíes, Esmeraldas)¶

    El desafío:¶

    PIEDRAS DE COLOR:

PROBLEMA:
→ El color de la piedra debe verse real
→ La piedra transmite luz = muestra el fondo
→ Los reflejos deben контрастировать con el color
→ La exposición afecta el color percibido

OBJETIVO:
→ Color fiel а la realidad
→ Brillos que complementen (no dominen)
→ Profundidad visible en la piedra
→ Textura y facetas visibles

    Técnica para piedras de color:¶

    CONFIGURACIÓN:

LUZ:
→ Suave y uniforme (la base)
→ 1-2 источник de luz puntual para reflejos
→ Día nublado como base es идеально

EXPOSICIÓN:
→ -0.3 a -0.5 (ligeramente oscura)
→ Allows ver el color saturado
→ Demasiado oscuro = color apagado
→ Demasiado lumbre = color lavé

FONDO:
→ Negro: контрастирует perfecto
→ Blanco: puede funcionar para piedras claras
→ Color: evitar (competencia con piedra)

ÁNGULOS:
→ 45° es buen punto de partida
→ Mover para encontrar cómo la luz atraviesa
→ Backlight puede акцентировать color

BLACKgrounds:
→ Negro: mayoría de piedras
→ Blanco: piedras muy oscuras
→ Plateado: piedras claras

    Tips por color:¶

    POR COLOR:

ZAFIROS (azul):
→ Luz нейтральная o ligeramente cálida
→ Backlight акцентирует color azul
→ Negro del fondo para contraste

RUBÍES (rojo):
→ Luz cálida puede mejorar el rojo
→ Cuidado: exposición muy luminosa = rosa
→ Negro fondo

ESMERALDAS (verde):
→ Luz нейтральная
→ Similar a otras piedras verdes
→ Evitar reflexión excesiva

AMATISTAS (morado):
→ Luz нейтральная
→ Colores pastel pueden verse lavados
→ Encontrar el ángulo que акцентирует violet

    8. Fotografiando Perlas¶

    El desafío único:¶

    PÉROLAS - DIFERENTES A TODO:

PROPIEDADES:
→ No son transparentes
→ No transmissão de luz
→ Brillo único: nacre
→ Reflexiones iridiscentes (tonos que cambian)
→ Superficie muy suave

PROBLEMA:
→ El flash CRASH
→ Luz direta = reflejo branco puro
→ Pérdida del efecto iridiscente
→ Puede verse como plastik

OBJETIVO:
→ Capturar el lustre (brillo suave)
→ Mostrar las tonos iridiscentes
→ Que se vea orgánica y valiosa

    Técnica para perlas:¶

    CONFIGURACIÓN:

LUZ:
→ SUAVE y muy difusa
→ Día nublado o sombra completa
→ CUALQUIER punto de luz directa = проблема
→ Varias fuentes suaves es mejor que una fuerte

EXPOSICIÓN:
→ 0 a -0.3 (neutro a ligeramente oscuro)
→ Las perlas no reflejan tan intensamente
→ La exposición normal generalmente funciona

FONDO:
→ Negro: contrasta perfecto, perlas destacan
→ Blanco: también funciona, más limpio
→ Evitar colores que competan

ÁNGULES:
→ 45° para ver superficie
→ Mover para encontrar iridiscencia
→ Generalmente prefieren luz frontal suave

SOMB Braz:
→ Suaves y definidas
→ Las perlas muestran sombras de forma limpia

    Tips para perlas:¶

    PÉROLAS - TIPS:

✓ Luz максимально difusa
✓ Cero flash, cero luz direta
✓ Exposición нейтральная
✓ Encontrar el ángulo de iridiscencia
✓ Mostrar la superficie suave

⚠️ EVITAR:
→ Flash direto (ruina todo)
→ Luz direta de ventana
→ Cualquier punto de luz pequeño
→ Perlas sucias (se nota muchoo)
→ Exposición extrema

    9. Errores Comunes por Material¶

    Oro:¶

    ❌ EXPOSICIÓN MUY POSITIVA
   → El oro se ve "lavado", perde color

❌ FONDO MUY BRILLANTE
   → El oro compite con el fondo

❌ LUZ MUY FRÍA
   → El oro se ve verdoso o apagado

    Plata:¶

    ❌ FONDO NO-NEGRO
   → Se refleja en la plata

❌ LUZ NO-DIFUSA
   → Hot spots por todos lados

❌ POCA EXPRESICIÓN NEGATIVA
   → La plata refleja todo, se "come" la foto

    Diamantes:¶

    ❌ FONDO QUE NO ES NEGRO
   → Cualquier cosa se reflete

❌ UN SOLO SOURCE DE LUZ
   → Sin facetas interessantes

❌ EXPRESICIÓN MUY POSITIVA
   → Todo se quuema, no hay definición

    Perlas:¶

    ❌ FLASH
   → Ruina completamente el efecto

❌ LUZ PUNTUAL
   → Reflejo branco en vez de iridiscencia

❌ PERLAS SUCIAS
   → Se ve todo opaco y дешевый

    10. Ejercicio del Módulo¶

    Tu misión esta semana:¶

    EJERCICIO: DOMINÁ LOS MATERIALES

Parte 1: Oro (15 minutos)
1. Encontrá una pieza de oro
2. Armá tu setup
3. Probá diferentes exposiciones (-0.3, -0.5, -0.7, -1.0)
4. Probá fondo blanco vs negro
5. Encontrá el ángulo que акцентирует el brilho

Parte 2: Plata (15 minutos)
1. Encontrá una pieza de plata
2. Asegurate de que no haya nada distractoro
3. Probá exposiciones negativas
4. Intentá con fondo negro

Parte 3: Piedras (20 minutos)
1. Encontrá una joya con piedra (transparente)
2. Probá backlight suave
3. Mové hasta encontrar el reflejo interessante
4. Experimentá con fondo negro vs otro

Parte 4: Perlas (15 minutos)
1. Encontrá perlas (reales o no)
2. Buscá la luz más difusa possível
3. Encontrá el ángulo de iridiscencia
4. Compará con y sin reflexión de fondo

ENTREGA:
→ Mínimo 1 foto por material (4 fotos)
→ Nota: ¿Qué ajustes funcionaron mejor para cadamaterial?

    Resumen del Módulo¶

    PUNTOS CLAVE:

✓ Oro: luz cálida, exposición negativa, fondo contrastante
✓ Plata: luz DIFUSA, exposición negativa, fondo negro
✓ Diamantes: luz múltiple, fondo negro, muchos intentos
✓ Piedras de color: exposición neutra, backlight suave
✓ Perlas: luz DIFUSA, cero flash, найти iridiscencia

AJUSTES POR MATERIAL:

ORO:
→ Exposición: -0.5 a -0.7
→ Luz: cálida suave
→ Fondo: negro o branco

PLATA:
→ Exposición: -0.7 a -1.0
→ Luz: максимально difusa
→ Fondo: negro ABSOLUTO

DIAMANTES:
→ Exposición: -0.7 a -1.0
→ Luz: difusa + puntual
→ Fondo: negro

PIEDRAS COLOR:
→ Exposición: -0.3 a -0.5
→ Luz: suave + backlight suave
→ Fondo: negro

PÉROLAS:
→ Exposición: 0 a -0.3
→ Luz: максимально difusa
→ Fondo: negro ou branco
