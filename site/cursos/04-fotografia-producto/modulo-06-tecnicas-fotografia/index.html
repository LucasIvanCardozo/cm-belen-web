    Módulo 06 — Técnicas de Fotografía: Ángulos, Composiciones y Estilos¶

    Fotografiá cada tipo de joya como un profesional¶

    "No existe un solo ángulo perfecto. Existe el ángulo correcto para cada joya, y aprender a encontrarlo es lo que te diferencia."

    1. Los Fundamentos del Ángulo¶

    Por qué el ángulo importa:¶

    ÁNGULO = PERCEPCIÓN

Misma joya, diferentes ángulos:
├── Desde arriba → ves la forma general
├── 45° → ves la profundidad y la piedra
├── Lateral → ves el perfil y el engaste
└── Micro → ves grabados y texturas

El ángulo correcto:
→ Resalta lo mejor de cada pieza
→ Oculta imperfecciones
→ Cuenta una historia

    Los 3 ángulos básicos:¶

    ÁNGULOS UNIVERSALES:

1. TOP DOWN (desde arriba):
   → Celular perpendicular a la joya
   → Como mirás un reloj
   → Para: anillos, aros, charms

2. 45 GRADOS:
   → El más común y versátil
   → Muestra frente + un lado
   → Para: TODO

3. PERFIL (lateral):
   → Vista desde el lado
   → Muestra el alto y el engaste
   → Para: anillos, aros

    Tips específicos para anillos:¶

    ANILLOS - TIPS:

✓ Apoyá el anillo en algo (dedal, clay, esponja)
✓ El lado con piedra generalmente va para la cámara
✓ Probá diferentes orientaciones (recto, rotado)
✓ Usá espejo para ver el reflejo de la cámara
✓ Para anillos con piedra: exponé para la piedra
✓ Incluí la mano para escala (fotos lifestyle)

    COLLERES - TIPS:

✓ Para flat lay: estirá el collar para que no se superponga
✓ Usá alfileres para mantener la forma
✓ El cierre generalmente va al frente o a un lado
✓ Para collares largos: probá diferentes "drops"
✓ Si el dije es muy pesado: sostené el collar de otro lado
✓ Para pearls: muy suave, sin apretar

    7. Técnicas de Composición¶

    Regla de los tercios:¶

    REGLA DE LOS TERCIOS:

Dividí tu foto en 9 partes iguales:
┌───────┬───────┬───────┐
│       │       │       │
│   A   │   B   │   C   │
│       │       │       │
├───────┼───────┼───────┤
│       │       │       │
│   D   │   E   │   F   │  ← Joyería generalmente
│       │       │       │    va en estos puntos
├───────┼───────┼───────┤
│       │       │       │
│   G   │   H   │   I   │
│       │       │       │
└───────┴───────┴───────┘

PARA JOYERÍA:
→ Poné la joya en los puntos B, D, E, F, o H
→ O alineá la joya con las líneas
→ Evitá el centro exacto (puede verse estático)

PERO:
→ A veces el centro SÍ funciona (especialmente simetría)
→ Experimentá

    Simetría y централизм:¶

    SIMETRÍA (CUÁNDO FUNCIONA):
→ Collares planos
→ Aros en par
→ Diseños geométricos
→ Piezas muy simétricas

┌─────────────────┐
│      JOYA       │  ← Centrado funciona
│     centrally   │
└─────────────────┘

DINAMISMO (CUÁNDO FUNCIONA):
→ Anillos únicos
→ Collares asimétricos
→ Piezas con dirección

┌─────────────────┐
│                 │
│    JOYA         │  ← Movida a un lado
│                 │    funciona mejor
│                 │
└─────────────────┘

    Negative space:¶

    NEGATIVE SPACE (ESPACIO VACÍO):

→ El espacio alrededor de la joya
→ No es "wasted space"
→ Ayuda a que la joya respire
→ Da sensación de lujo

EJEMPLO BUENO:
┌─────────────────────┐
│                     │
│                     │
│        JOYA         │  ← Mucho espacio
│                     │
│                     │
│                     │
└─────────────────────┘

EJEMPLO DEMASIADO:
┌─────────────────┐
│    JOYA    FONDO │
│    JOYA    FONDO │
│    JOYA    FONDO │
└─────────────────┘
  ← Joya muy grande
     en poco espacio

    8. Flat Lay: La Técnica Definitiva¶

    Por qué es esencial:¶

    FLAT LAY:
→ Foto tomada desde arriba
→ Todo está en una superficie plana
→ Estilo muy popular en Instagram
→ Funciona para TODO tipo de joyería

VENTAJAS:
✓ Consistencia (fácil de replicar)
✓ Muestra el producto completo
✓ Fácil de controlar la luz
✓ Permite creatividad en composición

    Cómo hacer un flat lay perfecto:¶

    PASO A PASO:

1. SUPERFICIE
   → Blanca, negra, o de color
   → Limpia, sin wrinkles
   → Foam board o cartulina funcionan

2. ILUMINACIÓN
   → Luz uniforme desde arriba
   → Sin sombras de un lado
   → Usa diffusers si es necesario

3. POSICIÓN DE LA JOYA
   → Centro o según regla de tercios
   → Recta, sin inclinar
   → Asegurada para que no se mueva

4. CÁMARA
   → Directly arriba (perpendicular)
   → Trípode ideal
   → Grid encendido para ayudar

5. DETALLES
   → Limpiá polvo antes de disparar
   → Revisá que no haya pelusas
   → El fondo debe estar perfecto

    Ideas de composición:¶

    COMPOSICIONES PARA FLAT LAY:

1. SOLO LA JOYA
   → Minimalista
   → Máximo foco en el producto

2. JOYA + HOJA DE MAGNOLIA
   → Verde контрастирует con metals
   → Natural y elegante

3. JOYA + FLORES SECAS
   → Más texture y color
   → Romántico

4. JOYA + LIBRO/TEXTURA
   → Más lifestyle
   → Contexto de uso

5. JOYA + MINERALES
   → Piedras naturales como base
   → Muy popular en 2026

6. JOYA + JOYA (colección)
   → Mostrá varios items
   → Para vender sets

    9. Macro: Capturando Detalles¶

    Por qué macro importa:¶

    MACRO:
→ Fotografía de muy cerca
→ Revela detalles que no ves a simple vista
→ En joyería: grabados, facetas, engastes, texturas

PARA QUÉ SIRVE:
→ Mostrar calidad de manufacture
→ Ver grabados personalizados
→ Apreciar el trabajo artesanal
→ Diferenciarte de fotos planas

    Cómo lograr buen macro:¶

    TÉCNICA:

CON CÁMARA DEL CELULAR:
1. Acercate a la joya (10-15cm)
2. Si tenés modo macro, se activa solo
3. Si no: usá zoom digital 2x
4. TRÍPODE obligatorio (blurs se amplifican)
5. Buena iluminación (más importante en macro)

PARA ANILLOS:
→ La piedra y sus facetas
→ El engaste (cómo está puesta la piedra)
→ Grabados en el interior
→ El pulido del metal

PARA COLLERES:
→ El cierre
→ Detalles del dije
→ Uniones de cadena
→ Texturas del metal

PARA PIEDRAS:
→ Las facetas (ángulos de corte)
→ Inclusiones naturales
→ El brillo en diferentes puntos

    10. Ejercicio del Módulo¶

    Tu misión esta semana:¶

    EJERCICIO: DOMINÁ LOS ÁNGULOS

Parte 1: Un anillo, 4 ángulos (20 minutos)
1. Poné un anillo
2. Tomá foto desde arriba
3. Tomá foto a 45°
4. Tomá foto de perfil
5. Tomá foto macro del detalle
6. Compará: ¿Cuál te gusta más? ¿Cuál muestra mejor la joya?

Parte 2: Diferentes joyas, diferentes técnicas (30 minutos)
1. Anillo: probá flat lay + inclinado
2. Collar: probá flat lay + colgando
3. Aros: probá par + individual + lifestyle
4. Pulsera: probá flat lay + en muñeca

Parte 3: Composición (20 minutos)
1. Hacé 3 flat lays con diferentes composiciones:
   a) Joya sola, centrada
   b) Joya + hojas
   c) Joya + otros elementos
2. Compará los resultados

ENTREGA:
→ Mínimo 8 fotos (2 por cada tipo de joya)
→ Nota: ¿Qué ángulos/tecnicas funcionaron mejor?
