    Módulo 08 — Edición Básica: De Crudo a Profesional¶

    Editá tus fotos de joyería paso a paso¶

    "Editar no significa manipular. Editar significa corregir lo que la cámara no puede capturar perfectamente y potenciar lo que ya está bien."

    1. Por qué editar fotos de joyería¶

    Lo que la cámara captura vs. lo que vemos:¶

    REALIDAD vs. CÁMARA:

LO QUE VEMOS:
→ Colores conmemoria real
→ Brillos tal como los percibimos
→ Balance de blancos нейтральный
→ Iluminación percibida como pareja

LO QUE CAPTURA LA CÁMARA:
→ Puede ser más oscuro o más lumbre
→ Los blancos pueden estar quemados
→ Los negros pueden estar aplastados
→ Los colores pueden estar inexactos

EDICIÓN:
→ Corrige lo que la cámara no pudo capturar
→ Potencia los puntos fuertes
→ Mantiene la fidelidad al producto real
→ Prepara para web/social media

    Qué buscar en una foto de joyería editada:¶

    FOTO DE JOYERÍA - CHECKLIST:

□ Exposición correcta (ni muy lumbre ni muy oscura)
□ Blancos que no estén quemados
□ Negros con detalle (no totalmente negros)
□ Colores fiéis a la realidad
□ Brillos controlados pero presentes
□ Fondo limpio y uniforme
□ Nitidez adecuada
□ Sin ruido digital excesivo

    2. Herramientas de Edición Recomendadas¶

    Apps gratuitas para celular:¶

    EDICIÓN MÓVIL - LAS MEJORES:

SNAPSEED (GRATIS - iOS/Android)
→ Potente y fácil
→ No tiene filtros automáticos raros
→ Control total sobre ajustes
→ MI RECOMENDADA para começar

LIGHTROOM MOBILE (GRATIS - iOS/Android)
→ Nivel profesional
→ Presets útiles
→ Control muy preciso
→ Si estás dispuesto a aprender

VSCO (GRATIS/PAID)
→ Filtros estéticos nice
→ Buenas para look "editorial"
→ Menos control técnico

CANVA (GRATIS - iOS/Android)
→ Más diseño que edición
→ Bueno para agregar textos/márgenes
→ No para edición técnica

APP NATIVA DEL CELULAR
→ Rápida
→ Limitada
→ Solo para ajustes básicos

    Mi recomendación:¶

    PARA COMENZAR: SNAPSEED
→ Gratis
→ Fácil de aprender
→ Tiene todo lo que necesitás
→ No tiene filters que arruinen la foto

PARA AVANZAR: LIGHTROOM MOBILE
→ Más opciones
→ Presets útiles
→ Curvas y ajustes avanzados
→ Vale la pena cuando domines Snapseed

    La interfaz:¶

    INTERFAZ DE SNAPSEED:

┌─────────────────────────────────────┐
│  Herramientas              Ajustes  │
├─────────────────────────────────────┤
│                                     │
│                                     │

TOOLS (Herramientas):
→ Selecciones específicas
→ Atenuar luz intensa
→ Detalles (nitidez, estructura)
→ Recortes, rotaciones

ADJUSTMENTS (Ajustes):
→ Brillo general
→ Contraste
→ Saturación
→ Atmosfera
→ Sombras
→ Luces
→ Temperatura
→ Nitidez

    Flujo de edición para joyería:¶

    PASO A PASO EN SNAPSEED:

1. ABRÍ LA FOTO
   → Abrí Snapseed
   → Tocá "Abrir"
   → Seleccioná la foto

2. AJUSTES BÁSICOS (primero)
   → Ir a "Ajustes" (parece un slider)
   → Empezar desde arriba

3. ORDEN RECOMENDADO:
   a) Exposición (brillo)
   b) Contraste
   c) Atmosfera /hdr
   d) Sombras
   e) Luces
   f) Saturación (poco)
   g) Temperatura
   h) Nitidez

    4. Ajustes Específicos: Explicación¶

    Qué hace cada ajuste:¶

    EXPOSICIÓN / BRILLO:
→ Qué tan luminosa es la foto completa
→ Ajusta el valor general
→ Para joyería: generalmente -10 a +10

CONTRASTE:
→ Diferencia entre luces y sombras
→ Alto contraste = más dramáticas
→ Bajo contraste = más planas
→ Para joyería: generalmente +5 a +15

ATÓSFERA / HDR:
→ Múltiples exposiciones combinadas
→ Recupera luces y sombras
→ Para joyería: +20 a +50 funciona bien
→ Cuidado: puede verse artificial si mucho

SOMBRAS:
→ Solo las zonas oscuras
→ Subir = más detalle en sombras
→ Bajar = sombras más profundas
→ Para joyería: +10 a +30

LUCES:
→ Solo las zonas claras
→ Bajar = recupera highlight quemados
→ Subir = más brillante
→ Para joyería: generalmente -10 a -30

SATURACIÓN:
→ Intensidad de los colores
→ Subir = colores más vivos
→ Bajar = colores más apagados
→ Para joyería: generalmente 0 a +10

TEMPERATURA:
→ Cálido (naranja) vs frío (azul)
→ Para joyería: depende de la luz original
→ Generalmente: 0 a +5 (ligeramente cálido)

    Valores de partida por material:¶

    VALORES RECOMENDADOS:

ORO:
→ Exposición: 0 a -5
→ Contraste: +10 a +15
→ Atmosfera: +20 a +40
→ Sombras: +10 a +20
→ Luces: -10 a -20
→ Saturación: +5 a +10
→ Temperatura: +5 a +10 (calidez)

PLATA:
→ Exposición: -10 a -20
→ Contraste: +15 a +25
→ Atmosfera: +10 a +20
→ Sombras: +20 a +30
→ Luces: -20 a -30
→ Saturación: 0 a +5
→ Temperatura: 0 a +5

PIEDRAS TRANSPARENTES:
→ Exposición: 0 a -10
→ Contraste: +10 a +20
→ Atmosfera: +20 a +40
→ Sombras: +10 a +20
→ Luces: -15 a -25
→ Saturación: +5 a +15
→ Temperatura: 0

PERLAS:
→ Exposición: +5 a +10
→ Contraste: +5 a +10
→ Atmosfera: +10 a +20
→ Sombras: +10 a +15
→ Luces: 0 a -10
→ Saturación: 0 a +5
→ Temperatura: +5 (ligeramente cálido)

    5. Tutorial Paso a Paso: Foto de Anillo¶

    Ejemplo práctico:¶

    EJEMPLO: ANILLO CON DIAMANTE

PASO 1: ABRIR
→ Abrí Snapseed
→ Tocá "Abrir"
→ Seleccioná tu foto

PASO 2: AJUSTES INICIALES
→ Tocá "Ajustes" (ícono de slider)
→ Exposición: -10 (proteger highlights)
→ Contraste: +15 (agregar definición)

PASO 3: RECUPERAR DETALLES
→ Atmosfera: +30 (HDR, recuperar luces y sombras)
→ Sombras: +20 (ver más en zonas oscuras)
→ Luces: -20 (recuperar brillos del diamante)

PASO 4: COLORES
→ Saturación: +5 (color ligeramente más vivo)
→ Temperatura: +3 (ligeramente cálido)

PASO 5: NITIDEZ
→ Volvé a Herramientas
→ Tocá "Detalles"
→ Nitidez: +20
→ Estructura: +10 (definiert bordes)

PASO 6: VERIFICAR
→ Mirar el resultado
→ Si algo no está bien, revertir ese paso
→ Guardar

PASO 7: GUARDAR
→ Tocá "Aplicar" (abajo a la derecha)
→ Luego "Exportar"
→ Elegí calidad (alta)
→ Guardar en galería

    6. Problemas Comunes y Cómo Arreglarlos¶

    Foto muy oscura:¶

    PROBLEMA: FOTO SUBEXPUESTA

SOLUCIÓN:
→ Subí Exposición: +15 a +30
→ Subí Sombras: +20 a +40
→ Bajá Contraste un poco: -5
→ Agregá Nitidez: +15

    Foto muy luminosa:¶

    PROBLEMA: FOTO SOBREXPUESTA

SOLUCIÓN:
→ Bajá Exposición: -15 a -30
→ Bajá Luces: -20 a -40
→ Subí Sombras: +10 a +20
→ Aumentá Contraste: +10 a +20

    Colores alterados:¶

    PROBLEMA: COLORES NO SON FIELES

PARA LUZ FRÍA (demasiado azul):
→ Subí Temperatura: +10 a +20

PARA LUZ CÁLIDA (demasiado naranja):
→ Bajá Temperatura: -5 a -15

PARA COLORES LAVADOS:
→ Subí Saturación: +10 a +20
→ Aumentá Contraste: +10 a +15

PARA COLORES MUY INTENSOS:
→ Bajá Saturación: -5 a -15

    Brillos quemados:¶

    PROBLEMA: BLANCOS SOBRESATURADOS

SOLUCIÓN:
→ Usá "Luces" para bajar: -20 a -40
→ O usá "Herramientas" → "Atenuar luz intensa"
→ Aplicar selectivamente en las zonas afectadas

    Fondo no uniforme:¶

    PROBLEMA: FONDO CON SOMBRAS O GRADIENTE

SOLUCIÓN:
→ Usar "Parche" o "Pincel" (en Herramientas)
→ Clonar zonas uniformes del fondo
→ O reducir sombras: Sombras +30 a +50

    7. Lightroom Mobile: Guía Rápida¶

    Si preferís Lightroom:¶

    INTERFAZ DE LIGHTROOM:

┌─────────────────────────────────────┐
│  Light   Color   Effects   Optics   │
├─────────────────────────────────────┤
│                                     │
│           TU FOTO                    │
│                                     │
│                                     │
├─────────────────────────────────────┤
│  [ Light ]  [ Color ]  [ More ]    │
└─────────────────────────────────────┘

LIGHT:
→ Exposición
→ Contraste
→ Luminancia (sombras/luces)
→ Temperatura
→ Tint

COLOR:
→ Saturación
→ Vibrance (menos agresivo que saturación)

OPTICS:
→ Nitidez
→ Ruido (granos)
→ Eliminación de viñeta

    Flujo similar a Snapseed:¶

    WORKFLOW EN LIGHTROOM:

1. LUZ
   → Exposición: ajustar general
   → Contraste: +10 a +20
   → Luminancia de sombras: +20 a +40
   → Luminancia de luces: -15 a -30

2. COLOR
   → Temperatura: ajustar según luz original
   → Vibrance: +10 a +20 (más sutil que saturación)
   → Saturación: 0 a +10

3. OPTICS
   → Nitidez: +20 a +40
   → Ruido: reducir si es necesario

    8. Presets y Filtros: Cuándo Usarlos¶

    Qué son los presets:¶

    PRESETS:
→ Configuración guardada que aplicás con 1 click
→ Lightroom y VSCO tienen presets
→ Aceleran el flujo de trabajo

PARA JOYERÍA:
→ Pueden funcionar si son sutiles
→ Pero generalmente es mejor ajustar manualmente
→ Los presets "artísticos" arruinan fotos de producto

    Cuándo SÍ usar presets:¶

    SÍ USAR PRESETS SI:
→ El preset es "ligero" y sutil
→ Lo ajustás después de aplicar
→ Es para mantener consistencia en tu feed
→ Ya probaste que funciona para tu tipo de joyería

NO USAR PRESETS SI:
→ Son muy extremos (looks vintage, cinematográfica)
→ No los ajustás después
→ No los probaste antes
→ Cambian los colores de forma unrealistic

    9. Errores de Edición en Joyería¶

    Errores fatales:¶

    ❌ SATURACIÓN MUY ALTA
   → Los colores se ven artificiales
   → La joya parece de plástico
   → Solución: Máximo +15, generalmente +5 a +10

❌ CONTRASTE MUY ALTO
   → Pierden detalle en luces y sombras
   → Look слишком драматичный
   → Solución: Máximo +25, generalmente +10 a +20

❌ TEMPERATURA MUY CÁLIDA
   → Todo se ve naranja
   → El metal se ve altered
   → Solución: Máximo +15, generalmente 0 a +10

❌ EXCESIVA NITIDEZ
   → Aparece ruido digital
   → Se ven artefactos
   → Solución: Máximo +30, generalmente +15 a +25

❌ EDITAR SIN VER EL RESULTADO EN PANTALLA GRANDE
   → Lo que ves en el celular puede ser diferente
   → Revisar en компьютер si podés
   → O hacer zoom al 100% en el celular

    10. Ejercicio del Módulo¶

    Tu misión esta semana:¶

    EJERCICIO: EDITA COMO PROFESIONAL

Parte 1: Edición básica (20 minutos)
1. Abrí una foto de joyería en Snapseed
2. Aplicá el flujo completo:
   a) Exposición
   b) Contraste
   c) Atmosfera
   d) Sombras
   e) Luces
   f) Saturación
   g) Temperatura
   h) Nitidez
3. Guardá el resultado

Parte 2: Comparación (15 minutos)
1. Tomá la misma foto
2. Editá 2 versiones:
   a) Versión 1: Ajustes conservadores (0 a +10 por ajuste)
   b) Versión 2: Ajustes más agresivos (+15 a +30 por ajuste)
3. Compará: ¿Cuál se ve mejor?

Parte 3: Diferentes materiales (20 minutos)
1. Editá 1 foto de oro
2. Editá 1 foto de plata
3. Editá 1 foto de piedra
4. Ajustá los valores según las recomendaciones del módulo
5. Compará con versiones sin editar

ENTREGA:
→ 1 foto editada (la que mejor te quedó)
→ Nota: ¿Qué ajustes fueron más importantes?
→ ¿Qué valores de partida funcionaron mejor?
