## 💡 Cómo usar este programa

### Paso a paso

1. **Leé los cursos en orden** — Cada curso te prepara para el siguiente
2. **Hacé los ejercicios** — La práctica hace al CM perfecto
3. **Usá las guías** — Consultalas cuando necesites ayuda con una herramienta
4. **Practica en tu celular** — Todo está pensado para dispositivos móviles

### 📖 Dentro de cada curso

Cada curso incluye:

- **Módulos** — Contenido teórico y práctico
- **Ejercicios** — Para poner en práctica lo aprendido
- **Recursos** — Plantillas, checklists y ejemplos
- **Resumen** — Repaso rápido de lo más importante
