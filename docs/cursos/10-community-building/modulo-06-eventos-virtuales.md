# Módulo 6 — Eventos Virtuales y Campañas

## Objetivos de aprendizaje

Al finalizar este módulo vas a:
- ✅ Planificar eventos virtuales efectivos
- ✅ Crear campañas de comunidad que generen participación
- ✅ Implementar rituales de comunidad
- ✅ Medir el éxito de eventos y campañas
- ✅ Mantener la energía de comunidad entre eventos

---

## 1. Por qué eventos virtuales funcionan

```
EVENTO = MOMENTO MEMORABLE

En un mundo de contenido constante,
un evento es un momento especial.

LO QUE UN EVENTO LOGRA:
✓ Crea sensación de "estábamos todas juntas"
✓ Genera contenido generado por la comunidad
✓ Refuerza pertenencia
✓ Atrae nuevos miembros (quieren estar en el próximo)
✓ Da material para contenido durante semanas

EJEMPLO:
"Juntada virtual de San Valentín"
→ 50 participantes
→ Fotos compartidas
→ Historias etiquetadas
→ 3 semanas de contenido derivado
→ 10 nuevas clientas potenciales

ROI: Altísimo.
```

## 2. Tipos de eventos virtuales

### 2.1 IG Live / TikTok Live

```
QUÉ ES:
Transmisión en vivo donde la comunidad interactúa en tiempo real.

PARA QUÉ FUNCIONA:
→ Talleres educativos
→ Preguntas y respuestas
→ Behind the scenes en tiempo real
→ Lanzamiento de productos
→ Entrevistas con expertos

DURACIÓN:
→ 15-30 min (óptimo para atención)
→ 45-60 min (para contenido denso)

EJEMPLO EN JOYERÍA:
"LIVE: Cómo elegir el anillo perfecto"
→ Explicás tipos de piedras
→ Mostrás anillos en vivo
→ Respondés preguntas en el momento
→ Anunciás oferta especial para asistentes
```

### 2.2 Juntadas virtuales

```
QUÉ ES:
Reunión social, no necesariamente con contenido educativo.
Simplemente "estar juntas".

PARA QUÉ FUNCIONA:
→ Construir conexión entre members
→ Conocer a la comunidad
→ Crear senso de pertenencia
→ Generar contenido generado por usuarios

FORMATO:
→ Llamada grupal (Zoom, Google Meet, WhatsApp)
→ Tema social (no trabajo)
→ Actividad simple (quiz, compartir fotos)

EJEMPLO:
"Juntada virtual: Mostrá tu joya favorita"
→ Cada persona muestra y cuenta la historia
→ Todas aplauden y comentan
→ Se van sintiendo parte de algo
```

### 2.3 Sorteos y concursos

```
QUÉ ES:
Premio para quien participe de cierta manera.

PARA QUÉ FUNCIONA:
→ Generar engagement rápido
→ Atraer nuevos seguidores
→ Recompensar a la comunidad
→ Crear contenido generado por usuarios

TIPOS:
→ Sorteo simple (comment to enter)
→ Concurso de fotos (UGC)
→ Quiz/cultural
→ Creativo ("Diseñá tu joya ideal")

EJEMPLO DE CONCURSO:
"Concurso de fotos: Tu joya favorita"
→ "Subí una foto de tu joya favorita con #MiJoyitaConHistoria"
→ "Las 3 más votadas ganan un cleaning gratis"
→ "Las fotos se comparten en stories"
```

### 2.4 Lanzamientos como eventos

```
QUÉ ES:
Cada nuevo producto es un momento, no solo un post.

PARA QUÉ FUNCIONA:
→ Generar expectación
→ Sentir exclusividad
→ Crear urgencia
→ Contenido para semanas

CÓMO ARMARLO:

DÍA -7: Teaser
"Algo viene... ¿Estás lista? 👀"

DÍA -3: Pistas
"Primer pista de la nueva colección. ¿Qué será?"

DÍA -1: Countdown
"Tomorrow we reveal. Make sure you're here."

DÍA 0: LANZAMIENTO
→ IG Live de presentación
→ Primeros en ver tienen 24h de ventaja
→ Descuento exclusivo para la comunidad

DÍA +1 a +7: Contenido derivativo
→ Fotos de clientas
→ Reviews
→ Detalles de cerca
```

## 3. Rituales de comunidad

### Qué son y por qué importan

```
RITUAL = Algo que pasa REGULARMENTE
         y la comunidad lo espera y disfruta.

EJEMPLOS:
→ "Tip del Martes"
→ "Foto del Domingo"
→ "Pregunta del Miércoles"
→ "Behind the Scenes Fridays"

POR QUÉ FUNCIONAN:
→ Generan predictibilidad
→ Crean hábito
→ La gente espera y planea alrededor
→ Unen a la comunidad por algo común
```

### Rituales para joyería

```
RITUALES QUE PODÉS IMPLEMENTAR:

1. "TRIVIA DEL LUNES"
   → Pregunta sobre joyería
   → La primera en responder bien gana algo
   → Genera participación early week

2. "FOTO DEL DOMINGO"
   → "¿Cuál es tu joya para esta semana?"
   → La comunidad comparte fotos
   → Se eligen favoritos para stories

3. "DESCUBRÍ LA PIEDRA"
   → "¿Qué piedra es esta?"
   → Educativo y divertido
   → Los expertos se sienten orgullosos

4. "MIÉRCOLES DE CONSEJO"
   → Tip de cuidado de joyas
   → Cada semana algo diferente
   → Acumulan conocimiento

5. "VIERNES DE behind the SCENES"
   → Mostrás cómo preparás pedidos
   → O el taller/proceso
   → Humaniza la marca

6. "SÁBADO DE STORIES COMPARTIDAS"
   → "Tag a alguien que te regala las mejores joyas"
   → Genera contenido UGC
   → Aumenta reach
```

## 4. Cómo planificar un evento

### Paso 1: Define el objetivo

```
ANTES DE PLANIFICAR, PREGUNTATE:

¿Qué quiero lograr con este evento?

□ Engagement (más interacción)
□ Ventas (lanzamiento o promoción)
□ Comunidad (conexión entre members)
□ Aprendizaje (educación sobre joyería)
□ Atracción (traer nuevos followers)
□ Retención (premiar a leales)

UN EVENTO PUEDE TENER MÚLTIPLES OBJETIVOS,
PERO TENÉ UNO PRINCIPAL.
```

### Paso 2: Define el formato

```
SEGÚN TU OBJETIVO:

SI ES ENGAGEMENT:
→ Quiz, trivia, concurso de fotos
→ Premio para participantes

SI ES APRENDIZAJE:
→ Taller, clase, IG Live educativa
→ Contenido de valor

SI ES COMUNIDAD:
→ Juntada social
→ space para conocerse

SI ES VENTA:
→ Lanzamiento con oferta
→ Live shopping
→ Exclusividad temporal
```

### Paso 3: Define logística

```
ANTES DEL EVENTO:

FECHA/HORA:
→ ¿Cuándo es más probable que tu audiencia esté activa?
→ ¿Dieron feedback sobre horarios antes?
→ ¿Es una fecha especial?

PLATAFORMA:
→ IG Live (fácil, integración con IG)
→ Zoom (más formal, hasta 100 personas)
→ WhatsApp (para grupo existente)

PROMOCIÓN:
→ ¿Cuántos días antes publicás sobre el evento?
→ ¿En stories? ¿En posts? ¿En DM?
→ "¿Marquen a una amiga que le gustaría venir?"

MATERIALES:
→ ¿Necesitás algo preparado?
→ ¿Tenés slides? ¿Productos? ¿Premios?
→ ¿Probaste la tecnología?

CONTINGENCIA:
→ ¿Y si no viene nadie?
→ ¿Y si se corta la conexión?
→ ¿Y si hay un problema técnico?
```

### Paso 4: Durante el evento

```
DURANTE:

1. LLEGÁ PRIMERO
   → Entrá 5-10 min antes
   → Probá audio/video
   → Prepará el espacio

2. DALE LA BIENVENIDA
   → "¡Bienvenidas todas!"
   → "Gracias por estar"
   → "Estamos [cantidad] conectadas"

3. MANTENÉ LA ENERGÍA
   → Hablá con entusiasmo
   → Involucrá a la audiencia
   → "¿Quién está ahí? Dejá un 👋"

4. PERMITÍ INTERACCIÓN
   → Preguntás en vivo
   → Respondé comentarios en voz alta
   → Hacé que se sientan incluidas

5. CIERRA CON NEXT STEPS
   → "¿Cuándo es el próximo evento?"
   → "¿Qué quieren ver?"
   → "¡Nos vemos pronto!"
```

### Paso 5: Después del evento

```
DESPUÉS:

1. AGRADECÉ
   → "¡Gracias a todas las que estuvieron!"
   → "Fue increíble compartir con ustedes"

2. COMPARTÍ CONTENIDO
   → Highlights de lo mejor
   → Fotos/videos del evento
   → Agradecimientos especiales

3. PEDÍ FEEDBACK
   → "¿Qué les pareció?"
   → "¿Para cuándo quieren el próximo?"

4. PLANIFICÁ EL PRÓXIMO
   → Mientras está fresco
   → Anunciá fecha tentativa

5. MIDE RESULTADOS
   → ¿Cuánta gente participó?
   → ¿Cuánto engagement generó?
   → ¿Se unieron nuevos miembros?
   → ¿Hubo ventas?
```

## 6. Errores comunes

### ❌ Error 1: No promocionar

```
MAL:
"Anunciamos el evento 2 horas antes. Vino poca gente."

BIEN:
"Teaser 1 semana antes.
Recordatorio 1 día antes.
Recordatorio 1 hora antes.
Vino muuuucha gente."
```

### ❌ Error 2: Sin estructura

```
MAL:
"Hablemos de lo que surja."
*Silencio incómodo*
*Nadie sabe qué esperar*

BIEN:
"Hoy vamos a:
1. Charlar sobre [tema]
2. Responder sus preguntas
3. Sortear algo especial

Duración: 30 minutos.
Empecemos."
```

### ❌ Error 3: No involucrar

```
MAL:
"Hablé por 40 minutos. Al final nadie participó."

BIEN:
"Pausás cada 5-10 min para preguntar.
Invitan a comentar.
Hacen participar a alguien.
Gamificás si es posible."
```

### ❌ Error 4: No dar seguimiento

```
MAL:
"Hicimos el evento. Fin."

BIEN:
"Gracias, posts de highlights,
feedback, próximos pasos."
```

## 7. Ejercicio práctico

```
EJERCICIO 1: Planificá tu primer evento

PASO 1: Objetivo
¿Qué querés lograr?

PASO 2: Formato
¿Qué tipo de evento?

PASO 3: Tema
¿De qué va a tratar?

PASO 4: Fecha/Hora
¿Cuál es la fecha y hora?

PASO 5: Promoción
¿Cómo lo vas a anunciar?
□ Posts (¿cuántos?)
□ Stories (¿cuántas?)
□ DM directo
□ Email/WhatsApp

PASO 6: Lo que necesitás
□ Materiales
□ Premio (si hay)
□ Tecnología

EJECUTÁ ESTE EVENTO EN LAS PRÓXIMAS 2 SEMANAS.

---

EJERCICIO 2: Crea un ritual

ELEGÍ UN RITUAL:
□ Trivia del Lunes
□ Foto del Domingo
□ Miércoles de Consejo
□ Viernes de BTS

NOMBRÁ:
→ ¿Cuándo? (día y hora)
→ ¿Qué consiste exactamente?
→ ¿Cómo participan?
→ ¿Qué obtienen?

LANZÁ EL RITUAL ESTA SEMANA.
Mantengólo durante 4 semanas mínimo.
```
