# Módulo 4 — Contenido Evergreen vs Trending

## Objetivos de Aprendizaje

Al finalizar este módulo vas a:

- ✅ Diferenciar contenido evergreen de contenido trending
- ✅ Crear contenido que funciona todo el año
- ✅ Aprovechar tendencias sin depender de ellas
- ✅ Equilibrar ambos tipos en tu estrategia
- ✅ Optimizar contenido evergreen para máximo alcance

---

## 1. Qué es contenido evergreen

```
DEFINICIÓN:

Contenido "perenne" que mantiene su valor y relevancia
independientemente de la fecha, estación, o momento actual.

Es como un árbol que no pierde sus hojas:
 siempre verde, siempre relevante.

EJEMPLOS DE CONTENIDO EVERGREEN PARA JOYERÍA:
• "Cómo limpiar tus joyas de plata"
• "Diferencias entre oro 18k, 14k y baño de oro"
• "Guía para elegir el regalo perfecto"
• "Cómo guardar tus joyas para que duren"
• "Significado de las piedras preciosas"
• "Cómo saber si una joya es de calidad"
```

---

## 2. Qué es contenido trending

```
DEFINICIÓN:

Contenido que aprovecha un momento, evento, o tema
que está siendo tema en este momento específico.

Es como una flor que florece y se marchita:
 tiene su momento, después no.

EJEMPLOS DE CONTENIDO TRENDING PARA JOYERÍA:
• "Joyas inspiradas en la nueva serie de Netflix"
• "El estilo de joya que viste en los Grammy"
• "Cuando la tendencia core aesthetics llega a la joyería"
• "Joyas para la era del quiet luxury"
• "La jewels que viste en la boda del año"
```

## 3. Comparación directa

```
┌───────────────────┬─────────────────────┬─────────────────────┐
│ Aspecto           │ Evergreen            │ Trending            │
├───────────────────┼─────────────────────┼─────────────────────┤
│ Duración          │ Meses o años        │ Días o semanas      │
│ Valor             │ Constante           │ Decreciente         │

│ Producción         │ Requiere investigación│ Requiere velocidad  │
│ Riesgo            │ Bajo                │ Alto (puede flop)   │

│ Ejemplo           │ Guía de materiales  │ Joyas de Taylor Swift│
└───────────────────┴─────────────────────┴─────────────────────┘
```

## 4. Por qué necesitás ambos

```
EQUILIBRIO NECESARIO:

EVERGREEN es tu BASE:
• Asegura que siempre tengas contenido relevante
• Construye autoridad y conocimiento
• Trae tráfico constante (no depende de tendencias)
• Se puede reutilizar y republicar

TRENDING es tu ACELERADOR:
• Da bursts de alcance y nuevos seguidores
• Muestra que estás atenta al momento
• Genera contenido "de época"
• Puede viralizar tu cuenta

SIN EVERGREEN:
→ Dependés de tendencias
→ Cuando no hay trending, no sabés qué publicar
→ Alcance errático

SIN TRENDING:
→ Tu contenido es siempre igual
→ Perdés oportunidades de alcance rápido
→ Parecés disconnected del momento
```

## 5. Distribución recomendada

```
REGLA PRÁCTICA:

70% EVERGREEN
30% TRENDING

O SEGÚN TU SITUACIÓN:

CUENTA NUEVA (< 1 año):
→ 80% evergreen, 20% trending
→ Necesitás construir base sólida primero

CUENTA ESTABLECIDA (> 1 año):
→ 60% evergreen, 40% trending
→ Ya tenés base, podés arriesgar más

CUENTA EN CRECIMIENTO:
→ 70% evergreen, 30% trending
→ Equilibrio para crecer steady

SI ESTÁS EN TEMPORADA (Navidad, San Valentín):
→ Puede ser 50/50 o más trending
→ Las fechas te dan trending natural
```

## 6. Cómo crear contenido evergreen perfecto

### 6.1 Características del evergreen efectivo

```
CHECKLIST PARA CONTENIDO EVERGREEN:

□ Resuelve un problema común
□ Es atemporal (no depende de fechas)
□ Está bien researched (información correcta)
□ Es completo (cubre el tema a fondo)
□ Es accionable (el lector puede hacer algo)
□ Es shareable (la gente quiere compartirlo)
□ Tiene una imagen o formato atractivo

EJEMPLO BUENO:
"Guía completa: Cómo cuidar tus joyas de oro para que duren años"
→ Resuelve problema: cómo cuidar oro
→ Atemporal: siempre aplica
→ Completo: guía completa
→ Accionable: pasos claros
→ Shareable: información útil que querés compartir
```

### 6.2 Temas evergreen para joyería

```
CATEGORÍAS DE CONTENIDO EVERGREEN:

EDUCACIÓN BÁSICA:
• Diferencias entre tipos de metales
• Cómo cuidar diferentes materiales
• Cómo almacenar joyas
• Significado de piedras
• Guía de tallas y medidas

INSPIRACIÓN PERENNE:
• Guía de estilos por ocasión
• Cómo combinar joyitas (layering)
• Joyas para cada tipo de outfit
• Classics que nunca pasan de moda

INFORMACIÓN DE MARCA:
• Proceso de creación
• Valores de la marca
• Historia de la marca
• Calidad y garantía

REGALOS:
• Guía de regalos por presupuesto
• Qué regalar según la relación
• Errores comunes al regalar joyas
• Cómo elegir el regalo perfecto
```

### 6.3 Optimizar evergreen para alcance

```
TIPS PARA QUE TU EVERGREEN LLEGUE LEJOS:

1. TÍTULOS QUE ATRAEN CLICKS
   → No: "Cómo limpiar plata"
   → Sí: "El método que nadie te cuenta para que tu plata brille como nueva"

2. IMÁGENES ATRACTIVAS
   → Antes y después
   • Pasos visuales
   • Infografías

3. CAPTIONS QUE AGREGAN VALOR
   → Contenido del post + tips extra en caption
   → CTA claro ("Guardá para después")

4. HASHTAGS RELEVANTES
   → Mezcla de hashtags grandes y de nicho
   → Hashtags de tu tema específico

5. REPUBLICAR PERIÓDICAMENTE
   → No es "spamming", es "recordar"
   → Cada vez con pequeños ajustes
   → 2-3 veces por año está bien
```

## 7. Cómo aprovechar tendencias

### 7.1 Tipos de tendencias

```
TIPOS DE TENDENCIAS:

TENDENCIAS DE MOMENTO:
• Eventos actuales (premios, películas, viral)
• Momen трендс в соцсетях
• Estilos virales

TENDENCIAS ESTACIONALES:
• Fechas específicas (Navidad, San Valentín)
• Cambios de estación
• Vueltas a la rutina

TENDENCIAS DE INDUSTRIA:
• Nuevos estilos en joyería
• Materiales que se ponen de moda
• Movimientos (sostenibilidad, joyería artesanal)

CÓMO IDENTIFICAR TENDENCIAS:
• TikTok (antes que nadie)
• Google Trends
• Lo que ves mucho en tu feed
• Lo que talkea tu audiencia
```

### 7.2 Cómo saltar sobre tendencias rápido

```
PROCESO RÁPIDO:

DÍA 1: Identificás la tendencia
→ Ves algo trending que conecta con tu nicho

DÍA 2: Decidís si vale la pena
→ ¿Conecta con mi audiencia?
→ ¿Puedo hacerlo bien?
→ ¿Tengo los recursos?

DÍA 3: Creás el contenido
→ Seja rápido pero no descuidado
→ Tu toque de marca, la tendencia como base

DÍA 4: Publicás
→ Timing es todo
→ Cuanto antes, mejor

EJEMPLO:

Tendencia: La serie "El Oso" tiene un personaje que usa un arete específico
Día 1: "Uy, ese arete es tendencia en TikTok"
Día 2: "¿Tengo algo similar en stock? Sí. ¿Puedo mostrarlo? Sí."
Día 3: Grabo reel "Cómo conseguir el look de [personaje]"
Día 4: Publico mientras la tendencia está hot
```

### 7.3 Cómo adaptar tendencias a joyería

```
TRADICCIÓN → TU NICHO:

Tendencia: Quiet luxury (lujo silencioso)
→ Joyería: Piezas minimalistas, oro simple, sin logotipos

Tendencia: Coastal grandmother (abuela de la costa)
→ Joyería: Perlas, jade, tonos suaves

Tendencia: Old money (dinero antiguo)
→ Joyería: Oro clásico, piezas heredadas, vintage

Tendencia: Dark academia (academia oscura)
→ Joyería: Plata oxidada, piedras oscuras, gemas

CÓMO HACERLO BIEN:
1. Entiende la tendencia (no solo copies la imagen)
2. Conectá con tu producto lógicamente
3. Agregá tu perspectiva de marca
4. No forceé - si no conecta, no lo hagas
```

→ Hashtags correctos

## 9. Errores comunes

### ❌ Error 1: Solo evergreen

```
MAL:
"Tengo guías educativas que funcionan bien.
No necesito publicar nada más."

PROBLEMA:
No estás aprovechando momentos de alcance rápido.
Tu cuenta se estanca.
Parecés "old school".

CORRECTO:
70% evergreen, 30% trending.
Usá trending para acelerar crecimiento.
```

### ❌ Error 2: Solo trending

```
MAL:
"Todo lo que no es trending no vale la pena.
Prefiero esperar la próxima tendencia."

PROBLEMA:
Dependés de external factors.
Si no hay tendencia relevante, no publicás.
No construís base de conocimiento.

CORRECTO:
Evergreen es tu base, trending es tu acelerador.
```

### ❌ Error 3: Forzar tendencias

```
MAL:
"Vi que está trending el estilo 'dark academia'.
No tengo nada que ver con eso, pero voy a publicar algo."

PROBLEMA:
Tu audiencia lo nota.
Parecés inauténtico.
Confusión de marca.

CORRECTO:
Solo adoptá tendencias que genuinamente conecten con tu marca.
```

### ❌ Error 4: Publicar trending tarde

```
MAL:
"Esa tendencia estuvo hot hace 3 semanas.
Voy a publicar ahora que tengo tiempo."

PROBLEMA:
Ya pasó el momento.
Alcance = mínimo.

CORRECTO:
Trending requiere velocidad.
Si no podés hacer rápido, mejor no hacerlo.
```
