# 🎀 Curso 09: Marketing de Contenido

**Creá contenido que tu audiencia busque, no solo que vea** 📝💗

---

> *"El contenido no es lo que publicás. Es el valor que dejás."*

---

## 📚 Descripción

Aprendé a crear una estrategia de contenido integral: desde pilares de contenido hasta calendarios editoriales, pasando por contenido evergreen y reutilización inteligente.

**Después de este curso vas a poder:**

- ✅ Entender qué es el marketing de contenido
- ✅ Definir pilares de contenido estratégicos
- ✅ Crear calendarios editoriales efectivos
- ✅ Producir contenido evergreen que trabaje siempre
- ✅ Reutilizar contenido en diferentes formatos
- ✅ Fomentar contenido generado por usuarios
- ✅ Distribuir contenido estratégicamente
- ✅ Medir el impacto de tu contenido

## 💡 Los 4 Pilares de Contenido

```
PARA JOYERÍA, SUGERIMOS:

1️⃣ EDUCATIVO
   → Cómo cuidar las joyas
   → Guía de tallas
   → Significado de piedras

2️⃣ INSPIRACIONAL
   → Looks con joyería
   → Tendencias de temporada
   → Estilos para ocasiones

3️⃣ EMOCIONAL
   → Historias detrás de las piezas
   → Proceso de creación
   → Significado personal

4️⃣ VENTA
   → Producto destacado
   → Novedades
   → Promociones (sutil)
```

---

## 🗓️ Calendario Editorial

```
EJEMPLO SEMANAL:

LUNES → Educativo (story + post)
MARTES → Inspiracional (reel)
MIÉRCOLES → Venta (carrusel producto)
JUEVES → Emocional (behind the scenes)
VIERNES → Inspiracional (reel)
SÁBADO → Interacción (encuesta, pregunta)
DOMINGO → Descanso o preparación
```
