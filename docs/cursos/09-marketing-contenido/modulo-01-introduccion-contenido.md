## 1. Qué es Marketing de Contenido

### Definición simple

```
MARKETING DE CONTENIDO ES:

Crear y distribuir contenido valioso, relevante y consistente
para atraer y retener a una audiencia claramente definida.

El objetivo final es generar acciones rentables.

EN PALABRAS SIMPLES:

No estás vendiendo directamente.
Estás dando valor.
A cambio, la gente te empieza a confiar.
Y cuando confían en vos, compran.
```

### No es lo mismo que...

```
PUBLICIDAD TRADICIONAL:
"Pagá $500 y tu anuncio llega a 10,000 personas AHORA."

CONTENIDO:
"Creá contenido valioso y, con el tiempo, 10,000 personas
van a buscarte porque confían en vos."

PUBLICIDAD = RESULTADO INMEDIATO, DINERO AHORA
CONTENIDO = RESULTADO A LARGO PLAZO, DINERO DESPUÉS

PERO:
Contenido bien hecho genera resultados DURADEROS.
Un buen post puede seguir trayendo clientes por meses.
```

## 2. Por qué el contenido importa para joyería

### El problema de vender joyería online

```
JOYERÍA ES UN PRODUCTO DIFÍCIL DE VENDER ONLINE:

❌ No la podés tocar
❌ No la podés probar
❌ No sabés cómo queda en vos
❌ Hay mucha competencia
❌ El precio puede parecer alto

POR ESTO EL CONTENIDO ES CRUCIAL:

El contenido hace lo que el producto no puede:
✓ Transmite la calidad
✓ Muestra cómo queda
✓ Genera deseo
✓ Crea conexión emocional
✓ Construye confianza
✓ Diferencia de la competencia
```

### La diferencia entre vender y enamorar

```
SIN CONTENIDO:
"Anillo de oro 18k. $150. Comprá ahora."
→ El cliente compara precios
→ Busca la opción más barata
→ No hay diferenciación

CON CONTENIDO:
"La historia detrás de este anillo.
Fue diseñado para Carolina, que buscaba algo especial
para celebrar sus 10 años de casada.
Cada detalle fue pensado para representar una década
de amor que solo crece.
Oro 18k. Hecho a mano. Único."
→ El cliente conecta emocionalmente
→ Ve el valor, no solo el precio
→ Está dispuesto a pagar por la experiencia
```

## 3. Tipos de contenido para joyería

### 3.1 Por su función

```
TIPOS SEGÚN PARA QUÉ SIRVEN:

📚 EDUCATIVO
   Enseña algo útil
   Ejemplo: "Cómo cuidar tus joyas de plata"

🎭 ENTRETENIMIENTO
   Divierte o emociona
   Ejemplo: "Transformación de este collar en 30 segundos"

💡 INSPIRADOR
   Genera deseo o aspiración
   Ejemplo: "El look perfecto para tu próxima cita"

🛒 DE VENTA
   Promueve directamente (usa con cuidado)
   Ejemplo: "Nuevo collar inspirado en el mar"

🤝 SOCIAL/PRUEBA SOCIAL
   Muestra que otros confían
   Ejemplo: Testimonios, fotos de clientes
```

### 3.2 Por su formato

```
TIPOS SEGÚN CÓMO SE PRESENTAN:

📸 CARRUSEL DE FOTOS
   Múltiples imágenes con información
   Mejor para: Mostrar variaciones, contar historias, tutorials

🎬 REELS/VIDEOS
   Videos cortos (15-90 segundos)
   Mejor para: Demostrar productos, detrás de escenas, ASMR

📝 POST CON TEXTO
   Imagen + caption largo
   Mejor para: Storytelling, reflexiones, CTAs claros

📖 STORIES
   Contenido efímero (24h)
   Mejor para: Interacción, encuestas, actualizaciones

🎙️ reels extensos
   Videos más largos (1-3 minutos)
   Mejor para: Tutoriales, historias de marca, testimonios
```

### 3.3 Por su temporalidad

```
CONTENIDO EVERGREEN (perenne):
   Funciona siempre, no caduca
   Ejemplo: "¿Cómo limpiar joyas de plata?"

CONTENIDO TRENDING (de tendencia):
   Aprovecha un momento o tema de moda
   Ejemplo: "Joyas inspiradas en la película X"

CONTENIDO ESTACIONAL:
   Relacionado con fechas específicas
   Ejemplo: "Regalos para San Valentín"
```

## 4. El Customer Journey y el contenido

### Qué es el Customer Journey

```
EL RECORRIDO DEL CLIENTE:

         ┌─────────────────────────────────────────────────────┐
         │                                                     │
    ┌────┴────┐    ┌─────────┐    ┌───────┐    ┌─────────┐   │
    │CONOCERTE│───→│CONFIARTE│───→│EVALUAR│───→│COMPRAR  │   │
    └─────────┘    └─────────┘    └───────┘    └─────────┘   │
         │              │             │             │          │
         ↓              ↓             ↓             ↓          │
    ┌─────────┐    ┌─────────┐  ┌─────────┐  ┌──────────┐   │
    │ Awareness│    │Consider │  │ Decision│  │ Purchase │   │
    └─────────┘    └─────────┘  └─────────┘  └──────────┘   │
                                                     │
         ┌──────────────────────────────────────────────┘
         │
         ↓
    ┌─────────┐    ┌─────────┐
    │LOYALIDAD│───→│DEFENSOR │
    └─────────┘    └─────────┘

CADA ETAPA NECESITA CONTENIDO DIFERENTE.
```

### Contenido para cada etapa

```
ETAPA 1: CONOCERTE (Awareness)
─────────────────────────────────
Tu objetivo: Ser descubierto

Contenido para esta etapa:
• Posts educativos sobre tu nicho
• Contenido trending o de tendencia
• Colaboraciones con otros
• Publicidad para reach

Ejemplo: "5 tendencias en joyas para este año"


ETAPA 2: CONFIARTE (Consideration)
──────────────────────────────────
Tu objetivo: Generar interés y confianza

Contenido para esta etapa:
• Historias de marca
• Detrás de escenas
• Testimonios
• Contenido de valor

Ejemplo: "Cómo nació esta colección - detrás de escenas"


ETAPA 3: EVALUARTE (Decision)
──────────────────────────────
Tu objetivo: Convencer de que sos la mejor opción

Contenido para esta etapa:
• Comparaciones
• Detalles de producto
• Testimonios específicos
• Ofertas o bonuses

Ejemplo: "Por qué elegir oro 18k vs baño de oro"


ETAPA 4: COMPRARTE (Purchase)
──────────────────────────────
Tu objetivo: Facilitar la compra

Contenido para esta etapa:
• CTAs claros
• Información de envío
• Facilidades de pago
• Urgencia legítima

Ejemplo: "2x1 en anillos hasta el viernes"


ETAPA 5: LOYALIDAD (Loyalty)
─────────────────────────────
Tu objetivo: Que vuelva a comprar

Contenido para esta etapa:
• Cómo usar el producto
• Contenido de cuidado
• Nuevos lanzamientos
• Beneficios exclusivos

Ejemplo: "Cómo combinar tu nueva pulsera con otras piezas"


ETAPA 6: DEFENSOR (Advocacy)
─────────────────────────────
Tu objetivo: Que recomiende a otros

Contenido para esta etapa:
• Invitaciones a compartir
• Programas de referidos
• Contenido generated por usuarios
• Comunidad

Ejemplo: "Etiquetanos cuando uses tu nueva joya"
```

## 5. La regla del 80/20

### Qué significa

```
REGLA 80/20 PARA CONTENIDO:

80% DEL CONTENIDO = VALOR (no vende directamente)
• Educativo
• Inspirador
• Entretenido
• De construcción de comunidad

20% DEL CONTENIDO = VENTA (puede vender directamente)
• Producto nuevo
• Promoción
• Llamada a la acción directa
• Lanzamiento

POR QUÉ ESTO FUNCIONA:

Si todo lo que publicás es venta,
la gente se cansa y deja de seguirte.

Si das valor el 80% del tiempo,
cuando vendés algo, te escuchan.
```

### Ejemplo para joyería

```
SEMANA TIPO CON REGLA 80/20:

LUN: 80% — "Cómo saber si una joya es de plata real"
MAR: 80% — Reel: transformaciones de joyas
MIÉ: 20% — Nuevo collar de la colección verano
JUE: 80% — Detrás de escenas del taller
VIE: 80% — Testimonio de clienta feliz
SÁB: 20% — Recordatorio: 2x1 termina mañana
DOM: 80% — Reflexión: "La joya que heredaste de tu abuela"

5 posts de valor, 2 posts de venta = ~71/29
```

## 6. Errores comunes en marketing de contenido

### ❌ Error 1: Solo vender

```
MAL:
"Tengo que publicar para vender."
90% de tus posts son productos con "COMPRÁ AHORA"

PROBLEMA:
La gente deja de ver tus posts.
El algoritmo baja tu alcance.
No hay confianza.

CORRECTO:
80% valor, 20% venta.
```

### ❌ Error 2: No tener estrategia

```
MAL:
"Publico lo que se me ocurre cada día."

PROBLEMA:
No hay consistencia.
No hay tema claro.
No hay progreso real.

CORRECTO:
Tenés pilares de contenido.
Tenés calendario.
Cada post tiene un propósito.
```

### ❌ Error 3: Copiar sin adaptar

```
MAL:
"Voy a copiar lo que hace esa marca gringa."

PROBLEMA:
No funciona para tu mercado.
Tu audiencia es diferente.
Tu marca es diferente.

CORRECTO:
Tomá inspiración.
Adaptá a tu contexto.
Agregá tu voz única.
```

### ❌ Error 4: Esperar resultados inmediatos

```
MAL:
"Llevo 2 semanas publicando y no vendí nada."

PROBLEMA:
Contenido necesita tiempo para funcionar.
Los algoritmos recompensan la consistencia.
La confianza se construye gradualmente.

CORRECTO:
Publicá consistente por 3-6 meses.
Medí progreso.
Ajustá basándote en datos.
```
