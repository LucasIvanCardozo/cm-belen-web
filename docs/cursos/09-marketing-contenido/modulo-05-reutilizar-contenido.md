# Módulo 5 — Reutilizar Contenido

## Objetivos de Aprendizaje

Al finalizar este módulo vas a:

- ✅ Entender por qué reutilizar contenido multiplica tu impacto
- ✅ Dominar técnicas para convertir un contenido en múltiples formatos
- ✅ Crear sistemas para maximizar cada pieza de contenido
- ✅ Evitar el ciclo de "crear desde cero" todo el tiempo

---

## 1. Por qué reutilizar contenido

```
EL PROBLEMA:
"Crear contenido toma mucho tiempo.
No puedo crear contenido nuevo para cada plataforma."

LA SOLUCIÓN:
Cada pieza de contenido que creás puede convertirse en múltiples
piezas para diferentes plataformas y formatos.

EJEMPLO REAL:
1 video de 60 segundos → Puede convertirse en:
├── 1 Reel para Instagram
├── 1 Short para TikTok
├── 1 carrusel de 5 imágenes
├── 1 post de blog
├── 1 newsletter
├── 3-5 posts para LinkedIn
└── 10+ tweets/hilos

TRADUCCIÓN:
1 hora de contenido = 10+ horas de output
```

---

## 2. La regla del 3x3x3

```
SISTEMA DE REUTILIZACIÓN:

1 PIEZA DE CONTENIDO → 3 PLATAFORMAS
                        ↓
           Cada contenido se publica en 3 lugares diferentes

3 FORMATOS POR PIEZA
                        ↓
           Cada contenido se presenta de 3 formas distintas

3 USOS EN EL TIEMPO
                        ↓
           Cada contenido se usa en diferentes momentos

EJEMPLO PRÁCTICO:

Pieza original: Video de 90 segundos
"Tres tips para elegir un anillo de compromiso"

3 PLATAFORMAS:
├── Instagram (Reel)
├── TikTok (Short)
└── YouTube Shorts

3 FORMATOS:
├── Video original (90 segundos)
├── Carrusel de 3 slides (resumen)
└── Infografía estática (tips escritos)

3 USOS:
├── Semana 1: Publicación inicial
├── Mes 3: Reutilización con actualización
└── Mes 6: Compilación de "mejores tips"
```

## 3. Técnicas de reutilización por tipo de contenido

### 3.1 De video a múltiples formatos

```
VIDEO ORIGINAL (2-5 minutos):
│
├── REEL/SHORT (15-60 seg)
│   └── Los mejores 60 segundos
│
├── CARRUSEL (5-10 slides)
│   └── Capturas de pantalla + texto
│
├── POST DE TEXTO
│   └── Resumen escrito con puntos clave
│
├── INFOGRAFÍA
│   └── Diseño estático con tips principales
│
├── AUDIO (podcast snippet)
│   └── Solo el audio + imagen
│
└── HILO DE POSTS
    └── Dividido en partes lógicas
```

### 3.2 De carrusel a múltiples formatos

```
CARRUSEL ORIGINAL (10 slides):
│
├── POST INDIVIDUAL
│   └── Cada slide como post separado
│
├── REEL ESTÁTICO
│   └── Slides en secuencia rápida
│
├── STORIES
│   └── Un slide por story
│
├── PDF DESCARGABLE
│   └── Carrusel como recurso
│
└── POST DE BLOG
    └── Expande cada slide en párrafos
```

### 3.3 De post a múltiples formatos

```
POST ORIGINAL:
"5 razones por las que el oro nunca pasa de moda"

├── TIKTOK
│   └── "Hablando de por qué el oro es para siempre"
│
├── CARRUSEL
│   └── Cada razón en un slide
│
├── VIDEO
│   └── Expandí cada punto con ejemplo
│
├── STORIES QUIZ
│   └── "¿Sabías que...? /swipe para saber más"
│
├── NEWSLETTER
│   └── Versión completa con imágenes
│
└── YOUTUBE
    └── "Deep dive" de 10 minutos
```

## 4. Sistema práctico paso a paso

### Paso 1: Creá contenido "master"

```
AL CREAR CONTENIDO, PENSA EN:
"¿Cómo puedo hacer que esto se transforme en más contenido?"

EJEMPLO:
Cuando grabés un video, pensá:
├── Los 15 segundos más interesantes = para Reel
├── Los mejores quotes = para posts estáticos
├── El concepto completo = para carrusel
└── Los comentarios/preguntas = para contenido futuro
```

### Paso 2: Creá tu "banco de contenido"

```
DESPUÉS DE CADA CONTENIDO:

TAREA 1: Editá y guardá las variaciones
├── Video completo → guardado
├── Clips cortos → guardados (3-5 clips)
├── Capturas de pantalla → guardadas
└── Transcripción → guardada

TAREA 2: Etiquet en tu sistema
├── Nombre del contenido
├── Tema/pilar
├── Fecha de creación
└── Formatos disponibles

HERRAMIENTAS PARA ORGANIZAR:
• Notion (base de datos de contenido)
• Google Drive (carpetas por tema)
• Carpeta en teléfono (por formato)
```

### Paso 3: Programa con inteligencia

```
DISTRIBUYE EL CONTENIDO EN EL TIEMPO:

SEMANA 1:
├── Lunes: Video completo (YouTube/IGTV)
├── Miércoles: Reel (best moments)
└── Viernes: Carrusel (expansión)

SEMANA 2:
├── Martes: Post estático (quote/tip)
└── Jueves: Stories (preguntas del contenido)

SEMANA 3:
├── Recircular el Reel original
└── Crear contenido nuevo basado en comentarios

SEMANA 4:
├── Compilación del contenido del mes
└── Nuevo contenido master
```

## 5. Reutilización inteligente para joyería

### 5.1 Ejemplo: Sesión de fotos

```
SESIÓN DE FOTOS: 20 fotos de nuevas piezas

OUTPUT DE CONTENIDO:
┌─────────────────────────────────────────────────┐
│ FOTO 1: Anillo dorado                          │
├─────────────────────────────────────────────────┤
│ → Post individual (detalle)                     │
│ → Parte de carrusel "Nuevas llegadas"          │
│ → Para stories (close-up shots)                │
│ → Para Reel (ASMR del brillo)                  │
│ → Para Pinterest                               │
└─────────────────────────────────────────────────┘

TOTAL DE UNA SESIÓN:
• 20 fotos
• = 20 posts individuales
• = 4 carruseles de 5 fotos
• = 10+ Reels/Stories
• = Contenido para 1 mes completo
```

### 5.2 Ejemplo: Captura de pantalla de comment

```
UN COMENTARIO INTERESANTE:
"¿Cómo sé si un anillo es de plata real?"

TU RESPUESTA EN VIDEO (60 segundos):
→ Puede convertirse en:
   • Reel educativo
   • Post tipo FAQ
   • Infografía "Cómo identificar plata real"
   • Stories con quiz
   • Sección de stories destacada
```

## 6. Fórmulas de reutilización

### Fórmula: CLIP-PULL-POST

```
C = Clip (cortá el mejor momento)
   → "El momento más interesante del video"

L = Lead (generá curiosidad)
   → "¿Sabías que...?"

I = Insight (dá valor)
   → "Por esto es importante"

P = Post (publicá en formatos múltiples)
   → "Aquí te explico más"

EJEMPLO:
C: "El 90% de las joyas que dicen 'plata' no lo son"
L: "¿Cómo distinguish lo real de lo falso?"
I: "Tres tests simples que podés hacer en casa"
P: Carrusel + Reel + Post
```

### Fórmula: ONE-MANY

```
ONE = Un contenido master
MANY = Muchas variaciones

ONE: Video de 3 minutos
"Cómo cuidar tus joyas de plata"

MANY:
├── 1 Reel (60 segundos)
├── 1 Short (60 segundos)
├── 1 Carrusel (5 tips)
├── 1 Infografía
├── 1 Post escrito
├── 1 Newsletter
├── 1 LinkedIn post
└── 3+ Tweets/posts cortos
```

## 7. Errores comunes

### ❌ Error 1: No guardar contenido raw

```
MAL:
"Ya publiqué el video, borro los archivos."

BIEN:
"Guardo video original + todos los clips + fotos
+ transcripción en carpeta organizada."

HÁBITO: Al final de cada contenido, guardá TODO.
```

### ❌ Error 2: Publicar igual en todos lados

```
MAL:
"El mismo post exacto en IG, FB, TikTok y LinkedIn."

BIEN:
"Adaptó el tono y formato a cada plataforma.
Lo que funciona en TikTok no funciona en LinkedIn."
```

### ❌ Error 3: Esperar a que se acabe el contenido

```
MAL:
"Ya publiqué todo, no tengo más ideas."

BIEN:
"Reviso mi banco de contenido no utilizado.
Convierto contenido antiguo en formatos nuevos."
```

### ❌ Error 4: No trackear qué funciona

```
MAL:
"Publiqué todo pero no sé qué funcionó."

BIEN:
"Trackeo qué formato/plataforma tuvo mejor resultado.
Me guío por los datos para decidir qué reutilizar."
```

## 9. Checklist de reutilización

```
DESPUÉS DE CADA CONTENIDO:

□ ¿Guarde el archivo original completo?
□ ¿Corté los mejores clips (3-5)?
□ ¿Saqué capturas/screenshots?
□ ¿Escribí una transcripción/resumen?
□ ¿Tengo 3+ formatos listos para usar?
□ ¿Esta en mi banco de contenido?
□ ¿Tengo fecha para publicar cada variación?
□ ¿Trackié qué formato funcionó mejor?
```

## 10. Ejercicio práctico

```
EJERCICIO: Reutilizá un contenido existente

PASO 1: Seleccioná
Elegí tu contenido más exitoso de los últimos 30 días.

PASO 2: Identificá formatos
¿Qué formatos podrías crear de ese contenido?
□ Reel/Short
□ Carrusel
□ Post estático
□ Infografía
□ Newsletter
□ Storie

PASO 3: Creá las variaciones
Creá al menos 3 variaciones diferentes.

PASO 4: Programá
Programá las variaciones para las próximas 2 semanas.

PASO 5: Medí
Después de 2 semanas, analizá:
¿Qué formato funcionó mejor?
¿Cuál tuvo menos engagement?
¿Qué aprendés para próximo contenido?

ENTREGA:
3+ variaciones publicadas
Análisis de resultados
```
