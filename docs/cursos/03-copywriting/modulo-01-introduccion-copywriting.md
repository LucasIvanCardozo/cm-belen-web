## 1. ¿Qué es copywriting?

**Copywriting** es el arte y la ciencia de escribir textos que venden, informan o generan una acción específica.

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   Copywriting ≠ Escritura creativa                  │
│   Copywriting = Escritura con PROPÓSITO             │
│                                                     │
│   Cada palabra tiene que cumplir una función:        │
│   • Captar atención                                 │
│   • Generar emoción                                 │
│   • Crear deseo                                     │
│   • Provocar una acción                             │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Ejemplo práctico:

**❌ Escribir sin copywriting:**

> "Les compartimos nuestra nueva colección de anillos. Están hermosos. Los pueden ver en nuestra tienda."

**✅ Escribir CON copywriting:**

> "Ese momento cuando encontrás el anillo que siempre soñaste ✨  
> No es solo plata. Es el recuerdo de cuando dijiste 'sí, acepto' 💍  
> Galería Luna — donde cada pieza cuenta una historia.  
> Link en bio 👆"

## 2. Copywriting vs. Otros tipos de escritura

| Tipo de escritura  | Objetivo          | Ejemplo                   |
| ------------------ | ----------------- | ------------------------- |
| **Copywriting**    | Vender, actuar    | "Comprá ahora"            |
| **Periodismo**     | Informar          | "Reportaje sobre joyería" |
| **Literatura**     | Entretener        | "Novela romántica"        |
| **Poesía**         | Emocionar         | "Soneto al oro"           |
| **CM Copywriting** | Conectar + Vender | "Ese momento cuando..."   |

## 3. ¿Por qué el copywriting es crucial para CM?

### La realidad del algoritmo en 2026

```
Instagram/TikTok en 2026:
├── Muestra contenido basado en engagement INMEDIATO
├── Los primeros 3 segundos del caption determinan si siguen leyendo
├── El engagement determina alcance orgánico
└── Los captions longos solo se leen si son atractivos

TU TRABAJO:
Hacer que alguien que scrollea SE DETENGA
Hacer que alguien que se detiene LEA TODO
Hacer que alguien que lee TODO interactúe
```

### Las estadísticas que importan

```
📊 OBSERVACIONES DE COPYWRITING PARA REDES SOCIALES

• Muchos expertos observan que la mayoría
  de los usuarios leen los captions completos
  (solo si las primeras líneas sonatractivos)

• Las marcas exitosas suelen mantener
  un copywriting consistente en sus publicaciones

• Un buen caption puede mejorar tu engagement
  sin cambiar nada del diseño

• Los captions con storytelling suelen generar
  más guardados que los informativos
```

## 4. Los 5 elementos del copywriting efectivo

### Para joyería y lujo, cada elemento es crítico:

```
1. 🎯 ATENCIÓN (Hook)
   → La primera línea tiene que detener el scroll

   "Ese momento cuando..."
   "¿Sabías que el 90% de las mujeres..."
   "No es solo una joya. Es..."

2. 💭 INTERÉS
   → Generar curiosidad o identificación

   "Cada pieza de nuestra colección..."
   "Cuando une una mujer decide..."

3. ❤️ EMOCIÓN
   → Conectar con sentimientos reales

   "Porque vos merecés..."
   "El día que la entregas..."

4. ✨ DESEO
   → Mostrar el valor, no solo el producto

   "Imaginate llevando esto..."
   "Una pieza que vas a usar siempre..."

5. 📢 ACCIÓN (CTA)
   → Decirles exactamente qué hacer

   "Escribinos ahora"
   "Guardá para después"
   "Compartí con alguien que lo necesite"
```

## 5. Copywriting para joyería: por qué es diferente

### El lujo tiene reglas especiales

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   JOYERÍA = ALTO VALOR EMOCIONAL + DECISIÓN LENTA      │
│                                                         │
│   Tu copy tiene que:                                    │
│   ✓ Hablar de emociones, no de características         │
│   ✓ Crear aspiración, no urgencia fake                 │
│   ✓ Usar palabras que denotan calidad y exclusividad    │
│   ✓ Contar historias, no solo informar                 │
│   ✓ Generar confianza, no presionar                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Vocabulario prohibido vs. recomendado

| ❌ No usar         | ✅ Mejor usar                |
| ------------------ | ---------------------------- |
| "Oferta"           | "Edición limitada"           |
| "Barato"           | "Accesible" / "A tu alcance" |
| "Comprá"           | "Hacé tuya" / "Llevátela"    |
| "Vendemos"         | "Creamos" / "Diseñamos"      |
| "No te lo pierdas" | "Es para quien sabe valorar" |
| "Solo $X"          | "Una pieza desde $X"         |

## 6. Tu primera lección como copywriter

### La mentalidad correcta:

```
MINDSET DE COPYWRITER PROFESIONAL

1. Cada palabra CUESTA
   → No escribas por llenar, escribí para comunicar

2. El lector está ocupado
   → hacé que valga su tiempo

3. No vendés productos
   → Vendés soluciones, emociones, identidades

4. Escribí como hablás
   → Tu voz authentic vale más que perfecto

5. Probá, medí, ajustá
   → Un caption nunca es definitivo
```

## 7. Ejercicio práctico: Tu primer análisis

### Actividad:

1. **Abre Instagram** en tu celular
2. **Buscá 5 cuentas de joyería** que sigas o que quieras analizar
3. **Leé los captions** de sus últimos 3 posts
4. **Analizá:**
   - ¿Cuál te hizo scrollear?
   - ¿Cuál te hizo guardar?

   - ¿Cuál te hizo comprar (o querer comprar)?

\*Engagement = (likes + comentarios + guardados) / seguidores x 100
