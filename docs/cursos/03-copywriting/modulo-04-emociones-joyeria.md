# Módulo 04: Emociones en Joyería y Lujo

## Cómo Escribir que el Corazón Compra

---

## 🎯 Objetivos de aprendizaje

Al terminar este módulo vas a:

1. **Identificar** los 6 emosi (emociones) trigger principales en marketing de lujo
2. **Reconocer** cómo cada emoción se traduce en copy específico
3. **Aplicar** vocabulario emocional en tus captions
4. **Crear** una biblioteca de emociones para tu nicho

---

## 1. La verdad incómoda sobre comprar joyería

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   LA GENTE NO COMPRA JOYAS.                            │
│                                                         │
│   Compra cómo se siente cuando la usa.                  │
│                                                         │
│   Compra recuerdos.                                      │
│   Compra identidad.                                     │
│   Compra pertenencia.                                    │
│   Compra merecerselo.                                   │
│                                                         │
│   Tu trabajo como CM es TRADUCIR                        │
│   el producto en esos sentimientos.                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Estadística clave:**

> El 85% de las decisiones de compra de lujo están basadas en emociones, no en lógica. La gente se convence a sí misma con lógica DESPUÉS de comprar por emoción.

## 3. EMOCIÓN 1: AMOR (Self-Love)

### Cuándo usarla:

```
•Productos para uno mismo
•Regalos para autocelebración
•Piezas statement
•Mensajes de empowerment
```

### Vocabulario para Amor:

| ❌ No decir           | ✅ Decir          |
| --------------------- | ----------------- |
| "Comprate algo lindo" | "Merecetelo"      |
| " gastá en vos"       | "Invertí en vos"  |
| "Ropero"              | "Tesoro personal" |
| "Vicio"               | "Autocuidado"     |
| "Compro impulsivo"    | "Me lo merecía"   |

### Ejemplo de caption - Amor:

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│ "No esperés a que alguien te regale lo que             │
│  vos misma te podés dar."                             │
│                                                         │
│ Ese pendiente que viste en la vidriera.                │
│ Ese collar que miraste 3 veces.                        │
│ Esa pulsera que te hizo sonreír cuando la probaste.    │
│                                                         │
│ No es vanidad. Es reconocimiento.                      │
│                                                         │
│ Es saber que merecés cosas bonitas.                     │
│ Y dártelas.                                            │
│                                                         │
│ Esta semana: 15% off en toda la colección               │
│ de autorregalo.                                        │
│ Porque vos también merecés una joya.                   │
│                                                         │
│ 👇 Escribinos y te ayudamos a elegir.                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 4. EMOCIÓN 2: ÉXITO

### Cuándo usarla:

```
•Logros profesionales
•Celebraciones
•Piezas que denotan achievement
•Ediciones limitadas
```

### Vocabulario para Éxito:

| ❌ No decir     | ✅ Decir                       |
| --------------- | ------------------------------ |
| "Rico"          | "Exclusivo"                    |
| "Bling bling"   | "Distinguido"                  |
| "Facha"         | "Presencia"                    |
| "Para presumir" | "Para dejar huella"            |
| "Status symbol" | "Piezas que cuentan historias" |

### Ejemplo de caption - Éxito:

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│ "No empezó con un premio.                               │
│  Empezó con una decisión."                             │
│                                                         │
│ La decisión de salir del guión.                         │
│ De hacer lo que nadie esperaba.                         │
│ De elegir el camino difícil porque sabías                │
│ que iba a llevar a algún lado.                          │
│                                                         │
│ Ese anillo que usás cuando tenés una reunión             │
│ importante no es un accesorio.                         │
│ Es un recordatorio de quién sos.                        │
│                                                         │
│ Colección Executives:                                    │
│ Piezas que se ganan el respeto                          │
│ antes de que digas una palabra.                         │
│                                                         │
│ 👇 Consultá por disponibilidad.                         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 5. EMOCIÓN 3: PERTENENCIA

### Cuándo usarla:

```
•Colecciones grupales
•Eventos especiales (bodas, egresos)
•Regalos entre amigos
•Sentido de comunidad
```

### Vocabulario para Pertenencia:

| ❌ No decir       | ✅ Decir                    |
| ----------------- | --------------------------- |
| "BFF"             | "Cercanía que se demuestra" |
| "Matching"        | "Vínculos que se lucean"    |
| "Para el grupo"   | "Para quienes importan"     |
| "Amigas forever"  | "Conexiones que duran"      |
| "Pedidas de mano" | "Momentos que se sellan"    |

### Ejemplo de caption - Pertenencia:

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│ "Hay amigas que te regalan una taza.                    │
│  Y hay amigas que te regalan una pieza                 │
│  que vas a usar durante años."                          │
│                                                         │
│ Las pulseras de la colección Vínculos                   │
│ están diseñadas para esas relaciones que no             │
│ necesitan verse todos los días para mantenerse          │
│ fuertes.                                               │
│                                                         │
│ Plata 925 con detalles que combinan.                     │
│ Pero lo más importante:                                 │
│ un mensaje que solo ustedes entienden.                  │
│                                                         │
│ Esta temporada: sets de 2 pulseras                      │
│ con grabado personalizado incluido.                      │
│ Porque los mejores regalos no se envuelven.             │
│ Se usan.                                                │
│                                                         │
│ 👇 Escribinos para consultar por tu set.                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 6. EMOCIÓN 4: LIBERTAD

### Cuándo usarla:

```
•Piezas versátiles
•Lifestyle aspiracional
•Concepto "minimalista"
•Joyas para el día a día
```

### Vocabulario para Libertad:

| ❌ No decir                | ✅ Decir                                      |
| -------------------------- | --------------------------------------------- |
| "Práctico"                 | "Sin límites"                                 |
| "Para usar todos los días" | "Piezas que van con todo"                     |
| "Simple"                   | "Esencial"                                    |
| "Cómodo"                   | "Que no te pesa, ni física ni emocionalmente" |
| "Minimalista"              | "Lo que está sin esfuerzo"                    |

### Ejemplo de caption - Libertad:

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│ "La joya que te ponés cuando no querés                  │
│  pensar en qué ponerte."                                │
│                                                         │
│ Piezas que no piden permiso.                            │
│ Que no te hacen elegir entre la oficina y el after.     │
│ Que van del café a la cena sin cambiar nada.           │
│                                                         │
│ La colección Esencial de Lunar tiene                   │
│ exactly eso: lo que necesitás.                          │
│ Ni más. Ni menos.                                       │
│                                                         │
│ Plata 925 con baño de rodio.                           │
│ Porque lo simple también merece ser forever.            │
│                                                         │
│ 👇 Ver toda la colección en el link de bio.            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 7. EMOCIÓN 5: PROTECCIÓN (Care)

### Cuándo usarla:

```
•Regalos para familia
•Joyas significativas
•Piezas con valor sentimental
•Concepto heredado
```

### Vocabulario para Protección:

| ❌ No decir       | ✅ Decir                                |
| ----------------- | --------------------------------------- |
| "Para tu mamá"    | "Para ella, que siempre cuidó de todos" |
| "Regalo útil"     | "Algo que se queda"                     |
| "Detalle"         | "Algo que marca"                        |
| "Para la familia" | "Para quienes son tu hogar"             |
| "Joyas"           | "Piezas que se heredan"                 |

### Ejemplo de caption - Protección:

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│ "Las mejores madres no se acuerdan                      │
│  de sus propios cumpleaños.                             │
│  Pero nunca olvidan el día                            │
│  que sus hijos les regalaron algo."                    │
│                                                         │
│ Por eso cuando elegís una joya para ella,               │
│ no pensés en qué se ve bonito.                         │
│ Pensá en qué se va a acordar.                          │
│                                                         │
│ La pulsera Aurora tiene un detalle especial:             │
│ una piedra que cambia de color según la luz.            │
│ Como ella, que es diferente según el momento,           │
│ pero siempre es la misma persona que te cuida.          │
│                                                         │
│ Día de la Madre: hasta 25% off en toda la              │
│ colección.                                              │
│ Porque las moms también merecen cosas que               │
│ no son para la casa.                                   │
│                                                         │
│ 👇 Consultá disponibilidad.                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 9. Cómo elegir la emoción correcta

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│ PREGUNTAS PARA ELEGIR LA EMOCIÓN:                     │
│                                                         │
│ 1. ¿Quién es mi público objetivo?                     │
│    → Soltera profesional → AMOR/ÉXITO                 │
│    → Mamás → PROTECCIÓN/PERTENENCIA                   │
│    → Parejas → AMOR/LEGADO                            │
│    → Empresarias → ÉXITO/LIBERTAD                     │
│                                                         │
│ 2. ¿Qué tipo de joya es?                             │
│    → Pieza diaria → LIBERTAD                          │
│    → Regalo especial → AMOR/PROTECCIÓN                │
│    → Alta gama → ÉXITO/LEGADO                        │
│    → Colección grupal → PERTENENCIA                  │
│                                                         │
│ 3. ¿Qué momento del año es?                          │
│    → San Valentín → AMOR                              │
│    → Día de la Madre → PROTECCIÓN                     │
│    → Navidad → PERTENENCIA/LEGADO                    │
│    → Graduaciones → ÉXITO                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 11. Tarea de práctica

### Nivel básico:

Elegí 3 productos distintos y asignales una emoción principal. Escribí un caption para cada uno enfocándote en esa emoción.

### Nivel avanzado:

Elegí 1 producto y escribí 3 versiones del mismo caption, cada una targeting una emoción diferente. Compará cuál te sale más natural.
