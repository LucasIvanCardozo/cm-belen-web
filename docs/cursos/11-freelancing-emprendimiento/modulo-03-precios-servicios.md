# Módulo 3 — Precios y Servicios

## Objetivos de Aprendizaje

Al finalizar este módulo vas a:

- ✅ Entender cómo funcionan los precios en servicios de CM
- ✅ Definir precios competitivos para tu mercado
- ✅ Crear paquetes de servicios claros
- ✅ Saber cuándo y cómo subir precios
- ✅ Negociar sin perder valor

---

## 1. Cómo pensar los precios

```
PRINCIPIO FUNDAMENTAL:
"No cobrás por tiempo. Cobrás por VALOR."

❌ "Trabajo 10 horas al mes"
✅ "Ayudo a la marca a vender más y conectarse mejor"

CUESTIONES:
• Una pyme que triplica sus ventas gracias a vos → Valor = alto
• Una marca que no vende nada por tu culpa → Valor = bajo
• Tu precio refleja el valor que generás, no las horas
```

## 2. Modelos de precio

### 2.1 Por proyecto

```
QUÉ ES:
Cobrás un monto fijo por un proyecto específico.

EJEMPLO:
"Diseño de estrategia de contenido para Instagram" = $300 USD

CUÁNDO USARLO:
✓ Sabés exactamente qué incluye
✓ El cliente sabe exactamente qué paga
✓ Proyectos definidos (lanzamiento, rebranding)

PRO:
→ Payment upfront usualmente
→ Sin sorpresas para ambos
→ Fácil de vender

CONTRA:
→ Scope creep puede comerte las ganancias
→ Si no definís bien = trabajás de más
```

### 2.2 Por mes/reTAINER

```
QUÉ ES:
Cobrás una tarifa mensual fija por servicios continuos.

EJEMPLO:
"Gestión completa de Instagram" = $400 USD/mes

CUÁNDO USARLO:
✓ Clientes que necesitan trabajo continuo
✓ Relación a largo plazo
✓ Servicios recurrentes

PRO:
→ Ingreso predecible mensual
→ Relación más profunda con cliente
→ Incentivo a que funcione bien (mantenés el cliente)

CONTRA:
→ Puede limitar tu crecimiento
→ Clientes pueden pedir más de lo contratado
```

### 2.3 Por hora

```
QUÉ ES:
Cobrás X USD por hora de trabajo.

EJEMPLO:
"Asesoría de 2 horas" = $50 USD/hora

CUÁNDO USARLO:
✓ Consultorías puntuales
✓ Trabajo indefinido
✓ Cuando no podés estimar el scope

PRO:
→ Podes cobrar por todo

CONTRA:
→ Limita tu ingreso máximo
→ Clientes pueden preferir otras opciones
→ No incentiva la eficiencia

⚠️ EVITÁ este modelo si podés.
```

### 2.4 Por resultados

```
QUÉ ES:
Cobrás basado en resultados (ventas, leads, etc.)

EJEMPLO:
"$200 fijo + 5% de ventas generadas"

CUÁNDO USARLO:
✓ Tenés track record que lo respalda
✓ Cliente está abierto a este modelo
✓ Trabajás con métricas claras

PRO:
→ Alinea incentivos tuyos con los del cliente

CONTRA:
→ Riesgo si no tenés control sobre las ventas
→ Complejo de medir
→ No todos los clientes lo aceptan
```

## 3. Cuánto cobrar

### 3.1 Rangos en LatAm (2026)

```
PARA CM JUNIOR (0-1 año experiencia):
• Por proyecto: $150-400 USD
• Por mes (básico): $150-300 USD/mes
• Por mes (intermedio): $300-500 USD/mes

PARA CM INTERMEDIO (1-3 años):
• Por proyecto: $400-800 USD
• Por mes (básico): $300-500 USD/mes
• Por mes (intermedio): $500-800 USD/mes
• Por mes (premium): $800-1200 USD/mes

PARA CM SENIOR (3+ años):
• Por proyecto: $800-2000+ USD
• Por mes: $1000-2500+ USD/mes
• Consultorías: $50-150 USD/hora

PARA ESPECIALIZACIÓN EN JOYERÍA:
+ 20-30% sobre los rangos base
(Porque el nicho es más exigente)
```

### 3.2 Cómo calcular tu precio mínimo

```
FÓRMULA:

Precio mínimo = (Gastos mensuales + Ahorro deseado) ÷ Horas lavorables

EJEMPLO:
Gastos: $500 USD/mes
Ahorro: $200 USD/mes
Horas trabajables: 80 horas/mes

Precio mínimo por hora = $700 ÷ 80 = $8.75/hora

PERO:
No cobrés por hora, cobrás por valor.
Si generás $1000 de valor, cobrás mínimo $300-500.

REGLA SIMPLE:
"Si no podés vivir de lo que cobrás, está muy bajo."
```

### 3.3 Factores que ajustan tu precio

```
SUBÍ EL PRECIO SI:

✓ Tenés especialización (joyería = +20-30%)
✓ Tenés resultados medibles (casos de estudio)
✓ El cliente es grande o premium
✓ Incluís extras (reportes, estrategias)
✓ Tenés poca disponibilidad

BAJÁ EL PRECIO SI:

✓ Empezás recién (no gratis, pero accesible)
✓ Es tu primer cliente real
✓ Tenés mucha demanda y poco tiempo
✓ El cliente es pyme muy chica

⚠️ NUNCA BAJES POR COMPETIR EN PRECIO.
Bajá si es una oportunidad estratégica.
```

## 4. Paquetes de servicios

### 4.1 Ejemplo de paquetes para CM

```
PAQUETE BRONZE — $300 USD/mes
────────────────────────────────
✓ 15 posts/mes (feed)
✓ 8 stories/semana
✓ 4 reels/mes
✓ Respuesta a comentarios y DMs (30 min/día)
✓ 1 reporte mensual de métricas
✗ Sin estrategia de contenido
✗ Sin publicidad
✗ Sin diseño de pauta

PAQUETE SILVER — $500 USD/mes
────────────────────────────────
✓ Todo lo de Bronze
✓ 20 posts/mes
✓ 12 stories/semana
✓ 8 reels/mes
✓ Estrategia de contenido mensual
✓ 2 reportes/quincena
✓ Gestión de 1 pauta/mes (hasta $100 de inversión)
✗ Sin community management profundo

PAQUETE GOLD — $900 USD/mes
────────────────────────────────
✓ Todo lo de Silver
✓ Gestión completa de redes
✓ 30 posts/mes
✓ 4 pautas/mes (hasta $300 de inversión)
✓ Community management activo
✓ 2 lives/mes
✓ Estrategia + contenido premium
✓ 4 reportes/quincena
✓ Gestión de influencers (básico)
```

### 4.2 Cómo presentar los paquetes

```
NO DIGAS:
"Tengo este precio porque..."

SÍ DECÍ:
"Este paquete incluye X, Y, Z.
Con este servicio, vas a conseguir [resultado específico]."

EJEMPLO:
"Por $500/mes, gestiono tus redes, creo contenido estratégico
y genero al menos 20% más engagement que tenés ahora.
Si no lo lográs en 3 meses, renegociamos."

EL CLIENTE NO COMPRA TIEMPO.
COMPRA RESULTADOS.
```

## 5. Cuándo y cómo subir precios

### 5.1 Señales de que está muy bajo

```
✓ Decís "SÍ" a todo porque necesitás el dinero
✓ Trabajás más de 50 horas y no te alcanza
✓ Otros CM te dicen que estás muy bajo
✓ Tenés más demanda de la que podés atender
✓ Nunca te rechazan por precio (señal de que está muy bajo)

⚠️ SI TODOS DICEN "SÍ", SUBÍ.
```

### 5.2 Cómo subir precios

```
PASO 1: Avisá con tiempo
"Desde [fecha], los precios van a ser [nuevos].
Pero para vos, mantengo [precio viejo] por 3 meses más."

PASO 2: Explicá el valor
"No es que cobro más. Es que doy más.
[Nuevo precio] incluye [nuevo servicio]."

PASO 3: Ofrecé opciones
"Podés mantener el servicio actual al precio actual
o subirte al nuevo paquete que incluye X."

PASO 4: No te disculpes
"No es que estaba muy barato. Es que ahora tengo más experiencia."
```

### 5.3 Subir precios a clientes actuales

```
REGLA DE ORO:
"NO bajés tu precio para mantener un cliente.
Si no puede pagar, es mejor que se vaya."

EJEMPLO DE COMUNICACIÓN:
"Hola [Cliente]. Quiero comentarte que a partir de [fecha],
mis precios van a ser [nuevos].
Para vos tengo dos opciones:
1) Mantenés tu paquete actual por [precio actual + 10%]
2) Pasás al paquete nuevo que incluye [X] por [nuevo precio]

Si ninguna funciona, entiendo perfectamente.
Fue un placer trabajar con vos."

RESULTADO:
→ Los que valoran tu trabajo, pagan más
→ Los que no, se van (y te liberan tiempo)
```

## 6. Negociación

### 6.1 Qué decir cuando piden descuento

```
❌ "Bueno, te lo dejo en..."
✅ "Entiendo tu presupuesto. Tenemos opciones:
   1) Bajamos el scope (menos posts, menos reels)
   2) Hacemos el servicio completo pero en 6 meses
   3) Empezamos con el paquete más chico y escalamos"

❌ "Es que estoy empezando..."
✅ "Entiendo. Por eso te ofrezco [opción adaptada].
   Mi precio refleja mi experiencia y resultados."

⚠️ NUNCA digas que tu precio es "por ser nuevo".
Tu precio es por el valor que das, no por tu experiencia.
```

### 6.2 Cuándo decir que no

```
DECÍ NO CUANDO:

• El cliente quiere trabajo gratis como "prueba"
  → "Entiendo, pero mi mejor trabajo es para clientes que pagan."

• Te piden todo por la mitad del precio
  → "No puedo hacer ese trabajo por ese precio. Así no me conviene."

• El scope es imposible
  → "Eso requiere [X] tiempo/inversión. A ese precio solo puedo hacer [Y]."

• El cliente es irrespetuoso
  → "Necesito que nos tratemos con respeto. Si no, no podemos trabajar juntos."

RECORDÁ:
Un cliente difícil no vale la pena.
Hay muchos clientes que van a valorar tu trabajo.
```

## 7. Errores comunes

### ❌ Error 1: Cobrar por hora

```
MAL:
"Cobro $10/hora. Esta semana trabajé 20 horas = $200."

PROBLEMA:
→ Tu ingreso máximo está limitado
→ Trabajás más pero ganás igual
→ No incentiva la eficiencia

CORRECTO:
"Cobro por proyecto. Este me toma 10 horas pero vale $400."
```

### ❌ Error 2: No tener paquetes definidos

```
MAL:
"¿Cuánto cobrás?" → "Mmm... depende de lo que necesites."

PROBLEMA:
→ Genera incertidumbre
→ El cliente no sabe qué está pagando
→ Te expose a que pidan cualquier cosa

CORRECTO:
"Tengo 3 paquetes: Bronze $300, Silver $500, Gold $900.
Elegí el que mejor se adapte a tu presupuesto."
```

### ❌ Error 3: Descontar por "amistad"

```
MAL:
"Ah, sos mi amiga, te lo dejo en la mitad."

PROBLEMA:
→ Desvalorizás tu trabajo
→ La amistad no paga tus facturas
→ Genera precedente para otros

CORRECTO:
"Te quiero, pero mis precios son mis precios.
¿Querés el paquete Bronze que es el más accesible?"
```

### ❌ Error 4: Aceptar cualquier cosa

```
MAL:
"Sí, sí, lo que necesites. Dale."

PROBLEMA:
→ Scope creep = trabajás de más
→ Agotamiento
→ Resentimiento hacia el cliente

CORRECTO:
"Eso no estaba en el paquete. ¿Querés agregar ese servicio?
Significa [ajuste de precio] o [reducimos algo del paquete]."
```

## 8. Checklist de precios

```
ANTES DE DEFINIR TUS PRECIOS:

□ ¿Sé cuánto necesito ganar por mes para vivir?
□ ¿Conozco los precios del mercado?
□ ¿Tengo mis servicios definidos?
□ ¿Tengo paquetes claros?
□ ¿Explico el valor, no solo el precio?
□ ¿Estoy dispuesto a decir "no" si el precio no conviene?
□ ¿Mis precios reflejan mi valor, no solo mi tiempo?
```

```
□ Gestión de redes
□ Creación de contenido
□ Diseño gráfico
□ Edición de video
□ Estrategia de contenido
□ Pauta digital
□ Respuesta a mensajes
□ Reportes y análisis
□ Community management
□ Others...

PASO 2: Definí 3 paquetes
□ BRONZE: Lo esencial ($X/mes)
□ SILVER: Lo completo ($Y/mes)
□ GOLD: Todo + extras ($Z/mes)

PASO 3: Escribí descripciones claras
"Este paquete incluye:
✓ [Servicio 1]
✓ [Servicio 2]
✓ [Servicio 3]
✗ [Lo que NO incluye]

¿Para quién es?: [Breve descripción]"

PASO 4: Definí tu precio mínimo
¿Qué es lo mínimo que cobrás?
¿Este precio cubre tus gastos + te deja vivir?

PASO 5: Testeá
Mostráselos a alguien de confianza.
¿Entienden lo que incluye cada uno?
¿El precio tiene sentido para lo que dan?

ENTREGA:
3 paquetes de servicios con precios y descripciones
```
