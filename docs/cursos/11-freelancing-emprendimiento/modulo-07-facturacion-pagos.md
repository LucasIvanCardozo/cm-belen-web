## 1. Métodos de pago en LatAm

### 1.1 Métodos por país

```
ARGENTINA:
• Mercado Pago
• Transferencia bancaria (CVU/Uala)
• Western Union
• Payoneer
• Efectivo (si es local)

MÉXICO:
• Mercado Pago
• Transferencia SPEI
• PayPal
• Western Union
• OXXO

COLOMBIA:
• Nequi
• Daviplata
• Mercado Pago
• Transferencia bancaria
• PayPal

CHILE:
• Mercado Pago
• Transferencia bancaria
• WebPay

PERÚ:
• Mercado Pago
• Transferencia bancaria
• Lukita
```

### 1.2 Recomendaciones por tipo de cliente

```
PARA CLIENTES LOCALES (mismo país):
• Transferencia bancaria directa
• Mercado Pago (con lowas o sin)
• Efectivo (si es cercano)

PARA CLIENTES INTERNACIONALES:
• PayPal (comisiones ~4%)
• Payoneer (comisiones más bajas)
• Wise (buen tipo de cambio)
• Transferencia SWIFT (caro, para montos grandes)

PARA PLATAFORMAS:
• Workana: Pago a través de plataforma
• Fiverr: Pago a través de Fiverr
• Upwork: Pago a través de Upwork (con cuestiones de seguridad)
```

## 2. Facturas

### 2.1 Por qué facturar

```
RAZONES PARA FACTURAR:

✓ LEGALIDAD
   → En la mayoría de países, es obligatorio
   → Evita problemas con impuestos

✓ PROFESIONALISMO
   → Muestra seriedad
   → El cliente toma más en serio

✓ PROTECCIÓN
   → Documenta el acuerdo
   → Prueba de pago

✓ CONTROL
   → Trackeas tus ingresos
   → Fácil para impuestos
```

```
ELEMENTOS DE FACTURA:
```

## 3. Aspectos legales básicos

### 3.1 Formalización del trabajo

```
OPCIONES DE FORMALIZACIÓN:

MONOTRIBUTISTA (Argentina):
• Facturás con CUIT propio
• Pagas monotributo mensual
• Incluya obra social y jubilación
• Simple de gestionar

REGISTRADO (Argentina):
• Facturás como autônomo o SA/SRL
• Más complejo
• Más gastos
• Mejor para ingresos altos

RÉGIMEN SIMPLificado (México):
• RFC con situación fiscal
• Factura electrónica
• Comprobante de CFDI

CONSULTÁ CON UN CONTADOR:
Un buen contador te ahorra dinero y problemas.
Invertí en consulta profesional.
```

### 3.2 Contratos simples

```
POR QUÉ CONTRATO:

✓ Define scope (qué incluí y qué no)
✓ Establece plazos de pago
✓ Protege a ambas partes
✓ Evita malentendidos

ELEMENTOS BÁSICOS:
1. Datos de ambas partes
2. Descripción del servicio
3. Frecuencia y duración
4. Precio y forma de pago
5. Plazos de pago
6. Plazos de entrega
7. Condiciones de terminación
8. Confidencialidad
9. Propiedad intelectual
10. Firmas

CONTRATO SIMPLE EJEMPLO (ver recursos):
Lo incluí en el archivo de recursos.
Adaptalo a tu situación.
```

## 4. Cómo cobrar de forma segura

### 4.1 Estrategias anti-impago

```
REGLA DE ORO:
"Si no cobrás, no trabajás."

TÉCNICAS:

1. ANTICIPO (Deposito inicial)
   → 30-50% antes de empezar
   → 50% a mitad del proyecto
   → 20% al finalizar

2. PAGO POR ADELANTADO
   → Mes completo pago anticipado
   → Trabaja solo si cobraste

3. PAGO AL FIN DE CADA MES
   → No dejés acumular más de 1 mes

4. MULTIPLICADORES DE PAGO
   → Clientes internacionales = pago anticipado
   → Nuevos clientes = depósito mayor
   → Clientes establecidos = confianza gradual
```

### 4.2 Qué hacer si no pagan

```
PASO 1: RECORDAR
"Un olvidito, no es falta de pago."
→ Mensaje amable recordatorio.

PASO 2: SEGUIMIENTO
"¿Pudiste hacer la transferencia?"
→ Mensaje más directo.

PASO 3: ÚLTIMO PLAZO
"Tengo que recibir el pago hasta el viernes.
De lo contrario, pausaré el trabajo."

PASO 4: ACCIÓN
• Suspendé servicios
• No entregues trabajo sin pago
• Considerá acciones legales (si vale la pena)

PASO 5: LEARNING
→ No trabajes con este cliente de nuevo
→ Implementá mejores prácticas
```

## 5. Manejo de precios internacionales

### 5.1 Cobrar en dólares

```
VENTAJAS:
✓ Dólares = más stable
✓ Acceso a más clientes
✓ Potencial de ganar más

DESAFÍOS:
✗ Tipo de cambio variable
✗ Comisiones de conversión
✗ Complejidad fiscal

TIPS:
• Fijá el precio en USD
• Cobrá en USD si podés
• Usá servicios como Wise para conversión
• Considerá el tipo de cambio al fijar precios
```

### 5.2 Herramientas para pagos internacionales

```
WISE (wise.com):
• Comisiones bajas
• Tipo de cambio real
• Ideal para recibir pagos internacionales

PAYONEER:
• Popular para freelancers
• Tarjeta prepaga
• Permite recibir de plataformas

PAYPAL:
• Universal pero caro (4% + FX)
• Proteção al comprador
• Todos conocen

WESTERN UNION:
• Para efectivo
• Sin cuenta bancaria
• Más caro
```

## 6. Errores comunes

### ❌ Error 1: Trabajar sin contrato

```
MAL:
"Somos amigos, no necesitamos papel."

PROBLEMA:
Sin contrato, no hay prueba de acuerdo.
Si hay problemas, no hay protección.

BIEN:
"Siempre tengo algún acuerdo por escrito,
aunque sea simple."
```

### ❌ Error 2: Dejar acumular meses de pago

```
MAL:
"El cliente es bueno, no quiero presionarlo."

PROBLEMA:
Cada mes sin pagar = más difícil cobrar.
El cliente se acostumbra a no pagar.

BIEN:
"Política clara: se cobra al final del mes.
Si hay problema, se habla ANTES del mes siguiente."
```

### ❌ Error 3: No guardar comprobantes

```
MAL:
"Transferencia hecha, listo."

PROBLEMA:
Sin comprobante, no hay prueba.
Problemas después = sin evidencia.

BIEN:
"Guardo screenshot de cada transferencia.
Lo guardo en carpeta por mes/año."
```
