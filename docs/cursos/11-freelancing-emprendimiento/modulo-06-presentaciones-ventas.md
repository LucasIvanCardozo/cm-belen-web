# Módulo 6 — Presentaciones y Ventas

## Objetivos de Aprendizaje

Al finalizar este módulo vas a:

- ✅ Entender el proceso de venta como freelancer
- ✅ Crear presentaciones profesionales
- ✅ Dominar el arte del pitch
- ✅ Manejar objeciones
- ✅ Cerrar clientes

---

## 1. El proceso de venta

```
ETAPAS DEL PROCESO:

1. PROSPECCIÓN
   → Encontrar leads
   → (Ya visto en módulos anteriores)

2. CALIFICACIÓN
   → ¿El lead es un buen cliente?
   → ¿Tiene presupuesto?
   → ¿Tiene necesidad real?

3. PRESENTACIÓN
   → Mostrar tu propuesta
   → Demostrar valor

4. MANEJO DE OBJECIONES
   → Responder dudas y miedos

5. CIERRE
   → Cerrar la venta

6. POST-VENTA
   → Entrega y relación
```

---

## 2. Calificación de leads

```
PREUNTAS DE CALIFICACIÓN:

1. "¿Cuál es tu presupuesto?"
   → Si no tiene presupuesto, no es cliente viable.

2. "¿Quién toma las decisiones?"
   → Necesitás hablar con quien decide.

3. "¿Cuál es tu urgencia?"
   → ¿Necesita para ayer o puede esperar?

4. "¿Qué esperás lograr?"
   → Entender el objetivo real.

5. "¿Ya trabajaste con CM antes?"
   → Si sí: ¿qué funcionó? ¿qué no?
   → Si no: Educación necesaria.

NO ES entrevista pero sí filtro.
```

## 3. Estructura de presentación

### 3.1 Pitch de 5 minutos

```
ESTRUCTURA DEL PITCH:

MINUTO 1: QUIÉN SOS
"Buenas, soy [nombre], CM especializada en joyería..."

MINUTO 2: QUÉ HACÉS
"Ayudo a marcas de joyería a construir
comunidades comprometidas en Instagram..."

MINUTO 3: RESULTADOS
"Resultados de mis clientes incluyen:
• Crecimiento de 200% en seguidores
• Engagement promedio de 4%..."

MINUTO 4: CÓMO FUNCIONAMOS
"Mi proceso es simple:
1. Diagnóstico de tu situación actual
2. Estrategia personalizada
3. Ejecución mensual con reportes..."

MINUTO 5: PRÓXIMOS PASOS
"¿Te parece si hacemos una llamada de 30 minutos
para hablar específicamente de tu marca?"
```

### 3.2 Presentación formal

```
ESTRUCTURA DE PRESENTACIÓN FORMAL:

SLIDE 1: TÍTULO
"[Nombre de la marca] - Propuesta de Gestión de Instagram"

SLIDE 2: DIAGNÓSTICO
• Tu situación actual
• Lo que está funcionando
• Lo que se puede mejorar

SLIDE 3: PROPUESTA
• Qué incluí
• Qué no incluí
• Por qué así

SLIDE 4: PROCESO
• Cómo vamos a trabajar
• Timeline
• Entregables

SLIDE 5: RESULTADOS ESPERADOS
• Métricas específicas
• Timeline realista

SLIDE 6: INVERSIÓN
• Precio claro
• Qué incluye
• Qué no incluye

SLIDE 7: PRÓXIMOS PASOS
• Qué sigue
• Timeline de inicio
• Qué necesito de ellos
```

## 4. Manejo de objeciones

### 4.1 Objeciones comunes

```
OBJECCIÓN 1: "Es muy caro"
RESPUESTA:
"Entiendo que puede parecer una inversión.
¿Te puedo preguntar qué esperabas gastar?
[Escuchá]
El precio refleja el tiempo y expertise
que voy a dedicar a tu marca.
Además, pagás resultados, no solo horas."

OBJECCIÓN 2: "Tengo que pensarlo"
RESPUESTA:
"Claro, es una decisión importante.
¿Qué necesitás para sentirte segura?
[Escuchá]
¿Hay algo que pueda clarificar?
¿Podemos hablar en [fecha] sobre tu decisión?"

OBJECCIÓN 3: "No tengo tiempo ahora"
RESPUESTA:
"Entiendo, el tiempo es valioso.
¿Te puedo ofrecer una opción más simple por ahora?
[Propuesta más pequeña]
Después, cuando tengas más capacidad,
escalamos a algo más completo."

OBJECCIÓN 4: "No sé si esto funciona"
RESPUESTA:
"Es una pregunta válida.
Te puedo mostrar casos de estudio de marcas similares.
¿Te gustaría ver los resultados de [marca similar]?
Mientras tanto, podemos empezar con un mes de prueba."
```

### 4.2 Técnica de empatía-escucha-estado

```
FRAMEWORK PARA CUALQUIER OBJECIÓN:

1. EMPATÍA
   "Entiendo cómo te sentís..."
   "Tiene sentido que pienses eso..."

2. ESCUCHA
   "¿Podrías contarme más sobre eso?"
   "¿Qué te hace pensar eso?"

3. ESTADO
   "Lo que pasa es que..."
   "Lo que realmente necesitas es..."

EJEMPLO COMPLETO:

CLIENTE: "Es muy caro para mi negocio."

EMP: "Entiendo completamente que el precio
es una consideración importante."

ESC: "¿Me podrías contar un poco más
sobre tu situación actual?"

CLIENTE: "Tengo un presupuesto muy limitado
para marketing este año."

EST: "Lo que necesitás es ver un retorno claro
de la inversión. Es totalmente razonable.
Por eso mi propuesta incluye métricas específicas
para que puedas medir exactamente qué devuelve
la inversión en redes sociales."

CIERRE: "¿Te parece si ajustamos la propuesta
para que se ajuste mejor a tu presupuesto
manteniendo los resultados clave?"
```

## 5. Técnicas de cierre

### 5.1 Cierre directo

```
CUÁNDO USARLO:
Cuando el cliente está interesado y listo.

EJEMPLO:
"¿Te parece si arrancamos el primero de mes?
Necesito que me confirmes hoy para preparar
el contrato y el onboarding."

USAR CUANDO:
✓ El cliente mostró interés genuino
✓ Las objeciones fueron resueltas
✓ Está en timing adecuado
```

### 5.2 Cierre por urgencia

```
CUÁNDO USARLO:
Cuando hay una razón legítima de urgencia.

EJEMPLO:
"Tengo disponibilidad para empezar con un cliente
nuevo solo este mes. Después agendo para mayo.
¿Querés asegurar tu lugar?"

USAR CUANDO:
✓ Realmente tenés limited availability
⚠️ NO FINGIR urgencia. Solo usarla si es real.
```

### 5.3 Cierre por alternativas

```
CUÁNDO USARLO:
Cuando el cliente no se decide.

EJEMPLO:
"¿Te parece si arrancamos con el paquete básico
por un mes, y si ves resultados, escalamos
al paquete completo?"

OFRECÉ:
✓ Opción más simple
✓ Trial/reto
✓ Escalamiento gradual
```

## 6. Errores comunes

### ❌ Error 1: Vender sin escuchar

```
MAL:
"Este es mi servicio, esto incluye, esto cuesta."

PROBLEMA:
No entendiste qué necesita el cliente.

BIEN:
"Contame más sobre tu situación.
[Escuchá]
Basándome en lo que me contás,
esta es mi propuesta..."
```

### ❌ Error 2: No establecer próximo paso

```
MAL:
"Bueno, cualquier cosa me decís."

PROBLEMA:
No hay compromiso. El cliente no hace nada.

BIEN:
"¿Te parece si hablamos el jueves a las 3pm
para avanzar con la propuesta?"
```

### ❌ Error 3: Vender a cualquiera

```
MAL:
"Cualquier cliente es bueno, necesito el dinero."

PROBLEMA:
Clientes malos = stress, no pago, burnout.

BIEN:
"Este cliente no es el indicado para mí.
Prefiero esperar al cliente que encaje mejor."
```

## 7. Ejercicio práctico

```
EJERCICIO: Prepará tu pitch y presentación

PASO 1: Tu pitch de 5 minutos
Escribí tu pitch usando la estructura:
1. Quién sos (1 min)
2. Qué hacés (1 min)
3. Resultados (1 min)
4. Proceso (1 min)
5. Próximos pasos (1 min)

PASO 2: Creá tu presentación
Usá la estructura de slides de arriba.
Incluí:
□ Diagnóstico
□ Propuesta
□ Proceso
□ Resultados esperados
□ Inversión
□ Próximos pasos

PASO 3: Prepará respuestas a objeciones
Escribí tu respuesta a:
□ "Es muy caro"
□ "Tengo que pensarlo"
□ "No tengo tiempo ahora"

PASO 4: Practicá
□ Grabá tu pitch
□ Escuchalo
□ Ajustá
□ Practicá de nuevo

PASO 5: Role-play
Pedile a alguien que te haga de cliente difícil.
Practicá manejar objeciones.

ENTREGA:
• Pitch escrito
• Presentación creada
• Respuestas a objeciones listas
```
