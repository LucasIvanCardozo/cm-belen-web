# Módulo 5 — Cómo Buscar Trabajo

## Objetivos de Aprendizaje

Al finalizar este módulo vas a:
- ✅ Dominar las plataformas freelance
- ✅ Crear propuestas ganadoras
- ✅ Entender qué buscan los clientes
- ✅ Aplicar a trabajos de forma estratégica
- ✅ Construir tu portfolio desde cero

---

## 1. Plataformas freelance en detalle

### 1.1 Workana

```
WORKANA (workana.com):
• Principal plataforma LatAm
• Proyectos de todo tipo
• Sistema de propuestas

CÓMO FUNCIONA:
1. Creás perfil 100% completo
2. Buscás proyectos de CM/Social Media
3. Enviás propuesta con propuesta
4. El cliente elige a quién contratar
5. Trabajás y cobrás a través de la plataforma

TIPS PARA DESTACAR:
✓ Perfil completo con foto profesional
✓ Portfolio con ejemplos (pueden ser ficticios)
✓ Propuestas personalizadas
✓ Respuesta rápida (< 2 horas)
```

### 1.2 Fiverr

```
FIVERR (fiverr.com):
• Gigs (servicios predefinidos)
• Clientes de todo el mundo
• Buenos para empezar

CÓMO FUNCIONA:
1. Creás "gigs" (servicios)
2. Esperás que clientes te contacten
3. O aplicás a "requests"
4. Cobrás cuando completás el trabajo

GIGS PARA CM:
• "I'll manage your jewelry Instagram account"
• "I'll create 20 Instagram posts for your brand"
• "I'll write engaging captions for your jewelry brand"
```

### 1.3 Upwork

```
UPWORK (upwork.com):
• Proyectos más grandes
• Freelancers con historial
• Más competitivo

CÓMO FUNCIONA:
1. Tenés que pasar pruebas
2. Completás perfil con portfolio
3. Aplicás a proyectos
4. Esperás a que te seleccionen
5. Cobrás después de aprobación

NOTA: Upwork es más competido.
Necesitás buenas reviews para destacar.
```

## 2. Cómo escribir propuestas ganadoras

### 2.1 Estructura de propuesta

```
ESTRUCTURA DE PROPUESTA GANADORA:

1. SALUDO PERSONAL
   "Hola [nombre],"

2. REFERENCIA AL PROYECTO
   "Vi que necesitás ayuda con [especificar].
   Es exactamente lo que hago."

3. ENTENDIMIENTO DEL PROBLEMA
   "Entiendo que [problema específico del cliente]."

4. TU SOLUCIÓN
   "Propongo [tu solución específica]."

5. POR QUÉ VOS
   "Tengo experiencia en [ejemplo relevante]."

6. RESULTADOS ESPERADOS
   "En [tiempo], vas a ver [resultado específico]."

7. CIERRE + PREGUNTA
   "¿Te parece si conversamos?
   Quedo atenta."

8. SALUDO
   "Saludos,
   [Tu nombre]"
```

### 2.2 Ejemplo de propuesta

```
ASUNTO: Propuesta para gestión de Instagram de tu joyería

"Hola María,

Vi que necesitás ayuda para gestionar tu cuenta de Instagram.
Es exactamente lo que hago como CM especializada en joyería.

Entiendo que con el día a día de la tienda,
no te queda tiempo para dedicarle a las redes sociales,
y eso está afectando tu presencia online.

Propongo:
• Gestión completa de tu Instagram (comentarios, DMs, menciones)
• 20 posts/mes con diseño profesional en Canva
• 10 reels/mes con edición en CapCut
• Captions optimizados para tu público objetivo

Tengo 2 años de experiencia gestionando
cuentas de joyería en Argentina,
con resultados como: [mencionar algo específico si tenés]

En el primer mes, ya deberías ver:
• Respuesta más rápida a mensajes
• Contenido más consistente
• Engagement aumentando

¿Te parece si agendamos una llamada de 15 minutos
para hablar sobre tu marca?

Quedo atenta,
Tu nombre
[Portfolio link]"
```

### 2.3 Tips para propuestas

```
✓ PERSONALIZÁ CADA PROPUESTA
  Mencioná algo específico del cliente.

✓ NO COPIES Y PEGUES
  Se nota. El cliente lo nota.

✓ SEGUI ESTRUCTURA
  Una propuesta desorganizada = freelancer desorganizado.

✓ MOSTRÁ CONFIANZA
  "Puedo ayudarte" > "Espero que pueda ayudarte"

✓ NO OFRECÉS DEMASIADO
  Scope creep es peligroso.

✓ PRECIO JUSTO
  Ni muy bajo (devalúa) ni muy alto (sin razón).
```

## 3. Qué buscan los clientes

### 3.1 Lo que más valoran

```
SEGÚN ESTUDIOS (2026):

1. COMUNICACIÓN (85%)
   → Respondes rápido
   → Explicás bien
   → Mantenés informado

2. RESULTADOS (80%)
   → Pueden medir el trabajo
   → Ven mejoras reales
   → Tienen métricas

3. PROFESIONALISMO (75%)
   → Cumplís plazos
   → Hacés lo que decís
   → Sos confiable

4. EXPERTISE (70%)
   → Sabés de lo que hablás
   → Entiendes el nicho
   → Tenés experiencia

5. FLEXIBILIDAD (65%)
   → Te adaptás a cambios
   → Trabajás con su ritmo
   → Sos solucionador de problemas
```

### 3.2 Lo que más rechazan

```
SEGÚN ESTUDIOS:

1. NO COBRAN LO QUE VALEN (90%)
   → "Te pago $50" → NO

2. NO TIENEN PORTFOLIO (85%)
   → Sin ejemplos = sin prueba

3. NO RESPONDEN RÁPIDO (80%)
   → >24 horas = desinteresado

4. PROPUESTAS GENÉRICAS (75%)
   → "Puedo hacer de todo" → No inspira confianza

5. MALAS COMUNICACIONES (70%)
   → Errores, descortesía, desorganización
```

## 4. Construir portfolio desde cero

### 4.1 Portfolio sin experiencia

```
OPCIONES PARA EMPEZAR:

OPCIÓN 1: PORTFOLIO FICTICIO
• Creá cuentas ficticias de joyería
• Diseñá contenido como si fuera real
• Usá fotos de stock (o las tuyas)
• Presentalo como "Caso de estudio"

EJEMPLO:
"Para una marca ficticia de joyería artesanal,
diseñé una estrategia de contenido que incluía:
• 20 posts/mes
• Calendario editorial
• Captions optimizados
[Ver ejemplos]"

OPCIÓN 2: CUENTA PROPIA
• Usá tu cuenta personal o creá una
• Aplicá todo lo que sabés
• Mostrá el proceso y resultados
• Incluí métricas aunque sean pequeñas

OPCIÓN 3: TRABAJO FREE/BARTER
• Ofrecé servicios gratis a cambio de testimonial
• Trabajá para una amiga con joyería
• Barter con una marca pequeña

OPCIÓN 4: TRABAJO VOLUNTARIO
• ONG necesita redes
• Pequeño negocio local
• Causa que te importe
```

### 4.2 Presentación de portfolio

```
FORMATOS DE PORTFOLIO:

1. PDF PRESENTABLE
   → Diseño profesional
   → 5-10 casos de estudio
   → Métricas y resultados
   → Contacto claro

2. LINKEDIN
   → Sección de Portfolio en perfil
   → Posts destacados
   → Recomendaciones

3. INSTAGRAM
   → Grid con trabajos
   → Highlights por cliente
   → Stories mostrando proceso

4. CANVA
   → Plantilla de portfolio
   → Fácil de actualizar
   → Compartible como PDF/link
```

### 4.3 Caso de estudio ejemplo

```
CASO DE ESTUDIO:

# Joyería "Luz de Luna" - Caso Ficticio

## Resumen
"Joya Luna necesitaba una presencia digital
profesional para competir con marcas más grandes."

## Desafío
• Sin estrategia de contenido
• Posts esporádicos
• 500 seguidores sin engagement

## Solución
• Calendario editorial semanal
• Contenido visual consistente
• Captions con storytelling
• Reels de producto

## Resultado (3 meses)
• Seguidores: 500 → 2,300
• Engagement: 1% → 4.5%
• Mensajes: +40% consultas
• Ventas atribuidas a IG: $1500 USD

## Mi rol
Gestión completa de Instagram.
"""

## 5. Errores comunes

### ❌ Error 1: Aplicar a todo

```
MAL:
"Aplico a 50 trabajos por día sin leer bien."

PROBLEMA:
• Propuestas genéricas = rechazo seguro
• Pérdida de tiempo
• Frustración

BIEN:
"Aplico a 5 trabajos bien pensados por día.
Cada propuesta es personalizada."
```

### ❌ Error 2: Cobrar muy bajo para conseguir trabajo

```
MAL:
"Voy a cobrar $50 para conseguir mis primeras reviews."

PROBLEMA:
• Atraés clientes que no valoran el trabajo
• Difícil subir precios después
• Desgaste emocional

BIEN:
"Cobro precios justos desde el principio.
Prefiero esperar al cliente correcto."
```

### ❌ Error 3: Sin portfolio pero aplicando igual

```
MAL:
"No tengo portfolio pero voy a aplicar igual."

PROBLEMA:
• No hay nada que ver
• El cliente tiene opciones con portfolio

BIEN:
"Primero construí portfolio (aunque ficticio).
Después aplicá a trabajos."
```
