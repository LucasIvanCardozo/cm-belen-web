## 1. Qué son los Pilares de Contenido

### Definición simple

```
PILARES DE CONTENIDO = Las 5-7 CATEGORÍAS principales
de contenido que tu marca va a publicar de forma regular.

Son los "pilares" sobre los que se sostiene toda tu estrategia.

EJEMPLO PARA JOYERÍA:

┌─────────────────────────────────────────────────┐
│ PILARES DE CONTENIDO                            │
├─────────────────────────────────────────────────┤
│ 1. Producto (las joyas en sí)                  │
│ 2. Lifestyle (cómo usar las joyas)             │
│ 3. Educativo (información sobre joyería)        │
│ 4. Behind the scenes (proceso, equipo)         │
│ 5. Storytelling (historias de marca)           │
└─────────────────────────────────────────────────┘
```

### Por qué importan

```
SIN PILARES: El contenido es RANDOM.
→ Publicás lo que se te ocurre
→ No hay coherencia
→ La audiencia no sabe qué esperar

CON PILARES: El contenido es ESTRATÉGICO.
→ Sabés exactamente qué publicar cada día
→ Mantenés variedad sin perder foco
→ La audiencia sabe qué esperar de vos
→ El algoritmo detecta coherencia
```

## 2. Los 5 Pilares Esenciales para Joyería

### 2.1 Pilar Producto (35%)

```
QUÉ ES: Fotos y videos de tus joyas en sí mismas.

OBJETIVO: Mostrar el producto de forma atractiva y detallada.

TIPOS DE CONTENIDO:
• Foto de producto solo (estudio/fondo limpio)
• Foto lifestyle (joya en contexto)
• Detalle/close-up (textura, piedras, craftsmanship)
• Colección completa (varias joyas juntas)
• Comparativa (antes/después, oro vs plata)
• Producto en movimiento (video, ASMR)

EJEMPLO DE POSTS:
📷 "Este collar tiene un mecanismo secreto..."
📹 Reel mostrando cómo se cierra el dije
📷 Colección completa para San Valentín
📷 Close-up del engraving en el inside

¿POR QUÉ 35%? Porque sos CM, vendés productos.
Pero no todo es producto: también necesitás conexión.
```

### 2.2 Pilar Lifestyle (25%)

```
QUÉ ES: Las joyas en contexto, cómo se usan, con quién.

OBJETIVO: Conectar emocionalmente, mostrar aplicación real.

TIPOS DE CONTENIDO:
• Outfit completo con la joya
• Joya en diferentes situaciones (trabajo, cena, playa)
• "Cómo usar esto para cada ocasión"
• Joya con ropa específica
• Moodboards con la joya como protagonista
• Celebraciones (cumpleaños, anniversary, graduación)

EJEMPLO DE POSTS:
📷 "De oficina a cena: este collar hace todo"
📹 Reel: 3 formas de usar el mismo collar
📷 Joya en mesa de restaurant
📷 Outfit de gala con pendientes statement
📷 Jewelry para tu wedding day

¿POR QUÉ 25%? Porque la gente compra el LIFESTYLE,
no solo la joya. Necesitan imaginarse USANDO eso.
```

### 2.3 Pilar Educativo (20%)

```
QUÉ ES: Información que ayuda a la audiencia a entender más.

OBJETIVO: Posicionarte como experta, dar valor real.

TIPOS DE CONTENIDO:
• "Cómo cuidar tu joyería"
• "Diferencia entre oro 14k y 18k"
• "Cómo saber tu talla de anillo"
• "Guía de piedras: significado y propiedades"
• "Cómo combinar joyas sin que se vea overloaded"
• "Significado de los gems"
• "Historia de los materials"

EJEMPLO DE POSTS:
📷 Carousel: "5 ошибок que hacés al limpiar tus joyas"
📹 Reel: "Así se hace el gold plating"
📷 Infografía: Tipos de cierres de collar
📹 Video: Comparando plata 925 vs chapada en oro
📷 Carousel: "Guía de tamaños de anillo" (con mano como referencia)

¿POR QUÉ 20%? Este contenido se GUARDA mucho.
Y los saves = señal fuerte para el algoritmo.
```

### 2.4 Pilar Behind the Scenes (10%)

```
QUÉ ES: El proceso, el equipo, el día a día.

OBJETIVO: Humanizar la marca, crear conexión personal.

TIPOS DE CONTENIDO:
• Proceso de creación de las joyas
• El atelier/taller
• Conociendo al equipo
• Preparando pedidos
• Packaging en acción
• El packaging como producto (unboxing)
• Días de fotos (cómo trabajamos)
• Errors y aprendizajes

EJEMPLO DE POSTS:
📹 Reel: "Así hacemos el packaging de este pedido"
📷 Nuestro taller cuando no hay clientes
📹 Time-lapse de un pedido siendo preparado
📷 El momento en que descubrimos esta nueva pieza
📹 "Un día en la vida de nuestra joyera"

¿POR QUÉ 10%? Es suficiente para humanizar.
No saturés con demasiado BTS o pierde valor especial.
```

### 2.5 Pilar Storytelling (10%)

```
QUÉ ES: Historias sobre la marca, los clientes, el origen.

OBJETIVO: Crear conexión emocional profunda.

TIPOS DE CONTENIDO:
• Historia de origen de la marca
• "Por qué hacemos lo que hacemos"
• Testimonios de clientas (con permiso)
• "La historia detrás de esta pieza"
• Valores de marca
• Comunidad (clientes usando las joyas)
• Colaboraciones especiales

EJEMPLO DE POSTS:
📷 "Esta pieza existió porque una clienta nos pidió..."
📹 Reel: "La historia de nuestro nombre"
📷 Post de testimonio de clienta satisfecha
📷 "3 generaciones de joyeras en mi familia"
📹 "Este es el por qué de todo esto"

¿POR QUÉ 10%? Este contenido es PODEROSO pero no diario.
Publicá historias cuando haya algo REAL que contar.
No inventés por inventar.
```

## 4. Cómo crear contenido para cada pilar

### 4.1 Para Producto

```
TIPS PARA FOTOS DE PRODUCTO:

📷 SIEMPRE:
✓ Buena iluminación (natural o artificial suave)
✓ Fondo limpio o coherente con tu estética
✓ Múltiples ángulos
✓ Detalle de cierre/mecanismo
✓ Foto de la pieza sola

📹 PARA REELS DE PRODUCTO:
✓ 15-30 segundos
✓ Muestrá la joya en movimiento
✓ Detalle del brillo
✓ El momento de "descubrimiento"
✓ Cómo se ve en alguien (antes/después)

📝 CAPTION PARA PRODUCTO:
→ No solo "hermoso collar"
→ Contá algo: el material, el proceso, para quién es
→ Incluye detalles que no se ven (material, medida)
```

### 4.2 Para Lifestyle

```
TIPS PARA CONTENIDO LIFESTYLE:

📷 SIEMPRE:
✓ La joya como protagonista
✓ Outfit coherente (no competition)
✓ Situación real (no demasiado staged)
✓ Calidad de imagen alta

📹 PARA REELS LIFESTYLE:
✓ "A day with this jewelry"
✓ 3 ways to style one piece
✓ Before and after adding the jewelry
✓ Context clues (mesa de restaurant, oficina, playa)

📝 CAPTION PARA LIFESTYLE:
→ Contexto: para qué ocasión es ideal
→ Tips de styling
→ Invitación a compartir situaciones propias
```

### 4.3 Para Educativo

```
TIPS PARA CONTENIDO EDUCATIVO:

📷 PARA CAROUSELS:
✓ 3-5 slides máximo
✓ Información clara y digerible
✓ Diseño limpio (no clutter)
✓ Fuentes legibles
✓ Cada slide = 1 idea

📹 PARA REELS EDUCATIVOS:
✓ 20-60 segundos
✓ "Did you know?" openings
✓ Quick tips
✓ Facts that surprise
✓ Tutorials

📝 CAPTION PARA EDUCATIVO:
→ Introducción al tema
→ Resumen del contenido
→ CTA para guardar/comentar
→ Pregunta para generar interacción
```

### 4.4 Para Behind the Scenes

```
TIPS PARA BTS:

📷 PARA FOTOS:
✓ No tiene que ser perfecto (de hecho, mejor si no lo es)
✓ Autenticidad > producción
✓ Detalles del proceso
✓ El equipo sonriendo
✓ El espacio de trabajo

📹 PARA REELS BTS:
✓ Time-lapses funcionan muy bien
✓ "Lo que no sabés de..." openings
✓ Proceso paso a paso
✓ Packaging en acción
✓ Preparando para un envío especial

📝 CAPTION PARA BTS:
→ Historia detrás de lo que muestro
→ "Esto tomó X horas"
→ Agradecimiento al equipo/clientes
```

### 4.5 Para Storytelling

```
TIPS PARA STORYTELLING:

📷 PARA FOTOS:
✓ Fotos reales (no stock)
✓ Momento capturado, no posed
✓ Emoción genuina
✓ Conexión visual con la audiencia

📹 PARA REELS STORYTELLING:
✓ El narrative es la estrella
✓ "Storytime" openings
✓ "Let me tell you about..." openings
✓ Emociones, no solo facts

📝 CAPTION PARA STORYTELLING:
→ Escribe como si estuvieras contando a una amiga
→ No vendas, CONECTÁ
→ Dejá que la historia hable por sí sola
→ Incluye pregunta/reflexión al final
```

## 7. Errores Comunes

### ❌ ERROR 1: Solo producto

```
PROBLEMA: 90% producto, 10% resto.

POR QUÉ FALLA:
→ La audiencia se aburre
→ No hay conexión emocional
→ No hay razón para guardar/comentar
→ Parece catálogo, no marca

SOLUCIÓN:
→ Audita tu contenido actual
→ Calculá qué % es cada pilar
→ Ajustá gradualmente
```

### ❌ ERROR 2: Demasiado educativo (aburrido)

```
PROBLEMA: 100% educativo, parece clase.

POR QUÉ FALLA:
→ Nadie sigue una cuenta de joyería para aprender
→ La gente quiere ENAMORARSE de las joyas
→ Lo educativo es ADD-ON, no el main event

SOLUCIÓN:
→ Balanceá: 1 educativo cada 3-4 posts de otro tipo
→ Hacé el contenido educativo VISUALMENTE atractivo
→ Usá reels para educativos (más engaging)
```

### ❌ ERROR 3: Perder la identidad

```
PROBLEMA: Pilares que no tienen nada que ver con joyería.

EJEMPLO:
→ Post de comida random
→ Foto de vacaciones sin joyería
→ Meme que no conecta con la marca

SOLUCIÓN:
→ Cada post, sin importar el pilar, debe connectar
   con joyería de alguna forma
→ Si no tiene que ver, no lo publiques
```

## 8. Cómo Inventar Contenido para cada Pilar

```
CUANDO NO SABÉS QUÉ PUBLICAR:

PARA PRODUCTO:
□ Nueva pieza (si tenés stock)
□ Pieza existente con nueva foto
□ Detalle específico (cierre, engraved, piedra)
□ Colección completa
□ Pieza "hero" (favorita de la semana)

PARA LIFESTYLE:
□ "Para qué ocasión es ideal esta pieza"
□ "Combinala con X outfit"
□ foto en X lugar (oficina, restaurant, casa)
□ "De día vs de noche"
□ "Apto para X actividad"

PARA EDUCATIVO:
□ Guía de cuidado
□ Significado de piedras/materiales
□ Cómo elegir la talla correcta
□ Diferencia entre tipos
□ Tips de styling
□ Datos curiosos

PARA BTS:
□ Proceso de packaging
□ Preparando pedidos
□ El espacio de trabajo
□ El equipo (si hay)
□ Un día en el taller
□ Recibido feedback de clienta (en packaging)

PARA STORYTELLING:
□ Historia de la marca
□ Por qué de una pieza especial
□ Testimonio (con permiso)
□ Valor de marca que querés compartir
□ Cliente usando la pieza (con permiso)
□ Colaboración especial
```
