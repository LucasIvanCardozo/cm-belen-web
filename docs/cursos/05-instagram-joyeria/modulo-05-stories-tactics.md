# Módulo 5 — Stories que Convierten para Joyería

## Objetivos de Aprendizaje

Al finalizar este módulo vas a:
- ✅ Entender cómo funcionan las Stories en 2026
- ✅ Dominar los stickers interactivos
- ✅ Crear sequences de Stories efectivas
- ✅ Usar Stories para construir comunidad
- ✅ Convertir viewers de Stories en clientes

---

## 1. Por qué las Stories Importan para Joyería

```
STORIES vs FEED:

STORIES:
✓ Más frecuentes (hasta 10-20 al día)
✓ Más informales, auténticas
✓ Permite interacción directa
✓ Vencen en 24 horas (urgencia)
✓ Funcionan para mantener comunidad activa

PARA JOYERÍA:

📱 Las Stories permiten:
→ Mostrar el día a día de la marca
→ Crear interacción constante
→ Dar "sneak peeks" de productos nuevos
→ Construir relación personal con audiencia
→ Vender de forma no agresiva
```

## 2. Los 8 Tipos de Stories para Joyería

### 2.1 Story de Producto

```
QUÉ ES: Mostrar un producto en Stories.

OBJETIVO: Dar a conocer productos de forma casual.

TÁCTICA:
📱 Foto/video del producto
📝 "Nuevo en la tienda..."
📝 "Este collar llegó hoy"
📱 Swipe up / Link sticker para más info

EJEMPLO:
Slide 1: 📷 Foto del collar nuevo
Slide 2: 📝 "Este llegó hoy y es ❤️"
Slide 3: 📝 "El detalle del dije es insane"
Slide 4: 📱 Link para ver más

TIPS:
✓ Usa good lighting
✓ Muestra el producto en contexto
✓ Include price if comfortable
✓ Don't oversell in every story
```

### 2.2 Story Interactiva (Poll/Quiz)

```
QUÉ ES: Usar stickers para generar interacción.

OBJETIVOS:
✓ Saber preferencias de tu audiencia
✓ Generar engagement
✓ Entretener
✓ Data sobre qué quieren

STICKERS INTERACTIVOS:

📊 POLL: "¿Plata u Oro?"
📝 QUESTION: "¿Qué preferís: collar o pendiente?"
🎵 QUIZ: "¿Qué piedra es esta?" (con imagen)
👍slider: "¿Cuánto vale este collar?" (1-10 scale)

EJEMPLO DE SEQUENCE:
Slide 1: 📷 "Quick question..."
Slide 2: 📊 Poll sticker: "Plata ⚪ u Oro 🟡"
Slide 3: 📝 "Mañana les muestro qué prefieren"
Slide 4: 📊 Resultados del poll
Slide 5: 📷 Producto baseado en el resultado

TIPS:
✓ Preguntá cosas relevantes para tu negocio
✓ Usá los resultados en tu estrategia
✓ Follow up con contenido basado en polls
```

### 2.3 Story Detrás de Cámaras (BTS)

```
QUÉ ES: Mostrar el proceso, no solo el resultado.

OBJETIVOS:
✓ Humanizar la marca
✓ Mostrar autenticidad
✓ Crear conexión

TIPO DE CONTENIDO:
📹 Preparando pedidos
📹 Empacando con cuidado
📹 El espacio de trabajo
📹 Arrancando el día
📹 El momento de descobrir algo nuevo

EJEMPLO DE SEQUENCE:
Slide 1: 📝 "Así arrancamos los lunes"
Slide 2: 📹 Video del espacio de trabajo
Slide 3: 📝 "Primero: buena luz ☀️"
Slide 4: 📹 Abriendo un pedido nuevo
Slide 5: 📝 "El packaging está listo"
Slide 6: 📹 Mostrando el resultado final

TIPS:
✓ No tiene que ser perfecto
✓ Lo raw/real funciona
✓ Behind the scenes = authentic connection
✓ 3-5 slides por sequence
```

### 2.4 Story de Valor/Educativo

```
QUÉ ES: Compartir conocimiento en formato Stories.

OBJETIVOS:
✓ Posicionarte como experta
✓ Dar valor real
✓ Hacer saves de tu content

FORMATO:
📱 Infografía simple
📝 "Tip del día..."
📱 Antes/después
📹 Quick tutorial

EJEMPLO DE SEQUENCE:
Slide 1: 📝 "Tip rápido sobre jewelry care"
Slide 2: 📷 Imagen del tip #1
Slide 3: 📝 "Tip #2"
Slide 4: 📷 Imagen del tip #2
Slide 5: 📝 "Guardá para después!"
Slide 6: 📱 CTA para seguir aprendiendo

TIPS:
✓ Keep it simple and visual
✓ Use text strategically
✓ End with save/remember reminder
✓ Connect to your products naturally
```

### 2.5 Story de Countdown

```
QUÉ ES: Crear anticipación para algo.

USOS:
📅 Lanzamiento de producto nuevo
📅 Venta especial
📅 Evento en tienda
📅 Restock de producto popular

TÁCTICA:
🔢 Countdown sticker (set days/hours)
📝 "Se viene algo especial..."
📷 Preview sutil
📝 "¿Ya adivinan qué es?"
📅 Reveal con Countdown sticker

EJEMPLO:
Slide 1: 📝 "Algo especial se viene..."
Slide 2: 🔢 Countdown sticker: 3 days
Slide 3: 📷 Silueta del producto (no se ve bien)
Slide 4: 📝 "Hint: Este finde se viene algo"
Slide 5: 📝 "¿Qué será? Comenta 👇"

TIPS:
✓ Don't reveal too much too soon
✓ Build anticipation with hints
✓ Share the reveal in real time
✓ Convert urgency to action
```

### 2.6 Story de Testimonio

```
QUÉ ES: Compartir feedback de clientas.

OBJETIVOS:
✓ Social proof
✓ Generate trust
✓ Show real results

TÁCTICA:
📸 Screenshot del mensaje/DM
📸 Story de la clienta (con permiso)
📝 Su nombre
📝 Su feedback
📱 Tag a la clienta (si okay)

EJEMPLO:
Slide 1: 📝 "Me llegó este mensaje y tenía que compartirlo 💛"
Slide 2: 📸 Screenshot del DM con review
Slide 3: 📝 "Gracias por confiar en nosotros"
Slide 4: 📝 "¿Ya conocés [producto]?"
Slide 5: 📱 Link a reviews

TIPS:
✓ Always get permission first
✓ Focus on specific feedback
✓ Don't show private info
✓ Combine with product mention
```

### 2.7 Story de Q&A / AMA

```
QUÉ ES: Responder preguntas de tu audiencia.

OBJETIVOS:
✓ Connect directly with followers
✓ Understand their needs
✓ Show expertise

TÁCTICA:
📝 "Preguntame lo que quieras"
❓ Question sticker
📝 Answer each question in a separate slide
📱 Use responses as content later

EJEMPLO:
Slide 1: 📝 "Preguntame lo que quieras sobre jewelry! 💎"
Slide 2: ❓ Question sticker: "What do you want to know?"
Slide 3: 📝 "Q: ¿Cómo saber si es plata real?"
Slide 4: 📹 Video answering the question
Slide 5: 📝 "Más tips como este? Follow! 👆"

TIPS:
✓ Post the Q&A prompt 24h before
✓ Answer in video, not just text
✓ Save best Q&A for feed content
✓ Follow up with related content
```

### 2.8 Story de Colaboración

```
QUÉ ES: Contenido compartido con otros.

USOS:
🤝 Con influencers
🤝 Con otras marcas
🤝 Con clientas
🤝 Con artesanos/colaboradores

TÁCTICA:
📸 Repost from collaborator
📝 "Thanks [name] for showing our jewelry"
📸 Your content featuring them
📱 Cross-promotion opportunity

EJEMPLO:
Slide 1: 📝 "La [@influencer] luciendo [tu producto] 🔥"
Slide 2: 📸 Screenshot/Repost de su Story
Slide 3: 📝 "Gracias por confiar!"
Slide 4: 📸 Tu versión del producto
Slide 5: 📱 "Visitá [@lacuenta] para más"

TIPS:
✓ Always credit the creator
✓ Make sure image quality is good
✓ Don't over-promote - natural integration
✓ Use collaboration to reach new audiences
```

## 3. Cómo estructurar tu día de Stories

```
JORNADA TÍPICA DE STORIES:

🌅 MAÑANA (opcional)
├── 1-2 slides: "Buenos días" / how you're doing
└── 1 slide: Motivation or quote

📦 MEDIODÍA (cuando tengas contenido)
├── 1-2 slides: Behind the scenes
├── 1-2 slides: Product showcase
└── 1-2 slides: BTS de algún pedido

📊 TARDE (interactive)
├── Poll or question sticker
├── 1-3 slides respondiendo a engagement
└── 1 slide: Preview de contenido del día

🌙 NOCHE (opcional)
├── 1 slide: Resumen del día
└── 1 slide: "Mañana se viene..."

NOTA: No necesitas hacer TODOS estos.
3-5 sequences de 3-5 slides está perfect.
```

## 4. Stickers Interactivos y Cómo Usarlos

```
STICKERS QUE IMPORTAN PARA JOYERÍA:

📊 POLL STICKER
Uso: "¿Plata u Oro?" "¿Anillo o collar?"
Mejor para: Quick preferences, opinions

❓ QUESTION BOX
Uso: "¿Qué pregunta tenés sobre jewelry?"
Mejor para: Q&A, gather ideas

🔢 SLIDER
Uso: "¿Cuánto vale?" [1-10 scale]
Mejor para: Engagement, fun interaction

📍 LOCATION TAG
Uso: "Estamos en [ciudad]"
Mejor para: Local reach, store visits

👤 MENTION
Uso: "@usuario"
Mejor para: Shoutouts, collaborations

📱 LINK STICKER
Uso: "Link in bio" o Swipe up
Mejor para: Traffic to website/shop
⚠️ Necesitás 10k+ followers o cuenta verificada

🎵 MUSIC
Uso: Add song to story
Mejor para: Mood, entertainment

📅 COUNTDOWN
Uso: "3 días para el lanzamiento"
Mejor para: Anticipation, events
```

## 5. Métricas de Stories que Importan

```
QUÉ MEDIR EN STORIES:

MÁS IMPORTANTE:
✓ Reach: Cuánta gente vio tus stories
✓ Replies: Cuánta gente respondió
✓ DMs: Cuántos mensajes directos generó
✓ Link clicks: Si tenés link sticker

MENOS IMPORTANTE:
• Impressions (veces que se mostró)
• Views (casi todos ven al menos 1 slide)

BENCHMARKS PARA JOYERÍA:
• Story reach: >30% de tus followers
• Replies: >2-3% de viewers
• Saves from stories: >1% de viewers

PATRUCONSEGUIDOS:
→ Más interaction = algoritmo muestra más
→ Más reach = más chances de convertir
```

## 7. Errores Comunes en Stories

### ❌ ERROR 1: Stories sin interacción

```
PROBLEMA: Solo publicás sin pedir nada a cambio.

SOLUCIÓN:
✓ Agregá siempre algún sticker interactivo
✓ Pedí opiniones
✓ Hacé preguntas
✓ Ask for input
```

### ❌ ERROR 2: Demasiadas Stories

```
PROBLEMA: Publicás 20 stories por día sin filtro.

SOLUCIÓN:
✓ 5-10 stories por día es suficiente
✓ Cada slide debe aportar algo
✓ Más no siempre es mejor
✓ La calidad > cantidad
```

### ❌ ERROR 3: Stories muy "salesy"

```
PROBLEMA: Cada story es "comprá ahora".

SOLUCIÓN:
✓ 80% de contenido de valor, 20% venta
✓ El contenido de valor genera confianza
✓ La venta llega natural después
✓ Las personas odian ser vendidas
```

### ❌ ERROR 4: Ignorar responses

```
PROBLEMA: La gente responde y no respondés.

SOLUCIÓN:
✓ Respondé TODOS los mensajes de stories
✓ Usá respuestas en futuras stories
✓ Make followers feel heard
✓ DMs from stories = hot leads
```
