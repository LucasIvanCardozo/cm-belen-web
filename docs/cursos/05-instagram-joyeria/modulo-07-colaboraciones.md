## 1. Tipos de Colaboraciones para Joyería

```
PARA JOYERÍA, LAS COLABORACIONES SON CLAVE PORQUE:

✓ Expanden tu alcance a nuevas audiencias
✓ Generan credibility social
✓ Traen contenido gratis
✓ Crean buzz alrededor de tu marca
✓ Pueden resultar en ventas directas
```

### Tipos principales

```
1. GIVEAWAYS CONJUNTOS
   → Con otras marcas complementarias
   → ej: Joyería + Florería + Restaurante

2. PARTNERSHIPS CON OTRAS CUENTAS
   → Posts cruzados
   → Stories compartidas
   → Conteúdo colaboración

3. INFLUENCER MARKETING
   → Micro-influencers (1k-50k)
   → Macro-influencers (50k+)
   →.client ambassadors

4. COLABORACIONES CON MARCAS
   → Brands no competidoras
   → Cross-promociones
   → Eventos conjuntos

5. AMBASSADORS
   → Programa formal de embajadores
   → Descuentos a cambio de contenido
```

## 2. Estrategia de Giveaways

### 2.1 Por qué funcionan los giveaways

```
BENEFICIOS DE GIVEAWAYS:

📈 SEGUIDORES:
• +500-2000 seguidores en 1 semana
• Crecimiento rápido de audiencia

💬 ENGAGEMENT:
•comments y shares masivos
• Activity en el algoritmo

👀 AWARENESS:
• Más personas conocen tu marca
• Visibility en el feed de otros

📸 UGC:
• Contenido generado por participantes
• Fotos con tu producto

⚠️ NOTA:
Los giveaways son una INVERSIÓN.
No son gratis: necesitás regalar producto o servicio.
Hacélos strategic, no por hacer.
```

### 2.2 Cómo estructurar un giveaway efectivo

```
ESTRUCTURA DE GIVEAWAY:

PARTE 1: PARTNERS (otros participantes)
├── Elegí 2-3 marcas complementarias
├── No competidores
├── Público similar
└── Aporte similar (todos regalan algo)

EJEMPLO PARA JOYERÍA:
• Joyería (tu marca)
• Florería local
• Café/restaurant
• Photo studio

PARTE 2: REGLAS DE PARTICIPACIÓN
├── Follow todas las cuentas participantes
├── Like el post
├── Comment con tag (1-3 personas)
└── Share/Guardar (opcional)

⚠️ NO PIDAS:
✗ Que te sigan para participar
✗ Que compartan en stories obligatoriamente
✗ Información personal excesiva

PARTE 3: DURACIÓN
├── Tiempo ideal: 5-7 días
├── Menos = poco alcance
├── Más = pierde momentum
└── Anunciá el winner dentro de 48h después

PARTE 4: PRIZES
├── Valor total: $50-500 USD
├── Cada marca pone algo
├── Prizes deben ser relevantes
└── El grand prize puede ser multi-marca
```

## 3. Partnerships con Otras Cuentas

### 3.1 Cómo encontrar partners

```
CRITERIOS PARA ELEGIR PARTNERS:

✓ audiencias similares (no necesariamente del mismo tamaño)
✓ marca complementaria (no competidora)
✓ engagement real (no seguidores comprados)
✓ valores alineados con tu marca
✓ cuentas activas y profesionales
✓ potencial para relación a largo plazo

EJEMPLOS DE PARTNERS PARA JOYERÍA:
• Florerías y regalos
• Boutiques de ropa
• Salones de belleza
• Estudios de fotografía
• Restaurantes/Cafés
• Bodegas/Productos gourmet
• Hoteles boutique
• Wedding planners
• Spas
```

### 3.2 Tipos de partnership

```
TIPO 1: POST COMPARTIDO
→ Creán un post juntos
→ Ambos publican
→ Cada uno tagging al otro

EJEMPLO:
@JoyeriaLujo x @BoutiqueElegante
"Un look completo para la fiesta de fin de año ✨
Collar de @JoyeriaLujo + Vestido de @BoutiqueElegante
¿Qué te parece el combo?"

TIPO 2: STORIES SHARING
→ Compartís stories de tu partner
→ Tu partner comparte tus stories
→ Más frecuente, menos esfuerzo

TIPO 3: LIVE/JELLY/CHAT
→ Live conjunto donde hablan de un tema
→ ej: "Cómo armar un look para San Valentín"
→ Más engagement, más personal

TIPO 4: CONTENIDO CREADO JUNTO
→ Creán contenido específico para la colaboración
→ fotoshoot conjunto
→ video reel juntos
→ Vale la pena si hay buen fit
```

### 3.3 Cómo contactar para partnerships

```
MENSAJE DE PRIMER CONTACTO:

Plantilla (DM o email):

"Hola [Nombre]!

Te escribo porque soy CM de [@TuJoyeria]
y admire tu cuenta @[CuentaPartner] desde hace tiempo.

Me parece que nuestras marcas tienen mucho en común:
[semejanza que notaste].

¿Estarías abierta a explorar una colaboración?
Pensé en [idea específica: giveaway/post conjunto/etc.].

No hay presión, solo quería proponerlo.
¿Tomamos un café virtual parachar?

¡Abrazo!
[Tu nombre]"

⚠️ TIPS:
✓ Personalizá cada mensaje
✓ Muestra que conocés su marca
✓ Sé específica con la propuesta
✓ No pidas demasiado de entrada
✓ Estás pidiendo, no exigiendo
```

## 4. Micro-Influencer Marketing

### 4.1 Por qué micro-influencers

```
MICRO-INFLUENCERS: 1,000 - 50,000 seguidores

¿POR QUÉ SON MEJORES QUE MACRO?

✓ MÁS engagement rate (3-8% vs 1-3%)
✓ Más auténticos y creíbles
✓ Costo accesible
✓ Más selectivos (mejor match)
✓ Audience más comprometido
✓ Easier to build relationships

PARA JOYERÍA ESPECÍFICAMENTE:
• Joyería es producto personal
• La persona que la usa importa mucho
• Pequeños influencers con buen engagement > grandes sin engagement
```

### 4.2 Cómo encontrar micro-influencers

```
DÓNDE BUSCAR:

1. HASHTAGS DE TU NICHO
   #joyeríaartesanal #joyasfinas #joyasartesanales
   #platayingenieria #joyaslocales

2. UBICACIÓN
   Si atendés localmente: buscá influencers de tu ciudad
   #joyeríaBsAs #joyasColombia

3. SEGUIDORES DE TUS COMPETIDORES
   Personas que siguen a otras joyerías similares

4. CLIENTES EXISTENTES
   Tus clientas que ya tienen seguidores
   pueden ser micro-influencers sin saberlo

5. HERRAMIENTAS
   • Heepsy (gratis)
   • Identify (gratis)
   • Modash (prueba gratis)
```

### 4.3 Criterios de evaluación

```
ANTES DE CONTACTAR, EVALUÁ:

ENGAGEMENT RATE:
→ Comments ÷ Followers = %
→ >3% es bueno para micro
→ >5% es excelente

AUDIENCE QUALITY:
→ ¿Los comentarios son genuinos?
→ ¿Los followers parecen reales?
→ ¿Hay actividad en stories?

CONTENT QUALITY:
→ Las fotos/videos son buenos?
→ Hay coherencia estética?
→ El tono de la cuenta es profesional?

AUDIENCE RELEVANCE:
→ ¿Son personas que podrían usar joyería?
→ ¿El demographico matchea con tu cliente ideal?
→ ¿Están en tu zona geográfica (si aplica)?

AUTHENTICITY:
→ ¿Parece genuina o muy "vendida"?
→ ¿Responde a comentarios?
→ ¿Interactúa con otros?
```

### 4.4 Modelos de compensación

```
OPCIONES:

1. PRODUCTO GRATIS (para micro-influencers pequeños)
→ Envías producto sin costo
→ Esperás contenido a cambio
→ Funciona con cuentas <5k

2. DESCUENTO + PRODUCTO
→ Descuento en tu catálogo
→ Pueden comprar para probar
→ Contenido a cambio

3. FEE POR CONTENIDO
→ Pagás por post específico
→ Range: $20-200 USD para micro
→ Depende del alcance y engagement

4. AMBASSADOR PROGRAM
→ Descuento permanente a cambio de contenido regular
→ ej: 20% off para siempre + 2 posts/mes

5. REVENUE SHARE
→ Comisión por ventas generadas
→ ej: 10% de las ventas con tu código
→ Menos común pero posible
```

## 5. Errores de Colaboraciones

### ❌ ERROR 1: Dar sin recibir

```
PROBLEMA: Colaboraciones desbalanceadas.

SOLUCIÓN:
✓ Siempre negociá reciprocidad
✓ Si das producto, esperás contenido
✓ Si hacés post, partner debe hacer lo mismo
✓ Todo por escrito antes de empezar
```

### ❌ ERROR 2: Partners con engagement fake

```
PROBLEMA: Colaborás con alguien que tiene seguidores comprados.

SOLUCIÓN:
✓ Always audit engagement antes de commit
✓ Si comments son spam = red flag
✓ Revisá followers/following ratio
✓ Mejor partner con 5% engagement que 500k seguidores fake
```

### ❌ ERROR 3: No formalizar el acuerdo

```
PROBLEMA: "Hicimos un trato" sin definir nada.

SOLUCIÓN:
✓ Todo por escrito (DM sirve si queda registro)
✓ Definí: qué, cuándo, cómo, cuánto
✓ Tiempo de entrega
✓ Permisos de uso de contenido
✓ Expectativas claras
```

### ❌ ERROR 4: Exceso de giveaways

```
PROBLEMA: Estás siempre haciendo giveaways.

SOLUCIÓN:
✓ Máximo 1 giveaway grande por trimestre
✓ Más = devalúa tu marca
✓ Alterná con contenido orgánico
✓ Los giveaways son boost, no estrategia base
```
