## 1. Qué es el Algoritmo de Instagram

### Definición simple

```
El algoritmo de Instagram es un conjunto de sistemas automatizados
que decidEN, en milisegundos, qué contenido le muestra a cada
usuario en su feed, Explore y Reels.

NO es una sola cosa. Son MUCHOS algoritmos trabajando juntos.

Cada sección de Instagram (Feed, Stories, Explore, Reels, Search)
tiene su propio algoritmo optimizado para diferentes objetivos.
```

### Por qué importa para joyería

```
PARA JOYERÍA, ENTENDER EL ALGORITMO ES CRUCIAL PORQUE:

1. El alcance orgánico es limitado pero posible
   → Saber cómo funciona maximiza tus posibilidades

2. Joyería es nicho competitivo
   → Necesitás que tu contenido llegue a las personas correctas

3. Las decisiones de compra en joyería son emocionales
   → Más alcance = más oportunidades de conectar

4. El algoritmo recompensa lo que genera conversación
   → En joyería, hay MUCHA historia para contar
```

## 2. Los 6 Factores Clave del Algoritmo en 2026

### 2.1 Interés (Interest)

```
QUÉ ES: Qué tanto le interesa a alguien tu contenido
basado en su historial de interacción.

CÓMO FUNCIONA:
• Si un usuario interactúa mucho con contenido de joyería,
  el algoritmo le muestra MÁS contenido de joyería
• Si interactúa con tus posts (guarda, comparte, comenta),
  te considera "relevante" para él

PARA JOYERÍA:
✓ Publicá contenido de nicho específico
✗ No mezcles joyería con contenido random sin relación
✓ Usá hashtags y palabras clave relacionadas a joyería
```

### 2.2 Información del Post

```
QUÉ ES: Señales sobre el contenido en sí mismo.

INCLUYE:
• Cantidad de likes, comentarios, shares, guardados
• Tiempo que la gente pasa en tu post
• Fecha de publicación
• Duración (para videos)
• Frecuencia de publicación del creador

PARA JOYERÍA:
✓ Reels de 15-30 segundos son los que más alcance tienen
✓ Posts que la gente guarda (tutorials, gift guides)
✓ Publicaciones en horarios pico de tu audiencia
✓ Carousels con información valuable (guías, comparativas)
```

### 2.3 Información del Creador

```
QUÉ ES: Datos sobre QUIÉN publica.

INCLUYE:
• Interacciones previas que el usuario tuvo con vos
• Frecuencia de publicación
• Seguimiento de seguimientos
• Cuentas que el usuario sigue relacionadas

PARA JOYERÍA:
✓ Mantené consistencia en publicar
✓ Interactuá con tu audiencia (respondé comentarios, DMs)
✓ Construí relación con seguidores (no solo publiques)
✓ Colaborá con otras cuentas del nicho
```

### 2.4 Actividad del Usuario

```
QUÉ ES: Cómo el usuario se comporta en Instagram.

INCLUYE:
• Posts que liked, commented, shared, saved
• Accounts followed recently
• Searches realizadas
• Tiempo spent en diferentes contenidos

PARA JOYERÍA:
✓ El contenido que genera acción = más alcance
✓ Guardar es LA SEÑAL más fuerte para el algoritmo
✓ Compartir a stories o DM = segunda señal más fuerte
✓ Los comentarios extensos son mejores que "hermoso"
```

### 2.5 Historial de Interacciones

```
QUÉ ES: Relación entre el usuario y el creador.

INCLUYE:
• Mensajes directos exchange
• Comments en posts del otro
• Reactions a stories
• Profile visits

PARA JOYERÍA:
✓ Respondé TODOS los mensajes y comentarios
✓ Usá Stories para generar interacción (polls, questions)
✓ Creá contenido que invite a comentar
✓ DMs personalizados a clientes reales
```

### 2.6 Signals de IA (2026 Update)

```
QUÉ ES: Meta AI analiza patrones de comportamiento.

INCLUYE:
• Análisis de imágenes con IA
• Reconocimiento de texto en imágenes
• Contextual understanding
• Content categorization

PARA JOYERÍA:
✓ Usá texto en las imágenes (para IA entienda)
✓ Incluí descripciones ALT TEXT en fotos
✓ Escribí captions descriptivos y ricos en keywords
✓ Categorizá tu contenido claramente
```

## 3. Cómo funciona en cada sección

### 3.1 Feed

```
PRIORIDADES DEL FEED:
1. Posts de cuentas con las que interactuás mucho
2. Posts que el algoritmo predice que te van a gustar
3. Posts de cuentas seguidas que no viste
4. Posts recomendados (ads o suggested)

PARA JOYERÍA:
→ Necesitás que la gente interactúe con vos ANTES
→ Comentá en cuentas de tu nicho
→ Interactuá genuinamente con tu audiencia
→ Tus mejores seguidores deben sentir que "TE CONOCEN"
```

### 3.2 Reels

```
PRIORIDADES DE REELS:
1. Videos que mantienen atención (replays, shares)
2. Videos que generan save y share
3. Contenido de cuentas con alto engagement
4. Videos originales (NO repost de otros)
5. Videos con audio popular/trending

PARA JOYERÍA:
✓ Reels de 15-30 segundos con información valuable
✓ Tutoriales de styling con joyería
✓ "Antes y después" de transforms
✓ Detalles de producto en video (el brillo, la textura)
✓ Tendencias de audio adaptadas a joyería
✗ NO publiques videos borrosos o de baja calidad
✗ NO uses contenido de otros sin permiso
```

### 3.3 Explore

```
PARA EXPLORE:
• Tu contenido necesita ser RELEVANTE y ORIGINAL
• Necesita generar ALTO engagement rápido
• Los saves son la señal más importante para Explore

PARA JOYERÍA:
✓ Gift guides estacionales
✓ "Cómo elegir la joya perfecta para..."
✓ Comparativas de materiales
✓ Contenido educativo sobre piedras
✓ "Top 10" o "Best of" posts
```

### 3.4 Stories

```
PARA STORIES:
• Interacción > Vistas
• DMs desde Stories > Swipe away
• Poll results, Q&A responses count

PARA JOYERÍA:
✓ Usa stickers interactivos (polls, questions, quizzes)
✓ Behind the scenes
✓ "This or That" sobre estilos
✓ Countdown para lanzamientos
✓ Exprésate de forma auténtica (el algoritmo premia esto)
```

## 4. Mitos vs Realidades

### ❌ MITO 1: "Publicar mucho hace que te shadowbanneen"

```
REALIDAD:
No hay evidencia de que publicar mucho cause shadowban.
Lo que SÍ puede causar problemas es:
• Contenido spam (megustacomenta sígueme)
• Hashtags prohibidos o truncados
• Violaciones repetidas de copyright
• Bot activity (comentarios automático)

PARA JOYERÍA:
→ Publicá con propósito, no por publicar
→ Si tenés mucho que mostrar, usá carousels
→ La calidad siempre > cantidad
```

### ❌ MITO 2: "Los hashtags ya no sirven"

```
REALIDAD:
Los hashtags siguen sirviendo, pero DIFERENTE.
• Menos hashtags genéricos (ahora funcionan los específicos)
• Hashtags de nicho > hashtags populares
• La cantidad bajó de relevancia (5-10 es suficiente)
• Los hashtags ayudan a CATEGORIZAR tu contenido

PARA JOYERÍA:
✓ Usa hashtags específicos de joyería (no solo #jewelry)
✓ Hashtags locales para clientes locales
✓ Hashtags de campaña específicos
✗ No uses los mismos 30 hashtags en todos los posts
```

### ❌ MITO 3: "El algoritmo esconde tu contenido si no pagás"

```
REALIDAD:
El algoritmo orgánico sigue funcionando.
El pago (ads) AMPLIFICA, no corrige.
Contenido sin engagement no se amplifica con ads.

PARA JOYERÍA:
→ Primero construí presencia orgánica
→ Después usá ads para AMPLIAR lo que ya funciona
→ Ads no hacen magia con contenido que no conecta
```

### ❌ MITO 4: "Solo los reels tienen alcance"

```
REALIDAD:
TODOS los formatos tienen alcance potencial.
• Reels: máximo alcance pero más competencia
• Carousels: alto engagement, guardados
• Static posts: menos alcance pero más durability
• Stories: mantenimiento de comunidad

PARA JOYERÍA:
✓ Usa TODOS los formatos estratégicamente
✓ Reels para alcance y descubrimiento
✓ Carousels para educación y valor
✓ Stories para comunidad y conversación
✓ Static posts para grid y referencia
```

## 5. Señales que el algoritmo AMAR (2026)

```
TOP 5 SEÑALES QUE EL ALGORITMO RECOMPENSA:

1. 💾 GUARDADO (Save)
   > La señal más fuerte de valor
   > Incluye: guardar post, agregar a colección

2. 📤 COMPARTIR (Share)
   > Enviar a stories o por DM
   > Significa: "esto vale la pena compartir"

3. 💬 COMENTARIO (Comment)
   > Más largo = más señales
   > +3 palabras = cuenta más que "lindo"

4. 👀 TIEMPO EN POST
   > Para videos: replay, tiempo de visualización
   > Para carousels: cuántas fotos viste

5. ❤️ LIKE (Like)
   > La señal más básica pero sigue importando
   > Combinada con otras señales es poderosa
```

## 6. Estrategia Práctica para Joyería

### 6.1 Para aumentar Guardados

```
TÁCTICAS QUE FUNCIONAN:

📌 "Gift Guides" para fechas especiales
📌 "How to style" posts
📌 "Complete the look" carousels
📌 "Care guide" para joyería
📌 "Compare" posts (oro vs plata, etc.)
📌 "Investment pieces" lists
📌 "Jewelry for..." posts (ocasiones, outfits)

EJEMPLO DE POST PARA GENERAR GUARDADOS:

"Jewelry para tu primera entrevista de trabajo 💼

Guardá este post porque:

1. Collar fino + blazer = poder sin exagerar
2. Aros pequeños + pelo recogido = atención al rostro
3. Anillo simple + manos sobre la mesa = confianza

Guardá para cuando la necesites 👆"
```

### 6.2 Para aumentar Shares

```
TÁCTICAS QUE FUNCIONAN:

🎁 "Regálale esto si..."
💝 "Si tu jewelry pudiera hablar..."
📊 "Datos que no sabías sobre..."
🎯 "Cómo elegir el regalo perfecto"
✨ "POV: Cuando encontrás la joya ideal"
💬 "Hot take: [opinión controversial sobre joyería]"

EJEMPLO:

"Hot take: Los aros de aro son el accessory
más subestimado de la historia.

Comment 'aro' si estás de acuerdo 👇

#jewelry #accesorios"
```

### 6.3 Para aumentar Comentarios

```
TÁCTICAS QUE FUNCIONAN:

❓ Preguntas específicas en captions
🎭 "This or That" posts
📊 Encuestas en posts ( "¿Plata u Oro?" )
💭 "Comenta tu opinión sobre..."
🔄 "Tag a alguien que necesita ver esto"
🎁 "Comentá X para participar del giveaway"

EJEMPLO:

"Mi suegra me dijo que mi anillo de plata
'es demasiado simple'.

¿Ustedes qué opinan?
Comment 1 si simple > elaborado
Comment 2 si elaborado > simple

Las respuestas van a ser interesantes 👀"
```

## 7. Errores que HINCHAN al algoritmo (en tu contra)

### ❌ ERROR 1: Contenido de baja calidad

```
PROBLEMA: Videos borrosos, fotos pixeladas, texto ilegible

EL ALGORITMO LO PENALIZA: Sí, pero más importante,
los USUARIOS lo penalizan (y el algoritmo detecta eso)

SOLUCIÓN:
✓ Invertí en buena iluminación
✓ Usá trípode para videos
✓ Revisá cada post antes de publicar
✓ editá antes de subir
✓ CapCut > Instagram native editor para reels
```

### ❌ ERROR 2: Engagement con bots

```
PROBLEMA: Comprar followers, usar bots de comentarios

EL ALGORITMO LO PENALIZA: MUCHO. Detecta patrones artificiales.

CONSECUENCIA: Tu cuenta puede ser penalizada o shadowbanned

SOLUCIÓN:
✓ Nunca compres seguidores
✓ Comentarios genuinos, no genéricos
✓ No uses apps de "auto-follow/unfollow"
✓ Crecer orgánico lleva tiempo pero es sostenible
```

### ❌ ERROR 3: Hashtags prohibidos

```
PROBLEMA: Usar hashtags como #beautyblog, #follow4follow

EL ALGORITMO LO PENALIZA: Sí. Algunos hashtags
están marcados como spam.

CONSECUENCIA: Tu post no se muestra a nadie

SOLUCIÓN:
✓ Antes de usar un hashtag, buscálo
✓ Si aparece " hashtags hidden" = no usarlo
✓ Usá herramientas como "Hashtag Expert" para validar
✓ Mezclá hashtags grandes con nicho específico
```

### ❌ ERROR 4: Publicar sin estrategia

```
PROBLEMA: Publicar random, sin consistencia, sin propósito

EL ALGORITMO LO PENALIZA: Indirectamente
(porque no generás engagement)

CONSECUENCIA: Alcance limitado, audiencia confundida

SOLUCIÓN:
✓ Tené un calendario editorial
✓ Seguí pilares de contenido
✓ Creá contenido con intención
✓ Analizá qué funciona y repetilo
```
