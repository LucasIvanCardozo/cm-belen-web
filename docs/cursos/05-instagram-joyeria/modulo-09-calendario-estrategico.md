# Módulo 9 — Calendario Estratégico para Joyería

## Objetivos de Aprendizaje

Al finalizar este módulo vas a:

- ✅ Planificar un mes completo de contenido
- ✅ Integrar todos los pilares de contenido
- ✅ Sincronizar con fechas especiales y campañas
- ✅ Crear un sistema sostenible de publicación
- ✅ Implementar el calendario en tu flujo de trabajo

---

## 1. Por qué un Calendario es Esencial

```
SIN CALENDARIO:
❌ Publicás random
❌ Olvidás fechas importantes
❌ No hay coherencia
❌ Falta de contenido en momentos clave
❌ Estrés constante de "qué publico hoy"

CON CALENDARIO:
✅ Sabés exactamente qué publicar
✅ Estás preparado para campañas
✅ Mantenés consistencia
✅ Tenés tiempo para crear contenido
✅ Reduces stress
✅ Mejor calidad porque hay planificación
```

## 2. Estructura del Calendario

### 2.1 Frecuencia de publicación

```
PARA JOYERÍA, RECOMENDAMOS:

MÍNIMO SOSTENIBLE:
• 1 post diario (7/semana)
• 3-5 stories por día
• 3-4 reels por semana

ÓPTIMO:
• 2-3 posts por día (14-21/semana)
• 5-10 stories por día
• 7+ reels por semana

⚠️ IMPORTANTE:
La consistencia > cantidad.
Mejor 1 post por día todos los días
que 5 posts un día y 0 la siguiente semana.

PARA EMPEZAR:
→ 1 post + 1 reel por día
→ Stories según disponibilidad
→ Gradualmente aumentar
```

### 2.2 Distribución semanal sugerida

```
EJEMPLO DE SEMANA (1 post + 1 reel):

LUNES:
• Mañana: Post (Educativo)
• Tarde: Reel (Producto)

MARTES:
• Mañana: Post (Lifestyle)
• Tarde: Stories interactivas

MIÉRCOLES:
• Mañana: Post (Producto)
• Tarde: Reel (Educativo)

JUEVES:
• Mañana: Post (BTS)
• Tarde: Stories

VIERNES:
• Mañana: Post (Lifestyle)
• Tarde: Reel (Producto/Trending)

SÁBADO:
• Mañana: Post (Storytelling)
• Tarde: Reel (Behind scenes)

DOMINGO:
• Mañana: Post (Educativo/Lifestyle)
• Tarde: Stories reflexivas

NOTA: Ajustá según tu audiencia y capacidad
```

## 4. Cómo Planificar por Semanas

### 4.1 Proceso semanal

```
CADA DOMINGO (o viernes tarde):

PASO 1: REVISÁ EL MES
• Calendario general del mes
• Qué fechas importantes vienen
• Qué campaigns hay

PASO 2: DEFINÍ TEMAS DE LA SEMANA
• Semana 1: Colección nueva
• Semana 2: Cuidado de joyas
• Semana 3: Behind the scenes
• Semana 4: Testimonios

PASO 3: ASIGNÁ DÍAS
┌─────────┬────────────────┬────────────────┐
│ DÍA     │ POST           │ REEL           │
├─────────┼────────────────┼────────────────┤
│ Lunes   │ Pilar: ______  │ Tipo: ______    │
│ Martes  │ Pilar: ______  │ Tipo: ______    │
│ ...     │ ...            │ ...             │
└─────────┴────────────────┴────────────────┘

PASO 4: CREÁ/CONSEGUÍ CONTENIDO
• Brief de lo que necesitás
• Fotografía nueva
• Edición
• Caption preparado

PASO 5: PROGRAMÁ EN META BUSINESS SUITE
• Agendá posts
• Prepará reels
• Tené todo listo
```

## 5. Sistema de Contenido

### 5.1 Batch Creation

```
QUÉ ES: Crear contenido en batches (lotes)

BENEFICIOS:
✓ Más eficiente
✓ Mejor calidad
✓ Menos estrés
✓ Consistencia garantizada

CÓMO HACERLO:

BATCH SEMANAL (2-3 horas):
• Lunes o domingo
• Crear todos los posts de la semana
• Editar todos los reels
• Preparar captions
• Programar en Meta Business Suite

BATCH MENSUAL (medio día):
• Una vez al mes
• Planificar todo el mes
• batch de fotoshoot si necesario
• Crear contenido evergreen

FLOW DE BATCH SEMANAL:
1. Hora 1: Fotos para la semana
2. Hora 2: Edición de fotos
3. Hora 3: Reels (grabar + editar)
4. Hora 4: Captions + programación
```

### 5.2 Contenido Evergreen vs Temporal

```
CONTENIDO EVERGREEN (siempre relevante):
• Cómo cuidar tus joyas
• Guía de materiales
• Cómo elegir la talla correcta
• Tips de styling
• Historia de la marca

→ Este contenido SE REUTILIZA
→ Lo podés republicar cada few meses
→ Actualizá levemente y listo

CONTENIDO TEMPORAL (fecha específica):
• Campaña de San Valentín
• Lanzamiento de colección nueva
• Tendencias de temporada
• Fecha especial

→ Solo para ese momento
→ No reutilizás tal cual
→ Pero podés adaptar la idea

RATIO SUGERIDO:
• 70% evergreen
• 30% temporal/específico
```

## 6. Herramientas de Calendario

### 6.1 Meta Business Suite (Gratis)

```
PROGRAMACIÓN EN META BUSINESS SUITE:

PASO 1: Creá el post
   → Photo/Video/Reel
   → Escribí caption
   → Agregá hashtags

PASO 2: Tocá "Programar"
   → Elegí fecha y hora
   → Confirmá

PASO 3: Repetí para cada post
   → Podés agendar hasta 75 posts

BENEFICIOS:
✓ Gratis
✓ Integrado con IG
✓ Análisis después de publicar
✓ Scheduling automático
```

### 6.2 Later (Gratis/Pago)

```
LATER:

✓ Visual planner de grid
✓ Scheduling multi-platform
✓ Análisis básico
✓ Link in bio tracking

PRECIO: Gratis hasta 1 perfil social
```

## 8. Errores de Planificación

### ❌ ERROR 1: Planificar muy poco o nada

```
PROBLEMA: "Ya veré qué público cuando me levante"

SOLUCIÓN:
✓ Planificá al menos 1 semana ahead
✓ Mejor aún: 2-4 semanas
✓ Lo ideal: todo el mes

```

### ❌ ERROR 2: Planificar contenido que no podés crear

```
PROBLEMA: "Voy a publicar 10 reels esta semana" pero no tenés tiempo.

SOLUCIÓN:
✓ Se honesto con tu capacidad
✓ Empezá con lo que podés sostener
✓ Aumentá gradualmente
```

### ❌ ERROR 3: No adaptarse a lo que pasa

```
PROBLEMA: "Tengo el calendario pero surgió algo y no lo uso"

SOLUCIÓN:
✓ El calendario es guía, no cárcel
✓ Ajustá cuando haya necesidad
✓ No abandones todo por una emergencia
✓ Periódicamente revisá y ajustá
```

### ❌ ERROR 4: No guardar contenido para después

```
PROBLEMA: "Usé todas las fotos y ahora no tengo nada"

SOLUCIÓN:
✓ Siempre tené banco de contenido
✓ Nunca publiques todo de una vez
✓ Reutilizá contenido evergreen
✓ Batch shoots para stock
```
