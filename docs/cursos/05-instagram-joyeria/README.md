## 📚 Descripción

Domina Instagram desde adentro, entendiendo el algoritmo, creando estética de lujo y construyendo una comunidad que compra. Este curso es **específicamente diseñado para marcas de joyería y lujo**.

**Después de este curso vas a poder:**

- ✅ Entender cómo funciona el algoritmo de Instagram
- ✅ Crear una estética de grid que impacte
- ✅ Planificar pilares de contenido estratégico
- ✅ Dominar Reels para joyería
- ✅ Usar Stories para conectar y vender
- ✅ Generar engagement real
- ✅ Medir y optimizar tu estrategia
