# Curso 06: Analítica y Métricas para CM

## Por qué este curso

```
Sin datos, estás ciega. Con datos, estás empoderada.

Como CM, tu trabajo no termina cuando publicás.
Empieza ahí: analizar qué funcionó, qué no, y cómo mejorar.
```

Este curso te enseña a transformar números en decisiones inteligentes.

---

## Qué vas a aprender

```
Al finalizar este curso vas a:

✓ Entender qué métricas importan y por qué
✓ Leer e interpretar Meta Business Suite
✓ Calcular engagement rate, reach, y otras métricas clave
✓ Crear reportes profesionales para clientes
✓ Definir KPIs específicos para joyería y lujo
✓ Usar datos para optimizar tu estrategia
✓ Tomar decisiones basadas en evidencia, no en intuición
```

---

## Estructura del curso

```
MÓDULOS:

1. Introducción a las Métricas
   Por qué los números importan en CM

2. Métricas Básicas Explicadas
   Engagement, reach, impressions, y más

3. Meta Business Suite en Profundidad
   Cómo leer y navegar las analytics de Meta

4. KPIs para Joyería y Lujo
   Métricas específicas para tu especialización

5. Cómo Interpretar Datos
   Pasar de números a insights accionables

6. Crear Reportes para Clientes
   Plantillas y estructura profesional

7. Optimizar Estrategia con Datos
   Usar analytics para mejorar resultados

8. Herramientas de Análisis
   Las mejores apps gratuitas para celular

RECURSOS:
• Plantilla de Reporte Mensual
• Diccionario de Métricas
• Calculadora de Métricas
```

---

## Para quién es este curso

```
✓ CM principiantes que quieren profesionalizarse
✓ Community managers de joyería/lujo
✓ Emprendedores que manejan sus propias redes
✓ Cualquiera que quiera entender qué funciona en redes

NO NECESITÁS:
✗ Ser buena en matemáticas (es más lógica que números)
✗ Un PC (todo se puede hacer desde el celular)
✗ Herramientas pagas (las gratuitas son suficientes)
```

---

## Tu perfil después de este curso

```
BEFORE (antes):
"No sé si lo que hago está funcionando.
Publico y rezo para que funcione."

AFTER (después):
"Mi engagement rate promedio es 4.2%.
Mis reels funcionan mejor los jueves.
Mis mejores hashtags son los de nicho.
Mi cliente entiende exactamente qué logra cada mes."

La diferencia: datos vs. intuición.
```

## Tiempo estimado

```
Lectura completa: ~4-5 horas
Práctica: ~2-3 horas
Total: ~7 horas

Podés hacer un módulo por día o terminarlo en un fin de semana.
Lo importante es que después de cada módulo practiques con datos reales.
```

---

## Cómo usar este curso

```
RECOMENDACIÓN:

1. Leé el módulo completo
2. Abrí tu Meta Business Suite (o la cuenta que vayas a analizar)
3. Buscá las métricas que se mencionan
4. Hacé los ejercicios prácticos
5. Aplicá lo aprendido a tu próximo reporte

Noal terminar = Al terminar de leer todo de golpe. Lo mejor es aprender + practicar + aplicar.
```
