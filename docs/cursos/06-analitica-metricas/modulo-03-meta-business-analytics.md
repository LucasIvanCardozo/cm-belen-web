```

COSTO: 100% GRATIS
REQUISITO: Tener cuenta de Instagram Business o Creator

## 3. Sección: Overview (Resumen)

### Qué encontrar acá

```
DASHBOARD DE RESUMEN:

┌─────────────────────────────────────────────┐
│ HOY          │ ESTA SEMANA   │ ESTE MES     │
├─────────────────────────────────────────────┤
│                                             │
│ 📊 ALCANCE                                 │
│ ¿Cuánta gente vio tu contenido?              │
│                                             │
│ 💬 INTERACCIONES                            │
│ Likes, comentarios, guardados, compartidos   │
│                                             │
│ 👥 SEGUIDORES                               │
│ Netos: +XX esta semana                     │
│                                             │
│ 🎬 REPRODUCCIONES DE REELS                  │
│ Vistas de videos esta semana                │
│                                             │
└─────────────────────────────────────────────┘
```

### Cómo interpretar el Overview

```
ELEMENTOS DEL DASHBOARD:

1. GRÁFICO DE TENDENCIA
   → Muestra cómo cambian tus métricas con el tiempo
   → Línea azul = Alcance
   → Línea verde = Interacciones

2. COMPARACIÓN CON PERÍODO ANTERIOR
   → "↑ 23% más alcance que la semana pasada"
   → Te dice si estás mejorando o bajando

3. BEST POST (Mejor publicación)
   → La publicación con mejor rendimiento
   → Clic para ver detalles

4. RECOMENDACIONES
   → Meta sugiere qué publicar según tendencias
   → Solo sugerencia, no obligación
```

## 4. Sección: Audience (Audiencia)

### Entender a tu comunidad

```
AUDiencia TAB:

┌─────────────────────────────────────────────┐
│ 👥 TU AUDIENCIA                             │
├─────────────────────────────────────────────┤
│                                             │
│ 📍 DÓNDE ESTÁN (Ubicación)                  │
│ • País y ciudad                             │
│ • Para jewelry: ¿Latinoamérica? ¿España?   │
│                                             │
│ 👤 EDADES                                   │
│ • Rango etario principal                    │
│ • Distribución                             │
│                                             │
│ 👩 GÉNERO                                   │
│ • % hombres vs. mujeres                     │
│                                             │
│ 🕐 CUÁNDO ESTÁN ACTIVOS                     │
│ • Mejores días y horarios                   │
│ • CRUCIAL para saber cuándo publicar       │
│                                             │
└─────────────────────────────────────────────┘
```

### Insights de Audiencia clave

```
DATOS MÁS IMPORTANTES DE AUDIENCIA:

1. UBICACIÓN
   → Te dice dónde está tu audiencia real
   → Para jewelry LatAm: esperás Argentina, México, Colombia
   → Si está muy dispersa, tu contenido puede no resonar

2. EDADES
   → Para jewelry de lujo: 25-45 es el sweet spot
   → Si tu audiencia es más joven o mayor, ajustá tono

3. HORARIOS DE ACTIVIDAD
   → Muestra cuándo tu audiencia está en IG
   → USALO para programar publicaciones
   → No es lo mismo, pero sí importa

   Ejemplo de output:
   ┌─────────────────────────────────────────┐
   │ TUS SEGUIDORES ESTÁN MÁS ACTIVOS EN:    │
   │                                          │
   │ 📅 Días: Martes, Jueves, Viernes        │
   │ 🕐 Horarios: 10am-12pm, 7pm-9pm        │
   │                                          │
   │ → Programa contenido en estos horarios   │
   └─────────────────────────────────────────┘

4. CRECIMIENTO
   → Cuántos seguidores ganaste/perdiste
   → De dónde vienen los nuevos seguidores
```

```

1. POR TIPO DE CONTENIDO

   • Videos
   • Reels
   • Carruseles
   • Stories

2. POR PERÍODO
   • Últimos 7 días
   • Últimos 30 días
   • Últimos 90 días
   • Rango personalizado

3. POR MÉTRICA
   • Ordenar por alcance (mayor a menor)
   • Ordenar por engagement
   • Ordenar por guardados

4. POR ESTADO
   • Solo impulsados (promocionados)
   • Solo orgánicos

COMO USAR LOS FILTROS:
→ Filtra por "Reels" → Ordená por "Alcance"
→ Vas a ver qué reels funcionaron mejor
→ Ese es tu aprendizaje para el futuro

### Qué buscar en Reels

```
CHECKLIST DE ANÁLISIS DE REEL:

□ ¿Cuál fue el reel con más reproducciones?
□ ¿Cuál tuvo mejor tiempo de visualización?
□ ¿Cuál generó más seguidores nuevos?
□ ¿Qué tipo de reel funciona mejor?
   - Tutoriales
   - Detalle de producto
   - Behind the scenes
   - Comedia/humor
   - Storytelling

□ ¿Qué duración funciona mejor?
□ ¿Hubo algún reel que "explotó"?
□ ¿Qué tenían en común los reels exitosos?

RESPUESTAS A ESTAS PREGUNTAS = TU ESTRATEGIA DE REELS
```

```
STORIES TAB:

MÉTRICAS DISPONIBLES:
┌─────────────────────────────────────────────┐
│ 📖 TU RENDIMIENTO DE STORIES                │
├─────────────────────────────────────────────┤
│                                             │
│ • Promedio de vistas por story              │
│ • Tasa de salida (cuántos se van antes)    │
│ • Respuestas                               │
│ • Clics en enlace                          │
│ • Encuestas/votaciones respondidas         │
│ • Capturas de pantalla                      │
│                                             │
│ 📊 POR STORY INDIVIDUAL:                    │
│ • Vistas totales                            │
│ • % que avanzó                             │
│ • Interacciones por tipo                    │
│                                             │
└─────────────────────────────────────────────┘
```

### Insights de Stories avanzados

```
MÉTRICAS DE STORY PROFUNDO:

1. PROGRESIÓN
   • ¿Qué % vio story 1?
   • ¿Qué % llegó al story 7?
   → Si muchos se van en story 3, ajustá el contenido

2. CAPTURAS DE PANTALLA
   • Indica contenido muy valioso
   • 1 captura = Muy alto valor
   • Las personas guardaron tu story de otra forma

3. RESPUESTAS
   • Contenido que genera conversación
   • Muy bueno para comunidad
   • Usá más preguntas/polos

4. CLICS EN ENLACE
   • Si tenés enlace, medí los clics
   • Tasa de clics = Interés en tu oferta
```

## 8. Cómo exportar datos

```
EXPORTAR DATOS:

Meta Business Suite permite exportar reportes.

COMO EXPORTAR:

1. Ve a Insights → Content
2. Toca "..." (más opciones)
3. Selecciona "Exportar datos"
4. Elige:
   • Período (rango de fechas)
   • Formato (CSV o PDF)
   • Métricas a incluir
5. Descargá el archivo

PARA QUÉ SIRVE:
• Análisis en Excel o Google Sheets
• Crear reportes más visuales
• Guardar histórico de datos
• Comparar períodos
```

## 9. Configurar alertas y notificaciones

```
ALERTAS ÚTILES:

En Configuración → Notificaciones podés ajustar:

RECOMENDADO ACTIVAR:
✓ Alertas cuando un post supera X interacciones
✓ Resumen diario de métricas
✓ Notificación si perdés seguidores de golpe
✓ Reporte semanal automático

NO ACTIVAR (ruido innecesario):
✗ Cada like individual
✗ Cada nuevo seguidor
✗ Comentarios uno por uno

CONFIGURÁ SEGÚN TU ESTILO:
Si revisás métricas 1 vez al día → Alertas de resumen
Si no revisás nunca → Alertas más frecuentes
```

## 10. Ejercicio práctico: Navegación completa

```
EJERCICIO: Explorá Meta Business Suite

PASO 1: Abrí la app
   → Navegá a cada sección principal
   → No toques nada, solo explorá

PASO 4: En Content
   → Filtrá por "Reels"
   → Ordená por "Alcance"
   → ¿Cuál es tu reel más visto?

PASO 5: En Insights → Actividad
   → ¿Cuántos seguidores ganaste esta semana?
   → ¿Hay algún día con pico de actividad?

ENTREGA: Resumen de 1 página con:
• Tus métricas principales
• Insights de audiencia
• Tu mejor publicación
• Área de mejora identificada
```
