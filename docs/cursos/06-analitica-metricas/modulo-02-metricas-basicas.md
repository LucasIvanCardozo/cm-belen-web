## 2. Alcance (Reach)

### Qué es

```
DEFINICIÓN:
Cantidad de cuentas únicas que vieron tu contenido al menos una vez.

Ejemplo:
"Este post tuvo un alcance de 2,500."
→ Significa: 2,500 personas distintas vieron este post.
```

### Por qué importa

```
• Muestra cuántas personas reales ven tu contenido
• Es la base: si nadie ve, nada más importa
• Indica si el algoritmo está mostrando tu contenido
• Permite comparar rendimiento entre posts
```

### Cómo interpretarlo

```
BENCHMARKS PARA JOYERÍA (cuentas pequeñas-medianas):

| Tipo de cuenta | Alcance promedio por post |
|---------------|--------------------------|
| Nueva (0-1K seguidores) | 200-500 |
| En crecimiento (1K-5K) | 500-2,000 |
| Estable (5K-20K) | 2,000-8,000 |
| Grande (20K+) | 8,000+ |

⚠️ IMPORTANTE: Estos son guías, no reglas.
Compará siempre con TU promedio, no con otros.
```

## 3. Impresiones

### Qué es

```
DEFINICIÓN:
Cantidad total de veces que tu contenido fue mostrado.
(Una persona puede ver el mismo post varias veces)

Ejemplo:
"Este reel tuvo 5,000 impresiones."
→ Significa: El reel se mostró 5,000 veces en total.
→ Puede ser las mismas 1,000 personas viéndolo 5 veces,
   o 5,000 personas viéndolo 1 vez.
```

### Alcance vs. Impresiones

```
LA DIFERENCIA CLAVE:

ALCANCE = Personas únicas
IMPRESIONES = Veces mostrado

VISUAL:

Persona A → vio 3 veces → Cuenta como 1 alcance, 3 impresiones
Persona B → vio 1 vez  → Cuenta como 1 alcance, 1 impresión
Persona C → vio 5 veces → Cuenta como 1 alcance, 5 impresiones
Persona D → vio 2 veces → Cuenta como 1 alcance, 2 impresiones

TOTAL: 4 personas (alcance) vs. 11 veces mostrado (impresiones)
```

### Por qué importa

```
• Impresiones altas + Alcance bajo = La misma gente ve mucho
  (Puede indicar que solo mostrás a tus seguidores actuales)

• Alcance alto + Impresiones bajas = Mucha gente ve una vez
  (Buen signe: estás alcanzando gente nueva)

• Lo ideal: Ambos altos, proporción ~2:1 (impresiones:alcance)
```

## 4. Engagement

### Qué es

```
DEFINICIÓN:
Todas las interacciones que la gente tuvo con tu contenido.

INCLUDE:
• Likes ❤️
• Comentarios 💬
• Guardados 💾
• Compartidos 🔄
• Clics en enlaces
• Vistas de video

NO INCLUYE:
• Solo ver el contenido (eso es alcance)
• Compartir sin ver
```

### Cómo se calcula

```
ENGAGEMENT = Likes + Comentarios + Guardados + Compartidos + Clics

Ejemplo:
Post con: 245 likes + 18 comentarios + 12 guardados + 5 compartidos
= 280 interacciones totales

Engagement = 280
```

## 5. Engagement Rate (La métrica más importante)

### Qué es

```
DEFINICIÓN:
El porcentaje de personas que interactuaron
vs. las que vieron tu contenido.

FÓRMULA:
Engagement Rate = (Interacciones / Alcance) × 100
```

### Ejemplo práctico

```
CALCULO PASO A PASO:

Tu post tiene:
• Alcance: 1,000 personas
• Likes: 85
• Comentarios: 12
• Guardados: 8
• Compartidos: 3
• Clics: 2

ENGAGEMENT TOTAL = 85 + 12 + 8 + 3 + 2 = 110

ENGAGEMENT RATE = (110 / 1,000) × 100 = 11%

¡11% es un engagement rate EXCELENTE!
```

### Benchmarks de Engagement Rate

```
ENGAGEMENT RATE PROMEDIO POR PLATAFORMA:

Instagram en general:
• Cuentas pequeñas (0-5K): 3-6%
• Cuentas medianas (5K-20K): 2-4%
• Cuentas grandes (20K+): 1-2%

Joyería/Lujo (nicho):
• Accounts enfocado: 3-8% (más comprometido)
• Accounts masivo: 1-3%

¿TU ENGAGEMENT RATE ES BUENO?
• < 1%: Necesitás mejorar tu contenido
• 1-3%: Normal, hay espacio para crecer
• 3-6%: Bueno, estás en el buen camino
• 6%+: Excelente, estás arriba del promedio
```

## 6. Guardados

### Por qué es GOLD

```
GUARDADOS = "Quiero ver esto después"

Esto es más valioso que un like porque:
1. La persona pensó: "Esto es valioso, quiero guardarlo"
2. Señal fuerte para el algoritmo: contenido de calidad
3. Potencial de retorno: la persona va a volver a verlo
4. Indicador de contenido evergreen (que no caduca)

UN GUARDADO VALE MÁS QUE 3 LIKES.
```

### Cómo medirlo

```
En Meta Business Suite:
→ Insights → Publicaciones → Ve el número de guardados

Benchmark:
• Posts normales: 1-3% de guardados sobre alcance
• Posts de valor (educativo): 5-10%
• Joyería premium: 3-5%
```

## 7. Compartidos

### Qué indica

```
COMPARTIDOS = "Quiero que otros vean esto"

Es la métrica de mayor valor porque:
1. Extiende tu alcance orgánicamente
2. Indica contenido altamente compartible
3. Genera "buzz" social
4. El algoritmo lo premia con más exposición

Tipo de contenido que más se comparte:
• Contenido educativo/informativo
• Humor (cuando aplica a marca)
• Emocional/inspiracional
• Contenido controversial (cuidado)
```

## 8. Seguidores

### Qué mirar realmente

```
SEGUIDORES TOTALES NO ES LA MÉTRICA PRINCIPAL.

Mirás:
├── SEGUIDORES NETOS = Nuevos - Perdidos
│   → Netos positivos = Crecimiento real
├── TASA DE CRECIMIENTO = (Netos / Total) × 100
│   → ¿Qué % crece tu cuenta por semana/mes?
└── FUENTE DE SEGUIDORES
    → ¿De dónde vienen? (Explorar, hashtags, otro perfil)

EJEMPLO:
Cuenta con 5,000 seguidores:
• Esta semana: +150 nuevos, -30 perdidos
• Netos: +120
• Tasa de crecimiento: 2.4% semanal

¡Eso es crecimiento saludable!
```

## 9. Métricas de Reels

### Específicas para video

```
MÉTRICAS DE REELS:

• VISTAS: Cuántas veces se reprodujo el video
  (Se cuenta a partir de 3 segundos de reproducción)

• PROMEDIO DE TIEMPO DE VISUALIZACIÓN:
  Cuánto tiempo miran antes de pasar
  → Más de 50% del video = Excelente
  → Más de 75% = WOW

• INTERACCIONES EN REEL:
  - Likes, comentarios, guardados, compartidos
  - Visitas al perfil desde el reel
  - Clics en enlace (si hay)

• ALCANCE DE REEL:
  - Similar a posts, pero generalmente mayor
  - Reels pueden alcanzar fuera de seguidores

• VISTA DE PERFIL:
  - Cuántas personas vieron tu perfil después del reel
```

### Tiempo de visualización = Rey

```
TIEMPO DE VISUALIZACIÓN ES LO QUE IMPORTA.

Por qué:
→ Si la gente mira todo el video, es contenido valioso
→ Instagram premia contenido que la gente ve completo
→ Más tiempo = Más señales para el algoritmo

BENCHMARK:
• < 25% del video → Bajo, la gente se va rápido
• 25-50% → Normal
• 50-75% → Bueno
• 75%+ → Excelente, Instagram te va a mostrar más

PARA JOYERÍA:
Los reels de close-up (detalle de la pieza) suelen tener
alto tiempo de visualización porque la gente quiere ver el brillo.
```

## 10. Métricas de Stories

```
MÉTRICAS DE STORIES:

• VISTAS: Cuántas personas vieron cada story
  (Diferente del alcance: muestra vistas totales)

• INTERACCIÓN:
  - Respuestas
  - Clics en enlaces
  - Votaciones/encuestas
  - Preguntas contestadas

• USUARIOS QUE AVANZARON:
  - Qué % llegó hasta el final
  - Story 1 vs. Story 7

• CAPTURAS DE PANTALLA:
  - Alguien guardó tu story = Muy valioso

BENCHMARK PARA STORIES:
• Vistas: 20-40% de tus seguidores
• Interacción: 1-3%
• Capturas: 0.5-2%
```
