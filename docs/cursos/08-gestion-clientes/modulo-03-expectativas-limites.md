# Módulo 3 — Expectativas y Límites

## La Línea entre el Servicio Excepcional y el Exceso

---

## Objetivos del Módulo

Al finalizar este módulo vas a:

- ✅ Entender por qué establecer límites es crucial
- ✅ Saber comunicar límites de forma profesional
- ✅ Identificar señales de cliente que pide demasiado
- ✅ Proteger tu tiempo y energía

---

## 1. Por qué los Límites importan

```
LA REALIDAD:

Un CM sin límites = Burnout garantizada.

Porque:
- Si decís que sí a todo → te explotan
- Si no ponés límites → el cliente abuse
- Si tenés miedo de decir que no → trabajás gratis

EJEMPLO REAL:
CM dice: "Lo que necesites, avisame."
Cliente: "¿Me podrías hacer este reel? Ah, ¿y
         меня llamás a mi mamá para coordinar?"
         "¿Y revisar las fotos del proveedor?"

CM que cobra: $300/mes
Horas extra no cobradas: 15hs/mes
Valor: ~$150 en trabajo gratis
```

---

## 2. Tipos de Límites

```
LÍMITES QUE DEBÉS TENER:

1. HORARIO DE TRABAJO
   → Cuándo estás disponible
   → Cuándo NO respondés mensajes

2. ALCANCE DEL SERVICIO
   → Qué está incluido
   → Qué NO está incluido

3. CANTIDAD DE REVISIONES
   → Cuántas rondas de cambios incluye
   → Cuánto cuesta cada cambio extra

4. CANALES DE COMUNICACIÓN
   → Qué herramientas usás
   → Qué NO es para vos (llamadas sin cita)

5. DISPONIBILIDAD
   → Días libres
   → Vacaciones
   → Feriados
```

## 3. Cómo Comunicar Límites

### Fórmula: Explicar + Ofrecer + Cerrar

```
PATRÓN:

EXPLICAR: Por qué es el límite
OFRECER: Qué sí podés hacer
CERRAR: Conclusión clara

EJEMPLO:

❌ MAL:
"No puedo hacer eso."

✓ BIEN:
"Entiendo que necesitas esto para hoy (EXPLICAR).
Mi horario de работы es L-V hasta las 6pm,
y hoy ya tengo compromisos (OFRECER).
Lo que sí puedo hacer es tenerlo listo
para mañana a primera hora (OFRECER).
¿Te parece? (CERRAR)"
```

### Ejemplos Prácticos

```
SITUACIÓN 1: Cambio de último momento

"Con gusto te ayudo con el cambio.
El servicio incluye hasta 2 rondas de revisión
por contenido (EXPLICAR).
Para este contenido ya vamos por la tercera ronda,
así que este cambio extra tiene un costo
adicional de $X (OFRECER).
¿Querés que lo incluya en la factura del mes?"
```

```
SITUACIÓN 2: Tarea fuera de alcance

"Gracias por pensar en mí para esto.
Esa tarea (redactar email a proveedores)
no está dentro del alcance de servicios de CM
que acordamos (EXPLICAR).
Lo que sí puedo hacer es redactarte un template
de mensaje que puedas usar (OFRECER).
¿Te sirve?"
```

```
SITUACIÓN 3: Llamada sin aviso

"Me encantaría ayudarte, pero no tengo
disponible una llamada ahora (EXPLICAR).
¿Podemos coordinar una para mañana?
Tengo slots disponibles a las 10am o 3pm (OFRECER).
¿Alguno te funciona?"
```

## 4. Expectativas Claras desde el Inicio

```
TODO EN EL CONTRATO:

Sección: ALCANCE DEL SERVICIO

ESTE ACUERDO INCLUYE:
✓ Gestión de Instagram (X posts/semana)
✓ Gestión de Facebook (X posts/semana)
✓ Creación de captions
✓ Diseño de gráficos (X plantillas)
✓ Programación de contenido
✓ Respuesta a comentarios y DMs (X/día)
✓ 2 rondas de revisión por contenido
✓ 1 reporte mensual

NO ESTÁ INCLUIDO:
✗ Fotografía de producto (salvo acuerdo especial)
✗ Videollamadas de emergencia
✗ Gestión de crisis de reputación
✗ Gestión de situaciones personales del dueño
✗ Investigación de proveedores
✗ Llamadas a clientes o proveedores

CAMBIOS Y ADICIONES:
Todo servicio adicional será cotizado por separado
con un mínimo de 48hs de anticipación.
```

## 5. Señales de Alerta

```
CLIENTE QUE NO RESPETA LÍMITES:

⚠️ Escribe fuera de horario constantemente
⚠️ Pide cosas "rápidas" que no son tu responsabilidad
⚠️ Cambia de opinión constantemente
⚠️ Espera respuesta inmediata
⚠️ Se enoja cuando decís que no
⚠️ "Pensé que eso estaba incluido..."
⚠️ Te contacta por múltiples canales a la vez

QUÉ HACER:
1. Identificá el patrón
2. Hablá claro en una conversación
3. Documentalo por escrito
4. Si continúa: reconsiderá el cliente
```

---

## 6. El Límite del Dinero

```
CUANDO TE PIDEN TRABAJO GRATIS:

❌ "Me podrías hacer este post de prueba?"
❌ "Si funcionает, te contrato"
❌ "Es solo 5 minutos"

✓ RESPUESTA:
"Entiendo la propuesta. Mi trabajo tiene
un valor de $X por post. Para darte una
idea de cómo sería trabajar juntos, te ofrezco
una llamada de 30 min donde te explico mi proceso
y vemos cómo podría ayudarte. ¿Te parece?"

El trabajo gratis normaliza que no valés nada.
```

---

## 7. Cuándo Decir No

```
SÍ DECÍS NO CUANDO:

1. Te piden algo ilegal o antiético
2. Te aleja de tus valores
3. Te compromete legalmente
4. No está en tus capacidades
5. Te va a hacer daño (tiempo, salud, energía)
6. No te pagan a tiempo
7. Te piden trabajo que no cotizaron

SÍ DECÍS SÍ (CON LÍMITES) CUANDO:

1. Es una expansión natural del servicio
2. Te pagan extra por ello
3. Tenés tiempo y energía
4. Aprendés algo nuevo
5. Construye la relación
```

## 9. Resumen

```
✓ Limits son protectores, no bloqueos
✓ Communicarlos temprano evita problemas
✓ Fórmula: Explicar + Ofrecer + Cerrar
✓ Todo fuera de alcance = cotizar aparte
✓ Señales de alerta = actuar antes de quemarte
✓ NO = respuesta completa, no necesita justificación
```
