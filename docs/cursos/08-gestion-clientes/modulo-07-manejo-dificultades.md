# Módulo 7 — Manejo de Dificultades

## Cuando las Cosas se Ponen Feas

---

## Objetivos del Módulo

Al finalizar este módulo vas a:
- ✅ Manejar clientes difíciles con profesionalismo
- ✅ Resolver conflictos antes de que escalen
- ✅ Saber cuándo terminar una relación profesional

---

## 1. Tipos de Clientes Difíciles

```
COMPORTAMIENTOS PROBLEMÁTICOS:

1. MICRO-GESTOR
   "¿Podrías cambiar el color de este botón?"
   "No me gusta, hacelo de nuevo."
   "Ese post no es lo que esperaba."

2. SIEMPRE URGENTE
   "Esto es para AYER"
   "Necesito que esté para hoy"
   "¿Por qué no me respondiste a las 11pm?"

3. CAMBIA CONSTANTEMENTE
   "Publicá esto... No, esperá, mejor esto otro."
   "Acción... No, mejor no publicamos nada."

4. NO PAGA A TIEMPO
   "¿Me das un par de días más?"
   "Lo tengo que transfers... se me olvidó"
   "No tengo efectivo hasta fin de mes"

5. PIDE MUCHO MÁS
   "Ah, ¿me podrías también...?"
   "Esto es solo 5 minutos, no?"
   "Pensé que eso estaba incluido"

6. DESPRECIA TU TRABAJO
   "Mi sobrina hace lo mismo gratis"
   "¿Tan complicado es hacer un post?"
   "Es solo escribir algo, ¿no?"
```

## 2. Estrategia General

```
ANTES DE REACCIONAR:

1. PAUSA
   → No respondas en el calor del momento
   → Mejor esperar 30 min - 2 horas

2. EVALUÁ
   → ¿Es un patrón o un incidente?
   → ¿Tiene razón?
   → ¿Es personal o profesional?

3. PLANIFICÁ
   → ¿Qué quiero lograr con esta respuesta?
   → ¿Cómo quiero que termine esta conversación?

4. RESPONDÉ
   → Profesional, claro, con límites
```

## 3. Respuestas por Situación

### Micro-gestor

```
SITUACIÓN:
"Nuevamente no me gusta el post. hacelo otra vez."

RESPUESTA:

"Entiendo que querés que el contenido sea perfecto.
Para asegurarnos de que quedás satisfecho,
necesitamos alinear las expectativas.

Este es el tercer cambio que hacemos en este post.
Según nuestro acuerdo, incluimos 2 rondas de revisión.
Este cambio adicional tiene un costo de $X.

Alternativamente, podemos hacer una llamada
para revisar juntos qué buscás específicamente
y así futuras piezas cumplan tus expectativas desde el inicio.

¿Cuál preferís?"
```

### Siempre Urgente

```
SITUACIÓN:
"¡Esto es URGENTE! ¡Necesito esto para hoy!"

RESPUESTA:

"Entiendo la urgencia. 
Pero 'urgente' significa que necesito
información/clienteo por parte de vos
para poder hacerlo bien.

Horario límite para publicable hoy: 2pm.
Después de eso, no puedo garantizar calidad.

¿Me confirmás la información antes de mediodía?"
```

### No Paga

```
SITUACIÓN:
"Me olvidó de pagar. ¿Me das unos días?"

RESPUESTA:

"Sin problema. 

Pero para mantener los servicios activos,
necesito el pago antes del [fecha límite].

Si no recibo el pago para ese día,
tendré que pausar la gestión de tu cuenta
hasta que esté al día.

¿Te confirmás cuándo hacés la transferencia?"
```

## 4. Cuándo Escalar

```
EScalar significa pedir ayuda cuando:
- Te sientas amenazada/o
- El cliente te insulta o humilla
- Hay presión illegal
- Te piden hacer algo antiético

OPCIONES:
1. Hablar con alguien de confianza
2. Consultar un abogado (por temas legales)
3. Terminar la relación profesionalmente
```

---

## 5. Cuándo Terminar

```
SEÑALES PARA TERMINAR:

⚠️ Pagos constantemente tarde
⚠️ Abuso verbal o presión excesiva
⚠️ Siempre insatisfecho sin razón
⚠️ Te hace trabajar fuera de tus valores
⚠️ Pide trabajo illegal
⚠️ No respeta límites después de hablar

CÓMO TERMINAR:

"Hola [Nombre],

Después de mucha reflexión, he decidido
terminar nuestra colaboración.

El motivo principal es [ser honestо pero diplomaticо].

Te ofrezco [X días/semanas] de support
para fazer la transición.

Te wish the best in your business."

- No te excuses demasiado
- No burned bridges
- Terminá professionnel
```

---

## 6. Resumen

```
✓ Antes de reaccionar: pausa, evalúa, planifica, responde
✓ Para cada tipo: respuestas templadas con límites
✓ Para pagos: fecha límite clara
✓ Para terminar: professionnel, sin burn bridges
✓ Tu bienestar > cualquier cliente
```
