# Módulo 5 — Contratos

## Protegé tu Trabajo Legalmente

---

## Objetivos del Módulo

Al finalizar este módulo vas a:

- ✅ Entender por qué necesitas un contrato
- ✅ Conocer los elementos esenciales de un contrato
- ✅ Usar plantillas efectivas
- ✅ Saber cuándo y cómo cobrar anticipo

---

## 1. Por qué un Contrato

```
SIN CONTRATO:
❌ El cliente cambia de opinión
❌ "Pensé que ibas a hacer más"
❌ No te pagan y no hay consecuencias
❌ Te explotan sin límite
❌ No hay forma de protegerte

CON CONTRATO:
✓ Todo queda claro desde el inicio
✓ Ambas partes saben qué esperar
✓ Hay consecuencias por incumplimiento
✓ Protegés tu propiedad intelectual
✓ Tenés base legal si hay problemas
```

---

## 2. Elementos Esenciales

```
TODO CONTRATO DEBE TENER:

1. PARTES INVOLUCRADAS
   → Tu nombre completo
   → Nombre del negocio del cliente
   → Datos de contacto

2. DESCRIPCIÓN DEL SERVICIO
   → Qué exactamente vas a hacer
   → Cuántas veces por semana
   → Qué canales
   → Qué incluye específicamente

3. NO INCLUIDO
   → Qué NO es parte del servicio
   → Evita malentendidos

4. DURACIÓN
   → Fecha de inicio
   → Fecha de fin (si aplica)
   → Período de prueba (si aplica)

5. PRECIO Y FORMA DE PAGO
   → Monto mensual
   → Cuándo se paga
   → Método de pago
   → Qué pasa si no paga

6. PROPIEDAD INTELECTUAL
   → Quién es dueño del contenido
   → Qué pasa si termina la relación

7. CONFIDENCIALIDAD
   → Qué información es privada
   → Qué hacés con la información del cliente

8. TERMINACIÓN
   → Cómo puede terminar cualquiera de las partes
   → Cuánto tiempo de aviso
   → Qué pasa con pagos pendientes

9. FIRMAS
   → Tu firma
   → Firma del cliente
   → Fecha
```

## 3. Cláusula de Propiedad Intelectual

```
IMPORTANTE PARA CM:

"El contenido creado durante este acuerdo
(escritos, captions, estrategias) es propiedad
del CLIENTE una vez que el pago esté completo.

El CM puede usar el contenido creado
para su portfolio y publicity, a menos que
el cliente solicite lo contrario por escrito."

¿POR QUÉ ESTO IMPORTA?
Si hacés un reel increíble, podés mostrarlo
como tu trabajo sin problemas.
```

---

## 4. Cláusula de Terminación

```
EJEMPLO DE TERMINACIÓN:

"Este acuerdo puede ser terminado por
cualquiera de las partes con 15 días de aviso
escrito por email.

En caso de terminación:
- El cliente debe pagar por servicios prestados
  hasta la fecha de terminación
- El CM dejará de gestionar las cuentas
  a partir de la fecha de terminación
- El CM предоставит acceso a la cuenta
  al cliente dentro de 48 horas

Si el cliente no paga durante 10 días,
el CM puede pausar los servicios
sin previo aviso adicional."

PARA JOYERÍA:
Agregar:
"Durante temporada alta (Nov-Feb),
se requiere 30 días de aviso
para terminación, para asegurar
transición adecuada."
```

## 5. Contrato Mínimo Viable

```
CONTRATO SIMPLE PARA FREELANCE CM:

═══════════════════════════════════════════
CONTRATO DE SERVICIOS DE COMMUNITY MANAGER

FECHA: [fecha]

PARTES:
PRESTADOR: [Tu nombre]
CLIENTE: [Nombre del negocio]

SERVICIO:
El PRESTADOR proveerá servicios de gestión
de Community Manager para [Nombre del Negocio],
incluyendo:
- [X] publicaciones semanales en [redes]
- Creación de contenido visual
- Redacción de captions
- Gestión de comunidad (comentarios y DMs)
- [Otros servicios]

NO INCLUIDO:
- Fotografía de producto
- Gestión de publicidad pagada
- [Otros]

DURACIÓN:
Inicio: [fecha]
Este acuerdo es mes a mes, con renewал automatيك
a menos que cualquiera de las partes notifique
con 15 días de anticipación.

PAGO:
Monto: $[monto] USD mensuales
Fecha de pago: día [X] de cada mes
Método: [transferencia, etc.]

PROPIEDAD INTELECTUAL:
El contenido creado es propiedad del CLIENTE
una vez recibido el pago correspondiente.
El PRESTADOR puede usar el contenido para
su portfolio con permiso del CLIENTE.

TERMINACIÓN:
Cualquiera de las partes puede terminar
con 15 días de aviso escrito.

FIRMAS:

_____________________        _____________________
[Tu nombre]               [Cliente]
Fecha: ___/___/___         Fecha: ___/___/___

═══════════════════════════════════════════
```

## 6. Cuando el Cliente No Quiere Contrato

```
ESCENARIO:
"¿Podemos trabajar sin contrato?
Confío en vos y vos confiás en mí."

RESPUESTA:

"Entiendo. Para mí también es más cómodo
cuando hay confianza. Te propongo algo:

Este no tiene que ser un documento legal
extenso. Puede ser un acuerdo simple
por email donde quede registrado:

1. Qué vamos a hacer
2. Cuánto cuesta
3. Cómo y cuándo pagás
4. Cuánto tiempo de aviso para terminar

¿Te parece si lo hacemos así?"

SI INSISTE EN NO FIRMAR NADA:
- Explicá que el acuerdo por email
  ES tu "contrato"
- Guardá screenshot de TODO
- Tené cuidado con cuánto invertís
  sin pago por delante
```

---

## 7. Resumen

```
✓ Contrato protege a ambas partes
✓ Elementos: partes, servicio, precio, duración, terminación
✓ Propiedad intelectual: protege tu portfolio
✓ Terminación: mínimo 15 días de aviso
✓ Si no quieren contrato: email como backup
✓ Siempre cobrá anticipo
```
