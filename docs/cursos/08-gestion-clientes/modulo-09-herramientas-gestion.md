# Módulo 9 — Herramientas de Gestión

## Apps y Sistemas para Organizarte

---

## Objetivos del Módulo

Al finalizar este módulo vas a:
- ✅ Conocer las mejores herramientas para gestión de clientes
- ✅ Implementar sistemas simples y efectivos
- ✅ Automatizar lo repetitivo

---

## 1. Herramientas Principales

```
PARA ORGANIZACIÓN:

NOTION (⭐ RECOMENDADO)
- Base de datos de clientes
- Templates personalizables
- Gratis hasta cierto uso
- Todo en un solo lugar

TRELLY
- Tableros kanban
- Bueno para visualizar tareas
- Gratis para uso básico
- Ideal para proyectos

GOOGLE WORKSPACE
- Google Drive: archivos
- Google Calendar: scheduling
- Google Docs/Sheets: documentos
- Todo integrado, gratis

COMUNICACIÓN:

WHATSAPP BUSINESS
- Respuestas automáticas
- Catálogo de productos
- Separa trabajo de personal

EMAIL PROFESIONAL
- Gmail con dominio propio: tu@tuemail.com
- Más profesional que gmail.com
- Herramientas de email marketing

PAGO:

MERCADO PAGO
- Para clientes LatAm
- Cobros fáciles
- Transferencias a banco local

PAYPAL / STRIPE
- Para clientes internacionales
- Más comisiones pero más reach
```

---

## 2. Sistema Mínimo Viable

```
SOLO 3 HERRAMIENTAS:

1. NOTION
   - Info de clientes
   - Tareas
   - Calendario

2. GOOGLE CALENDAR
   - Todas las citas
   - Recordatorios
   - Scheduling

3. WHATSAPP BUSINESS
   - Comunicación
   - Respuestas rápidas

¿POR QUÉ POCAS?
- Menos = más consistencia
- No te pierdas entre apps
- Solo agrega cuando NECESITES
```

## 3. Notion para CM

```
TEMPLATE MÍNIMO:

📊 Dashboard
├── 📁 Clientes
│   ├── Cliente A
│   ├── Cliente B
│   └── Cliente C
├── 📅 Calendario
│   └── [Calendario mensual]
└── 📋 Tareas
    └── [Tareas por semana]

CLIENTE TEMPLATE:
Propiedades:
- Nombre
- Servicio
- Precio
- Fecha de inicio
- Próximo pago
- Estado (activo/pausado/terminado)

Vista: Tabla + Calendario
```

## 6. Ejercicio Práctico

```
EJERCICIO: Configurá tu sistema

PASO 1: Creá cuenta Notion
   → Plantilla de cliente
   → Dashboard básico

PASO 2: Configurá Google Calendar
   → Calendario de trabajo
   → Recordatorios

PASO 3: WhatsApp Business
   → Respuestas automáticas
   → Info de contacto actualizada

PASO 4: Integración
   → Sincronizar calendario
   → Tareas en un solo lugar
```
