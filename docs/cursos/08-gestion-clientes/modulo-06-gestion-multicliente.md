```

Bloques específicos para cada cliente:

🟢 Cliente A (Joyería): 8-12am
🟡 Cliente B (Lifestyle): 1-4pm

2. El cliente sabe cuándo sos disponible
3. Cada bloque tiene una tarea específica
```

## 5. Cuántos Clientes Puedes Tener

```
CÁLCULO SIMPLE:

Si trabajás 40 horas/semana:
- Tiempo efectivo de trabajo: 30h (25% admin)
- 30 horas ÷ horas por cliente = clientes máximos

EJEMPLO:
CM Junior: 5h/cliente/mes
→ 30h ÷ 5h = 6 clientes máximo

CM Senior: 8-10h/cliente/mes (más eficiencia)
→ 30h ÷ 8h = 3-4 clientes

PARA JOYERÍA:
- Cada cliente necesita contenido de alta calidad
- Más fotos, más revisión
- Factor multiplicador: 1.3x el tiempo
→ Menos clientes, más calidad
```

## 7. Cuando un Cliente Te Absorbe

```
SEÑAL:
"Trabajo 40 horas solo para Cliente A
y tengo 3 más que apenas atiendo."

SOLUCIÓN:

1. HABLAR CLARO
   "María, valoro mucho trabajar con vos.
   Quiero darte el mejor servicio, pero
   actualmente tengo X proyectos.
   Necesitás: [más presupuesto / menos scope]."

2. REDUCIR SCOPE
   "De X posts, podemos ir a Y posts
   para que pueda darle la calidad que merece."

3. SUBIR PRECIO
   "Para mantener este nivel de atención,
   necesito ajustar el precio a $X."

4. DECIR NO
   "No puedo tomar más trabajo de este tipo
   actualmente. Te aviso cuando tenga espacio."
```

```
✓ Carpeta por cliente + info-general.md

✓ Regla: no cambiar bloques
✓ Matriz de urgencia vs importancia
✓ Saber decir "no" o "reduzcamos scope"
✓ Joyería requiere más tiempo = menos clientes
```
