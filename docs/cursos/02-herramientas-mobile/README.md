# 🎀 Curso 02: Herramientas Mobile para CM

**Tu celular es tu estudio — aprendé desde cualquier lugar** 📱💗

---

> *"La mejor herramienta es la que dominás. Y la podés llevar en tu bolsillo."*

---

## 📚 Descripción

Este curso te lleva paso a paso por las **herramientas esenciales** que todo CM necesita: Canva, CapCut, Later y más. Todo diseñado para aprenderse desde el celular, sin necesidad de una computadora.

**Después de este curso vas a poder:**

- ✅ Dominar Canva para crear contenido profesional
- ✅ Editar videos increíbles con CapCut
- ✅ Programar publicaciones con Later
- ✅ Crear flujos de trabajo eficientes
- ✅ Organizar tus archivos y proyectos
- ✅ Integrar herramientas para trabajo fluido
