## 1. Interfaz y Conceptos Fundamentales

### Arquitectura de CapCut Mobile

Antes de editar, entiende cómo funciona la interfaz:

```
┌─────────────────────────────────────────────┐
│  [←] Proyecto [Nombre]     [Exportar] [•••] │  ← Header: navegación + acciones
├─────────────────────────────────────────────┤
│                                             │
│           [VISTA PREVIA DE VIDEO]           │  ← Área principal: preview
│                                             │
│         ▶️ (reproducir/pausar)              │
├─────────────────────────────────────────────┤
│  [Agregar] [Texto] [Sticker] [Overlay]      │  ← Barra de herramientas
├─────────────────────────────────────────────┤
│                                             │
│    ════════════════════════════════════     │  ← Timeline (línea de tiempo)
│    [Clip 1] [Clip 2] [Clip 3] [+]           │
│                                             │
│    ════════════════════════════════════     │  ← Track de audio/música
│    [♪ Música]                                │
│                                             │
│    ════════════════════════════════════     │  ← Track de texto/anotaciones
│    [Texto] [Texto]                           │
│                                             │
└─────────────────────────────────────────────┘
```

### Conceptos Clave

| Concepto       | Definición                   | Para Joyería          |
| -------------- | ---------------------------- | --------------------- |
| **Clip**       | Segmento de video individual | Cada toma de una joya |
| **Timeline**   | Línea de tiempo donde editas | Donde armás tu reel   |
| **Track**      | Capa independiente           | Video + texto + audio |
| **Transición** | Cambio entre clips           | Efecto visual         |
| **Overlay**    | Elemento sobrepuesto         | Textos, stickers      |
| **Keyframe**   | Punto de animación           | Efectos de movimiento |

## 2. Tu Primer Reel de Joyería

### Preparación del Material

Antes de abrir CapCut, prepara tus archivos:

```
ORGANIZACIÓN SUGERIDA:

Carpeta: "Reels Joyería Mayo 2026"
├── Videos/
│   ├── anillo_luna_01.mp4
│   ├── anillo_luna_02.mp4
│   ├── collar_cierre.mp4
│   └── pulsera_toma.mp4
├── Fotos/
│   ├── producto_principal.jpg
│   ├── detalle_oro.jpg
│   └── estilo_vida.jpg
└── Audio/
    └── musica_trending.mp3
```

**Ideal para joyería:**

- Videos de 15-30 segundos por toma
- Múltiples ángulos del mismo producto
- Fotos en alta resolución (para insertar)
- Música sin copyright o con licencia

### Creación Paso a Paso

#### Paso 1: Nuevo Proyecto

1. Abre CapCut → Toca "Nuevo proyecto"
2. Selecciona tu primer video/clip
3. CapCut importa y muestra en preview

#### Paso 2: Agregar Clips al Timeline

**Opción A: Desde importación múltiple**

1. Toca el ícono "+" en timeline
2. Selecciona "Importar" → multiselección
3. Elige 3-5 clips
4. Se agregan secuencialmente

**Opción B: Desde fotos ( slideshow style)**

1. Toca "+" → "Importar"
2. Selecciona fotos (3-8)
3. Elige duración: 2-3 segundos por foto
4. CapCut crea clip tipo carrusel

#### Paso 3: Editar Posición y Duración

```
AJUSTES EN TIMELINE:

Mover clip: Arrastra horizontalmente
Ajustar inicio: Arrastra borde izquierdo
Ajustar fin: Arrastra borde derecho
Dividir clip: Toca clip + "Dividir" ( tijeras)
Eliminar: Selecciona + papelera

EJEMPLO PARA JOYERÍA:

Clip 1 (0-5s): Video panorámico del anillo
Clip 2 (5-8s): Close-up del detalle
Clip 3 (8-12s): Video de estilo de vida
Clip 4 (12-15s): Foto del producto final
```

#### Paso 4: Transiciones Entre Clips

**Cómo agregar transición:**

1. Toca el borde entre dos clips
2. Aparece menú: "Transición" o el ícono "+"
3. Selecciona categoría (Basic, Motion, etc.)
4. Elige transición
5. Ajusta duración (0.3-0.5s es ideal)

**Transiciones recomendadas para joyería:**

```
ELEGANTES (lujo, sofisticación):
- Fade (desvanecer)
- Dissolve (disolver)
- Smooth Zoom (zoom suave)
- Crossfade (fundido cruzado)

DINÁMICAS (energía, movimiento):
- Slide (deslizar)
- Rotate (rotar)
- Glitch (glitch - usar sparingly)
- Light Leaks (fugas de luz)

EVITAR PARA JOYERÍA:
- Bounce (rebote) - muy juguetón
- Shake (sacudida) - nervioso
- Warp (deformación) - distorsiona el producto
```

**Práctica:** Agrega transición fade entre tus clips.

## 3. Texto Profesional en Videos

### Tipos de Texto en CapCut

| Tipo               | Uso                       | Ejemplo para Joyería    |
| ------------------ | ------------------------- | ----------------------- |
| **Texto estático** | Títulos fijos             | "Colección Luna"        |
| **Texto animado**  | Títulos con movimiento    | "Nuevo arrivals"        |
| **Subtítulos**     | Transcripción             | Diálogos o narración    |
| **Caption**        | Texto que sigue tendencia | "watch till the end 💎" |

### Crear Texto Animado Estilo Luxe

#### El Estilo "Gold Elegance"

```
CONFIGURACIÓN PARA JOYERÍA PREMIUM:

Fuente: Playfair Display o similar serif
Tamaño: 40-60px para título principal
Color: Dorado (#D4AF37) o blanco (#FFFFFF)
Fondo: Negro con 30% transparencia

Animación recomendada:
- Fade in (entrada suave)
- Typewriter (máquina de escribir)
- Scale (escala desde pequeño)
- Float (flotar suavemente)

Duración: 1-2 segundos de entrada
Sostenimiento: 2-3 segundos
Salida: Fade out (0.5s)
```

#### Paso a Paso: Texto Título

1. Toca "Texto" en barra de herramientas
2. Selecciona "Agregar texto"
3. Escribe tu texto: "Colección Luna 2026"
4. Toca el texto en timeline para editar
5. En panel inferior: fuente, tamaño, color
6. Ve a "Animación" → selecciona entrada/salida
7. Ajusta posición en preview

#### Configuración Detallada

```
PANEL DE TEXTO EN CAPCUT:

Posición: Toca y arrastra en preview
Tamaño: Pellizco para agrandar/reducir
Fuente: 50+ opciones incluidas
Color: Selector + código hex personalizado
Glow: Efecto de brillo (recomendado: sutil)
Outline: Borde (evitar grueso)
Sombra: Drop shadow (suave, offset 2-3px)

ANIMACIÓN DE ENTRADA:
- Elegir de lista
- Preview automático
- Ajustar duración

ANIMACIÓN DE SALIDA:
- Opcional, puede omitirse
- Si se incluye: fade out o scale down
```

### Texto Inteligente (Auto Captions)

CapCut puede generar subtítulos automáticamente:

1. Toca "Texto" → "Auto captions" o similar
2. Selecciona idioma: Español
3. Espera procesamiento (IA transcribe)
4. Revisa y corrige texto si necesario
5. Ajusta estilo: fuente, color, posición

**Práctica:** Agrega subtítulos automáticos a tu reel.

## 4. Música y Audio

### Selección de Música

```
CRITERIOS PARA MÚSICA EN REELS DE JOYERÍA:

TEMA: Elegante, sofisticado, moderno
VELOCIDAD: Media (no muy rápida, no muy lenta)
ESTILO: Lo-fi, piano, beats minimalistas
TEMPORALIDAD: Actual, no anticuado

FUENTES DE MÚSICA:
- CapCut built-in library (gratis, sin copyright)
- Audios populares de tendencias
- Tu propia música licenciada

EVITAR:
- Música religiosa o política
- Songs muy reconocidas (problemas de copyright)
- Audio con voces que interfieran con tu mensaje
```

### Sincronización de Audio

**Alinear beats con cortes:**

1. Agrega música al timeline (pista inferior)
2. Reproduce y escucha donde están los beats
3. Ajusta clips para que cortes coincidan
4. Usa la función "Beat" si CapCut la tiene
   (marca automática de beats en audio)

```
EJEMPLO DE SINCRONIZACIÓN:

Tiempo 0s: - Intro / Logo aparece → beat 1
Tiempo 2s: - Primera joya → beat 2
Tiempo 4s: - Close-up detalle → beat 3
Tiempo 6s: - Precio o CTA → beat 4

El corte coincide con el beat = más profesional
```

### Voiceover y Grabación

Para narrar sobre tus productos:

1. Toca "Audio" en barra
2. Selecciona "Grabar voz" o "Voiceover"
3. Habla con tono profesional
4. Ajusta volumen: reducir música de fondo
5. Posiciona en timeline donde corresponda

**Tip de volumen:**

```
NIVELES PARA JOYERÍA:

Música de fondo: 20-30% de volumen
Voiceover: 70-80% de volumen
(Primero debe escucharse la voz, luego la música)
```

## 5. Efectos y Overlays Profesionales

### Efectos Recomendados para Joyería

| Efecto            | Cuándo Usar              | Intensidad   |
| ----------------- | ------------------------ | ------------ |
| **Beauty**        | Retoque de piel/rostro   | Baja (1-10%) |
| **Vignette**      | Enfoque dramático        | 15-25%       |
| **Glow**          | Resaltar brillo metálico | Sutil        |
| **Color grading** | Tono unificado           | Moderada     |
| **Blur**          | Desenfocar fondo         | 20-40%       |

### Cómo Aplicar Efectos

1. Toca clip en timeline
2. Toca "Efectos" en barra de herramientas
3. Explora categorías:
   - Básico (ajuste de color)
   - Filtros (looks predefinidos)
   - Belleza (retoque)
4. Selecciona y ajusta intensidad
5. Toggle para activar/desactivar

### Overlay de Imagen (Picture-in-Picture)

Para mostrar detalle de joya sobre video:

1. Toca clip principal
2. Toca "Overlay" → "Agregar overlay"
3. Selecciona imagen (foto close-up)
4. Ajusta posición (esquina, centro)
5. Tamaño: 20-30% del frame
6. Duración: solo donde necesitas

```
EJEMPLO OVERLAY PARA JOYERÍA:

Video principal: Modelo usando collar
Overlay: Close-up del cierre dorado
Resultado: Ver detalles sin cambiar video
```

### Stickers y Emojis

**Uso estratégico:**

- Logo/watermark: Esquina inferior (sutil)
- Emoji de joyas: 💎✨👑 (pocos, no saturar)
- Stickers de tendencias: Usar solo si relevante

**Evitar:**

- Stickers que distraigan del producto
- Emojis excesivos (1-2 máximo)
- Animaciones muy rápidas que distraigan

## 6. Exportación y Optimización

### Configuración de Exportación

```
PARA INSTAGRAM REELS:

Resolución: 1080 x 1920 (9:16 vertical)
Codec: H.264 (más compatible)
FPS: 30 fps (suave, no necesita 60)
Calidad: Alta (8-10 Mbps mínimo)

Audio:
- AAC 128-256 kbps
- Estéreo

PARA TIKTOK:

Mismo formato que Instagram
Revisar requisitos actuales de TikTok
(A veces cambian especificaciones)

PARA YOUTUBE SHORT:

1080 x 1920
Bitrate más alto (10-12 Mbps)
```

### Proceso de Exportación

1. Toca "Exportar" (esquina superior derecha)
2. Selecciona resolución (1080x1920)
3. Selecciona FPS (30)
4. Ajusta calidad si quieres reducir peso
5. Toca "Exportar" y espera
6. Guardar en galería o compartir directamente

**Tip:** Activa "Guardar sin marca de agua" si CapCut lo ofrece (requiere Pro).

## 💡 Pro Tips

### Tip 1: Regla de los 3 Segundos

Para joyería, cada clip debe tener algo nuevo en máximo 3 segundos:

- Cambio de ángulo
- Zoom in/out
- Nuevo texto aparece
- Nuevo producto entra en frame

Si el video no muestra nada nuevo en 3 segundos, el espectador se aburre.

### Tip 2: Prioriza el Producto

```
JERARQUÍA VISUAL EN REELS DE JOYERÍA:

1. Producto (60-70% del frame)
2. Texto informativo (20-25%)
3. Elementos decorativos (5-10%)

Si tu joya no es lo más visible,
el video no está cumpliendo su función.
```

### Tip 3: Calidad sobre Cantidad

Es mejor 1 reel profesional que 5 reels mediocres. Invierte tiempo en:

- Buena iluminación
- Fondo limpio (o interesante)
- Estabilización (usa trípode si es necesario)
- Edición cuidada

### Tip 4: Previsualiza en Móvil

Todo lo que crees en CapCut debe verse bien en la pantalla del celular. Lo que se ve perfecto en preview de tablet o desktop puede verse pixelado o mal alineado en Instagram.

### Tip 5: Guarda Versiones Alternativas

Antes de exportar, guarda una copia del proyecto sin exportar. Si después quieres cambiar algo, tienes el archivo original.

## ❌ Common Mistakes

### Mistake 1: Transiciones Excesivas

**Problema:** Usar 4-5 transiciones diferentes en un reel de 15 segundos
**Resultado:** Contenido parece amateur, distraído
**Solución:** Máximo 2-3 transiciones. Usa las mismas o similar estilo.

### Mistake 2: Texto Confuso

**Problema:** Demasiado texto, tamaños variados, colores clash
**Resultado:** Mensaje se pierde, difícil de leer
**Solución:** Máximo 2-3 elementos de texto visibles al mismo tiempo. Consistencia de estilo.

### Mistake 3: Audio Desbalanceado

**Problema:** Música muy alta (cubre todo) o muy baja (no se nota)
**Resultado:** Experiencia de visualización mala
**Solución:** Prueba en tu teléfono (no en headphones de estudio). El audio debe estar balanceado para speakers de móvil.

### Mistake 4: Cortes Torpes

**Problema:** Cortes abruptos sin transición, donde movimiento se corta mal
**Resultado:** Saltos visibles que rompen inmersión
**Solución:** Añade transiciones suaves o asegúrate de cortar en momentos sin movimiento.

### Mistake 5: Exportar en Mala Calidad

**Problema:** Exportar en 720p o con compresión alta
**Resultado:** Video pixelado o con artifacts
**Solución:** Siempre exporta en 1080x1920, alta calidad. Instagram comprime, pero si mandas mala calidad, empeora.
