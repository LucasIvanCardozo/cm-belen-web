## 1. Por qué medir es fundamental

```
SIN MEDICIÓN:

CON MEDICIÓN:
```

## 2. Métricas principales

### 2.1 Impresiones y Alcance

```
IMPRESIONES (Impressions):
→ Cuántas veces se mostró tu ad
→ Puede ser la misma persona 3 veces = 3 impresiones

ALCANCE (Reach):
→ Cuántas personas únicas vieron tu ad
→ La misma persona 3 veces = 1 alcance

EJEMPLO:
Alcance: 5,000 personas
Impresiones: 12,000

→ Cada persona vio tu ad ~2.4 veces en promedio

CUÁNDO USAR:
→ Para medir reconocimiento (awareness)
→ Para saber cuánto está llegando tu marca
```

### 2.2 Costo por Mil (CPM)

```
CPM = Costo por cada 1,000 impresiones

FÓRMULA:
CPM = (Gasto total ÷ Impresiones) × 1,000

EJEMPLO:
• Gastaste: $50 USD
• Impresiones: 20,000
• CPM = ($50 ÷ 20,000) × 1,000 = $2.50 USD

BENCHMARK PARA JOYERÍA:
• Bueno: $2-5 USD
• Promedio: $5-10 USD
• Caro: > $10 USD

CUÁNDO IMPORTAR:
→ Para campañas de alcance
→ Para comparar eficiencia entre campañas
```

### 2.3 Clics y CTR

```
CLICKS (Link Clicks):
→ Número de veces que alguien hizo click en tu ad
→ Va a tu web, WhatsApp, etc.

CTR (Click-Through Rate):
→ Porcentaje de personas que hicieron click
→ vs las que vieron el ad

FÓRMULA:
CTR = (Clics ÷ Impresiones) × 100

EJEMPLO:
• Clics: 200
• Impresiones: 10,000
• CTR = (200 ÷ 10,000) × 100 = 2%

BENCHMARK PARA JOYERÍA:
• Bueno: 1-3%
• Promedio: 0.5-1%
• Bajo: < 0.5%

CTR alto = Tu ad es relevante para la audiencia
```

### 2.4 Costo por Clic (CPC)

```
CPC = Cuánto pagás por cada click

FÓRMULA:
CPC = Gasto total ÷ Clics

EJEMPLO:
• Gastaste: $30 USD
• Clics: 150
• CPC = $30 ÷ 150 = $0.20 USD

BENCHMARK PARA JOYERÍA:
• Bueno: $0.10-0.30 USD
• Promedio: $0.30-0.60 USD
• Caro: > $0.60 USD

CUÁNDO IMPORTAR:
→ Para campañas de tráfico
→ Cuánto te cuesta llevar alguien a tu web
```

### 2.5 Costo por Lead (CPL)

```
CPL = Cuánto pagás por cada lead (contacto)

FÓRMULA:
CPL = Gasto total ÷ Leads

EJEMPLO:
• Gastaste: $100 USD
• Leads (mensajes): 25
• CPL = $100 ÷ 25 = $4 USD

BENCHMARK PARA JOYERÍA:
• Bueno: $2-5 USD
• Promedio: $5-10 USD
• Caro: > $10 USD

CUÁNDO IMPORTAR:
→ Para campañas de lead generation
→ Si vendés por WhatsApp/mensajes
```

### 2.6 Costo por Venta (CPA)

```
CPA = Cuánto pagás por cada venta

FÓRMULA:
CPA = Gasto total ÷ Ventas

EJEMPLO:
• Gastaste: $200 USD
• Ventas: 4
• CPA = $200 ÷ 4 = $50 USD

BENCHMARK PARA JOYERÍA:
• Depende del ticket promedio
• Si vendés a $100, un CPA de $30 es bueno (30%)
• Si vendés a $500, un CPA de $50 es excelente (10%)

CUÁNDO IMPORTAR:
→ Para campañas de conversión
→ Si tenés e-commerce o vendés directamente
```

### 2.7 ROAS (Return on Ad Spend)

```
ROAS = Retorno sobre la inversión en ads

FÓRMULA:
ROAS = Ingresos ÷ Gasto en ads

EJEMPLO:
• Gastaste en ads: $100 USD
• Ingresos por ventas: $400 USD
• ROAS = $400 ÷ $100 = 4x (o 400%)

INTERPRETACIÓN:
• ROAS 1x = No ganaste ni perdiste (profit = 0)
• ROAS 2x = Ganaste el doble de lo que invertiste
• ROAS 4x = Ganaste 4 veces lo invertido

BENCHMARK PARA JOYERÍA:
• Mínimo aceptable: 2x
• Bueno: 3-4x
• Excelente: 5x+

NOTA:
ROAS ≠ Profit
Si tu producto cuesta 50% en producción:
ROAS 3x = Profit 1.5x (sobre la inversión en ads)
```

## 4. Errores comunes

### ❌ Error 1: Solo mirar likes

```
MAL:
"Tuve 500 likes en mi ad. ¡Funcionó!"

PROBLEMA:
Likes no pagan las cuentas.
Si no medís clics, leads o ventas, no sabés el impacto real.

BIEN:
"Tuve 500 likes, 80 clics, 15 leads, 3 ventas = $150 USD.
El ROAS es 3x. Está funcionando."
```

### ❌ Error 2: No dar tiempo

```
MAL:
"Llevé 2 días, no hay ventas. Cancelo."

PROBLEMA:
Meta necesita tiempo para optimizar.
El primer día es el peor día de cualquier campaña.

BIEN:
"Dejé correr 7 días. Obtuve 3 ventas.
Ahora voy a evaluar si optimizo."
```

### ❌ Error 3: Comparar cosas incomparables

```
MAL:
"Mi ad A tiene 200 clics. Mi ad B tiene 5 ventas.
B es mejor."

PROBLEMA:
Son métricas diferentes.
A = tráfico (no midió ventas)
B = ventas (midió ventas)

BIEN:
"Ad A: 200 clics, 5 ventas, CPA $20
Ad B: 50 clics, 10 ventas, CPA $10
Ad B es mejor porque tiene menor CPA."
```
