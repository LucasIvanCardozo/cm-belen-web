## 1. Los dos caminos para hacer publicidad en Meta

```
META TE OFRECE DOS FORMAS DE HACER PUBLICIDAD:

CAMINO 1: POTENCIAR (Boost)
   → Simple, rápido, desde el celular

   → Ideal para campañas serias y optimización
```

## 2. Potenciar un post (Boost)

### 2.1 Qué es Potenciar

```
POTENCIAR (BOOST):
= Tomás un post que ya publicaste y lo mostrás a más personas.

EJEMPLO:
Publicaste una foto de un collar.
Lo vieron 200 personas orgánicamente.
Potenciás el post por $10 USD.
Ahora lo ven 2,000 personas.
```

### 2.2 Cuándo usar Boost

```
✅ USÁ BOOST CUANDO:
• Querés probar rápido si un post funciona
• Tenés poco tiempo y necesitás algo simple
• Estás empezando y no conocés Ads Manager
• Querés aumentar alcance de un post específico
• Tenés presupuesto bajo (< $20 USD)
• Querés hacer un test rápido antes de invertir más

❌ NO USÉS BOOST CUANDO:
• Necesitás targeting muy específico
• Querés hacer retargeting (mostrar ads a personas que ya te visitaron)
• Tenés varios ads y necesitás comparar resultados
• Querés optimizar campañas basadas en resultados
• Necesitás control total sobre presupuesto y timing
```

### 2.3 Cómo potenciar un post (paso a paso)

```
DESDE INSTAGRAM:

1. ENTRÁ a tu post en Instagram
2. TOCÁ "Potenciar" (botón debajo del post)
3. SELECCIONÁ el objetivo:
   • Más personas vean tu publicación
   • Más visitas al sitio web
   • Más mensajes
4. DEFINÍ tu audiencia:
   • Automático (Meta elige)
   • O creá una audiencia personalizada
5. SETEÁ presupuesto y duración:
   • Presupuesto diario o total
   • Duración (días que corre el ad)
6. REVISÁ y tocá "Potenciar"
7. ESPERÁ aprobación de Meta (usualmente minutos)

COSTO MÍNIMO:
• Recomendado: $5-10 USD/día
• Mínimo absoluto: $1-2 USD/día (muy poco alcance)
```

### 2.4 Configuración de audiencia en Boost

```
OPCIONES DE AUDIENCIA EN BOOST:

1. AUTOMÁTICO
   → Meta muestra tu ad a quien cree que es relevante
   → Bueno para empezar sin complicaciones

2. UBICACIÓN
   → Elegí país, ciudad, o radio específico
   → Ejemplo: Argentina, Buenos Aires

3. EDAD
   → Rango etario: 18-65+, pero para joyería: 25-55 ideal

4. GÉNERO
   → Todos, Hombres, o Mujeres
   → Para joyería: usualmente Mujeres

5. INTERESES
   → Meta sugiere intereses basados en lo que elegís
   → Ejemplo: Joyería, Moda, Lujo, Regalos
   → CUIDADO: no tan específico o no llegás a nadie

CONSEJO PARA JOYERÍA:
Edad: 25-50
Género: Mujeres
Ubicación: Tu país/ciudad objetivo
Intereses: Joyería, Moda, Lujo, Regalos, Jewelry
```

## 3. Ads Manager

### 3.1 Qué es Ads Manager

```
ADS MANAGER:
= La herramienta profesional de Meta para crear y gestionar campañas.

A diferencia de Boost, te permite:
• Crear múltiples ads en una campaña
• Targeting más específico
• Retargeting (volver a alcanzar personas)
• Control granular de presupuesto
• A/B testing (probar variaciones)
• Ver métricas detalladas
• Crear diferentes tipos de objetivos
```

### 3.2 Cómo acceder a Ads Manager desde el celular

```
DESDE INSTAGRAM:
1. Entrá a tu perfil profesional
2. Tocá las tres líneas (menú)
3. Seleccioná "Meta Business Suite"
4. En el menú, buscá "Ads" o "Anuncios"
5. Tocá "Crear anuncio"

O DESDE FACEBOOK:
1. Abrí Facebook
2. Tocá el menú (tres líneas)
3. Buscá "Ads Manager" o "Gestor de anuncios"
4. Tocá "Crear"

DESDE LA WEB (si tenés acceso a PC):
1. Visitá business.facebook.com
2. Iniciá sesión con tu cuenta business
3. Buscá "Ads Manager" en el menú
```

### 3.3 Estructura en Ads Manager

```
INTERFAZ DE ADS MANAGER:

┌─────────────────────────────────────────────────┐
│ 📊 CAMPAIGN (Campaña)                          │
│ └─ Objetivo de negocio                         │
│    └─ ON (activo) / OFF (pausado)              │
│                                                 │
│    📁 AD SET (Conjunto de Anuncios)            │
│    └─ audiencia específica                      │
│    └─ presupuesto y schedule                    │
│    └─ ubicación de los ads                      │
│                                                 │
│       📄 AD (Anuncio)                          │
│       └─ creatività (fotos, videos, texto)      │
│       └─ link de destino                        │
│       └─ CTA (call to action)                  │
└─────────────────────────────────────────────────┘
```

### 3.4 Crear tu primera campaña en Ads Manager

```
PASO A PASO (DESDE EL CELULAR):

PASO 1: CREAR CAMPAÑA
• Tocá el botón "+"
• Elegí tu objetivo de campaña
  (Te explain más en Módulo 3)
• Nombrá tu campaña
• Dejá " campaign budget optimization" activado

PASO 2: DEFINIR AUDIENCIA (Ad Set)
• Nombre: algo descriptivo
  Ejemplo: "Mujeres 25-40 Buenos Aires joyería"
• Ubicación: Argentina → Buenos Aires
• Edad: 25-40
• Género: Mujeres
• Segmentación detallada:
  → Intereses: Jewelry, Joyería, Luxury fashion, etc.
• Conexiones: "Excluir personas que ya te siguen"
  (para alcanzar nuevos)

PASO 3: PRESUPUESTO Y PROGRAMACIÓN
• Presupuesto: $5-10 USD/día
• Programación: Continuo o fechas específicas
• Puja: Automática (Meta optimiza por vos)

PASO 4: CREAR EL ANUNCIO
• Selección de formato:
  → Post único
  → Carrusel (varias imágenes)
  → Colección
  → Video
• Elejí las imágenes/videos
• Escribí el texto del anuncio
• Agregá el link de destino
• Elegí el CTA (Ver más, Comprar ahora, etc.)
• Agregá tracking (pixel si está disponible)

PASO 5: REVISAR Y PUBLICAR
• Meta revisa tu ad (minutos a 24 horas)
• Una vez aprobado, comienza a correr
```

## 5. Cuándo usar cada uno

```
MATRIZ DE DECISIÓN:

¿TENÉS POCO TIEMPO Y QUERÉS PROBAR?
→ Boost

¿ARRANCÁS Y NO CONOCÉS ADS MANAGER?
→ Empezá con Boost, depois pasá a Ads Manager

¿QUERÉS VOLVER A ALCANZAR A QUIENES YA TE VISITARON?
→ Ads Manager (con retargeting)

¿TIENÉS VARIOS ADS Y QUERÉS COMPARAR?
→ Ads Manager

¿QUERÉS HACER UNA CAMPAÑA SERIA PARA UN CLIENTE?
→ Ads Manager

¿NECESITÁS REPORTAR RESULTADOS A UN CLIENTE?
→ Ads Manager (mejor reporting)

¿ESTÁS ENCONTRANDO QUÉ FUNCIONA?
→ Ads Manager (A/B testing)
```

## 6. Errores comunes

### ❌ Error 1: Potenciar sin tener contenido bueno primero

```
MAL:
"Tengo un post con 3 likes. Lo voy a potenciar."

PROBLEMA:
Potenciar no arregla contenido malo.
Si tu post no genera engagement orgánico,
probablemente tampoco lo genere con ads.

BIEN:
"Tengo un post que tiene 8% engagement.
Lo voy a potenciar para llegar a más personas."
```

### ❌ Error 2: No definir objetivo claro

```
MAL:
"Potencié para más personas sin pensar en qué quiero."

BIEN:
"Potencié para más visitas a mi web porque quiero que
conozcan mi tienda online."
```

### ❌ Error 3: Gastar en Boost cuando necesitás Ads Manager

```
SITUACIÓN:
Necesitás volver a alcanzar a personas que ya visitaron tu web
(para recordarles que tienen algo en el carrito).

SOLUCIÓN:
Esto NO se puede hacer con Boost.
Necesitás Ads Manager + Pixel de Meta.

HACERLO CON BOOST:
= Gastar dinero erreichtando a personas random
= No llegará a quienes realmente importan
```

### ❌ Error 4: No revisar resultados

```
MAL:
"Potencié y me olvidé."

BIEN:
"Potencié por $20 USD. Revisá resultados después de 3 días.
Obtuve 5,000 impresiones, 120 clicks.
Costo por click: $0.17 USD. Aceptable."
```

## 7. Ejercicio práctico

```
EJERCICIO 1: Potenciá un post

REQUISITO:
Tené un post publicado con al menos decent engagement.

ACCIONES:
1. Entrá a tu post
2. Tocá "Potenciar"
3. Configurá:
   - Objetivo: Más visitas al sitio web (si tenés web)
   - Audiencia: Mujeres 25-45, Argentina
   - Presupuesto: $5 USD/día
   - Duración: 5 días
4. Revisá la vista previa
5. Publicá

DESPUÉS DE 3-5 DÍAS:
1. Volvé al post
2. Tocá "Ver estadísticas del anuncio"
3. Anotá:
   - Cuántas impresiones
   - Cuántos clicks
   - Costo por resultado
   - Engagement adicional

ENTREGA:
Screenshot de resultados + reflexión:
¿Valió la pena? ¿Qué cambiarías?

---

EJERCICIO 2: Explorá Ads Manager

ACCIONES:
1. Abrí Meta Business Suite o Ads Manager
2. Navegá las secciones sin miedo
3. Identificá:
   - Dónde ves tus campañas activas
   - Dónde ves las métricas
   - Dónde creás nuevo ad

NO PUBLIQUES NADA AÚN.
Solo explorá y familiarizate con la interfaz.
```
