## 1. Conceptos básicos de presupuesto

```
PREGUNTA CLAVE:
"¿Cuánto tengo que gastar en ads?"

RESPUESTA HONESTA:
"No existe un número mágico. Pero existe una forma
de saber si estás gastando bien."

El presupuesto depende de:
1. Tu objetivo (visitas, leads, ventas)
2. Tu público (más específico = más caro)
3. Tu nicho (joyería = competitivo)
4. Tu capacidad financiera
```

## 2. Presupuesto Diario vs Lifetime

### 2.1 Presupuesto Diario

```
QUÉ ES:
La cantidad promedio que gastás por día.

EJEMPLO:
Presupuesto diario: $5 USD/día
→ Meta puede gastar entre $3.50 y $7 por día
→ En 30 días: aproximadamente $150 USD

CUÁNDO USARLO:
✓ Cuando querés control constante del gasto
✓ Para campañas de prueba
✓ Cuando no tenés fecha de fin definida
✓ Para objetivos de engagement o awareness
```

### 2.2 Presupuesto Lifetime (duración)

```
QUÉ ES:
El monto total que gastás durante toda la campaña.

EJEMPLO:
Presupuesto lifetime: $150 USD
Duración: 30 días
→ Meta distribuye los $150 a lo largo de los 30 días
→ Puede gastar más o menos en ciertos días

CUÁNDO USARLO:
✓ Cuando tenés fecha de inicio y fin clara
✓ Para campañas de temporada (Black Friday, San Valentín)
✓ Cuando querés que Meta optimice la distribución
✓ Para launches o eventos específicos
```

### 2.3 Comparación rápida

```
┌─────────────────┬──────────────────────┬──────────────────────┐
│                 │ Diario               │ Lifetime             │
├─────────────────┼──────────────────────┼──────────────────────┤
│ Control         │ Más granular         │ Meta optimiza        │
│ Flexibilidad    │ Menos flexible       │ Más flexible         │
│ Mejor para     │ Pruebas, largo plazo  │ Eventos, campañas    │
│ Aprendizaje    │ Estadístico estable   │ Puede variar mucho   │
└─────────────────┴──────────────────────┴──────────────────────┘

MI RECOMENDACIÓN PARA PRINCIPIANTES:
Empezá con presupuesto DIARIO.
Es más fácil de controlar y aprender.
```

## 3. Cuánto gastar: Guía práctica

### 3.1 Presupuesto mínimo viable

```
PARA PRUEBAS:
Presupuesto mínimo recomendado: $5-10 USD/día

Por qué no menos de $5:
• Meta necesita data para optimizar
• Con $1-2/día casi no hay resultados
• Es mejor hacer 1 campaña de $10 que 10 de $1

CUIDADO CON "TESTEAR GRATIS":
Meta no es gratis. Si ponés $1/día, gastás $1/día.
```

### 3.2 Rangos por objetivo

```
PRESUPUESTOS ORIENTATIVOS (USD/mes):

PARA AWARENESS (que te conozcan):
• Mínimo: $50-100/mes
• Bueno: $150-300/mes
• Bueno+: $300-500/mes

PARA ENGAGEMENT (interacción):
• Mínimo: $75-150/mes
• Bueno: $200-400/mes
• Bueno+: $400-600/mes

PARA LEADS (contactos):
• Mínimo: $100-200/mes
• Bueno: $300-500/mes
• Bueno+: $500-800/mes

PARA VENTAS:
• Mínimo: $150-300/mes
• Bueno: $400-700/mes
• Bueno+: $700-1000+/mes

NOTA: Son rangos orientativos. Ajustá según tu mercado.
```

### 3.3 Regla del 10%

```
REGLA SEGURA PARA NEGOCIOS PEQUEÑOS:

"No gastes más del 10% de tu ingreso mensual en ads."

EJEMPLO:
Ingreso mensual: $1,500 USD
10% = $150 USD/mes para ads

¿No sabés si las ads generan ventas?
→ Empezá con $50-75/mes
→ Mide resultados por 2-4 semanas
→ Si generás ROI positivo, subí el presupuesto
→ Si no generás nada, ajustá la estrategia

RIESGO DE GASTAR MUCHO SIN MEDIR:
→ "Gasté $500 en ads y no vendí nada"
→ Esto pasa cuando no se mide ni se optimiza
```

## 4. Sistema de Bidding

### 4.1 Qué es el Bidding

```
QUÉ ES:
Cuando ponés un anuncio, hay una "subasta" con otros anunciantes.
El bidding es cómo decidís cuánto estás dispuesto a pagar por resultado.

META ANALIZA:
• Tu oferta (cuánto pagás)
• La probabilidad de que ocurra la acción (engagement, click, venta)
• La calidad de tu anuncio
```

### 4.2 Tipos de Bidding

```
TIPO 1: Lowest Cost (Menor Costo) ⭐ RECOMENDADO
────────────────────────────────────────────────────
Meta optimiza para conseguir el menor costo posible.

✓ Funciona bien para la mayoría
✓ No tenés que pensar mucho
✓ Meta ajusta automáticamente

USAR CUANDO:
• Empezaste recién
• Tenés presupuesto limitado
• Querés simplicidad


TIPO 2: Cost Cap (Tope de Costo)
────────────────────────────────────────────────────
Ponés un máximo que estás dispuesto a pagar por resultado.
Meta trata de mantenerte cerca de ese costo.

✓ Más control sobre el costo
⚠️ Puede limitar el alcance si el tope es muy bajo

USAR CUANDO:
• Tenés un costo objetivo específico
• Ya conocés tu costo promedio


TIPO 3: Bid Cap (Tope de Oferta)
────────────────────────────────────────────────────
Ponés cuánto ofrecés en la puja.

⚠️ Para usuarios avanzados
⚠️ Puede hacer que no ganas la puja

USAR CUANDO:
• Tenés experiencia y entendés el mercado
```

### 4.3 Cómo funciona en la práctica

```
EJEMPLO SIMPLIFICADO:

Vos ofrecés: $0.50 por cada 1000 impresiones
Meta evalúa:
  → Tu anuncio es relevante
  → Tu oferta es $0.50
  → La puja promedio es $0.30

Resultado:
→ Tu anuncio compite bien
→ Meta muestra tu anuncio

PERO:

Si Meta nota que tu anuncio genera ventas,
puede mostrarlo más aunque ofrecés menos.
(Meta premia resultados, no solo dinero)
```

## 5. Costos por objetivo

### 5.1 Costos promedio en LatAm (2026)

```
LOS NÚMEROS SON ESTIMACIONES - VARÍAN MUCHO:

ALCANCE:
• Costo por 1,000 personas alcanzadas: $2-8 USD
• Depende de la especificidad del público

ENGAGEMENT:
• Costo por like: $0.05-0.30 USD
• Costo por comentario: $0.30-1.50 USD
• Costo por guardado: $0.10-0.50 USD

TRÁFICO:
• Costo por click: $0.15-0.80 USD
• Depende de qué tan atractivo es tu anuncio

LEADS:
• Costo por lead (formulario): $1-8 USD
• Depende de la calidad del lead y el nicho

VENTAS:
• Costo por compra: $3-25 USD
• Joyería = nicho de ticket medio-alto
• Puede valer la pena si el producto tiene buen margen

NOTA: Estos son rangos orientativos.
En mercados más competitivos (BsAs, México) puede ser más caro.
```

### 5.2 Cómo calcular si vale la pena

```
FÓRMULA DE ROI BÁSICO:

ROI = (Ingreso - Costo Ads) ÷ Costo Ads × 100

EJEMPLO:
Gastaste $100 en ads
Generaste $400 en ventas
ROI = ($400 - $100) ÷ $100 × 100 = 300%

SIGNIFICA:
Por cada $1 que gastás, ganás $3.

¿CUÁNDO CONVIENE INVERTIR EN ADS?
✓ Si ROI > 100% (ganás más de lo que gastás)
△ Si ROI está entre 0-100% (cubres el costo, no ganás)
✗ Si ROI < 0% (perdés dinero)
```

## 6. Estrategia de presupuesto para principiantes

### 6.1 Empezar pequeño

```
FASE 1: TEST ($5-10/día)
─────────────────────────
• 1 campaña de prueba
• 1 conjunto de anuncios
• 1-2 creativos
• Duración: 7 días mínimo

Qué aprendés:
→ Qué tipo de contenido funciona
→ A qué público le llega mejor
→ Cuánto cuesta el resultado

FASE 2: OPTIMIZAR ($15-30/día)
─────────────────────────────────
• Te quedás con lo que funcionó en Fase 1
• Escalás el presupuesto de lo que funciona
• Eliminás lo que no rindió

FASE 3: ESCALAR ($30-50+/día)
────────────────────────────────
• Ya sabés qué funciona
• Aumentás presupuesto gradualmente
• Probás nuevas variaciones
```

### 6.2 Reglas de oro del presupuesto

```
1️⃣ NO EMPIECES GASTANDO MUCHO
   "Voy a poner $500 y ver qué pasa"
   → ERROR: Podés tirar plata
   → CORRECTO: Empezá con $5-10/día

2️⃣ DEJÁ CORRER LA CAMPAÑA
   "Puse $10 y no pasó nada en 2 horas"
   → ERROR: Apagás todo
   → CORRECTO: Esperá 3-7 días para data significativa

3️⃣ NO CAMBIES TODO A LA VEZ
   "No funciona, voy a cambiar target + creatives + presupuesto"
   → ERROR: No sabés qué funcionó o no
   → CORRECTO: Cambiá una variable a la vez

4️⃣ ESCALÁ LO QUE FUNCIONA
   "Este anuncio tiene $0.10 CPC y estoy gustando $5/día"
   → DUPLICÁ el presupuesto
   → Dejá que el ganador gane más

5️⃣ MIDE SIEMPRE
   "Gasté $100 y no sé si funcionó"
   → ERROR: Tiraste $100
   → CORRECTO: Definí metricas ANTES de empezar
```

## 7. Errores comunes con presupuesto

### ❌ Error 1: Gastar sin medir

```
MAL:
"Pongo $20/día y que sea lo que sea."

BIEN:
"Pongo $5/día por 7 días.
Mido: ¿Cuántos leads conseguí? ¿A qué costo?
Decido si sigo, optimizo, o paro."
```

### ❌ Error 2: Apagar campañas muy rápido

```
MAL:
"3 horas y solo gasté $2. No funciona."

BIEN:
"Dejé correr 7 días. Gasté $35. Tuve 3 leads.
Costo por lead: $11.67. Analizo si vale la pena."
```

### ❌ Error 3: No escalar lo que funciona

```
MAL:
"Tengo un anuncio con $0.15 CPC y está great.
Pero tengo miedo de poner más plata."

BIEN:
"Este anuncio funciona. Voy a duplicar presupuesto
y ver si sigue siendo eficiente."
```

### ❌ Error 4: Gastar en lo que no funciona

```
MAL:
"Tengo 5 anuncios y solo 1 funciona.
Pero no quiero apagar los otros."

BIEN:
"El que no funciona = tira dinero.
Lo apago. Invierto ese presupuesto en el que funciona."
```
