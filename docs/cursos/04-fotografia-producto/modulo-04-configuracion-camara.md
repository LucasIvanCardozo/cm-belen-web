# Módulo 04 — Configuración de Cámara para Joyería

## Ajustá tu celular como un profesional

---

> _"La diferencia entre una foto expuesta correctamente y una subexpuesta se ve en los detalles. Y en joyería, los detalles lo son todo."_

---

## 1. Por qué la configuración importa

### El problema del auto mode:

```
¿QUÉ PASA EN AUTO?

El celular intenta adivinar:
├── Qué hay en la foto (rostro? paisaje? joyería?)
├── Cuánta luz hay
├── Qué ajustar para que la foto "se vea bien"

PROBLEMA:
→ El celular prioriza rostros
→ En joyería no hay rostros
→ La exposición se descontrola
→ Los brillos se queman
→ Los detalles se pierden
```

### Lo que vas a aprender:

```
CONFIGURACIÓN MANUAL PARA JOYERÍA:

✓ Exposición correcta
✓ White balance ajustado
✓ Enfoque preciso
✓ ISO bajo
✓ Velocidad de obturación adecuada
```

## 2. Modos de Cámara: Cuál Usar

### Los modos de tu celular:

```
MODOS DISPONIBLES (la mayoría de celulares):

📷 FOTO / PHOTO
→ Modo automático básico
→ Ideal cuando sabés lo que hacés

🔍 MACRO
→ Para objetos cercanos
→ Activa el enfoque cercano automáticamente
→ Disponible en: iPhone 13+, Samsung S21+, Pixel 6+
→ Si no tenés macro, usá zoom 2x

🏛️ RETRATO / PORTRAIT
→ Desenfoca el fondo
→ PUEDE SER ÚTIL para joyería lifestyle
→ CUIDADO: a veces hace cosas raras con brillos

🌙 NOCHE / NIGHT
→ Usa exposición larga
→ Para poquísima luz
→ En joyería: CASI NUNCA lo necesitás (usamos luz natural)

📹 VIDEO
→ Para reels y videos
→ Misma configuración que foto

⚙️ PRO / MANUAL (avanzado)
→ Control total de todos los ajustes
→ Disponible en: Samsung Pro Mode, algunos iPhones con apps
→ LO VAMOS A USAR en este módulo
```

### Mi recomendación para empezar:

```
MI RECOMENDACIÓN:

→ iPhone: Modo FOTO con estos ajustes
  (los configuramos abajo)

→ Android: Modo FOTO con estos ajustes
  O buscar "Pro" si querés más control

→ NO USAR: Retrato para fotos de producto
  (el desenfoque no es útil aquí)
```

## 3. Los Ajustes Esenciales

### Explicación rápida:

```
AJUSTES DE CÁMARA:

EXPOSICIÓN:
→ Qué tan bright u oscura es la foto
→ Cuanto más alto = más luminosa
→ Para joyería: generalmente -0.3 a -0.7 (más oscuro)

WHITE BALANCE (BALANCE DE BLANCOS):
→ La temperatura de color de la foto
→ Warm = anaranjado/amarillo
→ Cool = azulado
→ Para joyería: dependemos de la luz natural

ENFOQUE (FOCUS):
→ Qué parte de la foto está nítida
→ Para joyería: LA JOYA debe estar enfocado
→ Tocar la pantalla donde está la joya

ISO:
→ Sensibilidad del sensor a la luz
→ Bajo (100-400) = menos ruido, mejor calidad
→ Alto (800+) = más ruido, peor calidad
→ Para joyería: LO MÁS BAJO POSIBLE

VELOCIDAD DE OBTURACIÓN:
→ Cuánto tiempo está abierto el sensor
→ Rápido = congela movimiento
→ Lento = más luz pero puede blur
→ Para joyería con trípode: más lento está bien
```

## 4. Configuración para iPhone

### Paso a paso:

```
IPHONE - AJUSTES DE CÁMARA:

PASO 1: Abrí Configuración → Cámara

PASO 2: Ajustá:
├── Formato de Video: HEVC (ahorra espacio)
├── Grabar Fotos: Mantené Photo (foto original)
├── Ajuste de Cámara:
│   ├── Volver a Ajustar Exposición: OFF
│   ├── Nivel de Sonido: apagado
│   └── Retícula: ENCENDIDA (grid)
└── Formatos: Más Pro (para ProRAW si tenés iPhone 12+)

PASO 3: EN LA APP DE CÁMARA:
├── HDR Inteligente: ENCENDIDO
├── Estilos Fotográficos: Elegí "Rico Contraste" o "Estándar"
└── Flash: SIEMPRE APAGADO

PASO 4: EXPOSICIÓN MANUAL:
├── Tocá donde está la joya para enfocar
├── Deslizá hacia abajo para oscurecer (-0.3 a -0.7)
└── Verás un indicador de sol con número

PASO 5: LOCK DE EXPOSICIÓN:
├── Después de ajustar, mantené presionado
├── Verás "AE/AF Lock"
├── Esto mantiene la exposición aunque muevas
└── Para desbloquear: tocá cualquier parte de la pantalla
```

### Ajustes recomendados por tipo de joya:

```
POR TIPO DE JOYA:

ANILLOS CON PIEDRAS BRILLANTES:
→ Exposición: -0.7 (más oscuro para no quemar el brillo)
→ HDR: Encendido
→ Foco: En la piedra

COLLERES (METAL SOLO):
→ Exposición: -0.3 a -0.5
→ HDR: Encendido
→ Foco: Centro de la cadena

PULSERAS CON PIEDRAS:
→ Exposición: -0.5
→ HDR: Encendido
→ Foco: Parte más importante

AROS CON BRILLO:
→ Exposición: -0.7
→ HDR: Encendido
→ Foco: El pendiente
```

## 5. Configuración para Android

### Samsung (similar para otros Android):

```
SAMSUNG - AJUSTES DE CÁMARA:

PASO 1: Abrí Cámara → Ajustes (⚙️)

PASO 2: Ajustá:
├── Resolución de foto: La más alta
├── Formato: HEIC (más eficiente) o JPG
├── Estabilización de video: Encendida
├── Líneas de cuadrícula: ENCENDIDAS
├── Snap de gesto: Encendido
├── Guardar opciones: Original
└── Flash: APAGADO

PASO 3: MODO PRO (si lo tenés):
├── ISO: 100-200 (lo más bajo)
├── Velocidad obturación: 1/125 o más rápido
├── White Balance: Automático o luz del día
├── Foco: Centro o táctil
└── Exposición: -0.3 a -0.7

PASO 4: MODO FOTO ESTÁNDAR:
├── Tocá la joya para enfocar
├── Deslizá el sol hacia abajo (oscurecer)
├── Esperá el "bloqueo de exposición" si está disponible
└── Tomá la foto
```

### Mi App Recomendada para Android:

```
PARA MÁS CONTROL (Opcional):

OPEN CAMERA (GRATIS):
├── Modo Pro real
├── Control de ISO, shutter, focus
├── Histograma en tiempo real
├── Guardar en RAW (mejor para edición)

DESCARGAR:
→ Play Store: "Open Camera"
→ Totalmente gratis, sin ads
```

## 6. Técnica: Bloqueo de Exposición y Enfoque

### Por qué es crucial:

```
EL PROBLEMA CON AUTO:

Cuando tocás la pantalla:
1. El celular enfoca donde tocaste
2. Ajusta exposición para esa zona
3. PERO... si movés un poco, vuelve a calcular

CON LOCK:
1. Tocás la joya para enfocar
2. Ajustás exposición
3. "Lockeás" los dos
4. Ahora aunque muevas, no cambia

→ Esto es CRUCIAL para joyería
→ Resultados más consistentes
```

### Cómo hacer el lock:

```
IPHONE:
1. Abrí la app de cámara
2. Tocá y mantené sobre la joya (1-2 segundos)
3. Verás "AE/AF Lock" en la pantalla
4. Ya está bloqueado
5. Para desbloquear: tocá en otro lado

SAMSUNG:
1. Abrí la app de cámara
2. Tocá la joya para enfocar
3. Deslizá el sol para ajustar exposición
4. Buscá un ícono de candado o "lock"
5. Tocalo para bloquear
6. Para desbloquear: tocá en otro lado

OTROS ANDROID:
→ Puede variar según marca
→ Buscá "AE/AF Lock" o "Exposure lock" en Ajustes
```

## 7. Modo Macro: Tu Mejor Amigo

### Cómo usarlo:

```
¿QUÉ ES MACRO?
→ Modo para fotografiar cosas muy cercanas
→ Revela detalles que no ves a simple vista
→ Cracks, grabados, texturas, facetas de piedras

¿CÓMO ACTIVARLO?
├── iPhone 13 Pro+: Automático cuando acercás
├── iPhone 13: Disponible en modo Foto
├── Samsung: Buscá ícono de flor
├── Otros: Buscá "Macro" o "close-up"
└── Si no tenés macro: Usá zoom 2x

¿CUÁNDO USARLO?
→ Anillos con grabados internos
→ Piedras con inclusiones
→ Texturas del metal
→ Detalles de engaste
→ Monogramas o iniciales
```

### Ajustes para macro:

```
PARA MACRO:

✓ Modo: Macro o Foto con zoom 2x
✓ Iluminación: MUY importante (macro amplifica todo)
✓ Estabilidad: TRÍPODE OBIGATORIO (blur amplificado)
✓ Distancia: La que permita el celular (generalmente 10-15cm)
✓ Flash: NO (crea reflejos horribles)
✓ Trípode: SÍ (reduce vibración)

EJERCICIO:
1. Poné un anillo con la piedra cerca
2. Acercá hasta que cambie a modo macro
3. Tomá fotos a diferentes distancias
4. Encontrá el "sweet spot" (punto óptimo)
```

## 8. Ajustes por Tipo de Luz

### Luz del día soleado (indirecta):

```
PARÁMETROS:
├── Exposición: -0.5 a -1.0 (oscurecer para proteger brillos)
├── ISO: Automático (generalmente bajo en luz)
├── White Balance: Luz del día o Automático
└── HDR: Encendido

NOTA:
→ Más exposición negativa para piedras con brillo
```

### Día nublado:

```
PARÁMETROS:
├── Exposición: -0.3 a -0.5 (la luz es más suave)
├── ISO: Automático (puede ser más alto por menos luz)
├── White Balance: Automático o Nublado
└── HDR: Encendido

NOTA:
→ Menos corrección de exposición necesaria
→ La luz es naturalmente más suave
```

### Hora dorada:

```
PARÁMETROS:
├── Exposición: -0.3 (la luz es cálida y fuerte)
├── ISO: Automático
├── White Balance: Sombra o Automático
└── HDR: Encendido o apagado (experimentá)

NOTA:
→ La calidez de la luz puede ser nice o no
→ Para oro: puede realzar el color
→ Para plata: puede hacerlo ver más amarillo
```

## 9. Ajustes para Diferentes Efectos

### Foto más oscura y dramática:

```
PARA EFECTO LUXURY OSCURO:
├── Exposición: -1.0 a -1.5
├── Fondo oscuro (cartulina negra)
├── Contraste: Alto
└── Resultado: Look premium, dramático

CUÁNDO USARLO:
→ Campañas de lujo
→ Fotos para website premium
→ Colecciones especiales
```

### Foto clara y limpia:

```
PARA EFECTO CLARO Y AIREADO:
├── Exposición: +0.3 a 0
├── Fondo blanco
├── Contraste: Bajo a medio
└── Resultado: Look clean, minimalista

CUÁNDO USARLO:
→ Fotos de producto generales
→ E-commerce
→ Fotos para Mercado Shops
```

### Foto con colores saturados:

```
PARA COLORES INTENSOS:
├── Exposición: -0.3
├── Ajustá después en edición (aumentar saturación)
├── HDR: Encendido
└── Resultado: Piedras más vibrantes

CUÁNDO USARLO:
→ Piedras de colores (amatista, esmeralda, zafiro)
→ Fotos para redes que tienen que "gritar"
→ Campañas con colores vibrantes
```
