## 2. El Elemento Más Importante: TU CELULAR

### Qué buscás en un celular para joyería:

```
REQUISITOS MÍNIMOS:
├── Cámara de al menos 12MP
├── Modo macro o enfoque cercano
├── Pantalla AMOLED o OLED (para ver colores reales)
└── Batería que dure (o cargador cerca)

MEJOR PARA JOYERÍA:
├── iPhone 13 o superior (mejor procesamiento de color)
├── Samsung Galaxy S21+ o superior (buen modo macro)
├── Google Pixel (increíble procesamiento HDR)
├── iPhone 12 también funciona bien

¿No sabés qué tenés?
→ Buscá "specs [tu celular]" en Google
```

### Configuración de cámara recomendada:

```
AJUSTES DE CÁMARA:

□ Resolución: La más alta disponible (عادة 12MP o más)
□ Formato: HEIF (iPhone) o JPEG (Android)
□ HDR: Apagado o automático (vamos a usar manualmente)
□ Flash: SIEMPRE apagado
□ Temporizador: 3 segundos (para usar con trípode)
□ Grid: Encendido (para composición)
□ Estabilización: Encendida
```

## 3. Foam Board (Tablas de Espuma)

### Por qué son esenciales:

```
FOAM BOARD = LUZ DIFUSA + FONDO + REFLECTOR

Funciones:
├── Actúa como fondo (superficie curva)
├── Difunde la luz natural de la ventana
├── Refleja luz hacia sombras
└── Crea el "infinito" (fondo sin curvas)
```

### Dónde comprarlo:

```
OPCIONES:
├── Papelerías (Ej: Paper Center, Libersur)
├── Librerías grandes (Cuspide, El Ateneo)
├── Ferreterías (algún tipo)
└── Amazon / MercadoLibre (envío)

ESPECIFICACIONES:
├── Tamaño: 50x70cm o 70x100cm
├── Color: Blanco (para la mayoría)
├── Grosor: 3mm o 5mm
└── Precio: $2-5 USD por tabla

CANTIDAD RECOMENDADA:
→ Mínimo 3 tablas (2 para formar "V", 1 de repuesto)
→ Ideal 5 tablas (para tener más opciones)
```

### Cómo usarlo:

```
FORMA "V" (la más común):
       ___/\___
      /   |    \
     /    |     \
    /     |      \
   /      |       \
  (       |        )

Colocá las tablas formando una "V"
La joya va en el medio, en el "suelo" de la V
La curva elimina las uniones y crea fondo infinito

FORMA PLANA:
Usa 1 o 2 tablas como fondo directo
Más simple, menos curvo
Bueno para jewelry más grande
```

## 4. Trípode

### Por qué lo necesitás:

```
BENEFICIOS DEL TRÍPODE:
├── Fotos nítidas (sin blur por movimiento)
├── Permite temporizador (manos libres)
├── Consistencia en ángulos
├── Más fácil encuadrar
└── Enables toma de fotos con manos libres

NO ES OBLIGATORIO AL PRINCIPIO:
→ Podés empezar apoyándote en algo
→ Pero lo vas a extrañar cuando lo pruebes
```

### Opciones por presupuesto:

```
PRESUPUESTO BAJO ($10-15):
├── Trípode de celular básico (Amazon/MercadoLibre)
├── Buscar: "trípode celular flexible" o "trípode mini"
└── Ejemplo: Trípode Joby Gorillapod (~$15)

PRESUPUESTO MEDIO ($20-30):
├── Trípode de aluminio con cabeza de burbuja
├── Más estable, más duradero
└── Ejemplo: Trípode Fancier / Ravelli

PRESUPUESTO $0 (TRAMPA):
❌ NO uses libros/apoyos inestables
✓ SÍ: Apoyá el celular en una pila de libros FIJA
✓ SÍ: Usá la pared para sostener
✓ SÍ: Pedí prestado un trípode
```

### Características importantes:

```
BUSCAR EN UN TRÍPODE:
├── Compatible con tu celular (clip o rosca)
├── Altura mínima 30cm (para tabletop)
├── Altura máxima 1.5m (bueno tener)
├── Cabeza removible o ajustable
├── Patas flexibles (Joby) o rígidas (trípode común)
└── Precio: $10-30 USD
```

## 5. Reflector DIY

### Cómo hacer uno por $2:

```
MATERIALES:
├── 1 cartón grueso (caja de zapatos sirve)
├── Papel aluminio (de cocina)
└── Pegamento o cinta adhesiva

PASOS:
1. Cortá el cartón del tamaño que necesites
   (aproximado: 20x30cm)
2. Forrá una cara con papel aluminio (lado brillante afuera)
3. Dejá la otra cara blanca (o forrá con cartulina blanca)
4. Listo: tenés 2 en 1 (reflector y difusor)

USO:
→ Lado aluminio → refleja luz hacia zonas oscuras
→ Lado blanco → suaviza la luz
```

### Alternativas rápidas:

```
SITUACIÓN → SOLUCIÓN:
├── No tenés aluminio → Usá una hoja de блонд blanca
├── No tenés cartón → Usá una tabla de foam sola
├── Necesitás reflector ahora → Usá una cuchara опасаться сторону
├── Solo necesitás diff → Sostene white foam board detrás
```

## 8. Tu Setup Mínimo Viable

### Cómo armar en 5 minutos:

```
PASO 1: UBICACIÓN
→ Buscá una mesa cerca de una ventana grande
→ Ideal: ventana con cortina translúcida
→ Sino: sombra directa de la ventana

PASO 2: FONDO
→ Colocá 2 tablas de foam en forma de V
→ La unión va detrás de la joya
→ El "suelo" de la V es donde apoyás la joya

PASO 3: CÁMARA
→ Poné el celular en el trípode
→ Ajustá la altura para que la cámara esté параллельно a la joya
→ Encendé el grid en la cámara

PASO 4: LUZ (ajuste fino)
→ Si hay sol directo: esperá nublado o usá cortina
→ Si la luz es muy dura: agregá una tabla de foam como difusor
→ Mové el setup hasta que no haya sombras duras

PASO 5: PRUEBA
→ Poné una joya
→ Tomá 5 fotos
→ Revisá en tu pantalla (o mejor, en компьютер)
→ Ajustá posición/luz si hace falta
```
