# Módulo 09 — Presentación para Instagram

## Convertí tus fotos en contenido que vende

---

## 1. Instagram en 2026

```
INSTAGRAM 2026:
→ Video/reels dominan el algoritmo
→ Carouseles tienen alto guardado
→ Stories para conexión directa
→ Feed para portfolio y primera impresión

PARA JOYERÍA:
→ Las fotos de producto SON el producto
→ La calidad visual = calidad percibida del producto
```

---

## 2. Especificaciones para Feed

```
FORMATOS INSTAGRAM:
→ Cuadrado: 1:1 (1080x1080)
→ Retrato: 4:5 (1080x1350) ← RECOMENDADO
→ Horizontal: 1.91:1 (1080x566)

CALIDAD:
→ Instagram acepta hasta 8MB
→ Óptimo: 1080px del lado más largo
→ Guardar como JPEG, calidad 80-90%
```

---

## 3. Cómo Preparar Fotos

```
PASO A PASO:

1. EDITÁ LA FOTO (Snapseed/Lightroom)
2. AJUSTÁ EL TAMAÑO
   → En Snapseed: Herramientas → Recortar
   → Elegí 4:5 o 1:1
3. GUARDÁ EN ALTA CALIDAD
   → Exportar → Original
4. SUBÍ A INSTAGRAM
```

---

## 4. Grilla Consistente

```
ESTILOS DE GRILLA:

1. MONOCROMÁTICO
   → Fondo del mismo color
   → Muy limpio y minimalista

2. MIXED BRIGHTS
   → Alternar fondos (blanco/negro)
   → Más dinámico

3. LIFESTYLE + PRODUCT
   → Mix de fotos producto y lifestyle

4. CLEAN PRODUCT ONLY
   → Solo fotos de producto
   → Máximo profesional
```

---

## 5. Carouseles

```
CARRUSEL TIPO "PRODUCT SHOWCASE":

SLIDE 1: GANCHO
→ Imagen impactante de la joya

SLIDE 2: VISTA COMPLETA
→ Foto de producto limpio

SLIDE 3: DETALLE
→ Macro de piedra o engaste

SLIDE 4: EN CONTEXTO
→ Lifestyle (mano/modelo)

SLIDE 5: ESPECIFICACIONES
→ Material, piedra, tamaño

SLIDE 6: CTA
→ Precio o "DM para comprar"
```

## 7. Reels

```
REEL IDEAS PARA JOYERÍA:

1. SPARKLE SHOT
   → Video macro del brillo
   → 5-10 segundos

2. PRODUCT REVEAL
   → Mano cubriendo la joya
   → Reveal dramático

3. LIFESTYLE
   → Joya en contexto
   → 10-20 segundos

4. PROCESS
   → Setup de fotos → resultado
   → 15-30 segundos
```

---

## 8. Caption que Venden

```
ESTRUCTURA DEL CAPTION:

GANCHO (1-2 líneas):
→ Pregunta o statement impactante

CUERPO:
→ Historia o información
→ 2-4 párrafos máximo

CTA:
→ "DM para más info"
→ "Link en bio"
→ "Guarda para después"
```
