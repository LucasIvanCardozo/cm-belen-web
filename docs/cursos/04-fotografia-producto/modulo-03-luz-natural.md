# Módulo 03 — Luz Natural: Domando el Sol

## La mejor fuente de luz para joyería está en tu ventana

---

> _"Los fotógrafos profesionales pagan miles de dólares por luces que replican la luz del sol. Vos la tenés gratis, todos los días."_

---

## 1. Por qué la luz natural es la mejor para joyería

### Ventajas sobre luz artificial:

```
LUZ NATURAL vs LUZ ARTIFICIAL:

✓ LUZ NATURAL:
├── Es difusa naturalmente (el sol pasa por la atmósfera)
├── Reproduce colores con fidelity real
├── Es gratis
├── Está disponible todos los días
├── Crea sombras suaves naturalmente
└── El ojo humano está acostumbrado a ella

✗ LUZ ARTIFICIAL:
├── Puede crear tonos raros (temperatura de color)
├── Necesita modificación para ser suave
├── Requiere inversión en equipment
├── Puede parpadear (luces LED baratas)
└── Puede no ser consistente (diferentes marcas)
```

### El problema: no toda luz natural es igual

```
LUZ NATURAL "MALA":
├── Sol directo → sombras duras, sobreexposición
├── Mediodía (12-3pm) → luz muy harsh
├── Sombra parcial →la mitad iluminado, mitad oscuro
└── Luz mixta (sol + lámpara) → tonos raros

LUZ NATURAL "BUENA":
├── Luz indirecta de ventana → suave y difusa
├── Día nublado → nubes como giant softbox
├── Hora dorada (temprano/tarde) → cálido y suave
└── Sombra completa → sin sombras duras
```

## 2. Las 4 Cualidades de la Luz

### Cómo evaluar la luz:

```
1. DIRECCIÓN
   ¿De dónde viene la luz?

   ├── Frontal (de frente) → iluminación pareja
   ├── Lateral (de lado) → crea volumen y sombras
   ├── Backlight (desde atrás) → silhouette o glow
   └── Desde arriba → sombras naturales hacia abajo

2. DUREZA
   ¿Qué tan suave o dura es?

   ├── Luz dura: sombras definidas, bordes sharp
   ├── Luz suave: sombras suaves, bordes difusos
   └── Para joyería: generalmente queremos suave

3. CALIDAD
   ¿Grande o pequeña la fuente?

   ├── Fuente grande (nube, ventana grande) → suave
   ├── Fuente pequeña (sol directo) → dura
   └── Para joyería: fuente grande = mejor

4. COLOR/TEMPERATURA
   ¿Cálida o fría?

   ├── Luz cálida (naranja/amarillo) → atardecer
   ├── Luz neutral → mediodía nublado
   ├── Luz fría (azul) → sombra, invierno
   └── Para joyería: depende del metal
```

## 3. Las Mejores Horas para Fotografiar

### El horario importa:

```
HORARIO ÓPTIMO PARA JOYERÍA:

🌅 HORA DORADA (IDEAL):
├── 6-9am o 5-7pm (depende de tu ubicación)
├── Luz cálida y suave
├── Sombras suaves y largas
├── Perfecta para: oro yellow/gold
└── Cuidado: cambia rápido, tenés poco tiempo

⛅ DÍA NUBLADO (SEGUNDO MEJOR):
├── Cualquier hora cuando está nublado
├── Las nubes actúan como giant softbox
├── Luz súper difusa y uniforme
├── Perfecta para: plata, piedras blancas
└── Consejo: sale afuera, no solo cerca de ventana

🌤️ MEDIODÍA NUBLADO (MI TERCER FAVORITO):
├── 11am-2pm cuando está nublado
├── Luz alta y difusa
├── Mínimo de sombras
└── Ideal para: todo tipo de joyería

❌ EVITAR:
├── Sol directo (cualquier hora)
├── Mediodía soleado (sombras duras)
├── Luz mixta (sol + lámpara)
└── Luz de tungsten (lámparas viejas)
```

### Cómo identifictu hora dorada:

```
MÉTODO SIMPLE:
1. Abrí una app de clima en tu celular
2. Buscá "hora del atardecer" para tu zona
3. La hora dorada de la mañana = 1-2 horas después del amanecer
4. La hora dorada del atardecer = 1-2 horas antes del atardecer

MÉTODO OBSERVACIÓN:
1. Salí a tu ventana cada hora
2. Notá dónde está el sol
3. Observá cómo cambia la calidad de la luz
4. Identificá cuándo la luz se ve más suave
```

## 4. Posicionamiento del Setup

### Cerca de ventana:

```
SETUP BÁSICO CERCA DE VENTANA:

         ☀️ SOL (fuera de la habitación)
            ↓ (no debe dar directo)
         ┌─────────┐
         │ VENTANA │
         └─────────┘
              ↓ (luz difusa)
         ┌─────────┐
         │  FOAM   │ ← difusor (si necesitás)
         └─────────┘
              ↓
         ┌─────────┐
         │  JOYA  │ ← en el centro
         └─────────┘
              ↓
         ┌─────────┐
         │  FOAM   │ ← V en el fondo
         └─────────┘
              ↓
            📱 CÁMARA

REGLAS:
→ La joya debe estar paralela a la ventana
→ El sol NO debe dar directo en la joya
→ La cámara debe estar del lado opuesto al sol
```

### Luz lateral vs frontal:

```
LUZ FRONTAL (más común):
Joyería iluminada desde la dirección de la cámara.
Resultado: iluminación pareja, pocos contrastes.
Mejor para: fotos donde ves la piedra central.

LUZ LATERAL (más dramática):
Joyería iluminada desde un lado.
Resultado: más volumen, sombras, profundidad.
Mejor para: mostrar textura y detalles del metal.

LUZ DESDE ATRÁS (backlight):
Luz detrás de la joya (la joya está entre luz y cámara).
Resultado: glow suave, silhouette, o reflection.
Mejor para: piedras transparentes, pearls.

EXPERIMENTÁ: Mové el setup y mirá cómo cambia la foto.
```

## 5. Controlando la Luz con lo que Tenés

### Técnicas de modificación:

```
PROBLEMA → SOLUCIÓN CASERA:

❌ Luz muy dura (sol directo):
→Solución: Poné una sábana blanca o foam board
  entre la ventana y la joya (difusor)
→ También: Esperá a que pase una nube

❌ Sombras muy oscuras:
→Solución: Poné un reflector blanco o aluminio
  en el lado opuesto a la ventana
→ Refleja luz hacia las sombras

❌ Luz muy fría (sombra azul):
→Solución: Usá una hoja de papel dorado
  como reflector (calienta la luz)

❌ Mucho contraste:
→Solución: Poné foam boards a ambos lados
  para fill shadows
```

### El poder del difusor DIY:

```
DIFUSOR CASERO (gratis si tenés foam):

OPCIONES:
├── Sábana blanca (de tu cama)
├── Cortina translúcida
├── Foam board blanco
├── Papel de seda
└── Bolsa de consorcio blanca (limpia)

CÓMO USARLO:
1. Ponelo entre la ventana y la joya
2. La luz que llega a la joya es más suave
3. Eliminá (o reducís) las sombras duras
4. IMPORTANTE: Debe estar cerca de la joya, no de la ventana
```

## 6. Fotografiando en Exterior

### Cuándo y dónde:

```
PARA QUÉ FUNCIONA BIEN:
├── Luz natural 100%
├── Flat lays grandes (necesitás espacio)
├── Fotos lifestyle (en contexto real)
├── Videos (mejor luz que interior)
└── Detrás de escenas (storytelling)

UBICACIONES RECOMENDADAS:
├── Bajo un árbol (luz filtrada)
├── Balcón con sombra
├── Patio con toldo
├── Galería cubierta
├── Puerta-ventana
└── Near grandes ventanas de vidrio

⚠️ PRECAUCIONES:
→ El viento puede mover la joya
→ Hay que llevar todo el setup
→ El polvo/viento puede ser un problema
→ Difícil de replicar exactamente
```

### Técnica para exterior:

```
PASOS PARA SESIÓN EN EXTERIOR:

1. ELEGÍ LA HORA
   → Hora dorada o día nublado
   → Nunca sol directo

2. PREPARÁ EL ÁREA
   → Buscá sombra completa
   → Usá una superficie limpia
   → Tené peso o pinzas para el viento

3. ARMÁ EL SETUP
   → Foam boards para fondo y reflector
   → La joya no se mueve sola (usá peso o clay)

4. DISPARÁ RÁPIDO
   → La luz cambia rápido afuera
   → Tené todo listo antes de sacar

5. TOMÁ MUCHO
   → Exterior = más variables
   → Más fotos = más opciones
```

## 7. Señales de que la Luz es Correcta

### Checklist visual:

```
¿LA LUZ ES BUENA?

□ ¿Las sombras son suaves (no tienen bordes duros)?
□ ¿No hay puntos de luz brillante en la joya?
□ ¿Los colores se ven naturales?
□ ¿La joya no está sobreexpuesta?
□ ¿Las sombras no son completamente negras?
□ ¿El fondo está iluminado uniformemente?

SI TODO SÍ → ¡Luz correcta!
SI ALGUNO NO → Ajustá posición o agregá difusor/reflector
```

## 8. Errores Comunes con Luz Natural

### Error 1: Sol directo

```
❌ QUÉ PASA:
Hot spots (puntos muy brillantes)
Sombras muy oscuras
La joya parece "lavada"
Reflejos blancos feos

✅ CÓMO ARREGLARLO:
→ Mover la joya a sombra
→ Usar un difusor (sábana, cortina)
→ Esperar nublado
→ Cambiar hora
```

### Error 2: Luz de solo un lado

```
❌ QUÉ PASA:
Un lado de la joya iluminado
El otro lado muy oscuro
No se ven detalles en la zona oscura

✅ CÓMO ARREGLARLO:
→ Agregar reflector en el lado oscuro
→ Mover la joya para que la luz venga de más direcciones
→ Usar foam board como fill
```

### Error 3: Luz mezclada

```
❌ QUÉ PASA:
Tonos raros en la foto
La joya se ve de un color y la sombra de otro
La temperatura de color no es consistente

✅ CÓMO ARREGLARLO:
→ Apagá todas las luces artificiales
→ Cerrá cortinas si entra luz mixta
→ Solo dejar la ventana como fuente
```

## 9. Patrón de Luz para Diferentes Joyas

### Cómo ajustar según el tipo:

```
ANILLOS:
→ Luz frontal suave
→ O luz lateral para mostrar el perfil
→ El brillo viene de arriba (ventana)

COLLARES:
→ Flat lay: luz uniforme desde arriba
→ Colgado: luz lateral suave
→ O backlight para piedras transparentes

PULSERAS:
→ Similar a anillos
→ Luz frontal + reflector debajo

AROS:
→ Luz frontal
→ Intentar capturar el shine desde el frente
→ A veces mejor con luz ligeramente lateral

PENDIENTES:
→ Colgando: luz de frente o ligeramente lateral
→ Capturar cómo oscilan con luz suave
```
