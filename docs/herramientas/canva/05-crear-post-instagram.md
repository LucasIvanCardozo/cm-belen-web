# 📸 05. Crear Posts para Instagram

---

## 🎯 Objetivo

Al terminar esta sección, vas a saber crear posts profesionales para Instagram desde Canva móvil: dimensiones correctas, estructura visual efectiva, texto overlay, y cómo diseñar para diferentes tipos de posts (producto, quote, lifestyle, carrusel intro).

---

## 📐 Dimensiones Correctas para Instagram

| Formato | Dimensiones | Uso |
|---------|------------|-----|
| **Cuadrado** | 1080 × 1080 px | Posts clásicos, más Engagement |
| **Vertical (4:5)** | 1080 × 1350 px | Más espacio para texto, tendencia actual |
| **Horizontal (1.91:1)** | 1080 × 566 px | Variación, menos común |
| **Retrato 4:3** | 1080 × 1350 px | Similar al vertical, menos usado |
| **Story** | 1080 × 1920 px | Para convertir a post (no recomendado) |

> **Pro tip:** El formato **vertical 4:5 (1080×1350)** está trending en 2025-2026 porque ocupa más espacio en el feed y genera más interacción. Pero la versión **cuadrada 1:1** sigue siendo segura y efectiva. Tené ambas plantillas listas.

### Cómo elegir el tamaño en Canva

1. Crear nuevo → **Diseño personalizado**.
2. Buscá abajo el selector de dimensiones.
3. Tocá el tamaño actual o el ícono de ⚙️ (ajustes).
4. Seleccioná:
   - **"Instagram Post (1:1)"** para cuadrado
   - **"Instagram Post (4:5)"** para vertical
   - O ingresá números manualmente: 1080 × 1080

## 🎨 Tipos de Posts para Joyería

### Tipo 1: Post de Producto (Single)

**Objetivo:** Mostrar una pieza específica.

**Estructura visual:**
```
┌─────────────────────────┐
│                         │
│       [FOTO DEL         │
│        PRODUCTO]        │
│                         │
│   ─────────────────     │
│                         │
│   💎 Nombre de la pieza │
│   $XXX                  │
│                         │
│   [Hashtag o CTA]       │
│                         │
└─────────────────────────┘
```

**Pasos para crear:**

1. Creá un diseño 1080×1080 (o 1080×1350).
2. Tocá **📷 Fotos** → **Subir** → seleccioná la foto de tu joya.
3. Ajustá la foto: tocá y arrastrá para posicionarla bien.
4. Tocá el borde de la foto → pellizcá para hacer zoom si es necesario.
5. Agregá el nombre del producto: tocá **🔤 Texto** → **Agregar encabezado**.
6. Escribí el nombre, elegí fuente elegante (Playfair, Montserrat Bold).
7. Agregá el precio debajo.
8. Si hay espacio, agregá un CTA: "Disponible en DM" o "Link en bio".

**Ejemplo para anillo de oro:**
```
💍 Anillo Solitario
18K Gold | Zafiro azul
$890

✨ Pedido por DM
#joyasdelujo
```

### Tipo 2: Post Quote/Motivacional

**Objetivo:** Conectar emocionalmente, no vender directamente.

**Estructura visual:**
```
┌─────────────────────────┐
│  [FONDO ELEGANTE]        │
│   (negro, dorado,        │
│    o foto blurred)       │
│                         │
│  ┌───────────────────┐  │
│  │                   │  │
│  │  "El lujo no es   │  │
│  │   sobre el precio │  │
│  │   sino sobre      │  │
│  │   lo que sientes"  │  │
│  │                   │  │
│  └───────────────────┘  │
│                         │
│  @nombredeMarca         │
└─────────────────────────┘
```

**Pasos para crear:**

1. Nuevo diseño → 1080×1080.
2. Fondo: elegí un color sólido (negro #1A1A1A) o textura.
   - Para textura: 📷 Fotos → buscar "marble texture" o "gold texture".
3. Tocá **🔤 Texto** → **Agregar encabezado**.
4. Escribí tu frase.
5. Ajustá: toca dos veces rápido → tipo de letra, tamaño, color.
6. Para citar: usá comillas tipográficas " " (las encontrás en el teclado).
7. Centrá el texto verticalmente.
8. Agregá tu usuario de marca abajo.

> **Pro tip:** Para quotes de joyería, usá frases que hablen de:
> - Amor y compromiso: "Un anillo que cuenta tu historia"
> - Valor intrínseco: "El oro nunca pierde su brillo, como tú"
> - Autoestima: "Merecés brillar"
> - Tradición: "Hecho a mano, pensado para durar"

### Tipo 3: Post Colección/Collage

**Objetivo:** Mostrar varias piezas juntas.

**Estructura visual:**
```
┌─────────────────────────┐
│ ┌─────────┬─────────┐   │
│ │         │         │   │
│ │   Foto  │   Foto  │   │
│ │    1    │    2    │   │
│ │         │         │   │
│ ├─────────┼─────────┤   │
│ │         │         │   │
│ │   Foto  │   Foto  │   │
│ │    3    │    4    │   │
│ └─────────┴─────────┘   │
│                         │
│   Nueva Colección       │
│   Primavera 2026        │
└─────────────────────────┘
```

**Pasos para crear:**

1. Nuevo diseño → 1080×1080.
2. Tocá **📐 Grids** → buscá "2x2" o "Grid collage".
3. Seleccioná el grid que más te guste.
4. Tocá cada celda del grid → subí las fotos de tus joyas.
5. Ajustá cada foto: pellizcá y arrastrá para encuadrar.
6. Agregá texto debajo o encima: nombre de la colección.
7. Agregá tu logo o usuario.

### Tipo 4: Post Before/After o Comparación

**Objetivo:** Mostrar transformación o diferencia de calidad.

**Estructura visual:**
```
┌─────────────────────────┐
│ ┌───────────┬───────────┐│
│ │           │           ││
│ │   ANTES   │  DESPUÉS  ││
│ │  (sin luz │ (con luz  ││
│ │   natural)│  natural)  ││
│ │           │           ││
│ └───────────┴───────────┘│
│                         │
│   ¿Ves la diferencia?   │
│   ➡️ Brillo real        │
│                         │
└─────────────────────────┘
```

**Pasos para crear:**

1. Nuevo diseño → 1080×1080.
2. Usá un **grid de 2 columnas** (buscá en Grids).
3. Subí la foto "antes" a la izquierda.
4. Subí la foto "después" a la derecha.
5. Agregá texto "ANTES" y "DESPUÉS" con íconos.
6. Agregá tu mensaje: "Diferencia de calidad", "Así se ve con luz natural".

> **Pro tip:** Este formato funciona muy bien para mostrar:
> - Calidad real vs fotos de baja resolución de otros
> - Pieza recién limpiada vs sucia
> - Con iluminación profesional vs sin ella

## ✏️ Añadir Texto Overlay Profesional

### Reglas de texto para Instagram

| Regla | Por qué |
|-------|---------|
| **Máximo 3 líneas** | En celular no se lee más |
| **Fuente grande** | Mínimo 10% del alto de la imagen |
| **Contraste alto** | Texto claro sobre fondo oscuro, o viceversa |
| **Máximo 2 fuentes** | Mezclar más es ruido visual |
| **Alineado al centro o bordes** | Evita texto "flotando" sin referencias |

### Paso a paso para texto overlay:

1. Tocá **🔤 Texto** en la barra de herramientas.
2. Elegí **"Agregar encabezado"** (título grande).
3. Escribí tu texto.
4. Tocá dos veces rápido para editar.
5. En el panel superior:
   - **Fuente:** elegí una elegante (Playfair, Cinzel, Montserrat Bold)
   - **Tamaño:** arrastrá el slider para agrandar
   - **Color:** tocá el color → elegí uno que contraste con el fondo
   - **Alineación:** ≡ centrado
6. Para agregar sombra al texto (mejor legibilidad):
   - Panel de texto → **Efectos** → **Sombra** → ajustá blur e intensidad

### Efectos de texto populares para joyería:

| Efecto | Cuándo usarlo |
|--------|--------------|
| **Neón glow** | Textos en foto oscura, estilo moderno |
| **Sombra suave** | Textos sobre fondos con textura |
| **Outline (contorno)** | Texto blanco sobre foto clara |
| **Gradient de color** | Títulos premium, combinar dorado y plateado |
| **Letter spacing** | Dar aire al texto, elegancia minimalista |

> **Pro tip:** Agregá una línea decorativa debajo o encima del texto para darle estructura. Buscá en **Elementos** → "line" o "divider". Usá líneas doradas finas para joyería.

## 🎯 Posición del Texto en el Canvas

### Guía de posicionamiento:

```
┌─────────────────────────────────────┐
│                                     │
│  ┌───────────────────────────────┐ │
│  │         ARRIBA (N3)            │ │  ← Línea de seguridad
│  │   No poner texto aquí          │ │
│  └───────────────────────────────┘ │
│                                     │
│                                     │
│         ZONA SEGURA (N5)            │  ← Donde va el texto
│    (evitar bordes externos 150px)    │
│                                     │
│                                     │
│  ┌───────────────────────────────┐ │
│  │         ABAJO (N7)            │ │  ← Para hashtags/copyright
│  │   Hashtags, logo, CTA         │ │
│  └───────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

**Zonas de texto recomendadas:**
- **Centro** → Quotes, mensajes principales
- **Centro superior** → Títulos, nombres de producto
- **Centro inferior** → Precio, CTA, hashtags

> **Pro tip:** No pongas texto en el margen inferior si ya tenés un CTA de Instagram (el de "Más" con los tres puntitos). Se superpone y no se lee bien. Dejá 200px libres abajo.

## 📏 Margen de Seguridad (Safe Zone)

Cuando diseñás para digital,hay elementos que se cortan según la pantalla:

| Plataforma | Margen seguro |
|------------|--------------|
| Instagram Feed | 150px de los bordes externos |
| Instagram Stories | 250px de los bordes (por las esquinas redondeadas del preview) |

**Regla:** Todo texto e información importante debe estar **dentro del 80% central** del canvas.

## ❌ Errores Comunes

### Error 1: Texto muy pequeño
**Problema:** Poner texto que se puede leer solo en desktop, no en celular.
**Solución:** El texto de título debe ser mínimo 80-100px de altura visual en el canvas 1080×1080. Si no se ve claramente en la preview de tu celular, agrandalo.

### Error 2: Demasiado texto
**Problema:** Escribir un párrafo completo sobre la joya.
**Solución:** Instagram es visual. Si necesitás explicar algo, usá el caption del post, no el diseño. En la imagen solo: nombre + precio + máximo 3 palabras de valor.

### Error 3: Mala calidad de imagen
**Problema:** Usar fotos pixeladas o borrosas.
**Solución:** Solo usá fotos en alta resolución (mínimo 1080px en el lado más corto). Si tu foto no es buena, mejor solo texto elegante que una foto fea.

### Error 4: No dejar espacio para el logo
**Problema:** Diseñar sin pensar en dónde va tu marca.
**Solución:** Siempre dejá espacio visible para tu logo o @usuario. Puede ir en una esquina (protección de marca) o abajo (contexto).

### Error 5: Fondo muy cargado
**Problema:** Poner texto sobre una foto muy compleja.
**Solución:** Agregá un overlay semi-transparente entre la foto y el texto (negro con 30-40% opacidad) o usa fondos sólidos/gradientes para textos.

---

## 📌 Resumen

- Usá **1080×1080** para posts cuadrados, **1080×1350** para vertical (trending).
- Tipos de posts para joyería: **producto, quote, colección, antes/después**.
- El texto debe ser **máximo 3 líneas, grande, con alto contraste**.
- Posicioná el texto en el **centro vertical** de la zona segura.
- Dejá **margen de 150px** de los bordes.
- La foto debe ser **mínimo 1080px** y de alta calidad.

---

## ✏️ Mini-Acción

1. Creá un diseño **1080×1080** para Instagram.
2. Subí una foto de una joya (o usá una de stock).
3. Agregá texto overlay:
   - Nombre de la pieza (arriba o centro)
   - Precio (si aplica)
   - Un CTA de una palabra: "Disponible" o "DM" o "Link en bio"
4. Agregá tu @usuario en la esquina.
5. Agregá una línea decorativa entre el nombre y el precio.
6. Exportá en **PNG** (para máxima calidad) o **JPG** (para menor peso).
7. Guardá este diseño como **"Plantilla post producto"**.

> *Tu primer post profesional de joyería está listo. Ajustalo, experimentá, y encontrá tu estilo.*
