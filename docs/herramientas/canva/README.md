# 🛠️ Guía de Canva

**Diseño profesional desde la palma de tu mano** 🎨💗

---

> *"No necesitás ser diseñadora para crear contenido visual increíble."*

---

## 📚 Descripción

Canva es la herramienta de diseño más popular del mundo, y con razón. Esta guía te transforma de usuaria básica a **diseñadora profesional de contenido** para redes sociales, todo desde tu celular.

**En esta guía vas a aprender:**

- ✅ Navegar la interfaz móvil de Canva
- ✅ Crear y configurar tu cuenta de negocio
- ✅ Usar y personalizar plantillas
- ✅ Crear posts para Instagram
- ✅ Diseñar carruseles irresistibles
- ✅ Hacer Stories y covers para Reels
- ✅ Editar fotos como profesional
- ✅ Trabajar con texto y tipografía
- ✅ Mantener consistencia de marca

- Teléfono Android (8+) o iPhone (iOS 14+)
- App Canva instalada (gratuita, con opción Pro)
- Conexión a internet para funciones completas
- Opcional: Canva Pro para Brand Kit avanzado
