# 🎠 06. Crear Carruseles para Instagram

---

## 🎯 Objetivo

Al terminar esta sección, vas a saber crear carruseles profesionales en Canva móvil: estructura de slides, transiciones visuales, tips de coherencia, y cómo diseñar carruseles educativos, de colección, y de storytelling para marcas de joyería.

---

## 📱 ¿Qué es un Carrusel de Instagram?

Un carrusel es una **serie de slides** (de 2 a 20 imágenes) que el usuario navega deslizando horizontalmente.

**Ventajas del carrusel:**
- Más tiempo de visualización del contenido
- Mayor tasa de guardado y compartidos
- Permite contar historias más completas
- Ideal para tutoriales, tips, y colecciones

**Desventajas:**
- Requiere más esfuerzo de diseño
- Cada slide debe mantener coherencia
- El primer slide debe ser muy atractivo para que quieran seguir deslizando

---

## 📐 Dimensiones para Carruseles

| Formato | Dimensiones | Uso |
|---------|------------|-----|
| **Cuadrado (1:1)** | 1080 × 1080 px | El más común, fácil de navegar |
| **Vertical (4:5)** | 1080 × 1350 px | Más espacio para texto |
| **Mixto (cuadrado + vertical)** | ⚠️ Evitar | Puede verse mal |

> **Pro tip:** Mantené el **mismo formato** para todos los slides del carrusel. Mezclar cuadrado con vertical puede causar que Instagram ajuste automáticamente y se vea desproporcionado.

## 🎨 Tipos de Carruseles para Joyería

### Tipo 1: Carrusel Educativo (Tips/Tutorial)

**Objetivo:** Aportar valor, posicionar como experta.

**Estructura (ejemplo de 5 slides):**

```
Slide 1 (Portada):
┌─────────────────────────┐
│                         │
│   ┌─────────────────┐   │
│   │ 💎              │   │
│   │                 │   │
│   │  5 Tips para    │   │
│   │  cuidar tus     │   │
│   │  joyas de oro   │   │
│   │                 │   │
│   └─────────────────┘   │
│                         │
│   Deslizá para más →    │
└─────────────────────────┘

Slide 2-5 (Tips individuales):
┌─────────────────────────┐
│                         │
│   💎 Tip [N]            │
│                         │
│   [Ilustración o foto]   │
│                         │
│   Descripción del tip    │
│   (1-2 líneas máximo)    │
│                         │
└─────────────────────────┘

Slide final (CTA):
┌─────────────────────────┐
│                         │
│   💎💎💎                 │
│                         │
│   Guardá para no        │
│   perder estos tips     │
│                         │
│   @tuMarca              │
│   #joyería #cuidado     │
└─────────────────────────┘
```

**Pasos para crear:**

1. Nuevo diseño → **1080×1080** (o 1080×1350).
2. Guardá este tamaño como tu plantilla base.
3. Creá el diseño.
4. Tocá el **➕** al final de las miniaturas de página (abajo).
5. Seleccioná **"Duplicar página"** → esto copia el slide exacto.
6. Repetí hasta tener todos los slides que necesitás.
7. Editá cada slide: cambiá el contenido, mantené la estructura.

> **Pro tip:** Duplicá el slide 1 varias veces primero, luego editá cada copia. Así mantenés la estructura coherente y solo cambiás el contenido específico de cada tip.

### Tipo 3: Carrusel de Historia/M Storytelling

**Objetivo:** Contar la historia de la marca, el artesano, o la pieza.

**Estructura:**
```
Slide 1: Pregunta/Hook
"¿Sabías que..."

Slides 2-4: Desarrollo
"Nuestra historia comenzó hace..."

Slide final: Conexión emocional
"Ahí está nuestra pasión. En cada pieza."
```

## 🔧 Cómo Crear un Carrusel Paso a Paso

### Paso 1: Crear el primer slide

1. **Crear nuevo** → Diseño personalizado → **1080×1080**.
2. Elegí un fondo: color sólido, gradiente, o foto.
3. Agregá los elementos base:
   - Logo o marca en una esquina
   - Número de slide (si aplica)
   - Estilo visual coherente con tu marca

### Paso 2: Duplicar para crear todos los slides

1. Tocá la **miniatura del slide** (abajo).
2. **Toque prolongado** → aparecieron opciones.
3. Tocá **"Duplicar"**.
4. Repetí hasta tener el número de slides necesario.

```
Miniatura actual:
┌─────┐
│  1  │  ← Slide actual
└─────┘
    ↓ Toque prolongado → Duplicar

Resultado:
┌─────┬─────┬─────┐
│  1  │  2  │  3  │  ← Ahora tenés copia del slide 1 como slide 2
└─────┴─────┴─────┘
```

### Paso 3: Editar cada slide

1. Tocá cada miniatura para ir a ese slide.
2. Cambiá el contenido de cada uno:
   - Textos
   - Imágenes
   - Números
   - Posición de elementos
3. Mantené coherencia visual:
   - Misma posición del logo
   - Mismo estilo de tipografía
   - Mismo tratamiento de fotos

### Paso 4: Agregar coherencia visual

Para que el carrusel se sienta como una unidad, todos los slides deben compartir:

| Elemento | Qué mantener igual |
|----------|-------------------|
| **Fondo** | Mismo color o estilo |
| **Tipografía** | Mismas fuentes y tamaños |
| **Logo** | Misma posición y tamaño |
| **Colores** | Solo cambiar el contenido, no los colores de marca |
| **Numeración** | Mismo estilo de número (si la usás) |

### Paso 5: Agregar número de slide (opcional)

1. Tocá **🔤 Texto** → **"Agregar texto pequeño"**.
2. Escribí "1/5" (o el total).
3. Posicionálo siempre en el mismo lugar (ej: esquina superior derecha).
4. Repetí en cada slide con su número correspondiente.

> **Pro tip:** Si ponés números, mantené la misma fuente, tamaño y posición en todos los slides. Esto ayuda al usuario a saber dónde está en el carrusel.

## 🖼️ Agregar Fotos a los Slides

### Método 1: Reemplazar placeholder

1. Tocá la imagen placeholder en el slide.
2. Seleccioná **"Reemplazar"**.
3. Elegí tu foto.

### Método 2: Agregar foto directamente

1. Tocá **📷 Fotos** → **Subir**.
2. Seleccioná la foto.
3. Arrastrá y redimensioná hasta que quede bien.

### Método 3: Grid dentro del slide

Para slides con varias fotos:

1. Tocá **📐 Grids**.
2. Elegí un grid (2×1, 2×2, etc.).
3. Tocá cada celda → subí una foto diferente.

> **Pro tip:** Para carruseles de colección, usá el mismo encuadre en todas las fotos (ej: todas tomadas de arriba, todas con fondo blanco). Esto le da profesionalismo sin necesidad de edición compleja.

## 📝 Tip para el Primer Slide (Portada)

El primer slide es crítico. Debe:

1. **Captar atención** → usa una imagen impactante o una pregunta.
2. **Dar contexto** → explicar qué es el carrusel.
3. **Invitar a deslizar** → "Deslizá para ver más" o "Swipe →".

```
Ejemplo de portada efectiva:

┌─────────────────────────┐
│                         │
│   [Foto de una joya      │
│    con fondo oscuro]    │
│                         │
│   ┌─────────────────┐   │
│   │ 💎              │   │
│   │                 │   │
│   │  Cómo elegir    │   │
│   │  tu anillo de   │   │
│   │  compromiso     │   │
│   │                 │   │
│   │  → Deslizá      │   │
│   └─────────────────┘   │
│                         │
└─────────────────────────┘
```

## ❌ Errores Comunes

### Error 1: Inconsistencia visual
**Problema:** Cada slide tiene un estilo diferente.
**Solución:** Usá la técnica de duplicar + editar. Así todos los slides comparten estructura. Antes de publicar, recorré todos los slides y verificá que mantengan coherencia.

### Error 2: Primer slide sin hook
**Problema:** El primer slide no motiva a deslizar.
**Solución:** Incluí un "Deslizá" o "Swipe" en el primer slide. Usá una pregunta o declaración intrigante. Mostrá que hay más contenido valioso.

### Error 3: Demasiados slides
**Problema:** Hacer carruseles de 15 slides que nadie termina.
**Solución:** Ideal: **5-7 slides**. Si necesitás más de 10, considerá dividir en dos carruseles temáticos.

### Error 4: No indicar número de slides
**Problema:** El usuario no sabe cuántos slides hay.
**Solución:** Poné "1/7" o "Swipe → (7 slides)" en algún lugar visible del primer slide.

### Error 5: Mezclar formatos (cuadrado + vertical)
**Problema:** Usar distintos tamaños en el mismo carrusel.
**Solución:** Elegí un formato al inicio y mantenelo. Si empezás en 1080×1080, todos los slides deben ser 1080×1080.

### Error 6: Demasiado texto por slide
**Problema:** Cada slide tiene párrafos completos.
**Solución:** Máximum 2-3 líneas de texto por slide. Si necesitás más información, usá más slides con menos texto cada uno.

## 📌 Resumen

- Carruseles: series de 2-20 slides para contar historias completas.
- Usá **mismo formato (1080×1080)** para todos los slides.
- Tipos para joyería: **educativo, colección, storytelling**.
- **Duplicá** el primer slide para crear todos, luego editá.
- Mantené **coherencia visual**: fondo, fuentes, posición del logo.
- Primer slide debe tener **hook + indicación de deslizar**.
- Ideal: **5-7 slides** por carrusel.

1. Creá un diseño **1080×1080**.

6. Guardá como **"Plantilla carrusel educativo"**.
