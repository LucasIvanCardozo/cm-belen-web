# 📱 07. Stories y Covers de Reels

---

## 🎯 Objetivo

Al terminar esta sección, vas a saber crear stories impactantes y covers de reels para Instagram desde Canva móvil. Vas a entender las diferencias de formato, cómo diseñar para la interacción en stories, y cómo crear covers consistency para tus reels de joyería.

---

## 📐 Dimensiones y Formatos

### Stories de Instagram

| Dimensión | Proporción | Uso |
|-----------|-----------|-----|
| **1080 × 1920** | 9:16 | Formato estándar de stories |
| **1080 × 1920** | 9:16 | Covers de reels (usar el mismo) |

> **Nota importante:** Los covers de reels usan el **mismo formato que las stories** (1080×1920). Pero ojo: lo que ponés en el cover debe ser lo que se vea cuando alguien toca el reel, no necesariamente lo que se ve en el feed.

## 🎨 Estructura de una Story para Joyería

### Zonas de una Story

```
┌─────────────────────────────────────┐
│ ▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲│
│                                     │  ← Zona segura superior
│           (logo)                    │     (evitar poner cosas
│                                     │      importantes aquí)
├─────────────────────────────────────┤
│                                     │
│                                     │
│                                     │
│         ZONA PRINCIPAL              │  ← Donde va el contenido
│         (foto/video)                │     principal
│                                     │
│                                     │
│                                     │
├─────────────────────────────────────┤
│                                     │
│         ZONA INFERIOR               │  ← Zona táctil
│         [Responder] [Enviar]        │     (dejar al menos
│                                     │      400px libres)
│                                     │
└─────────────────────────────────────┘
```

### Regla de zonas seguras:

| Zona | Margen | Por qué |
|------|--------|---------|
| **Superior** | 250px | Instagram muestra nombre/perfil aquí |
| **Inferior** | 400px | Botones de respuesta e interacción |
| **Laterales** | 50px | En algunos dispositivos se corta |

## 📝 Tipos de Stories para Joyería

### Tipo 1: Story Portada de Lanzamiento

**Objetivo:** Anunciar un nuevo producto o colección.

**Estructura:**
```
┌─────────────────────────────┐
│                             │
│  [Foto del producto          │
│   con fondo elegante         │
│   (oscuro, dorado, blanco)]  │
│                             │
│  ┌─────────────────────┐   │
│  │ ✨ NUEVO             │   │
│  │                     │   │
│  │ Anillo Aurora       │   │
│  │ Colección 2026      │   │
│  │                     │   │
│  │ 💬 Comentá "INFO"   │   │
│  │   para más detalles │   │
│  └─────────────────────┘   │
│                             │
│  ┌─────┐  ┌─────┐  ┌─────┐ │
│  │  ○  │  │  ○  │  │  ○  │ │  ← Indicador de slide
│  └─────┘  └─────┘  └─────┘ │
└─────────────────────────────┘
```

**Pasos para crear:**

1. Nuevo → **1080×1920**.
2. Agregá la foto del producto como fondo o elemento principal.
3. Agregá overlay oscuro si la foto es muy clara (para que el texto se lea).
4. Agregá el texto del anuncio.
5. Agregá el CTA: " Comentá INFO" o "DM para reservar".
6. Agregá tu logo discretamente.

### Tipo 2: Story Detalle del Producto

**Objetivo:** Mostrar-close up de una joya.

**Estructura:**
```
┌─────────────────────────────┐
│                             │
│  [Close-up de la joya        │
│   enfoque en brillo/textura]│
│                             │
│  ═══════════════════════   │  ← Línea decorativa
│                             │
│  💍 Anillo Solitario        │
│  18K Oro Amarillo           │
│  Zafiro azul de Ceilán      │
│                             │
│  🔍 Tocá para ver más        │
│                             │
└─────────────────────────────┘
```

> **Pro tip:** Para stories de detalle, usá fotos donde se vean los **reflejos y brillos** de la joya. El oro y los diamantes se ven mejor con iluminación lateral o desde arriba. Evitá fotos con fondo muy brillante que "quemen" los reflejos de la joya.

### Tipo 3: Story Countdown ( Lanzamiento)

**Objetión:** Crear urgencia para un lanzamiento.

**Estructura:**
```
┌─────────────────────────────┐
│                             │
│  [Fondo elegante             │
│   (negro con textura o        │
│    foto blurred)]            │
│                             │
│  ⏰ LLEGADA EN               │
│                             │
│    03                       │
│    DÍAS                      │
│                             │
│  ═══════════════════════   │
│                             │
│  💎 Nueva Colección         │
│     Primavera 2026          │
│                             │
│  🔔 Activá recordatorio     │
│                             │
└─────────────────────────────┘
```

**Pasos:**

1. Nuevo → **1080×1920**.
2. Fondo oscuro con textura sutil.
3. Agregá el countdown (podes usar elementos de números o Text).
4. Agregá nombre de la colección.
5. Agregá "Activar recordatorio" o similar.

## 🎬 Covers de Reels

### ¿Cuál es la diferencia?

| Elemento | Story | Cover de Reel |
|----------|-------|---------------|
| **Formato** | 1080×1920 | 1080×1920 |
| **Dónde se ve** | Barra superior del feed | En el perfil, en el reel mismo |
| **Objetivo** | Interacción inmediata | Captar atención en el feed |
| **Duración** | Segundos | 15-60+ segundos |
| **Elementos interactivos** | Encuestas, preguntas, stickers | Texto legible en silencio |

### Cómo crear un cover de reel efectivo

**Regla #1: El texto debe funcionar sin sonido**

En Instagram, los reels se ven primero en silencio. Tu cover debe comunicar sin audio:

```
┌─────────────────────────────┐
│                             │
│  [Foto/video del reel]      │
│                             │
│  ═══════════════════════   │
│                             │
│  CÓMO ELEGIR               │  ← Texto grande y claro
│  TU ANILLO DE               │
│  COMPROMISO                 │
│                             │
│  💎 Guía completa           │
│                             │
└─────────────────────────────┘
```

**Regla #2: Texto en el tercio superior**

Mucha gente ve reels en el feed sin tocar play. Poné el texto importante arriba para que se lea sin entrar al reel.

**Regla #3: Imagen impactante**

El primer frame del reel (o la imagen del cover) es lo que aparece en el feed. Usá:
- Close-up de la joya
- Foto con buena iluminación
- Contraste alto

**Regla #4: Consistencia de marca**

Si tenés varios reels, mantené el mismo estilo de cover:
- Misma posición del logo
- Misma tipografía
- Misma paleta de colores
- Mismo estilo de foto

## 🔧 Cómo Exportar para Stories y Reels

### Para Stories:

1. **Menú ⋮** → **Descargar**.
2. Seleccioná formato:
   - **JPG** (menor peso, suficiente para stories)
   - **PNG** (máxima calidad, mayor peso)
3. Verificá que la calidad esté en **"Alta"**.
4. Guardá.

### Para Covers de Reels:

1. Guardá el diseño como PNG de alta calidad.
2. En Instagram, al publicar el reel, subí la imagen como cover.
3. Podés tocar la imagen del cover para reposicionarla o cambiarla.

> **Pro tip:** Exportá en **PNG** si tu diseño tiene textos con efectos, gradientes, o si la foto tiene transparencias. Exportá en **JPG** si es una foto simple con poco texto.

## ❌ Errores Comunes

### Error 1: Texto muy abajo en stories
**Problema:** El texto queda tapado por los botones de interacción.
**Solución:** Dejá los **400px inferiores libres**. Poné texto arriba, en el tercio central.

### Error 2: Cover de reel sin texto
**Problema:** El reel no comunica nada sin sonido.
**Solución:** Agregá el título del reel en texto grande en el cover. Las personas deben entender de qué va el video sin escucharlo.

### Error 3: Demasiados elementos
**Problema:** Stories muy cargadas que confunden.
**Solución:** En stories, menos es más. Un mensaje claro, una acción clara. Si tenés mucho que decir, usá varios slides.

### Error 4: No probarlo en el celular antes
**Problema:** Se ve bien en Canva pero mal en Instagram.
**Solución:** Antes de descargar, usá la función de **"Vista previa"** o exportá una versión de baja calidad para ver cómo se ve en tu teléfono real.

### Error 5: Mezclar formats de stories y posts
**Problema:** Crear un diseño horizontal y pretender usarlo en stories.
**Solución:** Siempre verificá el tamaño antes de empezar. Para stories: 1080×1920.

---

## 📌 Resumen

- Stories y covers de reels usan el mismo formato: **1080×1920**.
- Zonas de seguridad: 250px arriba, 400px abajo, 50px laterales.
- Tipos de stories: **lanzamiento, detalle, countdown, pregunta, behind the scenes**.
- Covers de reels: texto debe funcionar **sin sonido**, en el tercio superior.
- Exportá en **PNG** para máxima calidad, **JPG** para peso menor.
- Mantené **consistencia** en todos tus covers.
