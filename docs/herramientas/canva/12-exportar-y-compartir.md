# 📤 12. Exportar y Compartir

---

## 🎯 Objetivo

Al terminar esta sección, vas a saber exportar tus diseños en los formatos correctos, con la mejor calidad, y compartir directamente a Instagram o guardar para publicar después. Vas a entender las diferencias entre PNG, JPG, PDF, y cuándo usar cada uno.

---

## 📁 Formatos de Exportación

### PNG (Portable Network Graphics)

| Característica | Detalle |
|----------------|---------|
| **Calidad** | Máxima, sin pérdida |
| **Fondo** | Transparente posible |
| **Peso** | Mayor que JPG |
| **Usos recomendados** | Diseños con transparencias, texto con efectos, logos |

**Cuándo usar PNG:**
- Diseño con logo sobre fondo de Instagram
- Fotos con elementos superpuestos
- Diseños con gradientes o efectos complejos
- Cualquier cosa que necesite transparencia

### JPG (Joint Photographic Experts Group)

| Característica | Detalle |
|----------------|---------|
| **Calidad** | Alta, con ligera compresión |
| **Fondo** | Siempre opaque (sin transparencia) |
| **Peso** | Menor que PNG |
| **Usos recomendados** | Fotos de productos, imágenes straightforward |

**Cuándo usar JPG:**
- Posts de producto con fondo sólido
- Stories con fondo completo (sin transparencias)
- Cuando el peso del archivo es importante

### PDF (Portable Document Format)

| Característica | Detalle |
|----------------|---------|
| **Calidad** | Máxima |
| **Usos recomendados** | Impresión, documentos, presentaciones |

**Cuándo usar PDF:**
- Diseños para imprimir (tarjetas, folletos)
- No para redes sociales

> **Pro tip para Belen:** Para posts de Instagram, usá **PNG** cuando tengas elementos con transparencias o efectos de texto complejos. Usá **JPG** para fotos straightforward de productos. El peso no es problema para Instagram (máximo 30MB por post).

## ⚙️ Opciones de Exportación

### Paso a paso para exportar:

1. En el editor, tocá el menú **⋮** (tres puntos) arriba a la derecha.
2. Seleccioná **"Descargar"** o **"Exportar"**.
3. Aparecerán las opciones:

```
┌─────────────────────────────────────┐
│  Descargar                          │
├─────────────────────────────────────┤
│                                     │
│  Formato:                           │
│  ○ PNG (mejor calidad)              │
│  ○ JPG (archivo más pequeño)        │
│  ○ PDF (documento)                  │
│                                     │
│  Calidad:                           │
│  ○ Alta (recomendado)              │
│  ○ Estándar                        │
│  ○ Para correo (archivo pequeño)   │
│                                     │
│  Páginas:                          │
│  ○ Todas las páginas               │
│  ○ Página actual                   │
│  ○ Seleccionar páginas             │
│                                     │
│  ─────────────────────────────────  │
│                                     │
│  [DESCARGAR]                        │
│                                     │
└─────────────────────────────────────┘
```

### Ajustes recomendados:

| Tipo de diseño | Formato | Calidad |
|----------------|---------|---------|
| **Posts de producto** | JPG | Alta |
| **Stories** | JPG | Alta |
| **Diseños con transparencias** | PNG | Alta |
| **Diseños con efectos de texto** | PNG | Alta |
| **Carruseles** | JPG o PNG | Alta |
| **Diseños simples solo foto** | JPG | Alta |

## 📱 Descargar para Instagram

### Configuración correcta:

1. **Formato:** JPG (para mayoría de casos) o PNG (para transparencias).
2. **Calidad:** Alta.
3. **Páginas:** 
   - Para posts/carruseles: "Todas las páginas" o selección.
   - Para stories: Página actual.
4. **Color de perfil:** sRGB (generalmente es el default).

### Descargar a tu teléfono:

1. Tocá **Descargar**.
2. Seleccioná **"Guardar en dispositivo"** o **"Guardar imagen"**.
3. Canva guarda en la **galería de fotos**.
4. El archivo aparece en tu carpeta de imágenes.

> **Pro tip:** Después de descargar, verificá en tu galería que la imagen se guardó correctamente y con la calidad esperada. Si es muy pixelada, volvé a descargar en mayor calidad.

---

## 📤 Compartir Directamente a Instagram

### Método 1: Compartir desde Canva

1. Tocá **"Compartir"** en lugar de "Descargar".
2. Seleccioná **"Instagram"** o **"Historias de Instagram"**.
3. Se abre la app de Instagram con la imagen lista.
4. Editá el caption y publicá.

> **Limitación:** Este método puede tener limitaciones dependiendo de tu dispositivo y configuración de Instagram.

### Método 2: Guardar y subir manualmente

1. Descargá la imagen en alta calidad.
2. Abrí Instagram.
3. Creá la publicación normalmente.
4. Seleccioná la imagen descargada.

**Para stories:**
1. Descargá la imagen.
2. Abrí Instagram → stories.
3. Tocá la galería (donde dice "Agregar desde galería").
4. Seleccioná tu imagen descargada.

### Método 3: Compartir a otras apps

Desde el menú de compartir, podés elegir:
- **WhatsApp** → enviar a clientes directamente
- **Email** → enviar a colaboradores
- **Drive** → guardar en la nube
- **AirDrop** → enviar a otro dispositivo (iPhone)

## 🎠 Exportar Carruseles

### Para carruseles de Instagram:

1. Verificá que tengas todas las páginas (slides) listas.
2. Descargar → **"Todas las páginas"**.
3. Canva descarga como:
   - **JPG/PDF separado por página** → cada slide se descarga individual
   - **PDF combinado** → todas las páginas en un PDF (no sirve para Instagram)

### Cómo obtener las imágenes individuales:

**Opción 1: Descargar página por página**
1. En el editor, navegá a cada slide.
2. Descargar → **"Página actual"**.
3. Repetí para cada slide.

**Opción 2: Usar la opción "Descargar todas las páginas"**
1. Descargar → Todas las páginas.
2. Buscá si hay opción de "Imágenes separadas" o similar.
3. Si Canva descarga como un ZIP, descomprimí en tu teléfono.

**Opción 3: Tomar screenshot de cada slide**
1. En Canva, usá la previsualización de cada slide.
2. Tomá screenshot de cada uno.
3. Recortá los screenshots.

> **Pro tip:** Para carruseles, es más rápido crear todos los slides en Canva y luego **descargar página por página**. Guardá los archivos con nombres secuenciales: "carrusel-01.jpg", "carrusel-02.jpg", etc.

## 💾 Gestión de Archivos Descargados

### Nomenclatura recomendada:

```
❌ Malos nombres:
- IMG_001.jpg
- Screenshot_2026.png
- Diseño sin título.jpg

✅ Buenos nombres:
- joyeria-aurora-post-mayo-01.jpg
- story-lanzamiento-anillo-01.png
- carrusel-tips-joyas-01.jpg
```

### Cómo organizar:

**En tu teléfono:**
- Creá una carpeta: "Canva - Joyería"
- Subcarpetas: "Posts", "Stories", "Carruseles", "Reels"

**En Google Drive o similar:**
- Guardá una copia de todos tus diseños originales (formato Canva .canva).
- Así no perdés los archivos editables.

> **Pro tip:** Guardá tus archivos originales de Canva (los .canva) en la nube. Si algún día necesitás editar un post que publicaste hace meses, vas a poder abrirlo y modificarlo.

## 🔧 Solución de Problemas de Exportación

### Problema: Imagen muy pixelada

**Causas posibles:**
- Exportaste en calidad baja
- La foto original era de baja resolución

**Solución:**
1. Volvé a exportar en calidad "Alta".
2. Si la foto es pixelada, reemplazá con una foto de mejor calidad.

### Problema: Fondo con borde blanco no deseado

**Causa:** Exportaste en JPG con fondo blanco en lugar de transparente.

**Solución:**
1. Exportá en PNG.
2. Si el diseño tiene fondo colored, verificá que no estés usando una foto con fondo blanco.

### Problema: Colores diferentes en Instagram

**Causas:**
- El perfil de color no es sRGB
- Instagram comprime la imagen

**Solución:**
1. Verificá que exportás en sRGB.
2. Aceptá que Instagram siempre comprime un poco. Por eso es importante usar alta resolución.

### Problema: No puedo descargar

**Causas:**
- Sin espacio en el teléfono
- Problema de conexión
- Error de la app

**Solución:**
1. Verificá espacio en almacenamiento.
2. Reiniciá la app.
3. Probá conectar a WiFi.
4. Intentá más tarde.

## ❌ Errores Comunes

### Error 1: Exportar en calidad baja
**Problema:** Las imágenes se ven pixeladas.
**Solución:** Siempre usá calidad "Alta" para contenido de Instagram.

### Error 2: No verificar la imagen después de descargar
**Problema:** La imagen quedó mal y no te das cuenta hasta publicar.
**Solución:** Siempre mirá la imagen en tu galería antes de publicar.

### Error 3: Mezclar transparencias y formatos
**Problema:** Querés PNG con transparencia pero el fondo se ve raro en Instagram.
**Solución:** Para posts, evitá transparencias. Solo usá PNG si necesitás el logo sin fondo, pero luego poné la imagen en un fondo sólido.

### Error 4: No guardar copias originales
**Problema:** Querés editar un post antiguo y no encontrás el archivo.
**Solución:** Subí tus diseños originales (.canva) a Google Drive o similar.

### Error 5: Descargar en el formato wrong
**Problema:** Necesitás PNG pero descargaste JPG.
**Solución:** Verificá el formato antes de descargar. Tocá "JPG" para cambiar a "PNG".

---

## 📌 Resumen

- **PNG** → máxima calidad, transparencias, efectos.
- **JPG** → fotos straightforward, menor peso.
- **PDF** → para impresión, no para redes.
- Exportá siempre en **calidad Alta**.
- Para carruseles, **descargá página por página**.
- Verificá la imagen en tu galería antes de publicar.
- Guardá copias originales (.canva) en la nube.

## ✏️ Mini-Acción

1. Abrí un diseño existente (o creá uno nuevo).
2. Exportalo de tres formas diferentes:
   - **PNG Alta** → guardá como "test PNG"
   - **JPG Alta** → guardá como "test JPG"
   - **JPG Estándar** → guardá como "test JPG bajo"
3. Compará los tres archivos en tu galería:
   - ¿Cuál tiene mejor calidad?
   - ¿Cuánto pesa cada uno?
   - ¿Se nota la diferencia?
4. Eliminá los archivos de prueba.
5. Descargá tu mejor diseño en PNG Alta y verificá que se guardó correctamente.

> *Saber exportar correctamente es tan importante como saber diseñar. Un mal export arruina un buen diseño.*
