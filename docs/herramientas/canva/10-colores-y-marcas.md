# 🎨 10. Colores y Consistencia de Marca

---

## 🎯 Objetivo

Al terminar esta sección, vas a saber construir y mantener una paleta de colores coherente para tu marca de joyería, aplicar colores consistentemente en todos tus diseños, y crear una identidad visual reconocible en Instagram.

---

## 🎨 Por Qué los Colores Son Cruciales en Joyería

Los colores en joyería no solo son estéticos: **transmiten valores y emociones**.

| Color | Emoción | Mejor para |
|-------|---------|-----------|
| **Negro** | Elegancia, misterio, lujo | Marcas alta gama, piezas statement |
| **Dorado** | Riqueza, sofisticación, calor | Oro, premium, clásicos |
| **Blanco** | Pureza, limpieza, modernidad | Plata, minimalismo, zirconias |
| **Champagne** | Calidez, romanticismo, feminidad | Colecciones románticas, novias |
| **Plateado** | Frío, precisión, modernidad | Plata, acero, joyería técnica |
| **Rosé** | Juventud, feminidad, frescura | Rose gold, moda, jóvenes |

### La regla del color en joyería:

```
❌ ERROR: Usar muchos colores sin coherencia
→ El público no reconoce tu marca
→ Parece amateur
→ No transmite profesionalismo

✅ CORRECTO: Usar 3-5 colores consistentes
→ Tu marca es reconocible
→ Transmite consistencia
→ Proyecta profesionalismo
```

## 🔧 Cómo Configurar Colores en Canva

### Agregar colores al Brand Kit (Pro)

1. Perfil → **Brand Kit**.
2. Tocá **"Colores"**.
3. Tocá **"Agregar color"**.
4. Ingresá el código HEX de tu color.
5. Nombra el color: "Negro Logo", "Dorado Acento", etc.

### Usar colores en el editor (Free)

1. Seleccioná el elemento.
2. Tocá el **selector de color**.
3. Ingresá el código HEX manualmente.
4. O guardá el color en "Favoritos" para reutilizarlo.

### Cómo obtener el código HEX de un color

**Desde una imagen:**
1. Usá una app como **"Color Grab"** o **"Color Picker"**.
2. Apuntá la cámara al color.
3. Te da el código HEX.

**Desde tu marca:**
1. Buscá en Google: "color picker".
2. Arrastrá el selector hasta encontrar tu color.
3. Copiá el código HEX.

**En Canva:**
1. Buscá "color picker online" en el navegador.
2. O usá apps de diseño que tengas.

> **Pro tip:** Guardá los códigos HEX de tu marca en una nota en tu teléfono para referencia rápida. Así no tenés que buscar cada vez.

## 🎯 Aplicar Color Consistentemente

### Regla 80/20

El **80% de tus posts** deben seguir tu paleta de colores exactamente.

El **20%** puede tener variación (fotos con fondos coloreados, decoraciones temporales, campañas específicas).

### Cómo asegurar consistencia:

**Método 1: Crear plantillas base**
1. Creá un diseño con tus colores de marca.
2. Guardalo como plantilla.
3. Cada vez que crees un post, partí de esa plantilla.

**Método 2: Usar el Brand Kit**
1. Configurá tu Brand Kit con tus colores.
2. Al crear un diseño, aplicá el Brand Kit primero.
3. Todos los elementos nuevos usarán tus colores.

**Método 3: Checklist de color**

```
□ Color de fondo: ¿es de mi paleta?
□ Color de texto principal: ¿es de mi paleta?
□ Color de texto secundario: ¿es de mi paleta?
□ Color de acentos/líneas: ¿es de mi paleta?
□ Color del logo: ¿es exacto?
```

## 🎨 Usar Color según el Contexto

### Fondos

| Tipo de fondo | Colores recomendados |
|--------------|---------------------|
| **Sólido oscuro** | Negro, gris muy oscuro, dorado oscuro |
| **Sólido claro** | Blanco, crema, champagne claro |
| **Foto con textura** | Transparente o overlay oscuro sutil |
| **Gradiente** | Dos colores de tu paleta |

### Textos

| Elemento | Color recomendado |
|---------|------------------|
| **Título principal** | Blanco sobre fondo oscuro, Negro sobre fondo claro, Dorado para énfasis |
| **Subtítulos** | Blanco, dorado, gris oscuro |
| **Body text** | Blanco 90% sobre fondo oscuro, gris oscuro sobre fondo claro |
| **Precios** | Dorado, blanco, negro |
| **Hashtags** | Blanco 50-60% de opacidad |

### Acentos y decoraciones

| Elemento | Color |
|---------|-------|
| **Líneas decorativas** | Dorado, blanco, gris |
| **Formas** | Transparente con borde dorado, o colores de paleta |
| **Logos** | Blanco o dorado sobre fondos oscuros |

## 📐 Aplicar Color en Diferentes Formatos

### Posts de producto (1080×1080)

```
□ Fondo: #1A1A1A (negro)
□ Texto título: #FFFFFF (blanco)
□ Texto precio: #D4AF37 (dorado)
□ Línea decorativa: #D4AF37 (dorado)
□ Logo: #FFFFFF (blanco)
```

### Stories (1080×1920)

```
□ Fondo: puede variar (fotos, overlays)
□ Texto principal: #FFFFFF con sombra
□ Texto secundario: #D4AF37 (dorado)
□ Elementos interactivos: #D4AF37 (dorado)
□ Botones/stickers: #FFFFFF o #D4AF37
```

### Carruseles

```
□ Todos los slides: mismo fondo o variaciones de tu paleta
□ Texto: misma paleta en todos los slides
□ Elementos decorativos: mismo color en todos los slides
□ Logo: misma posición y color en todos
```

## 🎯 Color en Fotos de Joyería

### Cómo elegir fondos según el material

| Material | Mejor fondo de foto | Por qué |
|----------|---------------------|---------|
| **Oro amarillo** | Negro, blanco, madera oscura | Resalta el calor del oro |
| **Oro rosa** | Negro, blanco, rosa muy claro | Complementa el tono rosado |
| **Plata** | Blanco, negro, gris claro | Resalta el brillo frío |
| **Zirconias** | Blanco, negro | Resalta el brillo y fuego |
| **Perlas** | Azul claro, rosa claro, negro | Complementa los tonos nacarados |
| **Piedras de colores** | Negro, blanco, o color complementario | Resalta el color de la piedra |

### Cómo editar colores en la foto

Si la foto tiene colores que no son de tu paleta:

1. **Ajustar saturación** → reducir colores no deseados
2. **Aplicar filtro** → imponer el tono de tu marca
3. **Overlay de color** → agregar overlay semi-transparente del color de tu marca

> **Pro tip:** Para fotos de joyería sobre fondos complejos, usá la función de **"Eliminar fondo"** (Pro) o agregá un fondo de color sólido de tu paleta. Esto asegura que el color de fondo siempre sea consistente.

## ❌ Errores Comunes

### Error 1: Usar la paleta completa en cada post
**Problema:** Se ve saturado, no hay jerarquía.
**Solución:** En cada post usá 2-3 colores máximo. Negro + dorado + blanco. No necesitas usar todos los colores de tu paleta en cada diseño.

### Error 2: Colores ligeramente diferentes cada vez
**Problema:** Tu marca no es reconocible porque los colores no son consistentes.
**Solución:** Usá siempre los **mismos códigos HEX**. No elijas "un dorado" cada vez. Usá exactamente #D4AF37. Guardá tus códigos HEX.

### Error 3: Color de fondo que compite con la joya
**Problema:** El fondo distrae de la joya.
**Solución:** El fondo debe ser **neutro o complementar** la joya. Negro o blanco son los ms seguros. Evitá fondos muy coloreados a menos que sea parte de tu estrategia de marca.

### Error 4: Usar muchos colores de relleno
**Problema:** El diseño se ve saturado y barato.
**Solución:** El color de relleno (fondos, formas) debe ser mayormente **neutro**. Los colores vibrantes deben ser **acentos pequeños**.

### Error 5: No documentar tu paleta
**Problema:** Olvidás los colores exactos.
**Solución:** Creá un documento o imagen con tus colores y códigos HEX. Guardálo en un lugar accesible (notas del teléfono, documento en la nube).

---

## 📌 Resumen

- Tu paleta de marca debe tener **3-5 colores máximo**.
- Elegí colores según la **personalidad de tu marca** (lujo, romántica, moderna).
- Usá los **mismos códigos HEX** siempre, no colores similares.
- El **80% de tus posts** debe seguir tu paleta exactamente.
- En cada post: máximo **2-3 colores** del total de tu paleta.
- Guardá tus códigos HEX para referencia rápida.
