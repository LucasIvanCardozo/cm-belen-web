# 🎨 Guía Completa de Canva para Community Managers
## Edición Móvil — Mayo 2026

---

## 📋 Descripción

Esta guía está diseñada para **Belen**, Community Manager principiante con especialización en marcas de joyería y lujo. El objetivo es transformar su conocimiento básico de Canva en dominio profesional, trabajando siempre desde el celular.

**Enfoque:**
- 100% móvil (interfaz, gestos, flujo de trabajo desde smartphone)
- Ejemplos del nicho joyería/lujo
- Progresión gradual: desde funciones básicas hasta técnicas avanzadas
- Español latino, tono cercano y natural

**Audiencia objetivo:** CM que ya sabe lo básico de Canva (agregar texto, cambiar colores) pero necesita dominar la creación profesional de contenido para redes sociales.

## 🎯 Cómo Usar Esta Guía

**Orden recomendado:**
1. Empieza por `01-interfaz-y-navegacion.md` si nunca usaste Canva móvil en profundidad
2. Continúa en orden numérico
3. `recursos/` son herramientas de consulta rápida

**Cada sección incluye:**
- 🎯 Objetivo de aprendizaje
- 📝 Contenido con pasos numerados
- 💡 Pro tip / Nota importante
- ⚠️ Errores comunes
- 📌 Resumen clave
- ✏️ Mini-acción para practicar

**Nivel de dificultad por sección:**

## 📱 Requisitos Previos

- Teléfono Android (8+) o iPhone (iOS 14+)
- App Canva instalada (gratuita, con opción Pro)
- Conexión a internet para funciones completas
- Opcional: Canva Pro para Brand Kit y funciones avanzadas
