## 🔮 Funciones Ocultas en Canva

### 1. Duplicar hasta la perfección

**Truco:** Duplicá un diseño completo para usarlo como base.

1. Menú ⋮ → **"Duplicar"**.
2. Se crea una copia exacta del diseño.
3. Abrí la copia y solo cambiá el contenido.

**Uso práctico:**
- Tenés un post de producto que quedó bien → duplicalo para el próximo producto.
- Tenés una story que funcionó → duplicala para la próxima story.

### 2. Plantillas mágicas (Magic Design)

**Truco:** Canva puede generar diseños automáticamente.

1. Subí una foto.
2. Tocá **"Ver magia"** o el ícono de varita mágica.
3. Canva genera múltiples plantillas con tu foto.
4. Elegí la que más te guste.
5. Personalizá.

**Disponibilidad:** Generalmente en Canva Pro, pero a veces disponible en Free.

### 3. Eliminar fondo con un toque

**Truco:** Eliminá el fondo de cualquier imagen automáticamente.

1. Agregá la imagen al canvas.
2. Tocá la imagen.
3. Buscá **"Quitar fondo"** o **"Eliminar fondo"**.
4. Esperá el procesamiento.
5. Tocá **"Aplicar"**.

**Disponibilidad:** Canva Pro.

### 4. Redimensionar diseños existentes

**Truco:** Tomá un diseño y cambiá su tamaño para otra plataforma.

1. Abrí un diseño existente.
2. Tocá el ícono de dimensiones (ej: 1080×1080).
3. Seleccioná un nuevo tamaño (ej: 1080×1920 para stories).
4. Canva ajusta el diseño automáticamente.

**Disponibilidad:** Canva Pro.

### 5. Grabación de pantalla

**Truco:** Creá videos animados mostrando cómo diseñas.

1. Creá tu diseño.
2. Tocá **"Presentar"** o **"Grabar"**.
3. La app graba mientras navegás el diseño.
4. Exportá como video.

**Uso:** Para tutoriales, para mostrar productos, para contenido de valor.

### 6. Ajustar múltiples elementos a la vez

**Truco:** Seleccioná varios elementos y editarlos juntos.

1. Mantené presionado un elemento.
2. Arrastrá para seleccionar más elementos.
3. Tocá cualquier elemento seleccionado.
4. Los cambios aplican a todos.

**Ejemplo:** Seleccioná varios textos y cambiarles el color juntos.

### 7. Bloquear elementos

**Truco:** Evitá mover elementos accidentalmente.

1. Seleccioná el elemento.
2. Tocá **⋮** → **"Bloquear"**.
3. El elemento no se puede mover hasta que lo desbloqueés.

**Uso:** Logo, watermarks, elementos decorativos que siempre van en el mismo lugar.

### 8. Reutilizar estilos

**Truco:** Copiá el estilo de un texto a otro.

1. Tocá un texto con el estilo que querés copiar.
2. Buscá **"Copiar estilo"** o el ícono de copiar.
3. Tocá el texto destino.
4. Buscá **"Pegar estilo"**.

**Alternativa:**
1. Tocá el texto con el estilo bueno.
2. Mantené presionado y arrastrá hasta el texto destino.
3. Suelta sobre el texto.

## 🎨 Técnicas de Diseño Avanzadas

### 1. Capas y profundidad

**Truco:** Creá profundidad usando capas.

```
┌─────────────────────────────────────┐
│  Capa 1 (fondo): Color sólido       │
│  Capa 2: Foto con overlay oscuro    │
│  Capa 3: Elementos decorativos      │
│  Capa 4: Texto principal            │
│  Capa 5: Logo                       │
└─────────────────────────────────────┘
```

**Cómo hacer capas en Canva:**
1. Agregá cada elemento en orden (primero el fondo, después la foto, etc).
2. Cada nuevo elemento queda "encima" del anterior.
3. Para cambiar el orden: toque prolongado → "Traer adelante" o "Enviar atrás".

### 2. Overlays de color

**Truco:** Agregá overlays para mejorar fotos o crear ambientes.

**Overlay oscuro (para texto legible):**
1. Agregá una forma (rectángulo) sobre la foto.
2. Elegí color negro.
3. Ajustá la **translucidez** a 30-40%.
4. El texto blanco se va a leer perfectamente.

**Overlay de color (para ambiente):**
1. Agregá una forma.
2. Elegí el color de tu marca (dorado, champagne, etc).
3. Ajustá la **translucidez** a 10-20%.
4. Dás un tono cohesive a toda la imagen.

### 3. Sombras profesionales

**Truco:** Creá sombras que hacen que los elementos "floten".

1. Seleccioná el elemento (foto, texto, shape).
2. Buscá **"Efectos"** → **"Sombra"**.
3. Ajustá:
   - **Desenfoque:** 5-10 para sombra suave
   - **Opacidad:** 30-50%
   - **Desplazamiento:** 3-5px
   - **Color:** negro o blanco según el fondo

### 4. Gradientes

**Truco:** Usá gradientes para fondos sofisticados.

**Cómo crear un gradiente:**
1. Agregá una forma (rectángulo).
2. Seleccioná el color.
3. Buscá opciones avanzadas → **"Gradiente"**.
4. Elegí la dirección (horizontal, vertical, radial).
5. Ajustá los colores de inicio y fin.

**Gradientes recomendados para joyería:**
- Negro a dorado oscuro
- Blanco a champagne
- Negro a transparente (fade out)

### 5. Texturas

**Truco:** Agregá texturas para dar carácter.

**Texturas disponibles en Canva:**
- mármol, madera, metal, tela, papel

**Cómo usar:**
1. 📷 Fotos → buscar "marble texture" o "gold texture".
2. Agregá la textura al diseño.
3. Ajustá la **translucidez** a 30-50%.
4. Poné la textura sobre un fondo sólido o sobre fotos.

> **Pro tip:** Las texturas sutiles son ms elegantes que las texturas obvias. Una textura de mármol al 20% de opacidad sobre un fondo negro es sofisticado. Al 80% parece un fondo de Windows 95.

## 🕐 Técnicas de Eficiencia

### 1. Workflow rápido para posts de producto

```
Tiempo objetivo: 5-8 minutos por post

1. [1 min] Abrir plantilla de post producto
2. [2 min] Cambiar foto del producto
3. [1 min] Cambiar nombre del producto
4. [1 min] Cambiar precio
5. [1 min] Agregar logo y verificar
6. [1 min] Exportar en PNG Alta
7. [1 min] Publicar en Instagram
```

### 2. Batch creation (creación en lotes)

**Truco:** Creá varios posts de una vez.

1. Abrí la plantilla de post producto.
2. **Duplicá** el diseño 5 veces.
3. Tenés 5 diseños abiertos (o en cola).
4. Editá cada uno con el contenido de diferentes productos.
5. Exportá todos.
6. Programá todos juntos.

**Resultado:** 10 posts en el tiempo que normalmente harías 2.

### 3. Pre-configurar colors y fonts

**Truco:** Antes de empezar a diseñar, configurá todo.

1. Perfil → Brand Kit → verificá que tus colores estén.
2. En el editor, cambiá la fuente default a tu fuente de marca.
3. Ahora todos los textos nuevos usan tu marca.

### 4. Usar el menú rápido

**Truco:** Agilizá el acceso a funciones frecuentes.

Cuando seleccionás un elemento, aparecen opciones rápidas en la barra inferior:

```
[Copiar] [Eliminar] [Girar] [Organizar] [Más...]
```

- **Copiar:** Duplicá el elemento sin ir al menú.
- **Girar:** Rotación rápida.
- **Organizar:** Traer adelante/atrás, alinear.

### 5. Preview en diferentes formatos

**Truco:** Verificá cómo se ve tu diseño en diferentes contextos.

1. Tocá el ícono de **"Vista previa"** o **"Preview"**.
2. Podés ver el diseño como se verá:
   - En el feed de Instagram
   - En stories
   - En perfil
   - En tamaño completo

> **Pro tip:** Siempre previsualizá antes de exportar. Si algo se ve mal en la preview, arreglalo antes de guardar.

## 🎨 Trucos para Joyería Específicos

### 1. Hacer que el oro brille

**Truco:** Ajustá la foto para que el oro se vea ms brillante.

1. Aumentá **brillo** +15.
2. Aumentá **contraste** +10.
3. Reducí **saturación** ligeramente -5.
4. Agregá un filtro de "Luz natural" a baja intensidad.

### 2. Fotos con reflejos

**Truco:** Cuando la foto tiene reflejos, ajustá para que se vean.

1. Reducí **highlights** -15.
2. Aumentá **shadows** +10.
3. Aumentá **claridad** levemente.
4. Esto revela ms detalle en las zonas brillantes.

### 3. Fondo limpio en fotos de producto

**Truco:** Cómo lograr que el fondo se vea profesional.

1. Usá la función **"Eliminar fondo"** (Pro).
2. Poné la joya sobre un fondo de color sólido (negro o blanco).
3. Agregá una sombra sutil debajo de la joya para que "flote".

### 4. Texto que complementa la joya

**Truco:** Marcar zonas donde el texto no compita con la joya.

```
┌─────────────────────────────────────┐
│                                     │
│  [ZONA SUPERIOR]                    │
│  ← Zona de texto segura            │
│                                     │
│         [JOYA]                      │
│                                     │
│  [ZONA INFERIOR]                    │
│  ← Otra zona de texto segura        │
│                                     │
└─────────────────────────────────────┘

El texto nunca debe estar encima de la joya.
```

### 5. Coherencia entre fotos diferentes

**Truco:** Cuando tenés fotos de diferentes sesiones, unificás el look.

1. Aplicá el mismo filtro a todas las fotos.
2. Usá los mismos ajustes de brillo/contraste.
3. Recortá todas en el mismo formato.
4. Agregá un overlay sutil del color de tu marca.

## ❌ Errores Comunes que Evitar

### Error 1: No usar plantillas
**Problema:** Empezás de cero cada vez.
**Solución:** Guardá tus mejores diseños como plantillas.

### Error 2: No duplicar antes de editar
**Problema:** Editás el original y perdés la versión anterior.
**Solución:** Siempre duplicá antes de hacer cambios importantes.

### Error 3: No guardar versiones
**Problema:** Hacés cambios y no podés volver atrás.
**Solución:** Usá "Duplicar" para guardar versiones antes de cambios grandes.

### Error 4: Ignorar el menú de más opciones
**Problema:** No sabés que existen funciones ms avanzadas.
**Solución:** Explorá el menú ⋮ y las opciones "Más" o "Avanzado" regularmente.

### Error 5: No experimentar
**Problema:** Siempre hacés lo mismo, nunca mejorás.
**Solución:** Cada semana, probá una función nueva que no conozcas.
