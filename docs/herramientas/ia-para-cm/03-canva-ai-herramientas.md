## 2. Magic Media — Generación de Imágenes

### Qué es

Magic Media crea imágenes a partir de texto. Genial para:

- Fondos únicos para posts
- Ilustraciones conceptuales
- Imágenes para campaigns
- Variaciones de productos
- Contenido sin needing fotos reales

### Cómo usarlo:

1. 📱 Abrí Canva y tocá **"+""** para crear diseño
2. 📱 Buscá **"Apps"** o **"Más"**
3. 📱 Encontrá **"Magic Media"** o **"Text to Image"**
4. ✏️ Escribí tu descripción
5. 🎨 Elegí estilo (fotografía, ilustración, etc.)
6. ⏱️ Esperá 10-30 segundos
7. ✅ Usá la imagen generada

### Prompts efectivos para joyería:

**Para fondos:**
```
"Minimalist marble background with soft gold accents, 
elegant luxury aesthetic, soft natural lighting, 
shallow depth of field, professional photography style"
```

**Para concepto:**
```
"Close up of elegant gold jewelry on velvet display, 
soft studio lighting, luxury jewelry photography, 
shallow depth of field, warm tones"
```

**Para contexto de uso:**
```
"Professional woman wearing delicate gold necklace in 
modern office setting, natural lighting, lifestyle 
jewelry photography, aspirational, warm tones"
```

### Tips para mejores resultados:

| Aspecto | Recomendación |
|---------|---------------|
| Sé específico | "anillo de oro con piedra verde" vs "joya" |
| Agregá estilo | "photography style", "oil painting", "digital art" |
| Mencioná iluminación | "soft natural lighting", "studio lighting" |
| Incluí colores dominantes | "warm gold tones", "cool silver palette" |
| Agregá calidad | "professional", "high quality", "4k" |

### Usos para joyería:

1. **Fondos para posts** cuando no tengas buenas fotos
2. **Conceptos visuales** para campaigns
3. **Imágenes de lifestyle** sin access a modelos
4. **Variaciones** de un mismo producto
5. **Contenido seasonal** (Navidad, San Valentín)

## 3. Magic Eraser — Eliminar Objetos

### Qué es

Magic Eraser elimina objetos no deseados de tus imágenes:

- Fondos desordenados
- Distracciones en la foto
- Imperfecciones
- Logos no deseados
- Personas que no deberían estar

### Cómo usarlo:

1. 📸 Insertá tu foto en Canva
2. 📱 Buscá **"Edit Photo"** o **"Editar"**
3. 📱 Encontrá **"Magic Eraser"** o **"Eraser"**
4. ✏️ Pintá sobre lo que querés eliminar
5. ✅ Listo, el objeto desaparece

### Casos de uso para joyería:

**Antes:**
```
❌ Foto de anillo con fondo desordenado de escritorio
❌ Foto de collar con etiqueta de precio visible
❌ Foto de joyero con reflejos no deseados
❌ Foto con fecha/hora del celular en la esquina
```

**Después:**
```
✅ Fondo limpio de mármol o tela elegante
✅ Solo el producto, sin distracciones
✅ Imagen profesional lista para publicar
```

### Tips:

- 🖌️ Pintá con trazos amplios para objetos grandes
- 🔍 Usá zoom para precisión en detalles pequeños
- 💾 Hacé una copia antes por si acaso
- 🔄 Podés usar varias veces hasta quedar satisfecho

## 4. Magic Edit — Reemplazar Elementos

### Qué es

Magic Edit te permite seleccionar un área y reemplazarla:

- Cambiar colores de objetos
- Reemplazar fondos
- Agregar elementos
- Cambiar texturas
- Transformar partes de la imagen

### Cómo usarlo:

1. 📸 Insertá tu foto
2. 📱 Buscá **"Edit Photo"** → **"Magic Edit"**
3. ✏️ Pintá sobre el área a cambiar
4. ✏️ Escribí qué querés que aparezca
5. ⏱️ Esperá el resultado
6. ✅ Aceptá o regenerá

### Ejemplos para joyería:

**Cambiar fondo:**
```
Seleccioná: el fondo
Prompt: "Elegant dark velvet jewelry display background"
```

**Cambiar color de joya:**
```
Seleccioná: el metal de la joya
Prompt: "Rose gold metal finish"
```

**Cambiar packaging:**
```
Seleccioná: la caja/bolsa
Prompt: "White luxury jewelry box with ribbon"
```

## 5. Magic Write — Textos con IA

### Qué es

Magic Write genera texto para tus designs:

- Captions para posts
- Textos para folletos
- Copy para historias
- Ideas de headlines
- Descripciones de productos

### Cómo usarlo:

1. 📱 Creá o abrí un diseño
2. 📱 Agregá una caja de texto
3. 📱 Buscá **"Magic Write"** o **"Suggest Text"**
4. ✏️ Escribí qué necesitás
5. ✨ Magic Write genera opciones
6. ✅ Tocá para insertar

### Prompts útiles:

**Para caption de post:**
```
"Short elegant caption for gold jewelry post, luxury 
tone, call to action included"
```

**Para descripción de producto:**
```
"Product description for handmade silver ring with 
turquoise stone, elegant and descriptive, 50 words"
```

**Para headline:**
```
"Attention grabbing headline for jewelry sale post, 
urgency without being pushy"
```

## 6. Translate — Traducción Automática

### Qué es

Canva AI puede traducir textos dentro de tus designs:

- Posts multilingües
- Adaptar contenido para diferentes mercados
- Traducir descripciones de productos
- Cambios rápidos de idioma

### Cómo usarlo:

1. 📱 Seleccioná el texto en tu diseño
2. 📱 Buscá el ícono de **traducir**
3. 🌐 Elegí el idioma destino
4. ✅ El texto se traduce automáticamente

### Usos para joyería:

```
Para joyería que vende internacionalmente:
- Español ↔ Inglés ↔ Portugués
- Adaptar campañas para diferentes países LatAm
- Contenido para tourists
```

## 8. Flujo de trabajo con Canva AI

### Sesión de diseño típica:

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│   SESIÓN DE DISEÑO (30-60 minutos)                      │
│                                                          │
│   1. Generá imágenes con Magic Media                    │
│   2. Limpiá fotos con Magic Eraser                     │
│   3. Creá 3-5 variaciones de un post                  │
│   4. Redimensioná para diferentes formatos             │
│   5. Traducí si necesitás multilingual                 │
│                                                          │
│   RESULTADO: 5-10 posts listos para programar          │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Ejemplo práctico: Post de collar nuevo

**Paso 1:** Sacá foto del collar
**Paso 2:** Abrí en Canva
**Paso 3:** Magic Eraser → eliminá fondo
**Paso 4:** Magic Media → generá 3 fondos diferentes
**Paso 5:** Agregá texto con Magic Write
**Paso 6:** Redimensioná para post, story, reel cover
**Paso 7:** Guardá → Programá en Later/Meta Business Suite

## 9. Limitaciones y consideraciones

### ⚠️ Tené en cuenta:

1. **Calidad variable** — Magic Media no siempre es perfecto, revisá los resultados
2. **Consistencia de marca** — Las imágenes generadas pueden no coincidir 100% con tu estilo
3. **Derechos** — Usá las imágenes generadas como apoyo, no como producto final principal
4. **No para producto real** — Para fotos de producto, siempre es mejor la foto real

### ✅ Mejores prácticas:

- Usá Magic Media para **apoyo visual**, no como imagen principal de producto
- Combiná fotos reales + imágenes AI
- Siempre revisá y ajustá los resultados
- Mantené coherencia con tu paleta de colores
