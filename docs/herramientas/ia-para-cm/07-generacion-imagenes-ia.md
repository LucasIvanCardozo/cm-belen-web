## 3. Prompts para Joyería

### 3.1 Anatomía de un buen prompt

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│   ESTRUCTURA DE PROMPT:                                  │
│                                                          │
│   1. SUJETO: qué es la imagen                          │
│   2. MEDIUM: fotografía, ilustración, etc.              │
│   3. STYLE: arte, cinematográfico, etc.                  │
│   4. LIGHTING: tipo de iluminación                     │
│   5. SETTING: fondo, ambiente                          │
│   6. QUALITY: details, high quality, etc.              │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 3.2 Prompts básicos

**Para fondo elegante:**

```
Elegant dark velvet jewelry display background,
professional photography, soft spotlight lighting,
shallow depth of field, luxury aesthetic,
dark charcoal and gold accents, high detail
```

**Para lifestyle:**

```
Professional woman wearing delicate gold necklace
in modern coffee shop, natural window lighting,
warm tones, lifestyle jewelry photography,
shallow depth of field, aspirational mood
```

**Para concepto de campaña:**

```
Elegant gold ring floating in cosmic void,
starry background with subtle sparkle,
dreamy ethereal lighting, luxury product photography,
cinematic composition, moody atmosphere
```

### 3.3 Prompts específicos

**Para oro:**

```
Close up of intricate gold jewelry piece on white
marble surface, warm golden hour lighting,
macro photography style, reflections and shimmer,
luxury jewelry photography, shallow depth of field
```

**Para plata:**

```
Minimalist silver bracelet on smooth concrete
background, cool lighting with subtle blue tones,
clean modern aesthetic, product photography style,
soft shadows, Scandinavian minimalism
```

**Para piedras:**

```
Emerald gemstone ring in natural light, showing
the deep green color and clarity, studio lighting,
luxury jewelry photography, detailed close up,
elegant dark background
```

**Para packaging:**

```
White luxury jewelry box with gold ribbon,
soft shadows on marble surface, top down view,
gift packaging photography, clean elegant aesthetic,
high end retail presentation style
```

### 3.4 Prompts con modelos

**⚠️ Importante:** Las imágenes con modelos realistas pueden tener restricciones y consideraciones éticas.

**Ejemplo (con precaución):**

```
Beautiful young woman with long dark hair,
wearing elegant gold necklace and earrings,
looking at camera, soft studio lighting,
fashion photography style, luxury jewelry campaign,
warm tones, professional quality
```

**Recomendación:** Usá prompts más abstractos o ilustrados para evitar problemas.

## 5. Ajustes importantes

### Guidance Scale

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│   GUIDANCE SCALE (qué tan fiel al prompt):             │
│                                                          │
│   3-5:  Más creativo, menos control                     │
│   7-9:  Buen balance (RECOMENDADO)                     │
│   12+: Muy fiel al prompt, puede perder naturalidad     │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

**Para joyería:** Usá 7-9

### Aspect Ratio

| Ratio | Uso                           |
| ----- | ----------------------------- |
| 1:1   | Posts de Instagram, cuadrados |
| 9:16  | Stories, reels                |
| 16:9  | Banners, website              |
| 4:5   | Posts portrait                |

### Image to Image

Subí una imagen y pedile variaciones:

1. 📷 Generá o conseguí imagen base
2. 🎨 Ajustá con Image to Image
3. 🔄 Generá variaciones
4. ✨ Obtené nuevas versiones

## 6. Usos prácticos para CM de joyería

### 6.1 Fondos para posts

```
WORKFLOW:
1. Generá fondo en Leonardo
2. Abrí en Canva
3. Agregá foto real de joya
4. Exportá
5. Publicá
```

### 6.2 Conceptos para campaigns

```
WORKFLOW:
1. Brief de campaña
2. Generá 3-5 conceptos visuales
3. Presentá a cliente
4. Con aprobación, usá como guía para fotos reales
```

### 6.3 Inspiración y mood boards

```
WORKFLOW:
1. Generá 10-15 imágenes concepto
2. Creá board en Pinterest o Notion
3. Usá como referencia para dirección creativa
4. No copies directamente, inspírate
```

### 6.4 Contenido para blog/website

```
WORKFLOW:
1. Artículo de blog necesita imagen header
2. Generá imagen relevante con IA
3. Agregá a tu CMS
4. ¡Listo!
```

### 6.5 Material para clientes

```
WORKFLOW:
1. Generá mockups de cómo se vería la joya
2. Usá para proposals sin inversión en fotos
3. Mostrá "antes" de comprometer presupuesto
```

## 8. Consideraciones éticas y legales

### ⚠️ Importante:

```
┌─────────────────────────────────────────────────────────┐
│                                                          │
│   TRANSPARENCIA:                                        │
│                                                          │
│   • Si publicás una imagen generada con IA,             │
│     considerá disclose ("imagen generada con IA")         │
│   • Para uso comercial, verificá términos de servicio     │
│   • Algunas plataformas tienen políticas específicas     │
│                                                          │
│   DERECHOS:                                            │
│                                                          │
│   • Leonardo: tus generaciones son tuyas                │
│   • Usá para inspiración, no copia directa              │
│   • No generes imágenes que infrinjan copyrights          │
│                                                          │
│   AUTENTICIDAD:                                        │
│                                                          │
│   • Fotos de productos reales = más confianza          │
│   • IA para apoyo, no para reemplazar producto real    │
│   • Tu audiencia valora autenticidad                    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Mejores prácticas:

1. **Sé transparente** cuando sea relevante
2. **No finjas** que imágenes IA son fotos reales de producto
3. **Mezclá** imágenes IA con fotos reales
4. **Crea valor** con IA, no soloen lugar de (reemplazando) inversión
