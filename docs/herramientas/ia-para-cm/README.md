## 💡 Cómo Usar la IA (y Cómo No)

```
✅ SÍ USA LA IA PARA:
• Inspiración y brainstorms
• Primeros borradores
• Generación de ideas
• Optimización de texto existente
• Investigación rápida
• Automatización de tareas

❌ NO USES LA IA PARA:
• Copiar y pegar sin revisar
• Contenido que no entendés
• Decisiones de marca importantes
• Reemplazar tu voz única
• Todo el trabajo sin supervisión
```

## 💎 Prompts para Joyería

```
EJEMPLOS EFECTIVOS:

❌ "Escribí un caption sobre joyería"
✅ "Escribí un caption de 150 palabras para una foto de un collar 
   dorado con dije de luna. El tono debe ser elegante y misterioso, 
   dirigido a mujeres de 25-40 años interesadas en joyería artesanal."

❌ "Dame hashtags para joyería"
✅ "Generá 15 hashtags en español para una marca de joyería de plata 
   artesanal. Incluí 3 hashtags de nicho, 5 de industria y 7 de 
   engagement. No incluyas hashtags genéricos."
```
