## 💎 Por qué templates específicos para joyería

La joyería tiene características únicas que requieren templates especializados:

```
JOYERÍA vs. OTROS NEGOCIOS
════════════════════════════

🏷️ PRODUCTO
→ Valor emocional alto
→ Detalles que importan (materiales, piedras)
→ Fotografía específica (luz, fondo, macro)
→ Inventario variable

👥 AUDIENCIA
→ Compra planificada (no impulsiva)
→ Necesita educación sobre el producto
→ Valora exclusividad
→ Decision time largo

📱 CONTENIDO
→ Mucho detalle visual
→ Storytelling de marca
→ Contexto de uso ( outfits, occasions)
→ Emocional > funcional
```

## 💡 Tips para joyería en Notion

### Documentación:
- **Guardá referencias visuales** (links a Instagram, Pinterest)
- **Anotá detalles de producto** que no querés olvidar
- **Mantené actualizado** el inventario (cambia seguido)

### Planificación:
- **Anticipate** las campañas estacionales (2-3 meses antes)
- **Tené un banco de ideas** siempre lleno (mínimo 20 ideas)
- **Documentá qué funciona** con métricas

### Organización:
- **Usá colores** para categorizar por tipo de joya
- **Linkeá** las piezas con el contenido que hacés sobre ellas
- **Archivá** campañas pasadas para referencia futura
