# Módulo 01: Instalación y Configuración de Notion

## 🎯 Objetivos del módulo

Al terminar este módulo vas a:

- ✅ Tener Notion instalado en tu celular
- ✅ Crear tu cuenta (gratis)
- ✅ Configurar el idioma español
- ✅ Conocer las funciones básicas
- ✅ Crear tu primer workspace

---

## 📲 Paso 1: Descargar Notion

### En Android:

1. Abrí la **Google Play Store**
2. Buscá "Notion"
3. Tocá **Instalar**
4. Esperá a que termine (puede tomar 1-2 minutos)

### En iPhone:

1. Abrí el **App Store**
2. Buscá "Notion"
3. Tocá **Obtener**
4. Autenticá con Face ID o Touch ID

**Tamaño:** Aproximadamente 50 MB

---

## 📝 Paso 2: Crear tu cuenta

Al abrir Notion por primera vez vas a ver la pantalla de bienvenida.

### Opción A: Registro con email (recomendado)

1. Tocá **"Registrarse"**
2. Elegí **"Email"**
3. Ingresá tu correo electrónico
4. Creá una contraseña segura (mínimo 8 caracteres, mezcla de letras y números)
5. Tocá **"Registrarse"**
6. Verificá tu email (revisa la carpeta de spam)

### Opción B: Registro con Google

1. Tocá **"Registrarse"**
2. Elegí **"Continuar con Google"**
3. Seleccioná tu cuenta de Google
4. Permití los permisos

**Tip:** Usá el mismo email que usás para tu trabajo de CM. Te va a servir para organizarte.

---

## 🌐 Paso 3: Configurar en español

Por defecto Notion viene en inglés. Para cambiarlo:

1. Tocá el **ícono de ajustes** (rueda dentada) en la esquina superior derecha
2. Buscá **"Language"** o **"Idioma"**
3. Seleccioná **"Español"**
4. Confirmá el cambio

¡Listo! Notion ahora habla español.

---

## 📱 Paso 4: Explorar la interfaz móvil

Cuando entrás a Notion, vas a ver:

### La barra inferior (navegación)

```
┌─────────────────────────────────────┐
│                                     │
│     [Tu contenido aquí]             │
│                                     │
├─────────────────────────────────────┤
│  🏠 Inicio  │  📁 Workspaces  │ ⚙️  │
└─────────────────────────────────────┘
```

| Ícono         | Función                       |
| ------------- | ----------------------------- |
| 🏠 Inicio     | Tu página principal personal  |
| 📁 Workspaces | Todos tus espacios de trabajo |
| ⚙️ Ajustes    | Configuración de cuenta       |

### La página principal

Desde "Inicio" vas a ver:

- Acceso rápido a páginas recientes
- widgets personalizables
- Tu actividad reciente

---

## 🗂️ Paso 5: Crear tu Workspace de CM

Un **Workspace** es tu espacio de trabajo. Vamos a crear uno dedicado solo para tu trabajo de CM.

1. Desde **"Workspaces"**, tocá el botón **"+"** (nuevo workspace)
2. Elegí **"Crear workspace"**
3. Ponele nombre: **"CM Belen"** o tu nombre real
4. Agregá una descripción (opcional): "Gestión de Community Manager"
5. Tocá **"Crear"**

**Tip:** Si ya tenés un workspace personal, podés usar sub-workspace o crear páginas dentro de uno solo. No necesitás crear uno nuevo si preferís mantener todo junto.

---

## 🎨 Paso 6: Configurar tu perfil

1. Tocá tu **avatar/foto** en la esquina superior izquierda
2. Elegí **"Configuración y miembros"**
3. Tocá **"Mi perfil"**
4. Completá:
   - **Nombre:** Tu nombre o "Belen CM"
   - **Foto:** Una imagen profesional
   - **Título:** "Community Manager | Joyería y Lujo"
   - **Bio:** Breve descripción de tus servicios

---

## 📊 Paso 7: Agregar widgets a tu inicio

Los **widgets** son tarjetas que muestran información importante en tu página de inicio.

Para agregar widgets:

1. En tu página de **Inicio**, tocá el botón **"+"** en una zona vacía
2. Elegí el tipo de widget:
   - **Páginas rápidas:** Acceso directo a páginas importantes
   - **Tareas:** Próximas tareas
   - **Hoy:** Lo que tenés para hoy
   - **Métricas:** (solo versión Pro)

**Recomendado para CM:**

- Acceso rápido a tu calendario editorial
- Acceso a lista de clientes
- Acceso a tareas de la semana

---

## ⚡ Atajos importantes

### Desde cualquier lugar:

| Acción       | Atajo                                      |
| ------------ | ------------------------------------------ |
| Nueva página | Botón "+" azul en esquina inferior derecha |
| Buscar       | Barra de búsqueda arriba                   |
| Menú         | Ícono de 3 líneas horizontales (≡)         |
| Compartir    | Ícono de compartir (>)                     |

### En una página:

| Acción         | Cómo                                |
| -------------- | ----------------------------------- |
| Agregar bloque | Escribí "/" para ver opciones       |
| Mover bloque   | Arrastrá el ícono "⋮⋮"              |
| Eliminar       | Seleccioná el bloque, luego eliminá |

```
□ Workspace de CM creado

```
