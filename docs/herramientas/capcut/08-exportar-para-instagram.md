# Módulo 08: Exportar para Instagram

## 🎯 Objetivo del módulo

Aprender a exportar tus videos con la configuración correcta para cada formato de Instagram (Reels, Stories, Feed) y entender cómo subirlos de forma óptima.

**Por qué es importante:**
- Un video mal exportado puede verse pixelado o cortado
- La configuración correcta maximiza la calidad
- Conocer los formatos te ayuda a planificar tus videos desde el inicio

---

## 📱 Formatos de Instagram: Guía rápida

### Los 3 formatos principales

| Formato | Dimensiones | Duración | Ubicación |
|---------|-------------|----------|-----------|
| **Reels** | 9:16 (1080x1920) | Hasta 90 segundos | Feed principal |
| **Stories** | 9:16 (1080x1920) | 24 horas visibles | Barra superior |
| **Feed** | 1:1 o 4:5 | Hasta 60 segundos | Feed principal |

### ¿Cuál elegir para qué?

| Contenido | Mejor formato | Por qué |
|-----------|---------------|---------|
| Video de producto | Reels o Feed 4:5 | Más tiempo de visualización |
| Detalle de joya | Reels | Formato vertical dominante |
| Comparación de productos | Stories (serie) | Formato secuencial |
| Unboxing | Reels | Permite hasta 90 segundos |
| Promo rápida | Stories | Urgencia, stickers de cuenta regresiva |
| Catálogo | Feed 1:1 | Se ve bien en grid |

## ⚙️ Configuración de exportación en CapCut

### Paso 1: Abrir exportar

1. Cuando tu video esté listo, tocá **"Exportar"** o **"Export"** (esquina superior derecha)
2. Tiene una flecha hacia arriba ↑

### Paso 2: Configurar opciones

```
┌─────────────────────────────────┐
│  📤 EXPORTAR                    │
├─────────────────────────────────┤
│                                 │
│  📐 Proporción                  │
│  ┌─────────────────────────┐   │
│  │ 9:16          ▼          │   │
│  └─────────────────────────┘   │
│  (Vertical para reels)         │
│                                 │
│  📹 Resolución                 │
│  ┌─────────────────────────┐   │
│  │ 1080P         ▼          │   │
│  └─────────────────────────┘   │
│  (1080P recomendado)           │
│                                 │
│  🎚️ Velocidad de fotogramas   │
│  ┌─────────────────────────┐   │
│  │ 30 fps         ▼          │   │
│  └─────────────────────────┘   │
│  (30 fps para la mayoría)      │
│                                 │
│  📁 Formato                    │
│  ┌─────────────────────────┐   │
│  │ MP4            ▼          │   │
│  └─────────────────────────┘   │
│  (Siempre MP4)                 │
│                                 │
├─────────────────────────────────┤
│                                 │
│      [ EXPORTAR ]              │
│                                 │
└─────────────────────────────────┘
```

### Detalle de cada opción

#### Proporción (Aspect Ratio)

| Proporción | Dimensiones | Para qué |
|------------|-------------|----------|
| **9:16** | 1080 x 1920 | Reels, Stories, TikTok |
| **16:9** | 1920 x 1080 | YouTube, videos horizontales |
| **1:1** | 1080 x 1080 | Feed cuadrado |
| **4:5** | 1080 x 1350 | Feed vertical (más espacio) |

**Mi recomendación para joyería:**
- **Reels:** 9:16 siempre
- **Stories:** 9:16 siempre
- **Feed:** 4:5 si es vertical, 1:1 si es cuadrado

#### Resolución

| Resolución | Calidad | Cuándo usarla |
|------------|---------|---------------|
| **4K** (2160P) | Máxima | Si tu celular lo soporta, para joyería premium |
| **1080P** | Alta | ✅ Recomendada para la mayoría |
| **720P** | Media | Solo si el archivo es muy pesado |

**Consejo:** 4K ocupa mucho espacio y tarda más en procesar. Para joyería, 1080P es más que suficiente y se ve excelente.

#### Velocidad de fotogramas (FPS)

| FPS | Suavidad | Cuándo usarla |
|-----|----------|---------------|
| **30 fps** | Normal | ✅ Recomendada para la mayoría |
| **60 fps** | Muy suave | Slow motion, contenido premium |

**Consejo:** 30 fps está perfecto. 60 fps solo si planeás hacer slow motion o querés máxima suavidad.

#### Formato

**Siempre MP4.** Es el formato universal que Instagram acepta.

## 🚀 Paso 3: Exportar

1. Verificá que todo esté configurado como arriba
2. Tocá **"Exportar"**
3. Esperá mientras procesa:
   - Barra de progreso aparece
   - Puede tomar 1-3 minutos dependiendo del video
   - Mantené la app abierta

### Opciones después de exportar

| Opción | Qué hace | Cuándo usarla |
|--------|----------|---------------|
| **Guardar** | Guarda en la galería del celular | Siempre, como backup |
| **Compartir** | Comparte por WhatsApp, etc. | Si alguien lo necesita ver |
| **Instagram** | Abre Instagram directamente | Si ya tenés el caption listo |

**Mi recomendación:** Siempre tocá **"Guardar"** primero. Después subilo manualmente a Instagram para tener más control sobre el caption y hashtags.

## 📤 Subir a Instagram manualmente

Subir manualmente te da más control. Es el método que recomiendo.

### Paso a paso

1. **Guardá el video** desde CapCut
2. **Abrí Instagram**
3. **Tocá el ícono "+"** (crear nueva publicación)
4. **Seleccioná el video** de tu galería
5. **Elegí el formato:**
   - **Reels:** Arrastrá para seleccionar "Reels"
   - **Feed:** Seleccioná la foto/video normal
6. **Editá el filtro** si querés (o tocá "Siguiente" sin cambios)
7. **Escribí el caption** (ver abajo)
8. **Agregá ubicación y personas etiquetadas** si corresponde
9. **Configurá Alt Text** (ver abajo)
10. **Publicá**

## ✍️ Cómo escribir captions que venden

El caption es tan importante como el video. Guidelines:

### Estructura de un caption efectivo

```
[GANCHO - línea que atrapa] (1-2 oraciones)
↓
[CONTEXTO - qué es el producto] (1-2 oraciones)
↓
[BENEFICIO - por qué lo quieren] (1-2 oraciones)
↓
[CTA - qué hacer] (1 oración)
↓
#hashtags #relevantes #para #joyería
```

### Ejemplo para un collar dorado

```
Este collar dorado que no vas a querer sacarte 💛

Diseño minimalista, detalle de cadena fina y
dije de círculo abierto. Perfeito para usar solo
o combinar con otros.

✨ Medidas: 45cm + extensible 5cm
✨ Material: Acero quirúrgico, no se oscurece
✨ Envío en 24-48hs

Guardá este post para cuando necesites el regalo
perfecto 🎁

#joyas #jewelry #accesorios #collar #gold
#joyasfinas #luxury #giftideas #mujeres
```

### Tips para captions

| Hacer | No hacer |
|-------|----------|
| Empezar con gancho fuerte | "Este producto..." o "Holaaaa" |
| Incluir beneficios, no solo características | Solo listar specs |
| Usar emojis con moderación | Demasiados emojis (max 5-7) |
| Dejar línea en blanco entre secciones | Texto todo junto |
| Incluir CTA claro | No decir qué hacer |

## ♿ Alt Text (Texto alternativo)

El Alt Text es una descripción de tu imagen para personas con discapacidad visual y cuando la imagen no carga.

### Cómo agregar Alt Text en Instagram

1. En la pantalla de edición, tocá **"Configuración avanzada"**
2. Buscá **"Texto alternativo"** o **"Alt text"**
3. Escribí una descripción de tu video/foto
4. Tocá **"Listo"**

### Cómo escribir buen Alt Text

| Bien | Mal |
|------|-----|
| "Collar dorado de acero quirúrgico con dije de círculo abierto, sobre fondo blanco" | "Joyita linda" |
| "Aretes plateados con piedra circulada, modelo elegante, presentados en caja de regalo" | "Aretes" |
| "Detalle de pulsera de acero rose gold con dijes de estrella y corazón" | "Pulsera" |

**Longitud ideal:** 1-2 oraciones describiendo el producto.

## 📱 Stories: Configuración especial

### Diferencias con Reels

| Aspecto | Stories | Reels |
|---------|---------|-------|
| Duración | 24 horas visibles | Permanente en feed |
| Formato | 9:16 | 9:16 |
| Interactividad | Stickers, encuestas, preguntas | Solo caption |
| Música | 15 segundos máx por clip | Hasta 90 segundos |

### Configuración para Stories

- **Proporción:** 9:16
- **Resolución:** 1080P
- **Duración por slide:** 5-7 segundos (más de eso y la gente se va)

### Tips para Stories de joyería

1. **Usa la serie de slides** para mostrar detalles:
   - Slide 1: Imagen principal
   - Slide 2: Detalle 1
   - Slide 3: Detalle 2
   - Slide 4: Precio/disponibilidad
   - Slide 5: CTA (compra desde link en bio)

2. **Stickers útiles:**
   - 🔗 Link (si tenés Instagram Shopping)
   - ⏰ Cuenta regresiva (para ofertas)
   - 💬 Pregunta (para engagement)
   - 📍 Ubicación (para alcance local)
