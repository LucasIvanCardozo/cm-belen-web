## 📚 Descripción

CapCut es el editor de video más potente para móviles, usado por creators profesionales. Esta guía te lleva desde cero hasta crear **videos increíbles para Instagram** con efectos, transiciones y audio profesional.

**En esta guía vas a aprender:**

- ✅ Instalar y configurar CapCut
- ✅ Navegar la interfaz de edición
- ✅ Crear tu primer proyecto de video
- ✅ Agregar texto y subtítulos profesionales
- ✅ Trabajar con audio, música y efectos
- ✅ Aplicar filtros y corrección de color
- ✅ Usar transiciones creativas
- ✅ Exportar videos optimizados para Instagram

## 💎 Tips para Joyería

```
PARA JOYERÍA, PRIORIZA:

• DETALLES CLOSE-UP
  → Muestra el brillo de las piedras

• MOVIMIENTOS LENTOS
  → Slow motion para impacto

• LUZ NATURAL
  → Resalta el brillo metálico

• MÚSICA ELEGANTE
  → Instrumental suave, no trending sounds

• SUBTÍTULOS
  → Muchos ven sin sonido
```

## 📱 Requisitos Previos

- Teléfono Android (8+) o iPhone (iOS 13+)
- App CapCut instalada (gratuita)
- Espacio de almacenamiento (~500MB mínimo)
- Videos o fotos para editar 🌟
