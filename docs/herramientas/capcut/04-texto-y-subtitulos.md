# Módulo 04: Texto y Subtítulos

## 🎯 Objetivo del módulo

Dominar todo lo relacionado con texto en CapCut: subtítulos automáticos, animaciones, fuentes profesionales y técnicas que hacen que el texto trabaje para vos.

**Por qué es importante:** El texto en video cumple tres funciones críticas:

1. **Transmite información** cuando el audio está muteado
2. **Refuerza el mensaje** para mayor retención
3. **Genera engagement** cuando es creativo o gracioso

---

## 📝 Tipos de texto en CapCut

CapCut tiene diferentes tipos de texto. Es importante saber cuándo usar cada uno:

| Tipo                    | Uso                                | Ejemplo                      |
| ----------------------- | ---------------------------------- | ---------------------------- |
| **Texto normal**        | Títulos, llamadas a la acción      | "Nueva colección 🌸"         |
| **Subtítulos**          | Transcripción de lo que se dice    | "esa pulsera es hermosa"     |
| **Subtítulos animados** | Subtítulos con estilo y movimiento | Texto que aparece con efecto |
| **Stickers con texto**  | Elementos decorativos              | Emoji con texto              |
| **Marquesina**          | Texto que se desplaza              | "🔥 Envío gratis hoy"        |

## 🤖 Subtítulos automáticos (la función más importante)

### Cómo funcionan

CapCut puede transcribir audio automáticamente. Detecta las palabras y las pone como subtítulos sincronizados con el video.

### Paso a paso

**1. Agregar subtítulos automáticos:**

1. Tocá **"Texto"** en la barra de herramientas
2. Buscá **"Subtítulos"** o **"Captions"**
3. Tocalos y CapCut analiza el audio
4. Si detecta palabras, las convierte en subtítulos

**2. Si no hay voz o no los detecta:**

CapCut también puede generar subtítulos "de máquina" que aparecen automáticamente aunque no haya voz hablada. Esta función detecta patrones de audio y pone subtítulos incluso en videos con solo música.

1. Tocá **"Texto"** → **"Auto captions"** o **"Subtítulos"**
2. Si aparece "No se detectó audio", buscá la opción de **"Subtítulos estáticos"** o **"Static captions"**
3. Estos subtítulos aparecen a intervalos regulares aunque no haya voz

**3. Si solo necesitás texto decorativo (sin voz):**

1. Usá texto normal (no subtítulos automáticos)
2. Agregá frases como: " swipe para ver más →" o " detalle 1 de 5"

## ✏️ Crear texto manual paso a paso

### 1. Agregar texto

1. Tocá **"Texto"** → **"Agregar texto"**
2. Escribí tu texto
3. Tocalo en el preview para posicionarlo

### 2. Editar el texto

Cuando tocás un texto, aparecen opciones:

```
A+ A-     Tamaño del texto
🗑️        Eliminar
💾        Duplicar
⏱️        Duración
🎨        Estilo
🔠        Tipografía (fuente)
🎭        Animación
⬜         Fondo (para el texto)
```

### 3. Opciones de formato

| Opción         | Qué hace                               |
| -------------- | -------------------------------------- |
| **A+ / A-**    | Agrandar o achicar el texto            |
| **Color**      | Cambiar el color del texto             |
| **Fondo**      | Agregar un rectángulo detrás del texto |
| **Borde**      | Agregar outline (contorno)             |
| **Sombra**     | Agregar sombra al texto                |
| **Alineación** | Alinear izquierda, centro o derecha    |
| **Espaciado**  | Espacio entre letras o líneas          |

3. Deslizá para ver todas las opciones
4. Tocá la que quieras

## 🎭 Animaciones de texto

Las animaciones hacen que el texto "entre" o "salga" de la pantalla.

### Animaciones de ENTRADA (lo más común)

| Animación      | Efecto                            | Para qué sirve       |
| -------------- | --------------------------------- | -------------------- |
| **Fade in**    | Aparece suavemente                | Subtítulos, elegante |
| **Typewriter** | Se va escribiendo letra por letra | Dramático            |
| **Bounce**     | Rebota al entrar                  | Casual, divertido    |
| **Slide in**   | Entra desde un lado               | Dinámico             |
| **Scale up**   | Aparece agrandándose              | Impacto              |
| **Blur in**    | Aparece desenfocado a nítido      | Moderno, elegante    |

### Animaciones de SALIDA

Lo mismo pero para cuando el texto termina. Usalas con moderación.

### Cómo aplicar animaciones

1. Seleccioná el texto
2. Tocá **"Animación"**
3. Elegí **"Entrada"** o **"Salida"**
4. Deslizá y tocá la animación que te guste
5. Ajustá la **duración** (generalmente 0.3-0.5 segundos)

### Tip para joyería elegante

Para un look de lujo, usá:

- **Entrada:** Fade in, 0.3 segundos
- **Salida:** Fade out, 0.3 segundos
- **Entre texto:** Nada (sin animación entre palabras)

Evitá animaciones muy llamativas o con sonido. Menos es más.

3. Deslizá y previsualizá tocando cada uno
4. Tocalo para aplicarlo

## ⏱️ Timing del texto (CUANDO aparece)

El timing es tan importante como el contenido. Guidelines:

### Duración ideal por frase

| Tipo de texto       | Duración sugerida |
| ------------------- | ----------------- |
| Subtítulo (leído)   | 2-4 segundos      |
| Título impactante   | 1-2 segundos      |
| Llamada a la acción | 3-5 segundos      |
| Información técnica | 3-4 segundos      |

### Sincronización con el contenido

**Reglas básicas:**

- ✅ El texto debe aparecer **antes** o **al mismo tiempo** que la imagen que describe
- ✅ Si mostrás una joya, el nombre aparece cuando se ve el detalle
- ✅ Si decís algo, el subtítulo aparece cuando hablás
- ❌ NO pongas texto que no tiene que ver con lo que se ve

### Ajustar duración

1. Tocá el texto en el timeline
2. Arrastrá los bordes:
   - **Borde izquierdo:** Cuándo empieza
   - **Borde derecho:** Cuándo termina

```
┌─────────────────────────────────┐
│                                 │  ← Zona donde el texto puede
│   [título o logo]               │     quedar tapado en algunos
│                                 │     celulares
├─────────────────────────────────┤
│                                 │
│                                 │
│         [contenido principal]   │
│                                 │
│                                 │
├─────────────────────────────────┤
│                                 │
│   [subtítulos o CTA]            │  ← Zona más segura para texto
│                                 │
└─────────────────────────────────┘
```

### Reglas de posicionamiento

| Posición               | Para qué                 | Precaución                     |
| ---------------------- | ------------------------ | ------------------------------ |
| **Arriba**             | Títulos, logos           | Puede quedar tapado en Stories |
| **Centro**             | Títulos principales      | Ojo con el contenido           |
| **Abajo (⅓ inferior)** | Subtítulos, CTAs         | ✅ La más segura               |
| **Esquina**            | Detalles, marcas de agua | Dejá margen                    |

## 🛠️ Técnicas avanzadas

### 1. Texto con fondo

Agregar un rectángulo detrás del texto mejora la legibilidad:

1. Seleccioná el texto
2. Buscá **"Fondo"** o **"Background"**
3. Elegí:
   - **Color sólido:** Negro o blanco con opacity 60-80%
   - **Degradado:** Negro a transparente
   - **Blur:** Desenfoque del video detrás

### 2. Texto animado con la música

Para que el texto "baile" con el ritmo:

1. Seleccioná el texto
2. Buscá **"Animación"** → **"Predeterminados"**
3. Encontrá opciones como "Beat" o "Music sync"
4. El texto se moverá con el ritmo

### 3. Highlights en el texto

Resaltar una palabra con color:

1. Escribí el texto completo
2. Seleccioná solo la palabra que querés resaltar
3. Cambiá el color de esa palabra específica
4. Ejemplo: "Joyas que **enamoran**" (donde "enamoran" está en dorado)

### 4. Contador o números

Para mostrar "1/5", "Paso 1", etc.:

1. Buscá el ícono de **"123"** o **"Números"**
2. Elegí el estilo de número
3. Agregalo al video

## ❌ Errores comunes con texto

| Error                         | Por qué está mal       | Solución                                |
| ----------------------------- | ---------------------- | --------------------------------------- |
| **Demasiado texto**           | Aburrido, nadie lo lee | Máximo 15-20 palabras en 15 segundos    |
| **Fuentesal azar (desordenado)** | Parece desordenado     | Usá máximo 2 fuentes distintas          |
| **Texto muy pequeño**         | Ilegible en celular    | Probá en vista previa a tamaño real     |
| **Colores que no contrastan** | No se lee              | Fondo oscuro + texto claro, o viceversa |
| **Animaciones excesivas**     | Distrae                | Usá animaciones sutiles o ninguna       |
| **Texto muy arriba**          | Se corta en Stories    | Ponelo en el tercio inferior            |

---

## ✅ Checklist de texto

- [ ] Subtítulos claros y legibles
- [ ] Máximo 2 fuentes diferentes
- [ ] Texto en zona segura (tercio inferior)
- [ ] Animaciones sutiles o ninguna
- [ ] Texto sincronizado con el contenido
- [ ] Contraste suficiente para leer
- [ ] Llamada a la acción incluida
