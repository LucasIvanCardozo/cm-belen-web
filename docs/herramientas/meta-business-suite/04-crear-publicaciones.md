## 1. Tipos de Contenido que Puedes Crear

Meta Business Suite te permite crear varios tipos de contenido:

### Tipos de Publicación (Feed)

| Tipo | Descripción | Mejor para |
|------|-------------|------------|
| **Foto** | Una sola imagen | Fotos de productos, lookbook |
| **Vídeo** | Video hasta 60 min | Reels, tutoriales, behind the scenes |
| **Carrusel** | Hasta 10 imágenes | Colecciones, antes/después, múltiples vistas |
| **Reels** | Videos cortos (hasta 90 seg) | Contenido viral, tendencias |
| **Texto único** | Solo texto, sin imagen | Anuncios, citas, preguntas |

> 💡 **PRO TIP para joyería:** El contenido de joyería funciona mejor con **fotos de alta calidad** y **carruseles** que muestran el producto desde múltiples ángulos. Los Reels están subiendo en importancia para el algoritmo.

## 2. Preparación: Antes de Empezar

### Checklist Antes de Publicar

Antes de abrir Business Suite, ten listos:

**✅ Contenido visual:**
- [ ] Foto(s) ya editada(s) y lista(s)
- [ ] Verificaste que la resolución es buena (mínimo 1080x1080 px)
- [ ] Las imágenes tienen el formato correcto (JPG o PNG)

**✅ Información del producto:**
- [ ] Precio (si lo vas a mencionar)
- [ ] Descripción breve del producto
- [ ] Características especiales (material, origen, garantía)

**✅ Copy (texto de la publicación):**
- [ ] Descripción atractiva lista
- [ ] Hashtags identificados
- [ ] Mención a la marca propia (@tu_marca)

**✅ Recursos adicionales:**
- [ ] Link del producto (si tienes tienda online)
- [ ] Imagen de portada para el post (si es video)

## 3. Crear una Publicación Básica

### Paso a Paso Completo

**Paso 1:** Abre Meta Business Suite y toca el ícono **✏️ Crear** en la barra inferior.

**Paso 2:** Verás el menú de creación:

```
┌────────────────────────────────────┐
│  ✏️ Crear                          │
├────────────────────────────────────┤
│                                    │

│  │  📱 CREAR HISTORIA           │ │
│  └──────────────────────────────┘ │
│  ┌──────────────────────────────┐ │
│  │  📢 CREAR ANUNCIO            │ │
│  └──────────────────────────────┘ │
│                                    │
└────────────────────────────────────┘
```

**Paso 3:** Toca **Crear publicación**.

**Paso 4:** Selecciona dónde quieres publicar:

```
┌────────────────────────────────────┐
│  Publicar en:                      │
│  ─────────────────────────────────│
│                                    │
│  [✓] 📷 Instagram                 │
│  [ ] 📘 Facebook                  │
│                                    │
│  ┌──────────────────────────────┐ │
│  │  Cuenta: @joyas_belen       │ │
│  └──────────────────────────────┘ │
│                                    │
└────────────────────────────────────┘
```

- **Instagram:** Marca el ícono si quieres publicar en tu Instagram
- **Facebook:** Marca si también quieres publicar en tu Fan Page

> 💡 **PRO TIP:** Para joyería, muchas veces conviene publicar en **ambos** (Instagram + Facebook). Pero verifica primero que el formato de tu imagen se vea bien en ambos. Facebook acepta más tamaños de imagen.

**Paso 5:** Toca **Siguiente**.

**Toca el cuadrado "Agregar medios"** o el ícono de **+** en el medio.

**Paso 9:** Navega hasta la carpeta donde tienes guardada tu foto de joyería.

**Paso 10:** Toca la foto para seleccionarla.

> ⚠️ **NOTA IMPORTANTE:** 
> - Para foto única: toca una imagen
> - Para carrusel: toca varias imágenes en orden (aparecerán numeradas)

### Paso 7: Escribir el Pie de Foto (Caption)

Este es crucial para marcas de joyería. El caption es donde vendes el producto.

**Toca el campo de texto "Escribe un pie de foto..."**

**Paso 13:** Escribe tu descripción:

```
┌────────────────────────────────────┐
│  Escribe un pie de foto...         │
│  ─────────────────────────────────│
│                                    │
│  ✨ Collar "Luna de Plata"         │
│                                    │
│  Diseño minimalista con lunar      │
│  de acero quirúrgico. Perfecto     │
│  para looks de diario.             │
│                                    │
│  📏 Largo: 45cm + extensible       │
│  💧 Resistente al agua             │
│  🎁 Incluye estuche de regalo      │
│                                    │
│  #joyas #collar #plata925          │
│  #joyeriadela #moda #accesorios   │
│                                    │
│  ─────────────────────────────    │
│  240 / 2,200 caracteres            │
└────────────────────────────────────┘
```

> 💡 **PRO TIP para captions de joyería:**
> 1. **Empieza con algo visual:** "Este collar...", "No podemos dejar de mirarlo..."
> 2. **Usa emojis estratégicamente:** ✨ 💎 🎁 pero no más de 5-8
> 3. **Incluye información clave:** Material, tamaño, qué incluye
> 4. **Hashtags al final:** No mezcles hashtags en medio del texto
> 5. **Llamado a la acción:** "Escribenos" o "Link en bio"

> - Reels (con sticker de enlace)

En Facebook sí puedes agregar enlaces clicables.

## 5. Filtros y Edición de Imagen

Desde Business Suite puedes hacer ajustes básicos a la imagen antes de publicar:

**Toca la imagen** y luego el ícono **✏️ Editar** o desliza para ver filtros.

### Opciones de edición:

| Herramienta | Qué hace | Ejemplo para joyería |
|-------------|----------|----------------------|
| **Brillo** | Aclara u oscurece | Oscurecer ligeramente para que brille |
| **Contraste** | Hace más profundos los colores | Subir para destacar el brillo del metal |
| ** Saturación** | Intensidad de colores | Bajar si los colores están muy vivos |
| **Calor** | Tono cálido o frío | Bajar para no distorsionar el oro/plata |
| **Tinta** | Equilibrio de colores | Ajuste fino si el dorado se ve naranja |
| **Viñeta** | Oscurece los bordes | Útil para fotos de producto centradas |
| **Marcos** | Agrega bordes | Solo si complementa el estilo |

> 💡 **PRO TIP:** Para joyería, los filtros deben ser sutiles. Quieres que el oro se vea dorado, no amarillo. Prueba mantener la imagen lo más cercana al color real posible. Un ligera reducción de saturación a veces ayuda.

## 8. Errores Comunes y Cómo Evitarlos

### Error 1: Imagen pixelada o borrosa

**Problema:** Subiste una imagen de baja resolución.

**Solución:**
- Usa imágenes de mínimo 1080x1080 pixeles
- Evita hacer zoom o crop en imágenes ya pequeñas
- Exporta desde tu editor en alta calidad

---

### Error 2: Texto que se corta

**Problema:** El caption es demasiado largo y no se ve completo.

**Solución:**
- La vista previa muestra cómo se ve realmente
- Revisa siempre antes de publicar
- Los "más" permiten expandir el texto

---

### Error 3: Hashtags no funcionan

**Problema:** El hashtag tiene un error de escritura.

**Solución:**
- Verifica que no tenga espacios
- Asegúrate que el hashtag exista
- Usa mayúsculas consistentes (#Joyas = #joyas)

---

### Error 4: No puedo etiquetar producto

**Problema:** No tienes Instagram Shopping configurado.

**Solución:**
- Necesitas solicitar aprobación de Instagram Shopping
- Puede tomar días o semanas
- Consulta el Módulo 11 para solución de problemas

---

### Error 5: Publicación solo en una plataforma

**Problema:** Olvidaste marcar la segunda plataforma.

**Solución:**
- Antes de publicar, verifica qué plataformas están seleccionadas
- Si te equivocas, puedes eliminar y volver a crear
