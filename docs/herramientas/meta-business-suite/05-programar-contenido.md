## 1. ¿Por Qué Programar Contenido?

Programar contenido significa **crear y programar una publicación para que se publique automáticamente en una fecha y hora futura**.

### Beneficios Principales

| Beneficio | Cómo te ayuda |
|-----------|----------------|
| **Consistencia** | Mantienes tu cuenta activa aunque estés ocupada |
| **Mejor horario** | Publica cuando tu audiencia está más activa |
| **Ahorro de tiempo** | Prepara todo el contenido en bloques |
| **Less stress** | No tienes que recordar publicar cada día |
| **Fines de semana** | Puedes preparar contenido el viernes para toda la semana |

> **Contexto joyería:** Imagina que estás en una boda el sábado. Con programación, el viernes preparaste un post de un nuevo colgante que se publica automáticamente el sábado a las 10am mientras tú disfrutas del evento.

---

### ¿Qué puedes programar?

- ✅ Publicaciones de feed (fotos, videos, carruseles)
- ✅ Historias de Instagram y Facebook
- ✅ Anuncios (pero esto es diferente, lo vemos en Módulo 9)

❌ No puedes programar:
- Reels (deben crearse en el momento o subirse manualmente)
- Lives (necesitan transmisión en vivo)

## 2. El Calendario de Contenido

Antes de programar, necesitas un **calendario de contenido**: un plan de qué publicarás cada día.

### Conceptos Básicos del Calendario

**Horarios recomendados para joyería:**

| Día | Mejor horario (LATAM) | Por qué |
|-----|------------------------|---------|
| Lunes | 9:00 AM | Inicio de semana, personas revisan redes |
| Martes | 10:00 AM | Audiencia activa pero no saturada |
| Miércoles | 11:00 AM | Mitad de semana, engagement alto |
| Jueves | 9:00 AM | Pre-fin de semana, planificación de compras |
| Viernes | 4:00 PM | Último momento antes del fin de semana |
| Sábado | 11:00 AM | Relax, gente ve contenido lifestyle |
| Domingo | 12:00 PM | Domingo de inspiración y exploración |

> ⚠️ **NOTA IMPORTANTE:** Estos horarios son referenciales. En Analytics (Módulo 8) verás cuándo está realmente más activa TU audiencia. Programa basado en datos reales.

---

### Estructura Semanal para Joyería

**Ejemplo de semana ideal:**

```
LUNES:     📷 Producto destacado (estático, alta calidad)
MARTES:    💡 Consejo/Tip de estilo (educativo)
MIÉRCOLES: 📷 Foto de look (outfit + joyería)
JUEVES:    📝 Engagement (pregunta, encuesta, trivia)
VIERNES:   🎉 Showroom/Vlog (detrás de cámaras)
SÁBADO:    📷 Reel o video (contenido dinámico)
DOMINGO:   💬 Inspiración (frase, cita, aspiracional)
```

## 3. Cómo Programar una Publicación Paso a Paso

### Paso a Paso Completo

**Paso 1:** Abre Meta Business Suite y toca **✏️ Crear**.

**Paso 2:** Toca **Crear publicación**.

**Paso 3:** Selecciona dónde publicar (Instagram, Facebook, o ambos).

**Paso 4:** Toca **Siguiente**.

**Paso 5:** Agrega el contenido visual (fotos, videos, carrusel).

**Paso 6:** Escribe el caption con hashtags, ubicación, etc.

**Paso 7:** Usa la **vista previa** para verificar que todo se vea bien.

**Paso 8:** En lugar de "Publicar ahora", toca la **flecha junto al botón**.

```
┌────────────────────────────────────┐
│  ← Nueva publicación               │
├────────────────────────────────────┤
│                                    │
│  [📷 Imagen] [Caption listo]       │
│                                    │
│  ┌──────────────────────────────┐ │
│  │                              │ │
│  │  [Publicar ahora        ▼]   │ │
│  │    Programar →               │ │
│  │    Guardar borrador         │ │
│  │                              │ │
│  └──────────────────────────────┘ │
│                                    │
└────────────────────────────────────┘
```

**Paso 9:** Toca **Programar**.

**Paso 10:** Selecciona la fecha:

```
┌────────────────────────────────────┐
│  📅 Programar                      │
│  ─────────────────────────────────│
│                                    │
│  Fecha:                           │
│  ┌────────────────────────────┐   │
│  │  🗓️ 23 Mayo 2026          │   │
│  │  ▼                         │   │
│  └────────────────────────────┘   │
│                                    │
│  Hora:                            │
│  ┌────────────────────────────┐   │
│  │  🕐 10:00 AM              │   │
│  │  ▼                        │   │
│  └────────────────────────────┘   │
│                                    │
│  [Programar publicación]           │
│                                    │
└────────────────────────────────────┘
```

**Paso 11:** Toca el campo de **Fecha** para abrir el calendario.

**Paso 12:** Navega al mes y día deseado, luego toca la fecha.

**Paso 13:** Toca el campo de **Hora**.

**Paso 14:** Selecciona la hora usando el selector (rueda de números o input).

> 💡 **PRO TIP:** Selecciona la zona horaria correcta. Si estás en Buenos Aires, asegúrate de que dice "-03:00" o "GMT-3". Si publicas para otra zona horaria, puede publicarse en horario que no esperabas.

**Paso 15:** Toca **Programar publicación**.

```
┌────────────────────────────────────┐
│  ✓ Programado                      │
│  ─────────────────────────────────│
│                                    │
│  Tu publicación está programada   │
│  para el 23 de Mayo a las 10:00 AM │
│                                    │
│  📷 Collar Luna de Plata          │
│  📘 Instagram: @joyas_belen       │
│                                    │
│  [Ver programación]               │
│  [Agregar otra]                   │
│                                    │
└────────────────────────────────────┘
```

**¡Listo!** Tu publicación se publicará automáticamente el día y hora seleccionados.

## 4. Programar Historias

Las historias tienen su propio proceso porque duran solo 24 horas.

### Paso a Paso para Programar Historia

**Paso 1:** Ve a **✏️ Crear** > **Crear historia**.

**Paso 2:** Selecciona **Instagram** o **Facebook** (o ambos).

**Paso 3:** Agrega el contenido (foto, video, texto).

**Paso 4:** Añade stickers, música, encuestas si lo deseas.

**Paso 5:** Toca **Siguiente** o el botón de programar.

**Paso 6:** Verás las opciones:

```
┌────────────────────────────────────┐
│  Historia                          │
│  ─────────────────────────────────│
│                                    │
│  ¿Cuándo quieres publicar?        │
│                                    │
│  ○ Publicar ahora                  │
│  ● Agregar a tu reserva de historias│ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │  La historia se publicará    │ │
│  │  cuando abras la app         │ │
│  │  y toques "Compartir ahora"  │ │
│  └──────────────────────────────┘ │
│                                    │
│  [Siguiente]                       │
│                                    │
└────────────────────────────────────┘
```

**Paso 7:** Selecciona **"Agregar a tu reserva de historias"**.

> ⚠️ **NOTA IMPORTANTE:** La "reserva de historias" significa que la historia queda guardada y te notificará cuando es momento de publicarla, pero NO se publica automáticamente. Necesitas abrir la app y confirmar.

## 6. Editar una Programación

Si necesitas cambiar algo de una publicación ya programada:

**Paso 1:** En la sección de Programados, toca la publicación.

**Paso 2:** Se abrirá el editor con los detalles.

**Paso 3:** Puedes cambiar:
- La imagen (toca la imagen > cambiar)
- El caption (toca el texto y edita)
- La fecha/hora (busca el ícono de calendario)

**Paso 4:** Toca **Guardar cambios**.

```
┌────────────────────────────────────┐
│  Editar programación               │
│  ─────────────────────────────────│
│                                    │
│  📷 Imagen: [Cambiar]             │
│  📝 Caption: [Editar]            │
│  📅 Fecha: 23 Mayo → 24 Mayo      │
│  🕐 Hora: 10:00 AM → 2:00 PM     │
│                                    │
│  [Cancelar cambios] [Guardar]     │
│                                    │
└────────────────────────────────────┘
```

---

### Cambiar Solo la Fecha/Hora

Si solo quieres cambiar cuándo se publica:

**Paso 1:** Toca los **tres puntos (...)** junto a la programación.

**Paso 2:** Selecciona **"Editar fecha y hora"**.

**Paso 3:** Selecciona la nueva fecha y hora.

**Paso 4:** Toca **Actualizar**.

## 7. Eliminar una Programación

### Eliminar (Cancelar) una Publicación

**Paso 1:** En la lista de Programados, toca **🗑️ (eliminar)** junto a la publicación.

**Paso 2:** Confirma que quieres eliminarla:

```
┌────────────────────────────────────┐
│  ¿Eliminar esta publicación?       │
│  ─────────────────────────────────│
│                                    │
│  Esta acción no se puede deshacer.│
│                                    │
│  [Cancelar]      [Eliminar]        │
│                                    │
└────────────────────────────────────┘
```

**Paso 3:** Toca **Eliminar**.

> ⚠️ **NOTA IMPORTANTE:** Eliminar una programación NO elimina la publicación si ya se publicó. Solo cancela futuras publicaciones que aún no han salido. Si la publicación ya salió, necesitas ir a Instagram/Facebook y eliminarla manualmente.

---

### Convertir Programación en Borrador

Si quieres "pausar" una publicación sin eliminarla:

**Paso 1:** Toca **... (tres puntos)** junto a la programación.

**Paso 2:** Selecciona **"Guardar como borrador"**.

**Paso 3:** La publicación se moverá a tus borradores y ya no se publicará automáticamente.

**Paso 4:** Para reprogramarla, ve a Borradores y programa de nuevo.

## 8. Errores y Soluciones

### Problema 1: Se publicó en horario incorrecto

**Causa:** Zona horaria mal configurada.

**Solución:**
1. Ve a **Configuración** > **Idioma y región**
2. Verifica que la zona horaria sea correcta
3. Al reprogramar, verifica la zona antes de confirmar

---

### Problema 2: Se me olvidó que programé algo

**Causa:** Falta de sistema de recordatorio.

**Solución:**
- Revisa la sección de Programados diariamente
- Añade un evento en tu calendario personal como recordatorio
- Meta envía notificación cuando se publica (aunque no siempre)

---

### Problema 3: Quiero cambiar una imagen pero no puedo

**Causa:** La programación ya está procesada.

**Solución:**
1. Elimina la programación actual
2. Crea una nueva publicación con la imagen corregida
3. Programa de nuevo

---

### Problema 4: Publicación no aparece en programados

**Causa:** Se guardó como borrador, no se programó.

**Solución:**
- Ve a **Borradores** en el menú
- Ábrelo y programa correctamente

## 9. Estrategia de Programación para Joyería

### Sistema Semanal Recomendado

**Paso 1: Define tus días de contenido**

Para una marca de joyería, mínimo recomendado:
- **4 publicaciones de feed por semana** (lunes, miércoles, viernes, domingo)
- **3-5 historias por semana** (variación diaria)

---

**Paso 2: Prepara contenido en lotes**

Dedica un día (por ejemplo, jueves) a preparar todo el contenido de la semana siguiente:

1. Captura/edita todas las fotos necesarias
2. Escribe todos los captions
3. Programa las 4+ publicaciones
4. Prepara las historias y agrégalas a la reserva

---

**Paso 3: Crea un banco de contenido**

Mantén un folder en tu teléfono con:
- Fotos de productos listas
- Diseños gráficos para ofertas
- Imágenes de lifestyle
- Captions pre-escritos

Esto hace que la preparación sea rápida y constante.

---

**Paso 4: Revisa y ajusta**

Cada fin de semana:
1. Revisa qué funcionó (Analytics)
2. Ajusta horarios si tu audiencia responde diferente
3. Prepara contenido para la semana siguiente
