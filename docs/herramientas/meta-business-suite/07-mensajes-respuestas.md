## 1. ¿Por Qué los Mensajes Son Críticos para Joyería?

Los mensajes directos (DMs) son donde **happens la venta en Instagram**. A diferencia del feed donde muestras productos, en los mensajes el cliente pregunta, negocia, y compra.

### Estadísticas Importantes

- 📱 **70% de los clientes** prefiere messaging sobre llamadas o emails
- 💬 **55% de los compradores** envía mensaje antes de comprar
- ⏰ **Responde en menos de 15 minutos** = 8x más probable que convierta

> **Contexto joyería:** Una clienta que ve tu collar en Instagram y te escribe "¿Lo tenés en dorado?" necesita una respuesta rápida. Si le contestas 2 horas después, puede que ya haya encontrado otro vendedor.

## 3. Tipos de Mensajes

### Mensaje Directo (DM)

Son mensajes privados entre tu cuenta y el cliente.

**Características:**

- Solo tú y el cliente lo ven
- Puede incluir texto, imágenes, videos
- Tienes 24 horas para responder sin coste (norma de Meta)
- Después de 24h, puedes seguir respondiendo pero no es "gratis"

---

### Comentarios con DM

Cuando alguien comenta en una publicación y tú respondes por mensaje:

**Ejemplo:**

1. Cliente comenta: "¡Qué hermoso! ¿Cómo lo compro?"
2. Tú tocas "Responder por mensaje privado"
3. Se abre DM para responder en privado

**Cómo responder desde comentarios:**

**Paso 1:** Ve a la publicación donde alguien comentó.

**Paso 2:** Toca el **💬 (comentario)**.

**Paso 3:** Toca el **ícono de mensaje** junto al comentario.

**Paso 4:** Escribe tu respuesta y envía.

```
┌────────────────────────────────────┐
│  💬 Responder a Laura M.           │
│  ─────────────────────────────────│
│                                    │
│  Laura M.:                         │
│  "¡Qué hermoso collar! ¿Lo tienen  │
│   en dorado?"                       │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ Tu respuesta aquí...          │ │
│  └──────────────────────────────┘ │
│                                    │
│  [Enviar mensaje]                  │
│                                    │
└────────────────────────────────────┘
```

---

### Comentarios Públicos

Puedes responder directamente en los comentarios de una publicación.

**Paso 1:** Ve a la publicación.

**Paso 2:** Toca el 💬 bajo la publicación.

**Paso 3:** Escribe tu respuesta y toca **Responder**.

> ⚠️ **NOTA IMPORTANTE:** Los comentarios públicos son visibles para todos. Úsalos para responder preguntas generales o agradecer. Para información personal (precio, dirección, detalles de compra), usa DM privado.

## 4. Cómo Responder Mensajes

### Paso a Paso: Responder un Mensaje

**Paso 1:** Toca **💬 Mensajes** en la barra inferior.

**Paso 2:** Toca la conversación que quieres abrir.

**Verás la pantalla de conversación:**

```
┌────────────────────────────────────┐
│  ←          Laura Martínez         │
│  👤        📷 @laura_m_           │
│             Instagram              │
├────────────────────────────────────┤
│                                    │
│  ┌──────────────────────────────┐ │
│  │      📷 (imagen del cliente) │ │
│  │      Muestra un collar        │ │
│  │      hace 2 días              │ │
│  └──────────────────────────────┘ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │  👤 Laura                    │ │
│  │  "Hola! Me encanta este      │ │
│  │  collar! ¿Lo tienen en       │ │
│  │  dorado también?"            │ │
│  │  10:30 AM                    │ │
│  └──────────────────────────────┘ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │  💬 Tú (enviado)             │ │
│  │  "¡Hola Laura! Gracias por   │ │
│  │  tu mensaje 💕 Sí, lo tenemos │ │
│  │  en dorado. Te paso fotos..."│ │
│  │  10:45 AM                    │ │
│  └──────────────────────────────┘ │
│                                    │
├────────────────────────────────────┤
│  ┌────────────────────────────┐    │
│  │ 💬 Escribe un mensaje...   │    │
│  └────────────────────────────┘    │
│  📷 📹 😊 ➕ 📤                   │
└────────────────────────────────────┘
```

**Paso 3:** Toca el campo de texto **"Escribe un mensaje..."**.

**Paso 4:** Escribe tu respuesta.

**Paso 5:** Opcional: Adjunta una imagen (📷), video (📹), o emoji (😊).

**Paso 6:** Toca **Enviar** (ícono de papeleta o flecha).

---

### Adjuntar Productos o Fotos

Para joyería, es muy útil enviar fotos del producto:

**Paso 7:** Toca **📷** junto al campo de texto.

**Paso 8:** Selecciona una foto de tu galería o toma una nueva.

**Paso 9:** Toca **Enviar**.

> 💡 **PRO TIP:** Cuando un cliente pregunta por un producto, siempre envía una foto. Un imagen vale más que mil palabras y aumenta significativamente la probabilidad de compra.

---

### Guardar Mensaje como Borrador

Si estás escribiendo un mensaje largo y necesitas parar:

**Paso 1:** Escribe tu mensaje.

**Paso 2:** Toca **X** o desliza hacia abajo para minimizar.

**Paso 3:** El mensaje queda como borrador guardado.

**Paso 4:** Cuando quieras continuar, abre la conversación y el borrador seguirá ahí.

## 5. Configurar Respuestas Rápidas

Las respuestas rápidas son **plantillas pre-escritas** que puedes enviar con un toque. Son esenciales para responder rápido cuando tienes muchas consultas.

### Crear una Respuesta Rápida

**Paso 1:** Ve a **⚙️ Configuración** > **Respuestas rápidas** (o busca en herramientas).

```
┌────────────────────────────────────┐
│  ⚡ Respuestas rápidas             │
│  ─────────────────────────────────│
│                                    │
│  [+ Nueva respuesta rápida]        │
│                                    │
│  ─────────────────────────────────│
│  Tus respuestas rápidas:           │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ 👋 Saludos                   │ │
│  │ "¡Hola! Gracias por escribir │ │
│  │ a @joyas_belen..."           │ │
│  │ [✏️] [🗑️]                   │ │
│  └──────────────────────────────┘ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ 📦 Información de envío      │ │
│  │ "¡Hola! Te comento sobre..." │ │
│  │ [✏️] [🗑️]                   │ │
│  └──────────────────────────────┘ │
│                                    │
└────────────────────────────────────┘
```

**Paso 2:** Toca **+ Nueva respuesta rápida**.

**Paso 3:** Configura tu respuesta:

```
┌────────────────────────────────────┐
│  Nueva respuesta rápida             │
│  ─────────────────────────────────│
│                                    │
│  Atajo (nombre):                    │
│  ┌────────────────────────────┐   │
│  │ /saludo                    │   │
│  └────────────────────────────┘   │
│                                    │
│  Mensaje:                          │
│  ┌────────────────────────────┐   │
│  │ ¡Hola! 👋                  │   │
│  │                           │   │
│  │ Gracias por escribir a     │   │
│  │ @joyas_belen 💎           │   │
│  │                           │   │
│  │ ¿En qué podemos ayudarte?  │   │
│  │                           │   │
│  └────────────────────────────┘   │
│                                    │
│  📝 Agregar nota (para ti)        │
│  ┌────────────────────────────┐   │
│  │ Usar al inicio de toda      │   │
│  │ conversación               │   │
│  └────────────────────────────┘   │
│                                    │
│  [Cancelar]        [Guardar]      │
│                                    │
└────────────────────────────────────┘
```

**Campos importantes:**

| Campo       | Descripción                            | Ejemplo                          |
| ----------- | -------------------------------------- | -------------------------------- |
| **Atajo**   | Código corto para invocar la respuesta | `/saludo`, `/envio`, `/precio`   |
| **Mensaje** | Texto completo que se envía            | "¡Hola! Gracias por escribir..." |
| **Nota**    | Solo para ti (no se envía)             | "Para ventas nuevas"             |

**Paso 4:** Toca **Guardar**.

---

### Usar una Respuesta Rápida

**Paso 1:** Abre una conversación.

**Paso 2:** Toca el campo de texto.

**Paso 3:** Escribe el **atajo** (por ejemplo, `/saludo`).

**Verás:**

```
┌────────────────────────────────────┐
│  Escribe un mensaje...             │
│  ─────────────────────────────────│
│                                    │
│  /saludo                           │
│  ─────────────────────────────────│
│  ┌──────────────────────────────┐ │
│  │ ⚡ Respuesta rápida           │ │
│  │ "¡Hola! Gracias por escribir │ │
│  │ a @joyas_belen..."           │ │
│  └──────────────────────────────┘ │
│                                    │
└────────────────────────────────────┘
```

**Paso 4:** Toca la respuesta rápida para insertarla.

**Paso 5:** La respuesta aparece en el campo de texto lista para enviar o editar.

**Paso 6:** Toca **Enviar**.

## 6. Plantillas de Respuestas para Joyería

### Plantilla 1: Saludo Inicial

**Atajo:** `/saludo`

**Mensaje:**

```
¡Hola! 👋✨

Gracias por escribir a @joyas_belen 💎

Somos una marca de joyería artesanal con piezas únicas. ¿En qué podemos ayudarte hoy?

💬 Responderemos lo antes posible.
```

---

### Plantilla 2: Información de Producto

**Atajo:** `/producto`

**Mensaje:**

```
¡Hola! Gracias por tu interés 💕

Te cuento sobre [NOMBRE DEL PRODUCTO]:

✨ Material: [acero/oro/baño]
📏 Medida: [detalles]
🎁 Incluye: [estuche/garantía]
💰 Precio: [PRECIO]

¿Te gustaría ver más fotos o videos? Con gusto te los envío.

📦 Envío a todo el país 🇦🇷
```

---

### Plantilla 3: Envío y Tiempo de Entrega

**Atajo:** `/envio`

**Mensaje:**

```
¡Hola! Con mucho gusto te explico 📦

📍 ENVÍOS:
• Envío estándar: 5-7 días hábiles
• Envío express: 2-3 días hábiles

💰 COSTO:
• Compras mayores a $XX: ENVÍO GRATIS
• Compras menores: según zona

📦 Todos los pedidos incluyen seguimiento.

¿Ya viste nuestra colección? Te paso el link ⬇️
```

---

### Plantilla 4: Precio y Pago

**Atajo:** `/precio`

**Mensaje:**

```
¡Hola! 💰

El precio de [PRODUCTO] es: [PRECIO]

💳 FORMAS DE PAGO:
• Transferencia bancaria
• Mercado Pago (cuotas sin interés)
• Efectivo (solo retiro en tienda)

🎁 ¿Es para regalo? Podemos incluir packaging premium + tarjeta personalizada sin costo adicional.

¡Decime cuando quieras hacer el pedido! 🛒
```

---

### Plantilla 5: Fuera de Stock / No Disponible

**Atajo:** `/n Stock`

**Mensaje:**

```
¡Hola! Gracias por tu interés 💕

Lamentablemente [PRODUCTO] se agotó por el momento.

Pero te tengo buenas noticias:
✨ Estamos produciendo más unidades
📅 Para finales de [MES] vamos a tener stock nuevamente

¿Querés que te avisemos cuando esté disponible? Solo necesito tu nombre y te mantenemos al tanto.

También podemos mostrarte alternativas similares 🪄
```

---

### Plantilla 6: Pedido Confirmado

**Atajo:** `/pedido`

**Mensaje:**

```
¡Tu pedido está confirmado! 🎉

📋 DETALLES:
• Producto: [NOMBRE]
• Cantidad: [CANTIDAD]
• Total: [PRECIO]
• Forma de pago: [PAGO]

📦 ESTADO:
En preparación → Te avisamos cuando se envíe.

🔍 Luego del envío, recibirás el número de seguimiento por aquí.

¡Gracias por confiar en @joyas_belen! 💎✨
```

---

### Plantilla 7: Seguimiento de Envío

**Atajo:** `/tracking`

**Mensaje:**

```
¡Hola! 💕

Tu pedido ya fue enviado 🚚✨

📦 Número de seguimiento: [CÓDIGO]
🔗 Link de seguimiento: [LINK]

📍 Tiempo estimado de llegada:
• CABA/GBA: [X] días
• Interior: [X] días

Cualquier consulta sobre el envío, escribinos aquí. ¡Estamos para ayudarte! 😊
```

---

### Plantilla 8: Despedida / Despedirse

**Atajo:** `/gracias`

**Mensaje:**

```
¡Fue un placer atenderte! 💎

Gracias por elegir @joyas_belen. Tu joyería estará lista pronto para brillar ✨

Si tenés alguna otra pregunta, no dudes en escribirnos. Estamos para ayudarte siempre.

¡Te esperamos de vuelta! 👋
```

## 7. Mejores Prácticas de Atención al Cliente

### Tiempo de Respuesta

**Meta ideal:**

- ⚡ **Primer respuesta:** < 15 minutos
- ✅ **Respuesta completa:** < 1 hora

**Si no puedes responder inmediatamente:**

1. Usa respuesta rápida de saludo para confirmar que leíste
2. Prometemos un tiempo de respuesta ("Te respondo en 1 hora")

```
💬 Respuesta automática (no configurar aún, lo vemos en Módulo 10):
"Hola! Recibimos tu mensaje y te respondemos lo antes posible. Mientras tanto, podés ver nuestra colección aquí 👉 [LINK]"
```

---

### Tono de Voz

**Para marcas de joyería y lujo:**

✅ **Haz:**

- Sé amable y profesional
- Usa emojis sutiles (💎✨😊)
- Personaliza el mensaje (usa su nombre)
- Da información completa
- Sé honesta (si no tienes algo, dilo)

❌ **No hagas:**

- Usar errores de ortografía
- Responder todo en mayúsculas
- Ser robotic con respuestas copiadas y pegadas sin personalizar
- Ignorar mensajes
- Dejar mensajes sin responder por días

---

### Proceso de Venta por DM

**Flujo recomendado:**

```
1. SALUDO
   /saludo

2. PREGUNTA
   "¿En qué te puedo ayudar?"

3. ESCUCHA
   Cliente describe lo que busca

4. MUESTRA
   Envía fotos/videos del producto

5. INFORMA
   Precio, disponibilidad, envío

6. CIERRA
   "¿Querés hacer el pedido?"

7. CONFIRMA
   /pedido

8. DESPEDIDA
   /gracias
```

---

### Guardar Conversaciones Importantes

Si una conversación tiene información relevante:

**Paso 1:** Abre la conversación.

**Paso 2:** Toca **... (tres puntos)** arriba.

**Paso 3:** Selecciona **"Archivar"** o **"Marcar como importante"**.

**Esto guarda la conversación y la saca de la bandeja principal sin eliminarla.**

## 8. Automatizaciones y Respuestas Automáticas

> ⚠️ **NOTA IMPORTANTE:** Las respuestas automáticas están disponibles solo en algunos casos:
>
> - Facebook Messenger: Respuestas automáticas completas
> - Instagram: Respuestas automáticas limitadas (necesitas Instagram Business)

### Ver Configuración de Automatización

**Paso 1:** Ve a **⚙️ Configuración** > **Automatizaciones** o **Respuestas automáticas**.

**Paso 2:** Verás opciones según lo que esté disponible para tu cuenta.

---
