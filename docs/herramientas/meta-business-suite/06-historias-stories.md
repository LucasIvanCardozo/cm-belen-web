## 1. ¿Qué Son las Stories y Por Qué Importan?

Las Stories son contenido efímero que dura **24 horas** antes de desaparecer. Aparecen en la barra superior de Instagram y en la parte superior del feed de Facebook.

### Características Principales

| Característica | Detalle                                   |
| -------------- | ----------------------------------------- |
| **Duración**   | 24 horas antes de desaparecer             |
| **Ubicación**  | Barra superior de IG / Superior de FB     |
| **Formatos**   | Foto, video, texto, collage               |
| **Elementos**  | Stickers, encuestas, preguntas, música    |
| **Métricas**   | Vistas, respuestas, salidas, seguimientos |

### ¿Por Qué Son Importantes para Joyería?

1. **Visibilidad alta:** Las Stories aparecen primero cuando alguien abre Instagram
2. **Engagement rápido:** Puedes recibir mensajes, respuestas, encuestas en tiempo real
3. **Humanizan la marca:** Muestras el "detrás de cámaras" que no cabe en posts de feed
4. **Formato informal:** Menos presión que el feed, más autenticidad
5. **Herramientas especiales:** Son el único lugar donde puedes usar stickers de enlace en Instagram (sin cuenta de negocio verificada)

> 💡 **PRO TIP:** Las marcas de lujo como Tiffany, Cartier, o Bulgari usan Stories para:
>
> - Mostrar el craftsmanship (el arte de crear la joya)
> - Dar sneak peeks (vistas previas) de nuevas piezas
> - Contar historias sobre los materiales y orígenes
> - Crear urgencia con cuenta regresivas para lanzamientos

## 2. Crear una Story desde Business Suite

### Paso a Paso

**Paso 1:** Abre Meta Business Suite y toca **✏️ Crear**.

**Paso 2:** Toca **Crear historia**.

**Paso 3:** Selecciona la plataforma:

```
┌────────────────────────────────────┐
│  ¿Dónde quieres publicar?          │
│  ─────────────────────────────────│
│                                    │
│  [✓] 📷 Instagram                 │
│  [ ] 📘 Facebook                  │
│                                    │
│  Cuenta: @joyas_belen            │
│                                    │
│  [Siguiente]                       │
│                                    │
└────────────────────────────────────┘
```

**Paso 4:** Toca **Siguiente**.

**Paso 5:** Selecciona el contenido:

```
┌────────────────────────────────────┐
│  Crear historia                    │
│  ─────────────────────────────────│
│                                    │
│  ┌─────────┐  ┌─────────┐         │
│  │ 📷     │  │ 📹     │         │
│  │ Galería │  │ Cámara │         │
│  │         │  │         │         │
│  └─────────┘  └─────────┘         │
│                                    │
│  Toca para agregar foto o video   │
│  de tu galería o tomar nueva      │
│                                    │
└────────────────────────────────────┘
```

**Opción A: Usar galería**

1. Toca **📷 Galería** o **📷 Galería de fotos**
2. Navega a tu carpeta de imágenes
3. Selecciona la foto/video
4. Toca **Siguiente**

**Opción B: Tomar nueva**

1. Toca **📹 Cámara**
2. Permite acceso a la cámara si se pide
3. Captura la foto o video
4. Toca **Siguiente**

## 3. Elementos de una Story

### 3.1 Agregar Texto

**Toca "Aa"** en la barra inferior.

**Opciones:**

- **Aa (texto normal):** Tipografía simple
- **Big Text:** Texto grande, ocupa más espacio
- **Neón:** Texto con efecto de luz neón
- **Marker:** Texto como si estuviera escrito a mano
- **Typewriter:** Texto con efecto de máquina de escribir

```
┌────────────────────────────────────┐
│  Aa                                 │
│  ─────────────────────────────────│
│                                    │
│  ┌──────────────────────────────┐ │
│  │ ✨ Nuevo Collar               │ │
│  │                              │ │
│  │ Selecciona un estilo:        │ │
│  │                              │ │
│  │ [ Moderno ] [ Clásico ]       │ │
│  │ [ Neón ] [Marker ]           │ │
│  └──────────────────────────────┘ │
│                                    │
│  [Aplicar] [Cancelar]             │
│                                    │
└────────────────────────────────────┘
```

> 💡 **PRO TIP para joyería:** Usa texto para destacar el nombre del producto, el precio (si aplica), o una frase de venta como "Edición limitada" o "Solo 5 disponibles".

---

### 3.2 Agregar Stickers

**Toca el ícono de sticker** (cuadrado con bordes redondeados o el emoji de sticker).

#### Stickers Más Útiles:

| Sticker              | Para qué sirve             | Ejemplo para joyería             |
| -------------------- | -------------------------- | -------------------------------- |
| **Ubicación**        | Marca dónde estás          | "Tienda Palermo"                 |
| **Mención**          | Menciona otra cuenta       | @joyas_belen                     |
| **Hashtag**          | Crea link a hashtag        | #JoyasBelen                      |
| ** emoji simple**    | Emojis flotantes           | 💎✨👑                           |
| ** GIF**             | Buscar GIFs animados       | Buscar "sparkle" o "diamond"     |
| **Encuesta**         | Pregunta a tu audiencia    | "¿Oro o Plata?"                  |
| **Deslizador**       | Pregunta con escala        | "Del 1 al 10, ¿cuánto te gusta?" |
| **Pregunta**         | Recibe preguntas abiertas  | "¿Qué Accessorio te falta?"      |
| **Respuesta**        | Recibe respuestas escritas | "Dime tu opinión"                |
| **Cuenta regresiva** | Crear urgencia temporal    | "Lanzamiento en 3 días"          |
| **Fecha**            | Mostrar fecha o evento     | "Navidad 2026"                   |

### 3.7 Agregar Filtros y Efectos

**Toca el ícono de cara (rostro)** o desliza a izquierda/derecha para ver filtros.

**Tipos de efectos:**

- Filtros de color (más cálido, más frío, etc.)
- Filtros de beauty (suavizan la piel)
- Efectos AR (realidad aumentada, cosas divertidas)
- Efectos de cara (transformaciones, masks)

> 💡 **PRO TIP para joyería:** Para fotos de productos, usa filtros sutiles. El objetivo es que el oro se vea dorado y la plata plateada, no distorsionado. Desliza suavemente y verifica que los colores no cambien demasiado.

## 4. Programar Stories con Anticipación

### Opción 1: Guardar en Reserva

**Paso 1:** Crea la Story normalmente.

**Paso 2:** En lugar de **"Compartir"**, toca **"Guardar en reserva"** o busca el menú de guardar.

```
┌────────────────────────────────────┐
│  Story lista para guardar          │
│  ─────────────────────────────────│
│                                    │
│  ┌──────────────────────────────┐ │
│  │ [Tu imagen]                  │ │
│  │ [Stickers]                   │ │
│  └──────────────────────────────┘ │
│                                    │
│  ¿Qué quieres hacer?              │
│                                    │
│  ○ Compartir ahora                 │
│  ● Agregar a tu reserva            │
│                                    │
│  [Siguiente]                       │
│                                    │
└────────────────────────────────────┘
```

**Paso 3:** Toca **Siguiente**.

**Paso 4:** Verás un calendario para seleccionar cuándo quieres que te notifique:

```
┌────────────────────────────────────┐
│  Programar recordatorio            │
│  ─────────────────────────────────│
│                                    │
│  Se te notificará para            │
│  compartir esta historia en:      │
│                                    │
│  📅 [Seleccionar fecha y hora]   │
│                                    │
│  ○ Mañana 9:00 AM                  │
│  ○ Jueves 6:00 PM                  │
│  ○ Personalizado                   │
│                                    │
│  [Guardar recordatorio]            │
│                                    │
└────────────────────────────────────┘
```

> ⚠️ **NOTA IMPORTANTE:** "Guardar en reserva" NO publica automáticamente. Solo te envía una notificación recordándote que es hora de publicar. Tú decides cuándo compartirla.

---

### Opción 2: Programar desde el Calendario de Contenido

**Paso 1:** Ve a **✏️ Crear** > **Crear historia**.

**Paso 2:** Crea y guarda la story.

**Paso 3:** En la notificación/recordatorio, toca **"Compartir ahora"**.

## 6. Mejores Prácticas y Errores a Evitar

### Mejores Prácticas

**✅ Hazlo consistently:**

- Publica al menos 3-5 stories por semana
- La consistencia mantiene tu marca en la mente

**✅ Mezcla formatos:**

- No solo fotos; incluye videos cortos
- Varía entre producto, lifestyle, y engagement

**✅ Usa todos los elementos:**

- Stickers, ubicaciones, hashtags
- Esto aumenta la visibilidad

**✅ Responde a mensajes:**

- Si alguien te responde, contesta
- Las respuestas generan más interacción

**✅ Analiza qué funciona:**

- Revisa quién vio cada story
- Identifica qué tipo de contenido genera más vistas

---

### Errores a Evitar

❌ **No pongas texto demasiado pequeño**

- Los usuarios ven Stories en pantallas chicas
- Usa textos grandes y legibles

❌ **No abuses de los stickers**

- Máximo 2-3 por story
- Muchos stickers distraen y confunden

❌ **No publiques solo venta directa**

- La gente huye de contenido 100% comercial
- Mezcla valor, educación y entretenimiento

❌ **No te olvides de la calidad**

- Fotos borrosas o pixeladas dañan tu marca
- Invierte en buena iluminación

❌ **No publiques a las 3am**

- Salvo que tu audiencia sea nocturna
- Verifica los horarios en Analytics

## 7. Analizar Rendimiento de Stories

### Ver Métricas de tus Stories

**Paso 1:** Ve a la sección de **📊 Análisis** (Analytics).

**Paso 2:** Busca **"Historias"** o **"Stories"**.

**Paso 3:** Verás una lista de tus stories recientes con métricas:

```
┌────────────────────────────────────┐
│  📱 Stories                        │
│  ─────────────────────────────────│
│                                    │
│  ┌──────────────────────────────┐ │
│  │ 📷 Collar Luna - nuevo      │ │
│  │  👁️ 234 vistas              │ │
│  │  💬 12 respuestas           │ │
│  │  ↗️ 3 seguimientos           │ │
│  │  📅 Ayer 10:00 AM            │ │
│  └──────────────────────────────┘ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ 📷 Tip de cuidado            │ │
│  │  👁️ 189 vistas              │ │
│  │  💬 5 respuestas            │ │
│  │  ↗️ 1 seguimiento            │ │
│  │  📅 Hace 2 días              │ │
│  └──────────────────────────────┘ │
│                                    │
└────────────────────────────────────┘
```

### Métricas Clave

| Métrica                   | Qué significa                  | Buena señal             |
| ------------------------- | ------------------------------ | ----------------------- |
| **Vistas**                | Personas que vieron tu story   | Más = mejor             |
| **Tasa de finalización**  | % que vio toda la story        | > 70% es bueno          |
| **Respuestas**            | Mensajes recibidos por story   | Engagement directo      |
| **Toques hacia adelante** | Personas que saltaron tu story | Muy bajo = malo         |
| **Toques hacia atrás**    | Personas que repasaron         | Interés en el contenido |
| **Nuevos seguidores**     | Cuántos follow desde la story  | Conversión              |

> 💡 **PRO TIP:** Las stories con encuestas, preguntas, o deslizadores suelen generar más engagement que las simples. Si tu tasa de respuesta es baja, prueba añadir interactivos.

## 🎯 Resumen del Módulo

| Elemento | Función                                          |
| -------- | ------------------------------------------------ |
| Stories  | Contenido efímero de 24 horas                    |
| Stickers | Herramientas interactivas (encuestas, preguntas) |
| Texto    | Títulos y textos sobre la imagen                 |
| Música   | Audio de fondo para la story                     |
| Reserva  | Guardar para publicar después                    |
| Métricas | Vistas, respuestas, engagement                   |

**Concepto clave:** Las Stories son tu espacio informal y de alta visibilidad. Úsalas para conectar, no solo para vender.
