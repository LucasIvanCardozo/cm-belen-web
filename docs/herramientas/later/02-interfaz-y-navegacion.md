## 1. Pantalla principal: el calendario

Cuando abrís Later, lo primero que ves es el **calendario visual**. Esta es tu herramienta principal.

### Vista del calendario

```
┌────────────────────────────────────────┐
│           ▼ Mayo 2026                   │  ← Mes actual
├────────────────────────────────────────┤
│  L    M    X    J    V    S    D       │
├────────────────────────────────────────┤
│  27   28   29   30    1    2    3      │
│  ░    ░    ▓    ░    ░    ░    ░      │  ← ▓ = hay post programado
│                                    4    │
│  4    5    6    7    8    9   10      │
│  ░    ▓    ░    ░    ░    ▓    ░      │
│                                    11  │
│  ...                                   │
└────────────────────────────────────────┘
│                                        │
│  ┌──────────────────────────────────┐  │
│  │ 📸 Collar Luna          9:00 AM │  │  ← Preview del post seleccionado
│  │ 💍 Anillo Solitario     9:00 AM │  │
│  │ 👑 Dijes Personalizados  7:30 PM │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
```

### Qué podés hacer en el calendario

| Acción                     | Cómo                                |
| -------------------------- | ----------------------------------- |
| Ver qué publicás cada día  | Mirar los puntos ▓ en el calendario |
| Agregar publicación        | Tocar **[+ ADD]**                   |
| Ver publicación específica | Tocar el día o el preview           |
| Cambiar mes                | Deslizar izquierda/derecha          |
| Filtrar por red social     | Tocar los íconos de redes arriba    |

## 2. Biblioteca de contenido

Later tiene una sección dedicada a guardar todo tu contenido multimedia.

### Acceder a la biblioteca

1. Tocá **[+ ADD]** para crear una publicación
2. O seleccioná **Media Library** en el menú lateral

### Qué contiene la biblioteca

```
┌────────────────────────────────────────┐
│  📁 Biblioteca de Contenido             │
├────────────────────────────────────────┤
│                                        │
│  📱 Subidos Recientemente              │
│  ├── IMG_2026_001.jpg                  │
│  ├── video_ring.mp4                    │
│  └── carousel_jewels.jpg               │
│                                        │
│  📦 Canva Designs                      │
│  ├── Post San Valentin.png             │
│  ├── Carrusel Tendencias.png           │
│  └── Stories Oferta.png                │
│                                        │
│  ☁️ Importar de Galería                │
│  └── → Ver más                        │
│                                        │
└────────────────────────────────────────┘
```

### Importar desde Canva

Una ventaja de Later es que conecta directamente con Canva:

1. En la biblioteca, tocá **"Canva"** (si está vinculado)
2. Verás todos tus diseños de Canva
3. Seleccioná el que quieras programar
4. Se importará en alta calidad listo para publicar

**Para joyería:** Esto es genial porque podés diseñar en Canva (desde tu celular) e importar directo a Later sin perder calidad.

## 4. Cómo usar el calendario visual

### Navegación básica

| Acción             | Gestual                       |
| ------------------ | ----------------------------- |
| Cambiar mes        | Deslizar izquierda/derecha    |
| Ver día específico | Tocar el número del día       |
| Volver a hoy       | Tocar **"Today"** o **"Hoy"** |
| Hacer zoom         | Pellizcar para ver semana     |

### Ver publicaciones de un día

1. Tocá un día con puntos (▓)
2. Se expandirá mostrando las publicaciones de ese día:

```
┌────────────────────────────────────────┐
│         Jueves 8 de Mayo               │
├────────────────────────────────────────┤
│                                        │
│  09:00 AM ━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  ┌────────────────────────────────┐   │
│  │ 📸 Collar Plata Luna          │   │
│  │    Imagen + caption program.   │   │
│  │    [Edit] [Delete] [Duplicate]│   │
│  └────────────────────────────────┘   │
│                                        │
│  07:30 PM ━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  ┌────────────────────────────────┐   │
│  │ 💍 Anillo Solitario           │   │
│  │    Reel + descripción          │   │
│  │    [Edit] [Delete] [Duplicate]│   │
│  └────────────────────────────────┘   │
│                                        │
│         [+ Add Post to this day]      │
└────────────────────────────────────────┘
```

### Estado de las publicaciones

Cada publicación tiene un estado visible:

| Estado          | Ícono | Color   | Significado              |
| --------------- | ----- | ------- | ------------------------ |
| **Programado**  | 📅    | Gris    | Esperando para publicar  |
| **Publicado**   | ✅    | Verde   | Ya se publicó            |
| **Fallido**     | ❌    | Rojo    | Error, necesita revisión |
| **Borrador**    | 📝    | Naranja | No terminado             |
| **En revisión** | 👁️    | Azul    | Esperando aprobación     |

## 5. Gestos y atajos móviles

### Gestos del calendario

| Gesto                    | Acción                             |
| ------------------------ | ---------------------------------- |
| **Un toque**             | Seleccionar día o publicación      |
| **Toque largo**          | Acciones rápidas (mover, duplicar) |
| **Deslizar izquierda**   | Mes siguiente                      |
| **Deslizar derecha**     | Mes anterior                       |
| **Pellizcar**            | Zoom in/out del calendario         |
| **Swipe arriba**         | Ver más publicaciones              |
| **Swipe en publicación** | Reveal de acciones rápidas         |

### Toques largos para acciones rápidas

**En una publicación del calendario:**

1. Hacé toque largo sobre el post
2. Aparecerá un menú:

```
┌────────────────────────────────┐
│  📋 Editar                    │
│  📄 Duplicar                  │
│  📅 Mover a otra fecha        │
│  🗑️ Eliminar                  │
│  📤 Compartir                 │
└────────────────────────────────┘
```

## 6. Reconocimiento de estados de publicaciones

### Publicación programada (estado normal)

```
┌────────────────────────────────────────┐
│  📅 Jueves 8 Mayo — 9:00 AM           │
│  ┌────────────────────────────────┐    │
│  │ 📸 Collar Plata Luna          │    │
│  │ "Dale brillo a tus días..."   │    │
│  │ #joyeria #plata #elegante     │    │
│  │                                │    │
│  │ Instagram ✓  Facebook ✓      │    │
│  │ [Edit]              [Delete]  │    │
│  └────────────────────────────────┘    │
└────────────────────────────────────────┘
```

### Publicación publicada exitosamente

```
┌────────────────────────────────────────┐
│  ✅ Martes 6 Mayo — 9:00 AM           │
│  ┌────────────────────────────────┐    │
│  │ 📸 Collar Plata Luna          │    │
│  │ "Dale brillo a tus días..."   │    │
│  │                                │    │
│  │ 💬 24  ❤️ 156  📤 12          │    │
│  │ (comentarios)(likes)(guardados)│   │
│  └────────────────────────────────┘    │
└────────────────────────────────────────┘
```

### Publicación fallida (requiere acción)

```
┌────────────────────────────────────────┐
│  ❌ Lunes 5 Mayo — 7:30 PM           │
│  ┌────────────────────────────────┐    │
│  │ 💍 Anillo Solitario           │    │
│  │ ⚠️ Error: Token de Instagram  │    │
│  │    expiró                     │    │
│  │                                │    │
│  │ ⚠️ Requerido: Reconectar      │    │
│  │ [Fix Now]         [Delete]     │    │
│  └────────────────────────────────┘    │
└────────────────────────────────────────┘
```

**Cómo resolver errores:**

1. Tocá **"Fix Now"** o **"Solucionar"**
2. Later te redirigirá a reconectar Instagram
3. Volvé a autorizar en Meta
4. Later intentará publicar nuevamente

## 7. Modo offline y sync

### Funcionamiento sin internet

| Función        | Offline    | Online |
| -------------- | ---------- | ------ |
| Ver calendario | ✅         | ✅     |
| Crear borrador | ✅         | ✅     |
| Programar post | ❌         | ✅     |
| Ver biblioteca | ✅ (cache) | ✅     |
| Analytics      | ❌         | ✅     |

**Tip:** Si estás diseñando en el subte o sin WiFi, podés preparar todo como borrador y cuando tengas conexión, Later sincronizará y publicará automáticamente.
