## 1. Flujo de programación en Later

El proceso completo tiene 6 pasos:

```
┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐
│ Media   │ → │ Editar  │ → │Caption  │ → │Hashtags │ → │ Schedule│ → │ Preview │
│ Select  │   │ Media   │   │ & Text  │   │         │   │  Time   │   │ & Send  │
└─────────┘   └─────────┘   └─────────┘   └─────────┘   └─────────┘   └─────────┘
```

## 2. Seleccionar contenido

### Opción A: Desde Galería del celular

1. Tocá **[+ ADD]** en Later
2. Seleccioná **"Upload Media"** o **"Subir Medios"**
3. Later abrirá tu galería
4. Seleccioná 1 foto, varias (carrusel), o video
5. Tocá **"Done"** o **"Hecho"**

**Tips:**

- Carrusel: seleccioná hasta 10 imágenes
- Video: máximo 60 segundos para Reels
- Later optimiza automáticamente para cada red

### Opción B: Desde Canva

1. Tocá **[+ ADD]**
2. Seleccioná **"Canva"**
3. Later conecta con tu cuenta de Canva
4. Navegá hasta tu diseño
5. Seleccioná y tocá **"Use This Design"**

```
┌────────────────────────────────────────┐
│  📐 Importar de Canva                  │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │ 📸 Post San Valentin            │  │

│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │ 📸 Carrusel Tendencias          │  │

```

### Opción C: Desde Media Library

1. ☰ → **"Media Library"**
2. Buscá contenido que ya subiste antes
3. Seleccioná y tocá **"Use Selected"**

## 3. Editar y ajustar medios

Después de seleccionar, Later muestra herramientas de edición:

### Filtros rápidos

| Filtro   | Uso                            |
| -------- | ------------------------------ |
| Original | Sin filtro                     |
| Vivid    | Colores vibrantes              |
| Warm     | Tonos cálidos (bueno para oro) |
| Cool     | Tonos fríos (bueno para plata) |
| B&W      | Blanco y negro elegante        |
| Fade     | Estilo vintage                 |

### Ajustes básicos

| Ajuste          | Qué hace                          |
| --------------- | --------------------------------- |
| **Brillo**      | Más o menos luz                   |
| **Contraste**   | Diferencia entre claros y oscuros |
| **Saturación**  | Intensidad del color              |
| **Temperatura** | Cálido (naranja) vs Frío (azul)   |
| **Foco**        | Desenfoque de bordes              |

### Recortar y redimensionar

**Importante:** Later muestra las dimensiones correctas:

| Red Social      | Formato         | Relación aspecto |
| --------------- | --------------- | ---------------- |
| Instagram Feed  | 1:1 (cuadrado)  | 1080x1080        |
| Instagram Story | 9:16 (vertical) | 1080x1920        |
| Instagram Reel  | 9:16 (vertical) | 1080x1920        |
| Facebook        | 1.91:1          | 1200x630         |
| Pinterest       | 2:3 (vertical)  | 1000x1500        |

Later te permite:

- **Smart Crop**: Later recorta automáticamente
- **Manual Crop**: Vos elegís qué se ve

## 4. Escribir caption efectivo

El caption es crucial. Later te da un editor completo:

```
┌────────────────────────────────────────┐
│  ✏️ Caption                            │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │ "Cada pieza cuenta una historia. │  │
│  │                                  │  │
│  │ Este collar de plata 925 está    │  │
│  │ inspirado en las noches de luna  │  │
│  │ llena, cuando el mundo brilla     │  │
│  │ igual que las zirconias que       │  │
│  │ decoran este diseño.              │  │
│  │                                  │  │
│  │ ✨ Características:               │  │
│  │ • Plata 925 genuina              │  │
│  │ • Zirconias AAA                  │  │
│  │ • Largo ajustable 40-50cm         │  │
│  │                                  │  │
│  │ 💬 ¿Ya viste nuestra nueva        │  │
│  │ colección? Comenta LUNA           │  │
│  │ y te enviamos el catálogo."       │  │
│  │                                  │  │
│  │ [124/2200]                       │  │
│  └──────────────────────────────────┘  │
│                                        │
│  📋 Hashtags: [Agregar]               │
│  📍 Ubicación: [Agregar]              │
│  👥 Etiquetar: [Agregar]              │
└────────────────────────────────────────┘
```

### Fórmula para captions de joyería

Usa esta estructura:

```
1. GANCHO (1-2 líneas)
   → Pregunta, dato o frase que atrape

2. HISTORIA (3-5 líneas)
   → Detrás de escena, inspiración, dato útil

3. PRODUCTO (2-3 líneas)
   → Características, material, tamaño

4. CTA (1-2 líneas)
   → Acción: comentar, DM, link en bio, compartir
```

### Ejemplo completo

```
✨"¿Sabías que la plata 925 es la más pura
para joyería everyday?

Cuando diseñamos este anillo, queríamos algo
que puedas usar todos los días sin que se oxide.
La plata esterlina 925 tiene un 92.5% de plata
pura, mezclada con cobre para darle durabilidad.

💍 Especificaciones:
• Plata 925 genuina
• Piedra central: ópalo sintético
• Talla ajustable

👉 Link en bio para ver más.
¿Cuál es tu piedra favorita? Dime en los comentarios 👇"
```

## 5. Agregar hashtags

Later tiene funciones especiales para hashtags:

### Opción A: Escribir manualmente

Simplemente escribí los hashtags después del caption:

```
#joyería #plata925 #anillo #moda #accesorios
```

### Opción B: Usar Saved Hashtag Sets

Guardá grupos de hashtags para reutilizar:

1. ☰ → **"#️⃣ Hashtag Sets"**
2. Tocá **"Create New Set"**
3. Creá grupos por tema:

```
┌────────────────────────────────────────┐
│  #️⃣ Crear Grupo de Hashtags           │
│                                        │
│  Nombre: [Joyería General      ]       │
│                                        │
│  Hashtags:                             │
│  ┌──────────────────────────────────┐  │
│  │ #joyeria                         │  │
│  │ #joyas                          │  │
│  │ #accesorios                    │  │
│  │ #moda                          │  │
│  │ #fashion                       │  │
│  │ #jewelry                       │  │
│  │ #jewelryaddict                 │  │
│  │ #silverjewelry                 │  │
│  │ #joyas.mx (o tu país)         │  │
│  └──────────────────────────────────┘  │
│                                        │
│  [+ Agregar más]                       │
│                                        │
│  💡 Máximo recomendado: 15-20          │
│     (Instagram es 30 máx)              │
│                                        │
│         [Guardar Grupo]               │

```

### Grupos de hashtags recomendados para joyería

| Grupo           | Hashtags                                                   |
| --------------- | ---------------------------------------------------------- |
| **General**     | #joyeria #joyas #accesorios #moda #fashion #jewelry        |

### Usar hashtag sets en posts

1. En el editor de caption
2. Tocá **"[Agregar]"** junto a Hashtags
3. Seleccioná el grupo:
4. Later pega los hashtags automáticamente

## 6. Seleccionar fecha y hora

### Programación individual

```
┌────────────────────────────────────────┐
│  📅 Programar para:                    │
│                                        │
│  Fecha:  [  Jueves 8 Mayo 2026  ▼ ]   │
│  Hora:   [  09:00 AM          ▼ ]     │
│                                        │
│  ⏰ Mejores horarios para vos:         │
│     • 09:00 AM — 11:00 AM             │
│     • 07:00 PM — 09:00 PM             │
│     • 11:00 AM — 01:00 PM             │
│                                        │
│  📍 Timezone: Buenos Aires (GMT-3)     │
│                                        │
│         [Programar Publicación]        │
└────────────────────────────────────────┘
```

### Programación múltiple (repeating)

Si querés publicar lo mismo cada semana:

1. Después de elegir fecha/hora
2. Buscá **"Repeat"** o **"Repetir"**
3. Opciones:
   - Daily (diario)
   - Weekly (semanal)
   - Custom (personalizado)

```
┌────────────────────────────────────────┐
│  🔄 Repetir publicación               │
│                                        │
│  ¿Querés repetir este post?           │
│                                        │
│  ○ No repetir                          │
│  ○ Diariamente                        │
│  ○ Semanalmente                       │
│  ● Personalizado                      │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │ Frecuencia: Cada 2 semanas       │  │
│  │ Días: Lunes, Miércoles, Viernes  │  │
│  │ Hora: 09:00 AM                   │  │
│  │ Termina: [1 Jul 2026      ▼]     │  │
│  └──────────────────────────────────┘  │
│                                        │
│  ⚠️ El contenido será el mismo.       │
│     Editá el caption para variar.      │
│                                        │
│         [Guardar Repetición]          │
└────────────────────────────────────────┘
```

## 9. Editar post ya programado

### Cambiar fecha/hora

1. Andá al calendario
2. Tocá el post
3. Cambiá fecha/hora
4. Tocá **"Update"** o **"Guardar"**

### Cambiar caption

1. Abrí el post
2. Editá el texto
3. Later actualiza automáticamente

### Cambiar imagen

1. Abrí el post
2. Tocá la imagen
3. Later permite:
   - Recortar diferente
   - Subir nueva imagen
   - Mantener caption y hashtags

### Eliminar post

1. Abrí el post
2. Tocá **"..."** o **"Eliminar"**
3. Confirmá

```
┌────────────────────────────────────────┐
│  ⚠️ Eliminar publicación               │
│                                        │
│  ¿Querés eliminar esta publicación   │
│  programada?                         │
│                                        │
│  Esta acción no se puede deshacer.    │
│                                        │
│  [Cancelar]       [Sí, Eliminar]      │
└────────────────────────────────────────┘
```

## 10. Programar en bulk (múltiples posts)

Para eficiencia, Later permite programar varios posts a la vez:

### Método 1: Cola de publicaciones

1. Creá varios posts sin programar (borradores)
2. ☰ → **"Drafts"** o **"Borradores"**
3. Seleccioná varios:
4. Tocá **"Schedule Selected"**
5. Later te deja asignar fechas a cada uno

### Método 2: Importar desde CSV

Para usuarios avanzados:

1. ☰ → **"Import"** o **"Importar"**
2. Later acepta archivos CSV con:
   - Fecha
   - Hora
   - Caption
   - Imagen (URL)
   - Red social

### Método 3: Copiar post existente

1. Abrí un post ya programado
2. Tocá **"..."** → **"Duplicate"** o **"Duplicar"**
3. Later crea una copia
4. Editá lo que necesites
5. Programalo para otra fecha
