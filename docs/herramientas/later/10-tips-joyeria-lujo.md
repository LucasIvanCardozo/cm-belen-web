## 1. Por qué joyería necesita estrategia especial

Joyería es un nicho visual y emocional. Tus posts compiten con marcas internacionales de lujo.

### Lo que hace diferente a la joyería

| Aspecto | Joyería | Otros nichos |
|---------|---------|--------------|
| **Detalle** | Millimeter importa | Puede ser más general |
| **Luz** | Brillo es todo | Luz menos crítica |
| **Contexto** | Lifestyle muy importante | Producto puede bastar |
| **Emoción** | Regalos, momentos especiales | Utilidad general |
| **Precio** | Mayor inversión = más duda | Compra más impulsiva |
| **Confianza** | Crítico para compra online | Menos crítico |

### El problema común

La mayoría de CM de joyería:
- Solo muestran el producto
- No muestran el brillo real
- Usan fotos planas sin luz
- Captions genéricos
-hashtags mezclados con todos

### La diferencia está en los detalles

```
❌ MAL:
📸 Foto del collar en la mesa
💬 "Hermoso collar. Link en bio #joyeria #accesorios"

✅ BIEN:
📸 Foto del collar con luz natural, brillo capturado
💬 "Este collar de plata 925 está inspirado en la luz
    de la luna llena. Cada zirconia está cortada a mano
    para maximizar el brillo. ✨
    
    ¿Tu look de esta noche merece algo especial?"
    #silverjewelry #lunacollection #moonlight
```

## 2. Fotografía de producto: lo básico

### Equipamiento mínimo para joyería

| Elemento | Opción económica | Opción mejor |
|----------|-----------------|--------------|
| **Trípode** | $10-15 USD | $30-50 USD |
| **Foam board** (reflector) | $2-5 USD | $10-15 USD |
| **Luz** | Luz natural de ventana | Ring light $20-30 |
| **Fondo** | Mesa blanca/simple | Lienzo profesional |
| **Celular** | El que tengas | iPhone Pro o Pixel |
| **Total mínimo** | ~$15 USD | ~$60 USD |

### Configuración básica de foto para joyería

**Luz natural (la mejor opción):**
```
┌────────────────────────────────────────┐
│  ☀️ LUZ NATURAL                       │
│                                        │
│  Ventana con luz indirecta            │
│                                        │
│  ┌──────────────────────────────┐      │
│  │                              │      │
│  │         💎 Joya              │      │
│  │                              │      │
│  │    ┌────────────┐           │      │
│  │    │   Foam     │           │      │
│  │    │   board    │           │      │
│  │    └────────────┘           │      │
│  │         ↑                   │      │
│  │    Reflector para           │      │
│  │    captar luz de abajo       │      │
│  └──────────────────────────────┘      │
│                                        │
│  📱 iPhone: hdr automático off         │
│  📐 Zoom: 2x (mejor calidad)         │
│  🔍 Focus: Tocar la joya              │
│                                        │
└────────────────────────────────────────┘
```

### Ajustes de cámara para joyería

| Ajuste | Valor recomendado |
|--------|-------------------|
| **Zoom** | 2x o 3x (no zoom digital) |
| **HDR** | OFF (evita sobreexposición) |
| **Flash** | OFF |
| **Enfoque** | Tocar la joya |
| **Exposición** | -0.3 a -0.7 (oscurecer ligeramente) |
| **ISO** | Bajo (100-400 si es posible) |
| **Balance de blancos** | Automático o luz del día |

## 3. Ajustes de color para diferentes metales

### Plata y metales blancos

| Aspecto | Ajuste | Por qué |
|---------|--------|---------|
| Brillo | +15 a +25 | Mostrar el reflejo |
| Contraste | +10 a +15 | Definir bordes |
| Temperatura | -5 a -10 | Evitar tonos amarillos |
| Sombras | +10 | Recuperar detalle |
| Highlights | +20 | Brillo característico |

**Preset Later/Canva para Plata:**
```
Brillo: +20
Contraste: +12
Saturación: -5
Temperatura: -8
Tinte: 0
```

### Oro y gold filled

| Aspecto | Ajuste | Por qué |
|---------|--------|---------|
| Brillo | +10 a +15 | Resaltar sin quemar |
| Contraste | +15 a +20 | Definir el oro |
| Temperatura | +10 a +15 | Calentar el tono |
| Saturación | +5 a +10 | Intensificar dorado |
| Sombras | +5 | Mantener detalle |

**Preset Later/Canva para Oro:**
```
Brillo: +12
Contraste: +18
Saturación: +8
Temperatura: +12
Tinte: 0
```

### Piedras de color

| Piedra | Ajuste específico |
|--------|------------------|
| Rubí/Escarlata | Saturación +15, Tinte rojo |
| Esmeralda/Verde | Saturación +12, Evitar saturación excesiva |
| Zafiro/Azul | Temperatura -5, Azul más profundo |
| Amatista/Púrpura | Saturación +10, Tinte púrpura |
| Ópalo/Rainbow | Aumentar vibrance, proteger highlights |

### Receta general para joyería

```
┌────────────────────────────────────────┐
│  ✨ RECETA GENERAL JOYERÍA            │
│                                        │
│  Brillo:       +15                     │
│  Contraste:   +15                     │
│  Saturación:  0 (o -5 para plata)    │
│  Temperatura: Ajustar según metal     │
│                (frío para plata,       │
│                 cálido para oro)       │
│  Vibrance:    +10                      │
│  Highlights:  +10                     │
│  Shadows:     +10                     │
│  Sharpness:   +5                      │
│                                        │
│  💡 Exportar como preset en Canva     │
│     para reutilizar                    │
└────────────────────────────────────────┘
```

## 4. Storytelling para joyería

La gente no compra joyería, compra emociones, historias y significado.

### Fórmula para captions de joyería

```
┌─────────────────────────────────────────────────────────┐
│  ESTRUCTURA DE CAPTION                                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  LÍNEA 1: GANCHO (pregunta o frase que atrape)       │
│  → "¿Alguna vez pensaste en cuánto brillo             │
│     tiene tu estilo?"                                 │
│                                                         │
│  LÍNEAS 2-4: HISTORIA/CONTEXTO                        │
│  → "Este anillo está inspirado en las noches          │
│     de luna llena en la costa argentina.               │
│     La piedra central es un ópalo de fuego            │
│     que captura exactamente ese brillo plateado."     │
│                                                         │
│  LÍNEAS 5-6: PRODUCTO (características objetivas)   │
│  → "• Plata 925 genuina                              │
│     • Ópalo de fuego sintético AAA                    │
│     • Piedra: 8mm                                    │
│     • Ajustable"                                     │
│                                                         │
│  LÍNEA 7: CTA (llamada a la acción)                   │
│  → "💬 ¿Luna llena o amanecer? Comment                │
│     tu momento favorito del día 👇"                   │
│                                                         │
│  LÍNEAS 8+: HASHTAGS                                  │
│  → #joyeria #plata925 #anillo #moda                  │
│    #opalo #nochesdeverano                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Tipos de historias para joyería

| Tipo | Ejemplo | Engagement |
|------|---------|------------|
| **Inspiración** | "Este collar lo usamos cuando..." | ⭐⭐⭐ |
| **Detrás de escena** | "Así armamos cada pedido..." | ⭐⭐⭐⭐ |
| **Significado** | "El jade representa..." | ⭐⭐⭐ |
| **Tutorial** | "Cómo cuidar tu plata..." | ⭐⭐⭐⭐ |
| **Testimonio** | "María lo recibió y nos dijo..." | ⭐⭐⭐⭐⭐ |

## 7. Errores comunes a evitar

### Errores de fotografía

| Error | Por qué está mal | Solución |
|-------|-----------------|----------|
| Fondo desordenado | Distrae de la joya | Fondo simple, limpio |
| Luz directa de flash | Crea sombras duras | Luz natural indirecta |
| Foto borrosa | No muestra detalles | Trípode, buena luz |
| Demasiado zoom digital | Pierde calidad | Zoom óptico o 2x |
| Colores incorrectos | No representa el producto | Ajustar temperatura |
| Sin contexto | Aburrido, sin inspiración | Mostrar en uso |

### Errores de caption

| Error | Por qué está mal | Solución |
|-------|-----------------|----------|
| Solo "Hermoso" | No dice nada | Storytelling real |
| Hashtags en caption | Arruina lectura | Primer comentario |
| Demasiados hashtags | Parece spam | 10-15 máximo |
| Sin CTA | No hay acción | "¿Cuál preferís?", "Link en bio" |
| Captions genéricos | No conecta | Historias personales |
| Errores de ortografía | Poco profesional | Revisar antes de publicar |

### Errores de estrategia

| Error | Por qué está mal | Solución |
|-------|-----------------|----------|
| Posts irregulares | Algoritmo baja | Consistencia |
| Solo producto | Aburrido, bajo engagement | Mix de contenido |
| Ignorar analytics | No sabés qué funciona | Revisar métricas |
| No responder comments | Pérdida de comunidad | Engagement diario |
| Hashtags mezclados | No atrapan nicho |grupos específicos |

## 8. Checklist de calidad para posts

### Antes de publicar en Later

```
┌────────────────────────────────────────┐
│  ✅ CHECKLIST DE CALIDAD               │
│  Joyería Luna — Before Publishing      │
├────────────────────────────────────────┤
│                                        │
│  📸 FOTO/VIDEO:                       │
│  □ Fondo limpio y simple               │
│  □ Luz que muestre el brillo           │
│  □ Joya en foco y nítida              │
│  □ Colores corregidos                 │
│  □ Resolucion suficiente (1080p+)     │
│  □ Formato correcto (1:1 o 9:16)       │
│                                        │
│  ✏️ CAPTION:                          │
│  □ Gancho en primeras líneas          │
│  □ Historia o contexto incluida       │
│  □ Características del producto       │
│  □ CTA claro                          │
│  □ Sin errores de ortografía          │
│                                        │
│  #️⃣ HASHTAGS:                        │
│  □ 10-15 hashtags                     │
│  □ Relevant al contenido               │
│  □ Mezcla de nicho + general          │
│  □ Ubicación incluido                 │
│  □ En primer comentario (Later)        │
│                                        │
│  📅 PROGRAMACIÓN:                     │
│  □ Fecha y hora correctas              │
│  □ Mejor horario seleccionado          │
│  □ Red social correcta                │
│  □ Ubicación agregada (si aplica)     │
│                                        │
│  📱 CUENTA:                           │
│  □ Conexión activa                    │
│  □ Token no expirado                  │
│  □ Notificaciones activadas            │
│                                        │
└────────────────────────────────────────┘
```

### Review rápido antes de programar

1. **¿La foto muestra el brillo real?** → Ajustar si no
2. **¿El caption cuenta una historia?** → Agregar contexto si no
3. **¿Los hashtags son relevantes?** → Revisar si son genéricos
4. **¿El CTA invita a interactuar?** → Agregar si no hay
5. **¿La hora es la mejor para mi audiencia?** → Usar Best Times
