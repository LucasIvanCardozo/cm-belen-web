## 1. Qué son workflows automatizados

Los workflows son secuencias de acciones que Later ejecuta automáticamente. Te ahorran tiempo repetitivo.

### Ejemplo sin vs. con workflow

| Sin workflow | Con workflow |
|-------------|--------------|
| Crear post | Later programa automáticamente |
| Copiar hashtags | Later pega hashtags solos |
| Publicar manualmente | Later publica por vos |
| Revisar analytics | Later te notifica métricas |

### Qué se puede automatizar en Later

| Función | Gratis | Starter | Growth |
|---------|--------|---------|--------|
| Programar posts | ✅ | ✅ | ✅ |
| Auto-publicar | ✅ | ✅ | ✅ |
| Hashtag sets | ✅ | ✅ | ✅ |
| Primer comentario | ❌ | ✅ | ✅ |
| Notificaciones analytics | ❌ | ✅ | ✅ |
| Calendario visual | ✅ | ✅ | ✅ |
| Múltiples cuentas | ❌ | ✅ | ✅ |
| RSS feed auto-post | ❌ | ❌ | ✅ |

## 2. Automatizaciones disponibles en Later

### Auto-publicación

**Cómo funciona:**
1. Programás tu post con fecha y hora
2. Later lo publica automáticamente
3. Recibís notificación cuando se publica

**Configuración:**
1. ☰ → **"Settings"**
2. Buscá **"Auto-Publishing"**
3. Activá **"Enable Auto-Publish"**

```
┌────────────────────────────────────────┐
│  ⚙️ Auto-Publishing                      │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │ 🔄 Enable Auto-Publish          │  │
│  │                                  │  │
│  │    Toggle: [====ON]             │  │
│  │                                  │  │
│  │ ✅ Later publicará por vos      │  │
│  │    automáticamente a la hora     │  │
│  │    programada                   │  │
│  └──────────────────────────────────┘  │
│                                        │
│  📱 Notificaciones:                   │
│  ┌──────────────────────────────────┐  │
│  │ ✅ Cuando se publique            │  │
│  │ ✅ Cuando falle                 │  │
│  │ ⬜ Resumen diario               │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
```

### Requisitos para auto-publicación

| Requisito | Detalle |
|-----------|---------|
| Cuenta profesional | Instagram Business o Creator |
| Token activo | Reconexión si expiró |
| Meta app approve | Para Facebook, necesita permisos |
| Imagen correcta | Formato soportado |

## 3. Crear flujo de trabajo semanal

### Flujo ideal para CM de joyería

```
┌─────────────────────────────────────────────────────────┐
│              SEMANA TIPO CON LATER                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  LUNES (Día de planificación)                         │
│  ├── Revisar analytics de semana anterior              │
│  ├── Planificar contenido de la semana                  │
│  ├── Diseñar en Canva lo que falte                     │
│  └── Programar todos los posts en Later                │
│                                                         │
│  MARTES                                                  │
│  ├── Later ejecuta posts programados automáticamente   │
│  ├── Revisar stories y responder comments              │
│  └── engagement en Meta Business Suite                  │
│                                                         │
│  MIÉRCOLES                                             │
│  ├── Later ejecuta posts programados                    │
│  ├── Crear contenido para Reels                        │
│  └── Programar Reels en Later                          │
│                                                         │
│  JUEVES                                                 │
│  ├── Later ejecuta posts                                │
│  ├── Análisis rápido de engagement                      │
│  └── Ajustar estrategia si necesario                   │
│                                                         │
│  VIERNES                                                │
│  ├── Later ejecuta posts                                │
│  ├── Preparar contenido weekend                        │
│  └── Planificar stories interactivos                   │
│                                                         │
│  SÁBADO-DOMINGO                                         │
│  └── Later ejecuta posts (mínimo)                        │
│      └── Vos: responder comments, engagement manual    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Cómo estructurar tu sesión de planificación

**Paso 1: Diseñar (Canva/CapCut)**
1. Creá todos los diseños de la semana
2. Exportalos en alta calidad
3. Guardalos en una carpeta "Para Later"

**Paso 2: Importar a Later**
1. ☰ → **"Media Library"**
2. Subí todas las imágenes/videos
3. Organizá por colección si querés

**Paso 3: Programar (Later)**
1. Creá cada post
2. Agregá caption y hashtags
3. Elegí fecha y hora
4. Programá

**Tiempo estimado:**
- Diseñar contenido: 2-3 horas (Canva/CapCut)
- Programar en Later: 30-45 minutos

## 4. Auto-publicación y sus límites

### Lo que funciona perfecto

| Tipo | Funciona |
|------|----------|
| Fotos de producto | ✅ Perfecto |
| Carruseles | ✅ Perfecto |
| Videos cortos (feed) | ✅ Perfecto |
| Stories con imagen | ✅ Funciona bien |
| Reels | ⚠️ A veces falla |

### Lo que puede fallar

| Tipo | Problema | Solución |
|------|----------|----------|
| Reels con música | La música no siempre se agrega | Publicar Reels manualmente |
| Videos largos | Timeout de conexión | Videos < 60 segundos |
| Stories con stickers interactivos | No funcionan | Agregar después manualmente |

### Checklist antes de confiar en auto-publicación

```
┌────────────────────────────────────────┐
│  ✅ Post programado correctamente     │
│  ✅ Hora y fecha correctas             │
│  ✅ Imagen/video en formato correcto  │
│  ✅ Caption completo                   │
│  ✅ Hashtags incluidos                │
│  ✅ Conexión de cuenta activa         │
│  ✅ Notificaciones activadas           │
│                                        │
│  📱 Recordá:                          │
│  ⚠️ Revisá Reels después de publicar │
│  ⚠️ Agregá stickers a Stories        │
│  ⚠️ Respondé comments en Meta Suite   │
└────────────────────────────────────────┘
```

## 5. Hashtag sets automáticos

Guardá grupos de hashtags para reutilizar rápido:

### Crear un hashtag set

1. ☰ → **"#️⃣ Hashtag Sets"**
2. Tocá **"Create New Set"**

```
┌────────────────────────────────────────┐
│  #️⃣ Crear Grupo de Hashtags           │
│                                        │
│  Nombre del grupo:                     │
│  [Joyería General              ]       │
│                                        │
│  Hashtags (uno por línea o separados   │
│  por coma):                            │
│  ┌──────────────────────────────────┐  │
│  │ #joyeria                          │  │
│  │ #joyas                            │  │
│  │ #accesorios                       │  │
│  │ #moda                             │  │
│  │ #fashion                          │  │
│  │ #jewelry                          │  │
│  │ #jewelryaddict                    │  │
│  │ #silverjewelry                    │  │
│  │ #instajewelry                     │  │
│  │ #joyeriamexico                    │  │
│  └──────────────────────────────────┘  │
│                                        │
│  Cantidad: 10 hashtags                │
│  💡 Máximo recomendado: 15-20        │
│                                        │
│         [Guardar Grupo]               │
└────────────────────────────────────────┘
```

### Grupos recomendados para joyería

| Grupo | Hashtags | Uso |
|-------|----------|-----|
| **General** | #joyeria #joyas #accesorios #moda #fashion #jewelry #jewelryaddict | Posts normales |
| **Producto Plata** | #silverjewelry #plata925 #silver925 #joyasdeplata #sterlingsilver | Posts de plata |
| **Producto Oro** | #goldjewelry #goldfilled #joyasdeoro #bijoux #gilded | Posts de oro |
| **Ubicación** | #joyeriabsas #joyeriabuenosaires #comprarjoyasonline (tu ciudad) | Posts locales |
| **Trending** | #viral #explore #reels #trending2026 | Sparingly |
| **Ofertas** | #oferta #descuento #enviogratis #saleday #promocion | Posts de venta |
| **Comunidad** | #soportelocal #compralocal #emprendedoreslatam | Posts comunidad |

### Usar hashtag sets en posts

1. Al crear post en Later
2. Tocá **"#️⃣ Hashtags"**
3. Verás tus grupos guardados:
4. Tocá el grupo que necesites
5. Later pega los hashtags

## 6. Primer comentario automático

**Disponible en planes Starter y Growth**

El primer comentario es ideal para hashtags sin que arruinen tu caption.

### Configurar primer comentario

1. ☰ → **"First Comment"** o **"Primer Comentario"**
2. Activá la función
3. Creá plantillas:

```
┌────────────────────────────────────────┐
│  💬 Primer Comentario Automático      │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │ Nombre: Hashtags Joyería         │  │
│  │ ─────────────────────────────── │  │
│  │ #joyeria #joyas #accesorios     │  │
│  │ #moda #fashion #jewelry         │  │
│  │ #silverjewelry #instajewelry    │  │
│  │ #joyeriabsas (tu ciudad)        │  │
│  │ #soportelocal                    │  │
│  │ [Activo ✓]                       │  │
│  └──────────────────────────────────┘  │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │ Nombre: Hashtags Oferta         │  │
│  │ ─────────────────────────────── │  │
│  │ #oferta #descuento              │  │
│  │ #enviogratis #saleday           │  │
│  │ #joyeriaenpromo                 │  │
│  │ #comprarjoyas                   │  │
│  │ [Activo ✓]                       │  │
│  └──────────────────────────────────┘  │
│                                        │
│         [+ Crear Nueva Plantilla]      │
└────────────────────────────────────────┘
```

### Cómo usar en posts

1. Creá post normalmente
2. En caption principal, NO pongas hashtags
3. Later automáticamente agregará el primer comentario con hashtags

## 7. Notifications y recordatorios

### Configurar notificaciones

1. ☰ → **"Settings"** → **"Notifications"**

| Notificación | Recomendación |
|--------------|----------------|
| **Post published** | ✅ Activar |
| **Post failed** | ✅ Activar |
| **Weekly summary** | ✅ Activar |
| **Marketing tips** | ⬜ A tu gusto |
| **New features** | ⬜ Desactivar |

### Recordatorios externos

Later no tiene recordatorios complejos, pero podés:

**1. Sincronizar con calendario:**
Después de programar, agregá manualmente a tu calendario personal:
- Recordatorio 1 día antes: "Revisar posts de mañana"
- Recordatorio 1 hora antes: "Stories pendientes de hacer"

**2. Usar herramientas externas:**
- Google Calendar
- Notion (ver módulo de Notion)
- Tarea/alarma del celular

## 8. Workflow para joyería: ejemplo completo

### Mi flujo de trabajo como CM de joyería

```
┌─────────────────────────────────────────────────────────┐
│  📅 FLUJO SEMANAL — Joyería Luna                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  🌅 LUNES MAÑANA (9:00 - 10:00)                       │
│  ─────────────────────────────────────────────────────  │
│  □ Abrir Later                                         │
│  □ Revisar analytics de semana anterior                 │
│  □ Ver qué posts funcionaron mejor                     │
│  □ Anotar insights                                    │
│                                                         │
│  🌞 LUNES TARDE (14:00 - 17:00)                       │
│  ─────────────────────────────────────────────────────  │
│  □ Abrir Canva (celular)                              │
│  □ Diseñar: 4 posts de la semana                      │
│    - 1 foto producto                                  │
│    - 1 carrusel educativo                             │
│    - 1 video/Reel (grabar en CapCut)                  │
│    - 1 post comunidad/testimonio                      │
│  □ Exportar en alta calidad                           │
│  □ Guardar en carpeta "Later - Semana"                │
│                                                         │
│  🌙 LUNES NOCHE (20:00 - 21:00)                       │
│  ─────────────────────────────────────────────────────  │
│  □ Abrir Later                                        │
│  □ Importar todas las imágenes                        │
│  □ Programar:                                          │
│    - Lunes: (ya publicado por Later si había)         │
│    - Martes: Post producto (9:00 AM)                  │
│    - Miércoles: Carrusel educativo (10:00 AM)        │
│    - Jueves: Reel (10:00 AM)                          │
│    - Viernes: Post comunidad (11:00 AM)               │
│    - Sábado: Oferta (10:00 AM)                        │
│    - Domingo: Post lifestyle (18:00 PM)               │
│  □ Agregar caption y hashtags (usar grupos)           │
│  □ Usar primer comentario automático                   │
│                                                         │
│  📱 MARTES A VIERNES                                   │
│  ─────────────────────────────────────────────────────  │
│  □ Later ejecuta posts automáticamente                │
│  □ 8:00 AM: Revisar qué se publicó OK                 │
│  □ Meta Business Suite: Responder comments/DMs        │
│  □ Crear stories del día (agregar manualmente)        │
│  □ Interactuar con comunidad (likear, comentar)        │
│                                                         │
│  🌟 SÁBADO                                              │
│  ─────────────────────────────────────────────────────  │
│  □ Análisis rápido de la semana                        │
│  □ Planificar temas para semana siguiente              │
│  □ Inspiration hunting para Canva                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Tiempo total semanal

| Actividad | Tiempo |
|-----------|--------|
| Analytics semanal | 30 min |
| Diseño en Canva/CapCut | 3 horas |
| Programación en Later | 45 min |
| Revisión diaria (5 días) | 30 min |
| Meta Business Suite | 30 min/día |
| **Total gestión** | ~3.5 horas/semana |
| **Total con engagement** | ~5 horas/semana |
