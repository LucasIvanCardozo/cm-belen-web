## 1. Por qué Stories y Reels son clave para joyería

### El poder de Stories para joyería

| Ventaja         | Por qué importa                                |
| --------------- | ---------------------------------------------- |
| **Visibilidad** | Stories te pone en primer plano en el feed     |
| **Emoción**     | Mostrás el brillo real de las joyas            |
| **Interacción** | Encuestas, preguntas, reacciones               |
| **Tráfico**     | "Swipe up" o link en sticker = tráfico directo |

### El poder de Reels para joyería

| Ventaja              | Por qué importa                              |
| -------------------- | -------------------------------------------- |
| **Alcance orgánico** | Algoritmo favorece Reels en 2026             |
| **Engagement alto**  | Videos de joyería tienen 2-3x más engagement |
| **Mostrar detalles** | Acercá para ver el brillo                    |
| **Viral potencial**  | Un buen Reel puede llegar a miles            |

### Estadísticas clave 2026

- 500+ millones de personas ven Stories diariamente
- Reels tienen 22% más alcance que posts estáticos
- 90% de usuarios siguen al menos una marca en Stories
- Joyería en Reels: 2.5x más engagement que fotos de producto

## 2. Programar Stories paso a paso

### Tipos de Stories que podés programar

| Tipo                 | Descripción               | Frecuencia  |
| -------------------- | ------------------------- | ----------- |
| **Foto estática**    | Imagen única con overlay  | Diario      |
| **Video corto**      | hasta 60 segundos         | 2-3x semana |
| **Carrusel Stories** | Múltiples imágenes        | Semanal     |
| **Encuesta/Quiz**    | Interacción con audiencia | 2x semana   |

### Paso a paso

#### Paso 1: Crear Story en Later

1. Tocá **[+ ADD]**
2. Seleccioná **"Instagram Story"** o **"Facebook Story"**
3. Subí tu imagen o video (9:16, 1080x1920)

#### Paso 2: Editar en Later

Later permite agregar elementos básicos:

```
┌────────────────────────────────────────┐
│  📱 Editar Story                       │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │                                  │  │
│  │      [Tu imagen/video]          │  │
│  │                                  │  │
│  │                                  │  │
│  └──────────────────────────────────┘  │
│                                        │
│  [Texto] [Sticker] [Filtro] [Música]  │
│                                        │
│  📍 Ubicación: [Joyería Luna]        │
│  #️⃣ Hashtags: #joyeria #lujo        │
│                                        │
│  Fecha: [ 8 Mayo 2026    ▼ ]          │
│  Hora:  [ 19:00         ▼ ]          │
│                                        │
│         [Programar Story]              │
└────────────────────────────────────────┘
```

#### Paso 3: Agregar elementos

**Texto:**

- Tocá **[T]** para agregar texto
- Escribí tu mensaje
- Elegí fuente y color
- Later permite fuentes limitadas en Story

**Stickers:**

- Later permite agregar stickers básicos
- Sticker de hashtag (#)
- Sticker de ubicación
- Sticker de hora

**Música:**

- ⚠️ **Limitación importante:** Later NO puede agregar música a Stories programadas
- Later publicará la Story sin música
- Si necesitás música, publicá manualmente después

#### Paso 4: Programar hora

```
┌────────────────────────────────────────┐
│  📅 Programar Story                    │
│                                        │
│  Fecha: [ 8 Mayo 2026    ▼ ]          │
│  Hora:  [ 19:00         ▼ ]          │
│                                        │
│  💡 Los Reels se ven mejor entre       │
│     18:00 - 21:00 en Latam            │
│                                        │
│         [Programar]                    │
└────────────────────────────────────────┘
```

#### Paso 5: Confirmar

1. Later publicará la Story automáticamente
2. Verás la Story en tu perfil por 24 horas
3. Later guardará el analytics

### ⚠️ Limitaciones de Stories en Later

| Funcionalidad        | Disponible en Later | Alternativa                        |
| -------------------- | ------------------- | ---------------------------------- |
| Programar foto/video | ✅                  | -                                  |
| Texto overlay        | ✅                  | -                                  |
| Stickers básicos     | ✅                  | -                                  |
| Ubicación            | ✅                  | -                                  |
| Hashtag sticker      | ✅                  | -                                  |
| **Música**           | ❌                  | Publicar manualmente o CapCut      |
| **Link sticker**     | ❌                  | Solo con cuenta profesional + 10k+ |
| **Encuestas**        | ❌                  | Crear en Instagram manualmente     |
| **Quiz**             | ❌                  | Crear en Instagram manualmente     |

**Recomendación:** Usá Later para la imagen/video base. Agregá música, encuestas y stickers interactivos manualmente en Instagram antes de que expire (24h).

## 3. Programar Reels desde Later

Reels son los videos cortos (15-60 segundos) de Instagram. Later facilita su programación.

### Por qué programar Reels

| Beneficio            | Descripción                         |
| -------------------- | ----------------------------------- |
| **Ahorro de tiempo** | No necesitás estar online           |
| **Consistencia**     | Publicás aunque no estés            |
| **Mejor alcance**    | Reels tienen prioridad en algoritmo |
| **Engagement**       | Videos de joyería = alto engagement |

### Paso a paso para Reels

#### Paso 1: Crear Reel en Later

1. Tocá **[+ ADD]**
2. Seleccioná **"Instagram Reel"**
3. Later abre el selector de video

#### Paso 2: Seleccionar video

1. Podés seleccionar desde:
   - Galería del celular (ya editado en CapCut)
   - Canva (videos de Canva)
   - Media Library

**Importante:** El video debe estar editado y listo. Later NO edita videos, solo los programa.

#### Paso 3: Editar metadata

```
┌────────────────────────────────────────┐
│  🎬 Configurar Reel                    │
│                                        │
│  📝 Caption:                          │
│  ┌──────────────────────────────────┐  │
│  │ "¿Conocés la diferencia entre    │  │
│  │  plata 925 y baño de plata?      │  │
│  │                                  │  │
│  │ En este Reel te lo muestro.      │  │
│  │ ♻️ Guardá para después"          │  │
│  │ [0/2200]                        │  │
│  └──────────────────────────────────┘  │
│                                        │
│  #️⃣ Hashtags:                         │
│  ┌──────────────────────────────────┐  │
│  │ #joyeria #reel #plata925         │  │
│  │ #educativo #joyas #moda         │  │
│  └──────────────────────────────────┘  │
│                                        │
│  📍 Ubicación: [Buenos Aires, AR]    │
│                                        │
│  🎵 Música: [Audio original    ▼ ]   │
│     ⚠️ Se publica sin audio          │
│     (Podés agregar música en IG)      │
│                                        │
└────────────────────────────────────────┘
```

#### Paso 4: Programar fecha y hora

```
┌────────────────────────────────────────┐
│  📅 Programar Reel                     │
│                                        │
│  Fecha: [ 6 Mayo 2026    ▼ ]          │
│  Hora:  [ 10:00 AM       ▼ ]          │
│                                        │
│  💡 Later encontró que tus Reels      │
│     funcionan mejor los martes y       │
│     jueves entre 10:00-12:00          │
│                                        │
│         [Programar Reel]              │
└────────────────────────────────────────┘
```

### Requisitos técnicos para Reels en Later

| Aspecto                | Requisito                                |
| ---------------------- | ---------------------------------------- |
| Duración               | 15 - 60 segundos                         |
| Formato                | MP4                                      |
| Resolución mínima      | 720p                                     |
| Resolución recomendada | 1080p                                    |
| Relación aspecto       | 9:16 (vertical) o 1:1                    |
| Audio                  | Se publicará sin audio (agregar después) |

### Consejos para Reels de joyería

**Qué grabar en CapCut antes de programar:**

1. **Detalle del producto** (5-15 seg)
   - Zoom lento al brillo
   - Mostrar textura
   - Detalle de piedras

2. **Antes/Después** (10-20 seg)
   - Joya en blister vs joyería elegante
   - Bolsa simple vs packaging premium

3. **Proceso** (15-30 seg)
   - Cómo se arma el collar
   - Empaque del pedido
   - Descubriendo el pedido (unboxing del cliente)

4. **Styling** (10-15 seg)
   - Cómo combinar la joya
   - Outfits con la pieza

## 5. Programación de Stories en serie

### Qué son Stories en serie

Son una secuencia de Stories que cuentan una historia o muestran un proceso. Later permite programarlas en orden.

### Ejemplo: Proceso de pedido

```
┌────────────────────────────────────────┐
│  📱 Stories Serie: "Tu pedido"         │
│                                        │
│  Story 1: [9:00 AM]                   │
│  "Gracias por tu compra, [Nombre]!"   │
│  📸 Imagen: Gracias + sticker ❤️      │
│                                        │
│  Story 2: [9:01 AM]                   │
│  "Lo estamos preparando con amor"      │
│  📹 Video: Empaquetando               │
│                                        │
│  Story 3: [9:02 AM]                   │
│  "Cada detalle importa"                │
│  📸 Imagen: Packaging final           │
│                                        │
│  Story 4: [9:03 AM]                   │
│  "¡Listo para enviar!"                │
│  📸 Imagen: Caja cerrada + etiqueta   │
│                                        │
│  Story 5: [9:04 AM]                   │
│  "Te avisamos cuando esté en camino"  │
│  📸 Imagen: Logo + sticker 📦         │
│                                        │
└────────────────────────────────────────┘
```

### Cómo programar serie en Later

1. Creá cada Story individualmente
2. Programalas con 1-2 minutos de diferencia
3. Later las publicará en orden
4. Tu audiencia verá la serie completa

**Tip:** Dejá 2-3 minutos entre cada Story para que aparezcan como serie fluida.

## 6. Análisis de engagement en Stories

### Métricas disponibles en Later

| Métrica          | Descripción             |
| ---------------- | ----------------------- |
| **Views**        | Cuántos vieron la Story |
| **Taps forward** | Cuántos avanzaron       |
| **Taps back**    | Cuántos volvieron       |
| **Exits**        | Cuántos salieron        |
| **Replies**      | Respuestas directas     |
| **Reactions**    | Reacciones emoji        |

### Cómo ver analytics de Stories

1. ☰ → **"Analytics"**
2. Seleccioná **"Stories"**
3. Verás todas tus Stories programadas y publicadas
