## 1. Por qué conectar redes sociales

Conectar tus redes sociales en Later te permite:

| Beneficio                  | Descripción                                 |
| -------------------------- | ------------------------------------------- |
| **Publicación automática** | Later publica por vos en el horario elegido |
| **Un solo lugar**          | Gestionás todas las redes desde Later       |
| **Ahorro de tiempo**       | No necesitás abrir cada app para publicar   |
| **Calendario unificado**   | Ves todo tu contenido en un solo lugar      |
| **Cross-posting**          | Publicar el mismo contenido en varias redes |

### ¿Qué se puede programar desde Later?

| Red Social    | Posts | Carruseles | Reels | Stories | Videos |
| ------------- | ----- | ---------- | ----- | ------- | ------ |
| **Instagram** | ✅    | ✅         | ✅    | ✅      | ✅     |
| **Facebook**  | ✅    | ✅         | ❌    | ✅      | ✅     |
| **Pinterest** | ✅    | ❌         | ❌    | ❌      | ❌     |
| **TikTok**    | ✅    | ❌         | ✅    | ❌      | ✅     |
| **Twitter/X** | ✅    | ❌         | ❌    | ❌      | ❌     |

## 2. Conectar Instagram

Esta es la conexión más importante para joyería. Later necesita autorización de Meta (Facebook) para publicar en tu nombre.

### Pasos para conectar Instagram

#### Paso 1: Ir a Connected Accounts

1. Abrí Later en tu celular
2. Tocá el menú hamburguesa (☰)
3. Seleccioná **"Connected Accounts"**
4. Verás una lista de redes sociales con íconos

#### Paso 2: Seleccionar Instagram

1. Buscá el ícono de **Instagram** (cámara violeta/naranja)
2. Tocá **"Connect"** o **"Conectar"**

#### Paso 3: Autorizar en Meta

```
┌────────────────────────────────────────┐
│  🔐 Meta autorización                    │
│                                        │
│  Later solicita acceso a tu            │
│  cuenta de Instagram para:            │
│                                        │
│  ✅ Publicar fotos y videos           │
│  ✅ Programar publicaciones           │
│  ✅ Ver análisis básicos              │
│  ✅ Gestionar comentarios              │
│                                        │
│  Later NO puede:                       │
│  ❌ Eliminar publicaciones              │
│  ❌ Cambiar contraseña                 │
│  ❌ Acceder a mensajes directos        │
│                                        │
│  [Cancelar]          [Autorizar]       │
└────────────────────────────────────────┘
```

#### Paso 4: Elegir la cuenta correcta

1. Si tenés varias cuentas de Instagram, seleccioná la correcta
2. Tocá **"Continue as [tu usuario]"**

#### Paso 5: Confirmación

1. Later mostrará: **"Instagram Connected Successfully!"**
2. Verás tu nombre de usuario y foto de perfil
3. Tocá **"Done"** o **"Listo"**

### ¿Qué permisos está dando?

| Permiso                                 | Para qué lo necesita Later |
| --------------------------------------- | -------------------------- |
| **Access your Instagram conversations** | Para gestionar comentarios |
| **Read and write your Instagram media** | Para subir fotos/videos    |
| **Read your Instagram profile**         | Para mostrar tu info       |
| **Read your Instagram insights**        | Para analytics             |

> **⚠️ Importante:** Later nunca puede leer tus mensajes directos (DMs). La configuración de Meta lo impide.

## 3. Conectar Facebook

Facebook funciona como "puente" para Instagram en algunos casos. Later necesita acceso a una Página de Facebook (no tu perfil personal).

### Requisito previo

Necesitás tener una **Página de Facebook** (no un perfil personal) vinculada a tu cuenta de Instagram Business o Creator.

**Cómo crear una Página de Facebook:**

1. Abrí la app de Facebook
2. Tocá ☰ → "Páginas" → "Crear"
3. Seguí los pasos
4. Vinculá tu Instagram desde Configuración → Configuración de Instagram

### Pasos para conectar Facebook en Later

#### Paso 1: Connected Accounts

1. Tocá ☰ → **"Connected Accounts"**
2. Buscá el ícono de **Facebook** (letra "f" azul)

#### Paso 2: Iniciar sesión en Meta

1. Later te redirigirá a Meta
2. Iniciá sesión con tu cuenta de Facebook
3. Ingresá tu email y contraseña

#### Paso 3: Seleccionar la Página

```
┌────────────────────────────────────────┐
│  📄 Seleccionar Página                │
│                                        │
│  Tocá la Página que querés conectar   │
│                                        │
│  ○ Mi Joyería Online                   │
│  ○ Belleza Natural (otra marca)       │
│  ○ Emprendedores Argentina            │
│                                        │
│         [Conectar Página]             │
└────────────────────────────────────────┘
```

4. Seleccioná solo la Página de tu negocio de joyería
5. Tocá **"Conectar Página"**

#### Paso 4: Permisos de Facebook

| Permiso               | Para qué lo necesita Later      |
| --------------------- | ------------------------------- |
| **Manage your pages** | Publicar en nombre de la Página |
| **Publish content**   | Subir fotos, videos, stories    |
| **Read insights**     | Ver métricas de la Página       |

#### Paso 5: Confirmación

1. Later mostrará: **"Facebook Connected!"**
2. Verás el nombre de tu Página
3. Tocá **"Done"**

## 4. Conectar Pinterest

Pinterest es **increíble para joyería** porque:

- Es un motor de búsqueda visual
- La gente busca regalos constantemente
- Los pins tienen vida útil larga (meses/años)
- Traffic de calidad a tu tienda online

### Pasos para conectar Pinterest

#### Paso 1: Connected Accounts

1. ☰ → **"Connected Accounts"**
2. Buscá el ícono de **Pinterest** (letra "P" roja)

#### Paso 2: Autorizar Pinterest

```
┌────────────────────────────────────────┐
│  📌 Autorizar Later en Pinterest      │
│                                        │
│  Later quiere acceso para:            │
│                                        │
│  ✅ Publicar Pins en tu nombre        │
│  ✅ Leer tus tableros                  │
│  ✅ Crear nuevos tableros              │
│  ✅ Ver estadísticas                   │
│                                        │
│  [Cancelar]          [Autorizar]       │
└────────────────────────────────────────┘
```

#### Paso 3: Seleccionar cuenta

1. Si tenés varias cuentas Pinterest, elegí la correcta
2. Confirmá la autorización

#### Paso 4: Configurar Pinterest Boards

Later te preguntará qué tableros querés usar:

| Opción                   | Descripción                         |
| ------------------------ | ----------------------------------- |
| **Todos los tableros**   | Later puede publicar en cualquiera  |
| **Tableros específicos** | Elegís cuáles usar                  |
| **Crear nuevos**         | Later crea tableros automáticamente |

**Recomendación para joyería:**

1. Tocá **"Tableros específicos"**
2. Seleccioná:
   - ✅ Joyería / Jewelry
   - ✅ Anillos / Rings
   - ✅ Collares / Necklaces
   - ✅ Regalos / Gifts
   - ✅ Moda / Fashion
   - (los que tengas en tu cuenta)

## 6. Gestionar múltiples cuentas

### Agregar más cuentas de Instagram

En el plan Free, Later permite **1 canal por red social**. Si necesitás más:

#### Para el plan Free:

- Solo 1 Instagram
- Solo 1 Facebook
- Solo 1 Pinterest

#### Para planes de pago:

- Starter: hasta 3 cuentas por red
- Growth: hasta 10 cuentas por red

### Cómo cambiar entre cuentas

Si gestionás varias cuentas (tu marca + clientes):

1. ☰ → **"Profiles"** o **"Perfiles"**
2. Verás todas las cuentas conectadas
3. Tocá la que quieras usar como "principal"

**Ejemplo:**

```
┌────────────────────────────────────────┐
│  👤 Perfiles Guardados                 │
│                                        │
│  ✅ Joyería Luna (principal)          │
│  ○ Belleza & Accesorios               │
│  ○ Mis Diseños                        │
│                                        │
│  [+ Agregar otro perfil]              │
└────────────────────────────────────────┘
```

## 7. Desconectar y reconectar cuentas

### Desconectar una cuenta

1. ☰ → **"Connected Accounts"**
2. Buscá la cuenta a desconectar
3. Tocá el ícono de **engranaje** ⚙️ o **"..."**
4. Seleccioná **"Disconnect"** o **"Desconectar"**
5. Confirmá

```
┌────────────────────────────────────────┐
│  ⚠️ Desconectar Instagram             │
│                                        │
│  ¿Estás segura de desconectar         │
│  @joyerialuna?                        │
│                                        │
│  Later ya no podrá publicar en        │
│  esta cuenta.                         │
│                                        │
│  [Cancelar]        [Desconectar]       │
└────────────────────────────────────────┘
```

### Reconectar una cuenta

Si Later pierde acceso (token expirado):

1. ☰ → **"Connected Accounts"**
2. Verás una marca ⚠️ o "Reconnect" en la cuenta
3. Tocá **"Reconnect"**
4. Autorizá nuevamente en Meta/Facebook/Pinterest

## 8. Permisos y seguridad

### Revisar permisos dados

Periódicamente, revisá qué apps tienen acceso a tus redes:

#### En Instagram:

1. Configuración → Aplicaciones → Later
2. Revisá los permisos
3. Podés revocar si algo cambió

#### En Facebook:

1. Configuración → Aplicaciones y sitios web
2. Buscá "Later"
3. Revisá y ajustá permisos

### Buenas prácticas de seguridad

| Práctica                  | Por qué                                               |
| ------------------------- | ----------------------------------------------------- |
| Usá contraseña única      | Si Later se compromete, solo esa                      |
| Actová 2FA en Meta        | Seguridad extra para tu cuenta                        |
| Revisá cuentas activas    | Revisión mensual de quién tiene acceso                |
| No compartas credenciales | Later funciona sin necesidad de compartir contraseñas |
